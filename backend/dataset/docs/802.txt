学术讲座（二十六）：静电纺丝纤维材料及其环境应用
化学与环境工程学院　2022/11/2 10:27:00
主讲人：
刘建允教授
时间：
11月4日上午10:00
地点：
腾讯会议
（
ID号：350 485 962
）
报告人简介：
刘建允，东华大学环境科学与工程学院教授，先后留学于德国马普高分子所及英国牛津大学。
2006-2010年就职于通用电气公司全球研发中心。长期从事能源材料、环境电化学、环境污染治理与检测等领域的教学与研究工作。在Anal Chem, J Mater Chem A, Appl Mater Today, Sep Puri Technol, Chem Eng J, Desalination, J Colloid Interface Sci, Carbon, Chem Commun等发表论文130余篇，获授权发明专利20余项。
讲座
摘要：
近年来，静电纺丝纳米纤维由于具有高长径比、高比表面积、纺丝成分灵活可调等优点，在化学化工、环境，能源材料等领域得到了快速发展。报告人将从静电纺丝技术的特点出发，围绕静电纺丝纳米材料的制备，介绍常用的纺丝纳米纤维材料的衍生方法；并结合本人的研究工作，分析其在电容器储能、电容法脱盐（离子选择性吸附）、环境污染物检测等方面的潜在的应用前景
。
