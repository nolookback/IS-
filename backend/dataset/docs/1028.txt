报告简介（
Abstract
）：
医学影像分割在疾病诊断、术前规划、术中导航以及术后评估等临床场景中发挥着重要的作用。在过去
3
0
多年里，以
“Snakes”为代表的变分模型和以U
-N
et为代表的卷积神经网络是两类备受关注的分割方法：基于经典变分模型的“浅”分割方法具有可解释性强、不需要大量训练样本等优点；当下流行的深度学习的“深”方法能从样本中自动学习高维数据的特征表达。融合两类方法的优势，能促进分割方法更好的落地走向临床应用。本次报告将分享三方面内容：(1
)
提出基于卷积阈值的迭代算法，提升经典变分模型的求解效率和鲁棒性，进一步与深度学习结合，实现医学影像的高效利用。
(2)
结合对抗学习和最优传输理论，提升分割网络在跨中心、跨序列和跨模态任务中的泛化能力；
(
3)
基于
M
ICCAI FLARE21
国际竞赛结果，介绍实现精准、快速和低显存消耗的腹部多器官分割优胜方案。
