讲座主要内容：
报告首先从国家战略层面解读提升质量已成为我国当前实现产业升级和高质量发展的战略举措。然后，在分析质量管理百年发展的历史上，对质量科学的理论发展和产业应用进行回顾与展望，基于工业
4.0
，探讨了质量
4.0
的定义和特点，分析了工业
4.0
和
IT
技术的发展对质量管理的促进作用，同时提出了随着智能制造和工业互联网、大数据的发展，质量管理面临的新的挑战，梳理了智能制造背景下质量管理需要亟需研究的新课题。最后，提出在质量
4.0
时代，需要质量与创新的融合与集成，并指出一些值得研究的科学问题。
