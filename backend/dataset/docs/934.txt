报告摘要：
Clustering is a fundamental unsupervised machine learning task which
has been widely used and studied    in many applications like community detection,
superpixel segmentation, web mining, recommendation systems, etc.    In this talk,
I will explain the basic idea of clustering,
how to use learning techniques to achieve better clustering     performance,
and present several works finished by my group that is related to clustering,
e.g., self-supervised clustering, clustering with multiple views, ensemble
clustering, and how to build deep clustering networks. I will also outline some
possible research issues in clustering.
