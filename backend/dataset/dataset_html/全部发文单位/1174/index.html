报告人简介：

朱永法,清华大学化学系教授、博导，国家电子能谱中心副主任。分别从南京大学、北京大学和清华大学获得学士、硕士和博士学位以及在日本爱媛大学从事博士后研究工作。1988.7月到现在，一直在清华大学化学系工作，从事能源光催化、环境光催化及光催化健康的研究。

获得教育部跨世纪优秀人才及国家自然科学基金委杰青年基金的资助。获得国家自然科学奖二等奖1项, 教育部自然科学奖一等奖2项、二等奖1项，教育部科技进步奖二等奖和三等奖各1次。发表SCI论文397篇，高被引论文35篇；论文总引30500余次，H因子为99。2014-2019年Elsevier高被引学者（化学），2016年Elsevier全球材料科学与工程学科高被引学者，2018-2020科睿唯安“高被引科学家”（化学）。

学术兼职有Applied Catalysis B 副主编，中国感光学会副理事长兼光催化专业委员会主任，北京市室内与车内环境净化行业协会会长。中国分析测试协会常务理事，中国化学会环境化学专业委员会委员；环境与能源光催化国家重点实验室学术委员会委员；教育部资源化学重点实验室学术委员会副主任。
