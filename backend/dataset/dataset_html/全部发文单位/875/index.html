学术讲座（二十九）：梧桐山植被特征及物种多样性
生命与海洋科学学院　2022/5/26 17:11:00
  （钟沛基 2022280095）

时间：2022年5月28日  下午15:00

地点：仙湖会议室

主持人：张永夏 副教授

主讲人：曾伟 副研究员

主讲人介绍：

    曾伟，男，1982年出生，博士，副研究员。2010年毕业于中国科学院植物所，现就职于深圳市仙湖植物园，主要从事城市森林生态站建设及森林碳汇、植物生理生态等模型模拟研究。2010年工作后主要从事的工作和研究有：深圳城市森林生态定位观测研究站建设、森林碳汇及调控措施、省级林业活动与土地利用变化温室气体清单编制、森林生态系统服务功能动态分析、森林林冠截留水文特性研究，及常绿阔叶林典型树种生理生态特性对环境因子胁迫的响应研究，主持完成了国家省部级以上项目8项，在国内外期刊发表论文15篇，获得奖项7项。
