学术讲座| 县级国土空间总体规划实践与思考——以山东省郓城县为例
建筑与城市规划学院　2021/11/25 22:47:00
  （钟沛基 2022280095）


 


讲座概要

   当前全国各地的国土空间规划编制工作全面展开，国土空间规划融合了多种规划类型，完成了“多规合一”，规划技术既有延承，也有创新。在近年的规划编制实践中相关规划思路、方法和技术逐渐稳定，系列规划技术标准依次发布，部分市县的国土空间规划公示稿也相继推出。作为国土空间规划体系的重要组成部分，县及乡镇层级的国土空间规划独具特色。本讲座以山东省郓城县为例，主要介绍县层级国土空间总体规划的实践，讲解县、乡镇层级国土空间规划编制的技术难点和突破方法，为国土空间规划的编制提供案例和技术参考。

 

主讲人简介

   杨箐丛，广州市城市规划勘测设计研究院国土与空间规划设计所总工，城市规划高级工程师，国家注册城市规划师，曾负责及参与昆明、大理、泉州、黄石、山东郓城、宁夏平罗、佛山南海等地的战略规划、国土空间规划、城市总体规划或“多规合一”工作。曾获全国优秀城乡规划奖10项，华夏建设科学技术奖1项，其他奖项20余项。参与自然资源部课题《城镇开发边界和城镇建设用地分类标准研究》、住建部课题《基于“多规合一”的规划协调机制研究》等省部级课题。合著《多规合一》1部，发表学术论文多篇。
