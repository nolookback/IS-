讲座题目：城市设计方案前的判断

讲座时间：2023年5月11日（星期四）10:00

讲座地点：建筑楼C205

讲座介绍：本次讲座从“多元是城市的核心价值”开始，以案例详细讲述城市之所以多元，是因为社会是由具有自由意志的人所构成。因个体意志和条件的特殊性形成的城市多样性是任何设计师都无法达到的。简单地想要透过空间设计规制安排人的生活，是有害的设计出发点；而想要摆脱千城一面的困境，需要拥抱个体意志给城市空间带来的变量，也需要从调研工具到设计方法进行全面创新。

主讲嘉宾：吴文媛，深圳雅克兰德设计有限公司总经理

嘉宾简介：吴文媛女士是资深规划师，深圳市雅克兰德设计有限公司总经理，“低冲击发展在中国”研究小组召集人，“中国城市化贡献力人物奖”获得者。吴女士是以存在优先为首要原则的规划理论——“消极规划”的主要倡导者，一直以来主张对社会、经济、自然、文化等方面展开综合研究之后提出规划设计的空间策略，并运用到大量实践案例之中。吴女士于2014年10月获得第六届世界健康城市联盟会议DWA（Design for Well-being Award）空间设计优异奖。
