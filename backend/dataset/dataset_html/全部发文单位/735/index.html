嘉宾简介：张先飞，二级教授、博士生导师、日本东京大学博士后研究员，现任《河南大学学报》（CSSCI）副主编。国家“万人计划”哲学社会科学领军人才、河南省首位教育部“青年知名学者”、中宣部文化名家暨“四个一批”人才、享受国务院政府特殊津贴专家、首批河南社科名家、日本东京大学集刊《赤门华风》主编。被钱理群先生誉为能够代表新一代中国现代文学研究发展方向的学术新人。主持国家社科基金重点项目1项、一般项目3项，霍英东教育基金会第十届高校青年教师基金项目、日本政府博士奖学金项目。获教育部第六届、第八届“高校科学研究优秀成果奖”三等奖，首届“唐弢青年文学研究奖”二等奖，河南省社会科学优秀成果奖一等奖2项、二等奖5项，河南省文学艺术优秀成果奖。出版系列专著《“人”的发现：“五四”文学现代人道主义思潮源流》《“人的文学”：“五四”现代人道主义与新文学的发生》《“人”的觉醒：现代中国文学“人学”思想探源》。
