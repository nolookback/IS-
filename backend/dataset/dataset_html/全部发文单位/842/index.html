学术讲座|城市认知的行与知
建筑与城市规划学院　2022/7/12 18:59:00
  （钟沛基 2022280095）

讲座时间：2022年7月19日（星期二）14:15

讲座地点：腾讯会议（154-134-053）

讲座介绍：城市认知是学习城市规划的重要一步，通过城市认知加深对城市发展现象和规律的认识，本讲座将对城市认知的方法、案例和经验进行介绍。

主讲嘉宾：赵虎，山东建筑大学建筑城规学院教授。本硕博分别毕业于济南大学、南京大学和东南大学，2012年入山东建筑大学从事教学科研工作至今，2014年担任学院第七工作室主持人，2017年任总规教研室副主任，2018年挂职长清区科技副镇长，2019年入选山东高校青创人才引育计划立项建设团队带头人，同时兼任中国城市规划学会城市规划历史与理论学术委员会委员。
