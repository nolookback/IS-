报告人简介：

唐惠儒，复旦大学特聘教授、国家杰青、“精准医学”及“前沿生物技术”国家重点研发计划项目首席、英国皇家化学会会士、伦敦大学博士。曾任英国帝国理工学院生物医学部Senior Scientist、“中国科学院生物磁共振分析重点实验室”创建主任、科技部973与蛋白质科学重大计划等项目评审专家。

研究代谢物功能及代谢组学30余年，发表Nature、Nat Microbiol、Cell Res、PNAS及JACS等SCI论文220余篇，被引1.4万余次（h指数~66）。获批国内外发明专利多项。

现任中国生物物理学会代谢组学分会会长；中国抗癌协会肿瘤代谢分会、中国营养学会基础营养分会、中国生物化学与分子生物学会脂质与脂蛋白委员会常务理事，国际实验磁共振大会（ENC）执委（大中华地区唯一）；Metabolomics、Arch Pharm、《基础医学与临床》等编委，Nutrition Metabolism及Phenomics等副主编。

讲座摘要：

机体小分子代谢物的异构体在疾病和生理中具有重要的功能，通过异构体分辨的定量代谢组学研究将为个性化医疗提供了重要的分子层面信息。本次讲座旨在介绍代谢物的异构体特征、异构体定量代谢组学新技术发展，以及在精准医学中的应用，探讨其如何助力糖尿病、高血脂、肥胖症等疾病的早期诊断、治疗反应的预测以及个性化治疗方案的制定，相信异构体分辨的定量代谢组学未来将是精准医学的重要组成部分。
