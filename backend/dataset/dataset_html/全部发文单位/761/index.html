【学术讲座】稀疏优化与基因调控网络
电子与信息工程学院　2022/12/13 10:39:00
  （钟沛基 2022280095）

报告时间：2022年12月15日下午3点

报告人：胡耀华 副教授，博士生导师

邀请人：李斌 教授

报告地点：沧海校区致信楼N710

 

摘要：稀疏优化是运筹优化的研究热点，在信息论、图像科学、机器学习等领域有着广泛的应用；非凸稀疏优化方法在应用中展现出强大的稀疏还原性与抗噪强健性。基因调控网络研究是通过基因之间相互作用来解释生命与进化的现象，是生物医药研究的重要工具。本报告将介绍稀疏优化与基因调控网络的桥梁，描述多种基因调控网络结构所对应的非凸稀疏优化模型。我们还将重点介绍非凸稀疏优化模型的数学理论，包括模型的相合性理论与渐进性理论，最优解的等价刻画，一阶算法的线性收敛速度与邻近点算子的显式迭代格式等。

 

报告人简介：胡耀华，深圳大学数学与统计学院副教授，博士生导师，广东省智能信息处理重点实验室学术骨干，国家优秀青年基金获得者，兼任中国运筹学会数学规划分会青年理事，中国运筹学会科普工作委员会委员。主要从事连续优化理论、算法与应用研究，尤其是非凸稀疏优化问题。先后主持国家自然科学基金4项，省市级科研项目9项。在SIAM Journal on Optimization, Journal of Machine Learning Research, Briefings in Bioinformatics等国际期刊发表论文40余篇，授权3项国家发明专利，开发多个生物信息学工具包与网页服务器。
