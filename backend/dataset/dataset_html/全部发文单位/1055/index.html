学术讲座——加拿大双语现象的前世今生
外国语学院　2021/6/17 9:09:00
  （钟沛基 2022280095）

主讲人：熊伟

时间：2021年6月17日（周四）下午14:30—16:00

地点：深圳大学汇文楼办公区2楼1200会议室

 

主讲人简介：熊伟，深圳大学外国语学院西语系讲师，中加访问学者，主要研究领域法语语言学、法语国家与地区社会与文化研究。

 

讲座摘要：

英语和法语同为加拿大官方语言，这是加拿大有别于其他国家的一项特殊政策，这一政策的制定有着深刻的社会、历史原因。它是加拿大英裔和法裔两大民族间矛盾难以调和的产物，互相妥协的结果。实行双语政策是加拿大维护国家团结统一、民族平等的基石。然而近些年随着其他族群移民数量的增加，走在多元文化交叉路口的加拿大该如何维系双语政策？在魁北克捍卫法语权力愈演愈烈的背景下，由语言折射出的文化和身份认同又使得加拿大双语政策的未来充满挑战。让我们从加拿大身上来体验语言，文化和身份认同的微妙联系。
