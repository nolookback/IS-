讲座题目：可持续生产与可再生能源协同优化研究——组合优化与机器学习相结合

讲座时间：2022年12月22日（周四）10:00-11：00

讲座地点：腾讯会议(会议号：163-755-123)

主讲嘉宾：崔维伟副教授　　上海大学

主讲嘉宾简介：

　　崔维伟，上海大学管理学院管理科学与工程系副教授/副系主任，2017年于上海交通大学获得工学博士学位，主要研究领域为可持续生产运作与绿色供应链管理，长期致力于将运筹优化理论应用于解决生产调度、设备维护、能源分配的集成问题。主持国家自然科学基金1项、上海市软科学领域重点研究课题2项、上海市浦江人才计划1项，以第一作者在NRL、COR、IJPE、JORS、系统工程理论与实践等国内外一流期刊发表论文十余篇，所发表成果曾获得国家一级学会“青年论坛新锐奖”、高被引论文、高影响力论文等荣誉，并多次在IEEM、ICPR等国际学术会议中组织分论坛并担任分会场主席，同时担任EJOR、CIE、Energy等多个国内外期刊审稿人。

讲座主要内容：

　　可再生能源的推广与普及为解决化石能源带来的环境污染问题提供了有效途径，然而可再生能源发电的随机性与动态性为其在生产制造领域的应用带来很大困难，如何从调度层面实现生产计划与能量分配的有效匹配成为了发挥其最大效用的关键。本报告将从两个研究问题出发，阐述该类问题具有的一些共同特点，介绍组合优化、机器学习两类方法的融合契机以及算法设计中的难点，并就该领域中逐渐兴起的研究方向进行简要探讨。
