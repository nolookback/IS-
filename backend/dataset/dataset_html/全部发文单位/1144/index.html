化学与环境工程学院　2021/3/25 13:12:00

主讲人: 董源教授

时间: 3月25日下午3:00

地点: 丽湖校区守信楼（原B1栋）518会议室

 

报告人简介：

董源，杭州电子科技大学机械工程学院教授，国家海外高层次人才引进计划青年项目入选者，本科与博士毕业于清华大学，美国麻省理工学院访问学者，2014-2019 年在美国密苏里大学哥伦比亚校区担任研究助理教授等职。主要从事智能制造、人工智能、工程热物理、微纳材料、高性能计算等交叉学科领域研究工作。以第一作者、通讯作者在国际顶级期刊发表 SCI 论文20 余篇。 兼任中国工程热物理学会传热传质分会青年工作委员会委员，深圳市人工智能产业专家委员会人工智能材料应用首席科学家，宁波市能源材料与技术重点实验室学术委员会委员。任《2020 年深圳市人工智能产业发展白皮书》、《福田区人工智能产业现状和发展战略研究》编委。曾获清华大学优秀毕业生，清华大学优秀博士论文一等奖，中国工程热物理学会吴仲华优秀学生奖等荣誉。
