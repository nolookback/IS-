“社群阐释与实践常规：文学新闻的美国与中国经验”学术讲座在传播学院举办
传播学院　2023/3/15 14:42:00
  （钟沛基 2022280095）

2023年3月3日，陕西师范大学刘蒙之教授受邀为传播学院研究生主讲了一场以“社群阐释与实践常规：文学新闻的美国与中国经验”为主题的学术讲座。本次讲座由传播学院彭华新副教授主持。

刘蒙之老师曾任中国写作学会理事、澎湃非虚构写作大赛学术评委、腾讯谷雨丹枫推荐委员会委员、阅文探照灯书评人奖评委、海峡两岸出版与文化创意论坛常务理事等职。刘老师通过本次讲座与大家分享其关于中美文学新闻实践的学术研究。

讲座伊始，刘蒙之老师向大家介绍了研究缘起。他指出，在新闻业向技术主义靠拢的背景下，其社会科学模式扩张与计算转向的趋势越发忽略了新闻报道中的人文与道德传统，使得新闻业逐渐变成一个寻求控制的领域。无论在西方还是国内，文学新闻都被寄予拯救人文与道德传统缺失的想象性厚望。

随后，刘蒙之老师分别从美国文学新闻的社群阐释、中国文学新闻实践社群的实践常规、中美文学新闻的公约数与反思这三个方面详细阐释了其对文学新闻的思考。在第一部分，刘老师回归到文学新闻的“原产地”美国，介绍了文学新闻的定义、边界设定与面对质疑时的正当话语；在第二部分，刘老师将文学新闻落到本土语境，阐述了中国文学新闻的公共价值与采写规范；在第三部分，刘老师针对中美文学新闻共同存在的三个问题——解释模式、文学技巧和情感使用进行了深刻反思。

最后，刘蒙之老师引用Hartsock的隽语“并不是说一种新闻形式优于另一种，‘考虑到两者的优势’才是我们的责任”作为本场讲座的总结，引发了在场同学们的共鸣。

彭华新副教授对此次讲座进行总结发言，表示刘老师对文学新闻及传统新闻的比较思考能够极好地启发同学们未来的学术研究。参会同学纷纷表达了对刘老师的衷心感谢。至此，本次讲座圆满结束。
