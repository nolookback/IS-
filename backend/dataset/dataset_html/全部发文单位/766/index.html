报告人简介：

林章凛教授，教育部知名学者特聘教授，全国优秀科技工作者（2014）；国家973项目首席、国家重点研发专项项目负责人。清华大学化学工程和经济学双学士、美国马里兰大学分子与细胞生物学博士；博士后师从加州理工学院Frances Arnold教授。一直专注于分子生物学和工程学（数学）交叉研究。近5年主要从事蛋白质设计、生物制药、合成生物学的应用基础研究和商用技术研究。现任中国生物工程学会合成生物学专业委员会副主任、省政协委员等。曾任工业酶国家工程实验室副主任等。关注国家科技创新与创业，合著有《科技创新与强国之路》中“美国篇”章节（张先恩教授主编）。
