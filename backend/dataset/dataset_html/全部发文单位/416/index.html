【学术讲座】面向癫痫致痫灶定位的脑MRI影像数据分析
电子与信息工程学院　2024/4/29 9:27:00
  （钟沛基 2022280095）

报  告 人：康莉 博士 深圳大学

报告时间：2024年4月29日周一上午10：00-11：00

报告地点：汇星楼1504


报告人简介：主要从事信号与信息处理等领域的教学和科研工作，近年来重点关注智能信息处理技术、压缩感知技术、DSP技术等在低空、超低空防空预警和通用航空飞行器监视等领域的应用研究，完成了国家科技支撑计划项目“通用航空综合运行支持系统”和深圳市科技计划项目"基于贝叶斯压缩感知的无线传感器网络数据压缩方法"、广东省自然科学基金"城市低空目标多源协同探测关键技术研究"等课题，突破了低空目标的多传感器组网信息融合、目标检测、目标识别和目标跟踪和精确定位等多项关键技术和装备。

报告摘要：医学影像处理是应用计算机技术和图像处理算法对医学影像进行分析、增强、分割及识别等操作的过程。利用计算机对医学影像进行后处理，可客观、量化、准确地分析影像结构和功能异常，减少医生主观经验造成的影响，已成为医学领域数据分析的发展趋势。本报告以癫痫致痫灶的定位为例，对医学影像处理中涉及到的一些关键问题进行讨论，包括医学影像的小样本处理、弱监督学习、多视角分析等，并对后续工作进行展望。
