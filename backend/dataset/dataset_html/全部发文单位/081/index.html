深圳民族宗教研究基地学术讲座第十六场 数字时代的人论：历史纵深与当前挑战
人文学院　2025/3/25 14:23:00
  （钟沛基 2022280095）

主讲人：洪亮 教授　（华中科技大学）

时 间：2025年3月28日　 10:00

地 点：深圳大学粤海校区汇文楼H1-401

　

主讲人简介：

洪亮，德国图宾根大学博士，华中科技大学哲学学院教授，博士生导师，楚天学者，中国政法大学宗教与法律研究中心研究员，汉语基督教文化研究所学术委员暨特聘教授，《道风》学刊（A&HCI）编委，主编商务印书馆“数字时代的人论”（2025-）文库。 

 

内容简介：

数字技术的加速发展催生出诸多新议题，也使“人是什么”这个经典的人论议题呈现出迥异于近现代的问题聚焦，本次讲座将从历史渊源和当前状况两个角度，结合“数字时代的人论”商务文库规划，剖析这一复杂议题，总结其对基础性人文学科的挑战与意义。
