学术讲座：华南理工大学牛保庄教授—竞合供应链中的技术依存和生产时序决策研究

讲座题目：竞合供应链中的技术依存和生产时序决策研究

讲座时间：2021年11月19日（周五）10:00-10：40

讲座地点：明德楼402

主讲嘉宾：牛保庄教授  华南理工大学

主讲嘉宾简介：

　　牛保庄，华南理工大学教授、博导。国家杰青，教育部重要人才计划入选者，国家优青，广东省政协委员。在中国跨境供应链管理领域做出系列研究，已有11篇论文发表于国际顶级期刊，包括MSOM 2篇，POM 6篇，TRB 3篇，另有60余篇文章发表在Risk Analysis, EJOR, Omega等知名期刊。成果曾获第八届高等学校科学研究优秀成果（人文社会科学）二等奖、中国管理学青年奖，中国信息经济学优秀成果奖、广东省哲学社会科学优秀成果一等奖、二等奖等奖项。培养的博士生应届获聘海外名校教职，为华南地区首次。

讲座主要内容：

　　本文刻画的是中国代工企业推出自主品牌产品时，自主创新程度以及带来的消费者需求不确定性风险之间的权衡。在该代工企业和它服务的品牌商形成的的竞合供应链中，以生产时间决策为关键决策，本文重点刻画了在自主创新需求波动较大情形下，代工商与品牌商如何共生共赢的可行区间。研究发现，由于存在合作-竞争的关系，推迟生产策略（PS）并不总是使代工企业或品牌商受益。企业对生产时间的决策取决于其自主创新产品的市场需求不确定性程度和成品市场上的竞争激烈强度。当需求不确定性很高时，两家公司都选择事后生产策略（PS）；当需求不确定性为中等或低时，只有其中一家选择事后生产策略（PS）。有趣的是，当成品市场竞争激烈时，代工商倾向于选择事后生产策略（PS），这种策略可以使它解决市场需求不确定性，获得信息价值，并通过溢出效应从品牌商的先动优势中受益。
