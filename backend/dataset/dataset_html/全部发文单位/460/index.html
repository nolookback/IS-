【生物医学工程学院学术讲座】脑-计算机接口一变革性的人机交互新技术：人类的梦想
医学部　2024/4/7 15:43:00
  （钟沛基 2022280095）

主讲嘉宾：伏云发 教授

时间：2024年4月10日   周三晚上19:00-21:00 

地点：深圳大学丽湖校区A2-517

主持人：梁臻 副教授


报告内容简介：

包括1）人类的梦想之一：利用脑电波直接控制外部设备；脑-计算机接口（BCI）是什么？为什么是变革性的？ 脑-计算机接口（BCI）能做什么？做到了什么程度？不能做什么？ 脑-计算机接口（BCI）的产业转化现状如何？如何转化？ 脑-计算机接口（BCI）不准确或偏差认知有哪些？ 脑-计算机接口（BCI）伦理规范应考量哪些？ 2）基于脑电的神经反馈训练（NFT）是干预创伤后应激障碍（PTSD）的一种具有潜力的方法。然而，目前鲜有评估NFT 干预PTSD患者期间脑状态动态变化的方法。为此，本研究针对卢旺达种族屠杀事件引起的女性PTSD患者的NFT数据，提出基于过程噪声动态自适应的中游介观网络（Process Noise Dynamic Adaptation Based Midstream Mesoscopic Network, PNDA-MMNet），以表征并评价NFT期间PTSD患者脑状态的动态变化。本研究分类结果表明了所提新模型PNDA-MMNet可有效表征NFT对PTSD患者脑状态的改变，可望用于量化评估NFT对PTSD患者脑状态的作用。
