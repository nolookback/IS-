【学术讲座】非均匀阵列信源到达角估计
电子与信息工程学院　2022/6/29 10:25:00
  （钟沛基 2022280095）

报告人：邱伟，国防科技大学 硕士生导师，

报告时间：2022年7月2日（周六）晚上19:00-22:00

报告地点：致信楼N710，腾讯会议：161-205-891，会议密码：220702

邀请人：孙维泽 助理教授

报告摘要：

利用传感器阵列获取信源的方位在雷达、声纳、通信等领域具有重要的应用，方位估计的分辨率与阵列的孔径大小有关。非均匀阵列能够以较少的阵元获取较大的阵列孔径，因此受到了很多关注。本报告针对非均匀阵列的信源到达角估计问题，介绍了两种阵列构型及相应的到达角估计方法。第一种从均匀阵列中随机选取阵元构成非均匀阵列，并利用压缩感知原理，通过协方差矩阵的空域稀疏性实现信源的到达角估计；第二种是采用一种阵元分布满足平方律的非均匀阵列，通过分数阶傅里叶变换的思想实现信源到达角重构。此外，对水声信号处理以及其他物理场测量与应用的相关问题做了介绍与展望。

 

 

 

报告人：邱伟，博士, 硕士生导师，国防科技大学气象海洋学院副研究员，感兴趣研究方向包括：多源信息融合、阵列信号处理以及海洋信息处理等，发表SCI/EI论文20余篇（包括IEEE TGRS, TIP, TAES,TCI，GRSL等），主持国家自然科学基金青年1项、面上项目1项，博士论文获批2017年湖南省优秀博士论文。
