学术讲座（二十九）：“新课改后对中学教师素养的新要求”
化学与环境工程学院　2021/4/22 11:22:00
  （钟沛基 2022280095）

主讲人:江伟

时间: 4月29日下午3:30

地点: 丽湖校区守信楼（原B1栋）420会议室

 

报告人简介：

江伟，1990年西南师范大学全日制本科毕业，深圳市红岭中学化学教师，深圳市红岭中学高中部副部长，中学高级职称。深圳市化学学科兼职教研员，福田区化学兼职教研员，曾获深圳市优秀教师、深圳市高考工作先进个人、全国高中化学竞赛优秀教练员等荣誉称号，现担任福田区2020年“未来教育名师”工作室主持人，广东省基础教育化学学科教研基地负责人。2014年至2020年，连续担任15次深圳市高三年级调研考试化学学科命题人，多次在深圳市高考备考经验交流会上做主题发言，多次承担深圳市模拟考试的全市质量报告撰写和分析发言任务。受邀到深圳市各区及广东省各地进行高考备考和命题方面的专题讲座（二十余场次），具有较大的影响力。积极响应政府号召，去广东省教育贫困市县进行帮扶，通过展示公开课、专题讲座等多种形式，提升当地教师的科研能力。主持省课题1项、市课题5项、区课题3项，在《化学教育》等核心杂志上发表论文28篇，出版专著1部，参编著作5部。
