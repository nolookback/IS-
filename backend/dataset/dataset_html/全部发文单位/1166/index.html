讲座题目：温度会降低市场热度吗？——来自 A 股市场的证据

讲座时间：2021年1月21日10: 00

讲座地点：腾讯会议  会议ID：773 835 283

主讲嘉宾：熊熊教授  天津大学管理与经济学部


主讲嘉宾简介：

　　熊熊，天津大学管理与经济学部教授、博士生导师，副主任。研究领域：大数据金融，计算实验金融学，企业发展与金融策略等。在国内外高水平学术期刊上发表学术论文80余篇，出版专著2部。主持国家自科基金重点项目“基于大数据的金融创新及其风险分析理论”等10余项国家、省部级项目。获得省部级以上学术奖励3次。2007年获得教育部“新世纪优秀人才支持计划。2010年获得天津市青年科技奖。任中国系统工程学会副秘书长，金融系统工程专业委员会主任，青年工作委员会副主任。中国优选法统筹法与经济数学研究会青年工作委员会副主任，中国运筹学会决策分会副理事长。

讲座主要内容：

　　温度的持续上升一方面造成极端天气事件发生频率增加，破坏公司的正 常生产秩序，另一方面会引起环保政策的变化，导致公司生产成本上升甚至停业 整顿，因此是上市公司面临的一个不可忽视的风险因素。文章运用了 2007 年至 2019 年中国上市公司所在地的温度数据以及上市公司的股价数据和财务数据， 研究了温度持续上升对上市公司的产生的影响以及影响程度的决定因素和作用 机理。实证结果表明，上市公司所在地温度的持续上升会给上市公司带来一个显 著的负向冲击，并且当公司的规模越小，账面市值比越高的时候，这种负向冲击 产生的后果更加明显。文章采用了多种方法对上述结果的稳健性进行了分析。
