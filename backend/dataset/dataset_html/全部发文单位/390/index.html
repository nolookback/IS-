学术讲座（四十三）：中国总膳食研究及应用
化学与环境工程学院　2024/5/20 10:46:00
  （钟沛基 2022280095）

主讲人：赵云峰研究员

时间：5月23日下午16:00

地点：丽湖校区守信楼420会议室

 

报告人简介：

赵云峰，国家食品安全风险评估中心研究员，化学实验室主任，第二届食品安全国家标准审评委员会理化检验方法与规程专业委员会主任委员，国家卫生计生突出贡献中青年专家。

报告摘要：

报告将简要介绍中国总膳食研究和母乳监测，举例分析全氟和多氟烷基化合物的膳食暴露。通过本次讲座，参与者将获得中国总膳食研究的前沿成果，理解其在食品安全、医学研究和临床实践中的应用价值，以及面临的挑战和机遇。

欢迎有兴趣的师生参加！
