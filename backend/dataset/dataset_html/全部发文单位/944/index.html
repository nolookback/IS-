学术讲座：星火沙龙----视听技术多场景应用
信息中心　2021/12/7 15:54:00
  （钟沛基 2022280095）

讲座题目：视听技术与应用环境

主讲嘉宾：钟晓流（清华大学深圳国际研究生院）

讲座时间：2021年12月8日（周三）下午2:30

讲座地点：深圳大学沧海校区致理楼C座L3-1115会议室

 

主讲嘉宾简介：

钟晓流，清华大学信息技术中心副总工程师，《现代教育技术》主编，中国教育技术协会副会长、中国教育技术协会人工智能专业委员会理事长，全国信息化标准委员会教育技术分会专家，教育部信息化标准委员会专家。

作为主要技术负责人或总设计师，钟教授主持规划设计的重大工程建设项目包括：清华大学信息化教学环境，深圳大学城清华大学研究生院、北京大学研究生院和哈工大研究生院会议中心及多媒体教学环境，国家会计学院国际会议中心、人民大会堂多媒体视听环境、华中科技大学智慧教学环境、浙江大学智慧教学环境等。作为标准化工作组组长或首席专家，钟教授主持了《智慧校园总体框架》、《多媒体教学环境设计要求》、《数字化语言学习环境设计规范》四项国家标准和教育部《信息化教学环境视听健康设计规范》、《深圳市云端学校设计及装备配置》等标准的研制工作。他既有扎实的技术理论基础，也有丰富的实践经验，著述颇丰，较有影响的专著及文章包括：《多媒体视听技术与应用环境》、《实用录播技术与应用系统》、《教学环境设计》、《e-control技术及其在教学环境中的应用》、《i-时代的教学环境与评价体系》和《智慧校园大数据平台顶层设计》等等。

 

讲座主要内容：

本讲座全方位剖析影响各类型信息化应用环境视听效果的建筑物理因素、信息装备因素及其之间的关联性，从视觉效果设计、听觉效果设计等维度诠释各类应用环境的构建与集成。
