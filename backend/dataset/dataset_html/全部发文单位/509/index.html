学术讲座（七）：不对称还原加成反应
化学与环境工程学院　2024/1/22 15:33:00
  （钟沛基 2022280095）

主讲人：陈宜峰教授

时间：1月23日下午15:30

地点：丽湖校区守信楼420会议室

报告人简介：

陈宜峰，华东理工大学教授，博士生导师，入选国家海外高层次人才（2017年），上海市启明星项目（2020年）。陈宜峰教授2007年本科毕业于苏州大学（导师 : 邹建平教授），2012年博士毕业于中科院上海有机所（导师 : 刘元红研究员），2013年麻省理工博士后工作（合作导师：: Stephen L. Buchwald教授），2014-2017年耶鲁大学博士后工作（合作导师 : Timothy R. Newhouse教授）。长期围绕立体专一性偶联研究领域，聚焦于多手性中心精准构筑和不饱和烃可控官能团化等关键科学问题从事有机合成新反应、新配体的开发与创制，取得了以下成绩：开发了手性8-喹啉咪唑啉配体，并将其应用于不对称还原偶联反应高效合成含多手性中心内酰胺骨架；发展了内烯烃参与的三组分不对称偶联反应，模块化地构筑含多个手性中心高度官能团化的线性烷烃分子；实现了丰产金属镍催化条件温和的羰基化偶联反应，创制了不饱和烃的酰基金属化新反应范式。迄今已发表SCI论文50余篇，其中以第一或通讯作者发表包括Nature. Chem. 1篇、Nature Synth. 1篇、Angew. Chem. 8篇、JACS 4篇、Nature Commun. 2篇。
