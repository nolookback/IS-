【学术讲座】光网络物理层信道安全编解码技术研究
电子与信息工程学院　2023/12/11 15:03:00
  （钟沛基 2022280095）

报告嘉宾：李沼云，中国电子科技集团公司第三十四研究所教授级高级工程师

邀 请 人：王可 副研究员

时    间：12月17号上午10：00

地   点： 致信楼N710

 

报告摘要：针对光物理层高速、大容量信息抗截获传输需求，分析了光网络物理层信道安全编解码技术发展趋势，提出了基于时间和幅度两个维度对含噪声光信号进行编码的方法，将信号有规律的0和1数字特征改变为复杂的物理波形，窃听方通过直接探测方式对这种编码波形所携带的信息状态难以进行识别和提取，从而增加了窃听者对光信息的破译难度。

 

主讲人简介：李沼云，中国电子科技集团公司第三十四研究所教授级高级工程师。多年来一直致力于光网络物理安全技术研究，先后承担了国防预先研究和型号研制、军科委国防科技创新特区、173基础增强研究、测试仪器型号、广西科技创新驱动发展重大专项等多个项目；发表论文20余篇，申请专利10余项，其中国防专利8项；获得广西科技进步二等奖2项，军科委“XX行动”网络安全竞赛全军一等奖1项。
