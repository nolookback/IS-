“从地点到地方：新闻的在地”学术讲座在传播学院举办
传播学院　2024/1/19 14:46:00
  （钟沛基 2022280095）

美国圣克劳德州立大学教授彭增军教授于2023年12月13日受邀为传播学院师生主讲了一场主题为“从地点到地方：新闻的在地”的学术讲座。讲座由传播学院副院长杨洸主持。

讲座中，彭增军教授主要讨论了新闻的距离与公众信任问题。他首先说明了新闻的现状：失信的新闻与失落的真相，之后具体解释了什么是在地和在场，认为新闻的“三在”中“在地”最为优先。在场和在地涉及到新闻五W中的where，新闻学的在场就是新闻生产过程中的现身，在场可以理解为是新闻的标签，新闻是真实的在场，是在场的此时此地。他认为，新闻的在地虽然与在场有许多重合，却比在场更进一层。在场是主体（记者）同客体（报道对象）距离的缩短，在地是主体同客体时间上的同步和空间的一体。

彭教授重点分析了在地的重要性。在地性是新闻机构作为一方水土的政治和文化象征。在地就是将一个空间地点转化为地方，再而变为公共领域的过程。在地可以看见在场看不到的东西，能引起共情从而进行公共服务。彭教授认为在地性的削弱和缺失恰恰是新闻业的软肋。赢得信任，不仅仅需要在线、在场，更需要的是在地。在地性是恢复和重建公众信任的重要途径，有了在地性，有了共享的现实，才能有认同。没有在地的主观，新闻的客观无从谈起。

最后，彭教授与学院老师和同学们进行了深入交流，解答了师生们的问题。
