报告人简介：

Robert Young，英国皇家科学院、皇家工程院、欧洲科学院院士，曼彻斯特大学材料学院创院院长。研究涵盖高分子和复合材料的结构与力学性能之间的各个方面，他在该领域出版了多本书籍和超过450篇论文，总被引50000余次，H指数90。他与英国、欧洲大陆和中国的学术界和工业界合作开展了许多研究项目。他应用拉曼光谱技术监测高性能纤维和复合材料的变形过程已经得到了广泛的发展，并扩展到其他材料，如碳纳米管和蜘蛛丝与石墨烯。
