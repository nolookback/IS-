主讲人介绍：

李玲，华南师范大学生命科学学院教授，博士生导师，教育硕士导师。现在是华南师范大学研究生督导委员会委员。

主持国家自然科学基金面上基金项目、广东省自然科学基金重点项目、教育部博士点基金项目、广东省科技计划项目等多个项目。发表学术论文130篇，其中SCI收录40多篇，获得国家发明专利8项，获广东省科学技术奖励二等奖1项。出版教材5部，2021年获教育部全国优秀教材（高等教育类）二等奖，主持国家级一流课程、国家级精品课程、国家级精品资源共享课程“植物生理学”。主特与完成多项教育部、广东省教学研究项目，发表教学研究论文20余篇。荣获广东省高等教育省级教学成果一等奖共3项， 第二届广东省高等学校教学名师。曾任教育部生物技术专业、生物工程专业教学指导委员会委员，教育部本科教学工作审核评估专家。
