经济学院　2022/1/10 9:10:00
  （钟沛基 2022280095）

讲座题目：数字经济时代背景下供应链转型

讲座时间：2022年1月18日（周二）20：00

讲座地点：腾讯会议（会议号：803-661-066）

主讲嘉宾：窦志武教授　　云南财经大学

主讲嘉宾简介：

　　窦志武，教授，博士生导师，教育部物流管理与工程类教指委委员，云南省物流管理与工程类教指委副主任委员，国家自然科学基金委项目评审专家，教育部学科评估评审专家，云南省教学成果一等奖获得者，行业科技进步奖二等奖获得者。

讲座主要内容：

　　全球数字经济的迅猛发展驱动基础关键领域供应链变革。数字经济的非线性作用将通过严密逻辑体系、有序演化的复杂系统作用与供应链创新。数字经济驱动的路径在于通过“互联网+”“人工智能+”“区块链+”为供应链协同创新赋能，不断增强面向现代复杂性经济的敏捷性、适应性和协调性。数字经济驱动的路径在于供应链组织与人的创新。

　　欢迎广大师生参加！
