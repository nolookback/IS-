“文化自信成为公共道德责任的理由”学术讲座在马克思主义学院顺利举办
马克思主义学院　2024/1/16 0:08:00
  （钟沛基 2022280095）

2024年1月4日，湖南师范大学二级教授、博士生导师、湖南省伦理学学会副会长彭定光教授受邀为深大师生带来了一场题为“文化自信成为公共道德责任的理由”的精彩讲座。本次讲座由深圳大学马克思主义学院副院长张守奎教授主持，众多师生到场聆听。

在讲座中，彭定光教授首先强调了自觉自愿承担文化自信这一公共道德责任的重要性，他指出，文化自信是每个中国人都应该承担的公共道德责任，事关人类文明新形态的确立和中国式现代化建设。这一公共道德责任不是纯粹外在的要求或者强制，而是每个中国人都应该发自内心地承担的。

彭定光教授进而从三个方面进行了详细的阐述。第一，自觉自愿承担文化自信源自文化与人的一体性。具体而言，人与文化的一体性指的是人与文化的内在相关性和不可分割性，人离不开文化，文化也离不开人。所以，每个中国人都不仅应承担文化自信这一道德责任，而且应该自觉自愿地承担。这表面上是对于与人内在相关的文化的道德责任，实际上既是对于人类和中华民族的责任，又是对于个人自己之所以成为人的责任；既是对于人的生存的责任，又是对于人的发展的责任。第二，自觉自愿承担文化自信基于文化的实践性。他指出，文化的实践性指的是文化与实践是内在一体、密切相关和同构共存的，实践需要文化，实践也内生文化和传承文化。每个中国人都应该对让自己得以安身立命的文化予以充分的肯定和坚定的自信，都应该依赖和依据中国文化来保持定力，激发动力，提高能力，当仁不让地在走向中国未来的实践中发挥自己的聪明才智，坚定不移地对实现中华民族伟大复兴这一实践充满自信。第三，自觉自愿承担文化自信出于文化作用的为人性。彭教授指出，文化既是人为的，是由人创造出来的，又是为人的。就中国文化来说，它在根本上是为人的，既为了每个中国人的生存发展，又为了中国国家共同体的建构和中华民族对于自由、幸福的追求。文化的为人性有多方面的表现，在文化作用上的表现最明显也最容易被人直接感受。文化作用的为人性不仅使每个人都坚定文化自信，而且为其文化自信提供动力，使人们都自觉自愿地承担文化自信这一公共道德责任。总之，文化与人的一体性、文化的实践性和文化作用的为人性，既是每个中国人都应该自觉自愿地承担这一公共道德责任的理由，又是其履行该道德责任的动力。

讲座最后，张守奎教授对彭定光教授的精彩讲述表示感谢并做总结发言，指出这是一场内涵丰富、具有启发性的学术讲座。此次讲座内容政治站位高、逻辑严谨并且见解独特，提升了师生们的专业素养，同时也营造出良好的学术氛围。在热烈的讨论氛围中，讲座圆满落下帷幕。
