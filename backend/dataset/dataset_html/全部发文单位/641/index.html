报告人简介：

方千荣，吉林大学化学学院教授，国家杰出青年科学基金获得者。2001年，吉林大学化学学院本科毕业。2007年，获吉林大学化学学院无机化学专业博士学位，师从裘式纶教授。从2007年到2014，先后在美国加州大学洛杉矶分校、美国德州农工大学、美国加州大学河滨分校和美国特拉华大学从事博士后工作。2015年全职回吉林大学无机合成与制备化学国家重点实验室工作。主要从事新型多孔晶体材料的结构设计、定向合成及应用研究。先后在Nat. Chem., J. Am. Chem. Soc., Angew. Chem. Int. Ed., Nat. Commun., Adv. Mater. 等学术期刊发表SCI收录论文120余篇，论文被他引超10000次。
