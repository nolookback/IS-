嘉宾名称：沈悦，日本兵库大学风景园林管理学院院长、教授

 

讲座主持：何昉 教授

况达 助理教授

 

讲座时间：2022年3月15日10：15

腾讯会议号：788-6146-3737

 

讲座提要：

本讲座为风景园林的基本认知上的梳理，解释不同的概念，并逐层地阐述在不同背景下所形成的风景现象以及它们的固有特征。讲座还将通过大量实例分析来解释风景园林在设计方面所应进行的思考，阐述如何从文化中汲营养，如何推动实践，同时对行业的前景进行了展望。

 

嘉宾简介：

日本兵库县立大学景观设计管理学院院长 教授，日本兵库县立大学自然环境科学研究所次长。东京大学博士，北京林业大学园林学院兼职教授，日本神户国际艺术节召集人，日本淡路花园都市未来规划委员，淡路景观园艺学校教育研究部长。日本都市计划学会会员，日本造园学会会员，在研究领域获日本造园学会奖，设计领域获日本建设大臣奖。在实际设计工作中讲求地域特征，设计作品于日本，中国，澳大利亚东各地展开。先后于日本大型景观设计机构东京景观设计研究所、PREC研究所从事公园绿地等景观规划设计，项目涉及东京，横滨，长野，福冈，冲绳等日本各地，从国立公园，自然公园，都市绿地，社区公园设计，到世博会景区规划，花卉博览会出展，旅游度假区的规划及设施设计，以及农村空间活性化，传统街区再生规划设计等多方面。在近几年，沈悦先生开始参与一些国内项目，以多文化圈互融的视点展开着独自的理念。在理论研究方面以景观形成论为轴心，在景观构成解析，城乡景观建设以及传统文化景观与当代设计融合的手法上有独到的成果，论著多数。
