学术讲座：不可靠通信下车辆列队控制性能研究
机电与控制工程学院　2021/4/9 16:59:00
  （钟沛基 2022280095）

讲座题目：不可靠通信下车辆列队控制性能研究

演讲嘉宾：赵成成 特聘副研究员，浙江大学控制学院

讲座时间：4月15日（星期四）下午 14：00－15：30

讲座地点：深圳大学南校区机电大楼S601会议室

邀请人：邱丽老师

讲座内容简介：

       在智能交通系统中，车辆列队行驶需要基于分布式控制算法实现对车辆之间距离的有效控制。具有移动性、无物理耦合的车辆列队行驶控制，需要通过无线通信来实现分布式算法。这使得车辆列队行驶控制性能极大地依赖于车辆之间无线通信的质量。相比与有线通信，干扰、信号衰落迅速等因素使得无线通信的可靠性更差。因此，研究车辆列队系统在不可靠通信环境下的控制性能是具有极大的理论和应用价值的。在本次报告中，考虑了包括通信范围有限和时变延迟在内的不可靠因素，研究了基于均匀恒定时距间隔策略的车辆列队行驶控制问题。首先，对于每辆车利用多个前后车辆信息的情况，我们基于矩阵多项式的稳定性和矩阵特征值扰动理论，得到了保证车辆列队系统的内部稳定性的充分性条件。接着，我们证明了存在正实数上界当时变通信延迟在上界之内时，车辆列队仍可以保持内部稳定性。最后，我们通过传递函数来分析理想通信情况和具有均匀恒定延迟情况下系统L2- 弦稳定性。

赵成成特聘副研究员简介：

       赵成成博士2013年6月毕业于湖南大学电气学院，获工学学士学位，同年免试进入浙江大学控制学院直攻博士。2018年6月获控制科学与工程专业博士学位。2018年12月，入职浙江大学控制学院，从事博士后研究工作。2019年5月至2020年12月，赴加拿大维多利亚大学电子与计算机工程系从事博士后研究工作。目前在浙江大学控制科学与工程学院和网络空间安全学院担任特聘副研究员。赵成成博士长期致力于分布式控制与优化，网络系统安全与隐私、智能电网、车辆协同控制等方向的研究，承担国家自然科学青年基金、中国博士后面上基金等项目。发表控制/通信领域旗舰期刊/会议论文20余篇，其中IEEE/ACM汇刊论文10余篇（包括TAC、Automatica、TSG、TSP、TIE、TVT等）。研究成果获中国自动化学会优秀博士学位论文奖、中国电子学会优秀博士学位论文奖、IEEE PES-GM最佳会议论文奖等。受邀担任IEEE ICC'18和IEEE GLOBECOM'20)会议程序委员会委员等，并长期担任 IEEE TAC/TIE/TDSC/TVT 等数个国际权威期刊审稿人。
