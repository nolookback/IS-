主讲嘉宾介绍：

约翰·扎卡赖亚斯

自2012年起担任北京大学建筑与景观学院讲座教授，接受过城市规划专业的系统培训，并在该领域拥有丰富的专业经验。其研究关注于设计环境如何在各种城市尺度上影响行为，研究成果在ResearchGate上备受关注，并刊登在城市规划、城市研究、环境心理学以及环境-行为关系等领域的领先期刊中。


讲座介绍：

城市规划通常被认为是一个自上而下的过程，即私营部门的行动者通过实施计划、规则和条例相应地采取行动。尽管发展方向和指导原则是必要的，但由于机构之间复杂的相互作用和动态的环境，使城市政策很难控制一切。根据Hayek的自下而上组织原则，有时在经济理论中被称为“隐藏的手”，提议城市规划必须理解并与这些自下而上的过程合作。在本讲座中，利用在中国几个城市研究的案例来汲取经验以改进城市规划实践。
