学术讲座｜流域城水空间耦合关系及人类活动影响研究—以京津冀与莆田为例
建筑与城市规划学院　2021/4/20 17:54:00
  （钟沛基 2022280095）

讲座内容：流域作为相对封闭的地理单元，内部自然要素循环有其自身规律。城市作为人类活动的集中体现，与流域内以水为代表的自然要素相互耦合，形成今天的地表人类活动圈层空间格局。通过对华北平原及莆田平原长时序城水空间耦合关系研究，可以深入理解流域城市及其所在环境的演变规律，更好地引导流域城镇体系设计，提高区域可持续发展水平。

主讲人：杜龙江，中国水利水电科学研究院高级工程师，中国地理学会、中国城市规划学会、中国水利学会会员，长期从事流域规划、水利遥感、水利史、城市规划设计等相关研究。先后作为技术负责人或技术骨干主持完成多项国家重大科技支撑计划、”863”计划、城市总体规划、流域综合治理等咨询及规划设计项目，包括宁夏自治区水土保持规划、国家发改委太湖流域治理课题、天津战略规划水专题、沈阳水生态修复与保护规划设计等，获省部级科技进步二等奖2项，三等奖1项，目前主要研究方向为基于流域的城镇与水系统研究与设计。

时间：2021年4月23日

          周五 下午2：30-4：00

讲座地点：建筑与城市规划学院A103
