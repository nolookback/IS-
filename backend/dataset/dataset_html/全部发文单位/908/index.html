讲座题目：

《从规划引领到数据驱动治理——深圳实践与展望》


讲座简介：

　　　　深圳建市40年以来，始终坚持城市与交通协同发展，通过交通系统建设支撑并引领经济形态、产业结构、劳动要素、空间资源转型升级，并以城市交通规划理念和方法上的不断创新，创造了若干个“全国领先”。当前，随着我国超大城市相继迈入复杂综合立体基建协同建设、融合运营时代，亟需在科学化、精细化、智能化上下功夫，推动交通管理手段、管理模式、管理理念创新，让城市交通运转更聪明、更智慧。数据驱动的治理模式，必将成为城市交通高质量发展的全新引擎。
