【深圳大学40周年校庆800场学术活动（第420场）】“‘正能量’与‘大流量’：全媒体时代重大主题报道的视听生产”学术讲座在传播学院举办
传播学院　2023/4/4 16:05:00
  （钟沛基 2022280095）

2023年3月22日，深圳报业集团视听中心首席运营官张延、深圳报业集团视听中心栏目部负责人刘兴萍、深圳报业集团视听中心项目部负责人谢婷受邀为传播学院师生主讲了一场以“‘正能量’与‘大流量’：全媒体时代重大主题报道的视听生产”为主题的马克思主义新闻观学术讲座。本次讲座由传播学院副院长王建磊副教授主持。

3位主讲嘉宾在本次讲座上与大家分享了全媒体时代重大主题报道生产的宝贵经验。张延以深圳报业集团视听中心的发展历程为切入点，介绍了近年来视听中心获得的主要成就和产出的优秀作品，并重点阐释了“以新闻为先”的重要使命和以创新与传播效果为中心的生产理念。随后，刘兴萍发表了题为“围绕中心，服务大局：从‘思想、艺术、技术’层面深化重大主题报道融合创新”的主旨演讲，她特别指出，思想引领是重大主题报道的核心特征，艺术表达是拉近与用户距离的有效途径，技术运用是融合创新的重要引擎。紧接着，谢婷向在场的师生介绍了视听项目制作从前期策划到后期管理的全过程。之后，张延围绕新闻四力、媒体融合四力、国际传播三力这几个方面，对讲座内容进行了总结。

在提问与讨论环节，同学们踊跃发言，3位主讲嘉宾认真回答了同学们提出的问题。最后，王建磊老师对本场讲座进行了总结发言。至此，本次讲座圆满结束。
