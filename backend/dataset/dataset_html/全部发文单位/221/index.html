主讲人简介：

阿隆南（Yashadatta Somaji Alone），印度人。印度尼赫鲁大学美学与视觉研究学院教授。研究领域包括:古印度艺术、现代印度艺术评论、新佛教通俗视觉文化等。曾数百次受邀在印度国内和德国、西班牙、美国、中国等国家进行学术讲座，曾为中国深圳大学ICCR客座教授，并被北京人民大学、上海华东师范大学、马德里自治大学、德国海德堡大学聘为短期客座教授。出版专著《早期西印度佛教洞穴：形制和赞助人》（2014年），发表数百篇论文。他近期策划了以社会关注为中心的当代艺术实践展览，发展了“安贝德卡尔美学”（“Ambedkarian Aesthetics”）一词。他对哲学和文化批评而闻名，提出了一个关键的概念表述：“受保护的无知”（“protected ignorance”）。

 

讲座简介：

印度在殖民时期所发现的玄奘的文献，对重述印度佛教历史起到关键性作用。有趣的是，印度古代的视觉艺术为理解佛教叙事模式的形成也提供了重要信息。作为古印度佛教艺术遗址阿旃陀石窟是融合建筑、雕刻、绘画为一体的视觉艺术表现形式，其雕塑和壁画对中国和东南亚的佛教石窟艺术影响深远。本讲座结合玄奘的行记叙述和政治表达，通过解读阿旃陀的雕塑和绘画，探讨印度佛教视觉文化特征和图像叙事形式。
