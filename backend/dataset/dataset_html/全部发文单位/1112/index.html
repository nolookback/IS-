学术讲座：中国人民大学王玉涛教授—中国分析师报告有效性研究：特定信息与投资者有限关注
经济学院　2021/4/23 17:07:00
  （钟沛基 2022280095）


讲座题目：中国分析师报告有效性研究：特定信息与投资者有限关注

讲座时间：2021年4月26日15:00-16：30

讲座地点：汇文楼1500会议室

主讲嘉宾：王玉涛教授  中国人民大学

主讲嘉宾简介：

　　王玉涛，中国人民大学会计系教授。本科毕业于中南财经政法大学；并于中山大学取得管理学硕士学位；2010年于清华大学取得工商管理博士学位。王玉涛教授毕业后就职于中央财经大学，并于2018年加入中国人民大学。王玉涛教授主持并参与了多项国家级科研项目；在国内外重要期刊发表多篇论文；主要研究方向为证券分析师行为、投资者关系、公司估值于投资决策、盈余管理与信息披露以及企业会计准则。

讲座主要内容：

　　王玉涛教授将报告其最新的工作论文：《中国分析师报告有效性研究：特定信息与投资者有限关注》。该研究主要关注投资者是否能够有效区分和理解分析师报告中的多种信息。研究发现分析师报告中的盈余预测和股票推荐对投资者有不同含义：相较于盈余预测修正，分析师推荐评级调整具有更强的价值相关性；而盈余预测在分析师推荐评级调整的基础上提供依赖于评级调整信息的增量信息。
