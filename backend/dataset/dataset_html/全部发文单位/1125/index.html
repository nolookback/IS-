学术讲座（二十一）：从超疏水表面到液体橡皮泥

主讲人: 李晓光副教授

时间: 4月16日上午10:00

地点: 丽湖校区守信楼（原B1栋）420会议室

报告人简介：

李晓光，西北工业大学副教授。2008年本科毕业于同济大学物理学院，2013年获得同济大学凝聚态物理学博士学位，2010-2012年赴澳大利亚联邦科工组织接受联合培养。先后在同济大学材料学院和北京大学工学院开展博士后研究，2015年至今任职于西北工业大学物理学院。主要从事超疏水表面、液体弹珠、液体橡皮泥等非浸润体系的基础和应用研究。主持国家级科研项目3项，发表论文四十余篇，在液体塑形方面取得重要成果，提出了“液体橡皮泥”的概念，相关工作被 Discovery探索频道等媒体报导。

讲座摘要：

通常情况下液滴会润湿固体表面并粘附在上面，若能避免浸润，无论是对于固体材料还是液滴的应用而言都具有重要意义。为此有两种途径：一是着眼于固体表面，赋予其超疏水或超滑特性；二是着眼于液滴，利用疏水颗粒将液滴包裹起来形成“液体弹珠”。传统液体弹珠的制备以粉末为颗粒源，难以避免颗粒团聚、粗糙度大、透明度低、形态单一等问题。本报告介绍一种新的液体弹珠制备思路，及由其衍生出的液体塑形技术。利用一种结合力极弱的超疏水薄膜为颗粒源，获得了具有单层纳米颗粒结构（Monolayer nanoparticle-covered, mNPc）的超高透明度液体弹珠。利用界面堵塞效应，获得了像固体一样可任意塑形的“液体橡皮泥”。以mNPc液体弹珠为理想模型，系统阐明了液体弹珠有效表面张力的机制和测量原则；以mNPc液体弹珠和液体橡皮泥为自支撑微容器，成功进行了干细胞培养、气体传感、蛋白分离等应用探索。
