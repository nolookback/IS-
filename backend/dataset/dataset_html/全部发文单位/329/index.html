学术讲座（五十二）：南海近岸海域中新型替代阻燃剂的生态效应及研究挑战
化学与环境工程学院　2024/6/25 17:21:00
  （钟沛基 2022280095）

主讲人：徐向荣研究员

时间：6月27日下午14:00

地点：丽湖校区守信楼518会议室

报告人简介：

徐向荣，女，中国科学院南海海洋研究所研究员，博士生导师，中国科学院“引进国外杰出人才”（百人）计划入选者。长期致力于探索人类活动影响下的新污染物在南海典型热带河口、海湾及岛礁等近海生态系统中的环境地球化学过程和生态风险。主持和参加20多项国家或省部级科研项目。已在国内外重要学术期刊上发表论文200多篇，被SCI总引11000多次，H指数60。目前是中国环境科学学会沉积物环境专业委员会委员、中国土壤学会环境微塑料工作组第二届工作委员会主任委员、中国海洋与湖沼学会海岸带可持续发展委员会委员、中国海洋与湖沼学会化学分会理事、全国环境（海洋）微塑料污染与管控学术研讨会学术委员会成员，广东省环境保护厅环境咨询专家委员会专家、广东省应急管理专家委员会专家、巴塞尔公约亚太区域中心化学品和废物环境管理智库专家。现担任海洋环境领域重要SCI期刊Marine Environmental Research主编和其它5个期刊编委。入选科瑞维安2020年、2021年、2023年全球高被引科学家名单，2022和2023年度地球与环境科学领域全球前2%顶尖科学家榜单。
