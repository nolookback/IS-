主讲人：周路明  国家海外人才离岸创新创业基地（深圳）总裁

主讲人简介：

国家海外人才离岸创新创业基地（深圳）总裁、深圳市源创力离岸创新中心总裁、深圳市源创力清源投资基金董事长、香港大学前海智慧交通研究院理事长、中国源头创新百人会秘书长、大湾区数字技术创新青年百人会名誉主席。周路明先生历任深圳市科技局副局长、深圳清华研究院副院长、深圳市科协主席等职，任内主持制定了深圳市一系列科技创新重大立法、决策研究工作，推动了民办科研机构的理论研究与实践，扶持了一批顶尖技术团队成功实现产业化。
