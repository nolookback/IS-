[学术讲座]Learning to Clustering

  （钟沛基 2022280095）

报 告 人: 东南大学计算机科学与工程学院   贾育衡 副教授

邀请人：邬文慧助理教授

报告时间：12月17日（周五）下午5:00-6:00

报告地点：沧海校区-致信楼-N901会议室

 

报告摘要：Clustering is a fundamental unsupervised machine learning task which has been widely used and studied    in many applications like community detection, superpixel segmentation, web mining, recommendation systems, etc.    In this talk, I will explain the basic idea of clustering,  how to use learning techniques to achieve better clustering     performance, and present several works finished by my group that is related to clustering, e.g., self-supervised clustering, clustering with multiple views, ensemble clustering, and how to build deep clustering networks. I will also outline some possible research issues in clustering.

 

嘉宾简介：

贾育衡，江苏省双创博士，2019年获得香港城市大学博士学位，2020年加入东南大学计算机科学与工程学院任职副教授。曾任斯坦福大学访问学者。研究内容广泛涉及机器学习和数据表示的多个子领域，包括高维数据分析与建模、张量表示与建模、图机器学习等。在相关研究领域的国际会议和期刊上发表学术论文40+，其中CCF-A/IEEE Trans 论文25+。担任欧洲机器学习大会（ECML-PKDD）2021分会主席。
