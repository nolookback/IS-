教育学部　2022/6/18 10:02:00
  （钟沛基 2022280095）

主讲人：张华教授

时间：2022年6月18日（周六）下午15:00-17:00

地点：深圳大学汇德楼A408

 

主讲人简介：

张华，杭州师范大学教育科学研究院院长、教授、博士生导师。教育部义务教育课程修订指导组成员。美国富布赖特学者（Fulbright Scholar）。兼任教育部基础教育课程教材专家工作委员会执行委员；中国管理科学学会常务理事、基础教育管理专业委员会主任；中国教育学会中小学综合实践分会学术委员会主任；国际课程研究促进协会荣誉主席。

张华教授长期从事课程与教学论、教育哲学、道德教育、教师教育等方向的研究，为我国基础教育课程改革做出重大贡献。现已出版学术论著12部，发表学术论文200余篇，出版译著7部。学术成果被广泛引用与转载，并多次获奖。学术研究致力于教育与社会启蒙，教师与学生解放。
