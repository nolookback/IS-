学术讲座：隐写码的设计原理初探
电子与信息工程学院　2021/5/28 16:36:00
  （钟沛基 2022280095）

主讲嘉宾：李伟祥 博士 中国科学技术大学

邀请人： 李斌 教授

时间： 2021年6月1日 上午9:30

地点：深圳大学沧海校区致信楼电子与信息工程学院N710

 

报告摘要：隐写技术是将消息隐藏在图像、音视频等各种载体，实现隐蔽通信和存储的一类信息隐藏技术。隐写码（Steganographic codes）是隐写技术的重要理论基础和核心底层技术，用于减小消息隐藏带来的嵌入影响。回顾隐写码20余年的发展历程，大多数隐写码方案是利用纠错码的校验矩阵实现消息的嵌入和提取，并基于纠错码的译码算法设计隐写码的编码算法。本报告从经典的矩阵编码出发，分析纠错码与隐写码的对偶关系，并给出卷积码和极化码用于构造隐写码的设计思路。报告将重点讨论将极化码译码算法改造为隐写码编码算法的若干关键问题及解法，并对极化隐写码的性能进行分析。

 

嘉宾简介：李伟祥，中国科学技术大学网络空间安全专业博士，本科毕业于西安电子科技大学信息安全专业。研究方向为隐写与隐写分析，已在IEEE TCOMM/TIFS/TCSVT/TVCG、ACM IH&MMSec等期刊和会议上发表学术论文十余篇，曾获得国际信息隐藏与多媒体安全学术会议ACM IH&MMSec2018最佳学生论文奖。
