“现象学与近代哲学传统”学术讲座在马克思主义学院顺利举办
马克思主义学院　2024/6/26 13:21:00
  （钟沛基 2022280095）

2024年6月14日下午，广东外语外贸大学马克思主义学院博士生导师李云飞教授受邀为深大师生带来了一场题为“现象学与近代哲学传统”的精彩讲座。本次讲座由深圳大学马克思主义学院毛家骥助理教授主持，众多师生到场聆听。

          　

在讲座中，李云飞教授从四个方面对胡塞尔与近代哲学传统进行了详细的阐述。第一，胡塞尔的现象学观念。李云飞教授指出，与传统的经验主义的直观主义不同，现象学的直观主义不是基于自然的世界经验的明见性，而是基于反思经验的明见性。它克服了自然的世界经验的素朴性，纯粹根据自身被给予性展显原初的实事。因此，胡塞尔又称之为“先验的直观主义”。第二，胡塞尔的传统批判的动机和原则。李教授认为，胡塞尔的哲学批判旨在揭示近代哲学发展的统一动机和“隐蔽的目的论”，认为近代哲学发展贯穿着一个唯一的任务，即指向现象学的实现。他通过撇开历史制约的特殊性，探究哲学的根本意义，并试图冲破哲学史表面的“历史事实”外壳，揭示其内在意义和目的。在胡塞尔看来，现象学是近代哲学的隐秘憧憬，所有过去的哲学虽未意识到，但均内在地指向这一新哲学意义。这一观点体现了胡塞尔对哲学传统和未来发展的深刻洞察。第三，现象学是近代哲学的隐秘憧憬。现象学作为近代哲学的核心议题，自笛卡尔以来就受到关注。笛卡尔的普遍怀疑与“我思”的提出，为现象学的发展注入了“先验的动机”，但受限于客观主义，他未能完全理解先验意识的重要性。洛克虽认识到“我思”的绝对性，但同样因客观主义立场而未能深入探究先验问题，反而将意识自然化，陷入心理主义误区。胡塞尔指出，近代哲学的发展本质上是从客观主义向先验主体主义的转变，而笛卡尔与洛克虽有所贡献，但均未能完全摆脱客观主义的影响，从而限制了他们对现象学的真正理解。洛克的“假的直观主义”也表明，经验主义需进一步突破客观主义与自然主义，才能真正迈向内在“实证主义”。第四，胡塞尔的传统批判的意义。李云飞教授指出，胡塞尔的历史-目的论反思旨在揭示其先验现象学的历史的必然性和正当性，深化了现象学的自身理解，凸显了其先验现象学的哲学史地位，为我们如何对待传统提供了宝贵启示。

讲座最后，毛家骥助理教授对李云飞教授的精彩讲述表示感谢并做总结发言，指出这是一场内涵丰富、具有启发性的学术讲座。此次讲座内容政治站位高、逻辑严谨并且见解独特，提升了师生们的专业素养，同时也营造出良好的学术氛围。在热烈的讨论氛围中，讲座圆满落下帷幕。
