  （钟沛基 2022280095）

报告题目：甘蔗育种技术创新研究进展与展望

主讲嘉宾：王勤南 所长/研究员

报告主持：刘宏涛 院长/教授

报告时间：2025年4月17日（星期四）15:30

报告地点：深圳大学丽湖校区守慧楼A6-501

报告人简介：

王勤南，研究员，现任广东省科学院南繁种业研究所所长，致力于作物种质资源创新利用及新品种创制与优异基因发掘应用，全球首创甘蔗单倍体育种技术体系。担任国家糖料产业技术体系岗位科学家、中国作物学会甘蔗专委会副主任委员、广东省南繁服务中心副主任、海南南繁协会副会长;国家自然科学基金评审专家;华南农业大学等高校硕士生导师;担任Euphytica等国际期刊编委，2019年获得全国南繁工作先进个人，中国热科院特聘研究员。先后主持国家基金、省级重点研发专项、省级科技计划等项目15项，参与国家及省级等项目26项;获授权发明专利9件(第一发明人)，其中国际授权3件;发表论文100多篇，其中以第一作者或通讯作者在Plant Physiology、The Crop journal、Critical review in Biotechnology等SCI期刊发表论文26篇;获甘蔗新品种登记6件，获得广东省农业农村厅主导品种2个、主推技术1项;获神农中华农业科技奖一等奖和广东省科技进步二等奖等各类奖项12项。
