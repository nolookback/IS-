讲座摘要：

圆偏振发光是近年来手性研究领域非常活跃的一个方向。圆偏振发光可以用发光的左旋还是右旋、发光波长、不对称因子以及量子效率等参量来描述。其主要问题是如何产生圆偏振发光，如何来调控这些参量。圆偏振发光体系一般要求手性体系与发光体系的耦合，通过共价合成的手性分子发光不对称发光因子往往比较小，而通过超分子组装是实现不对称因子放大的重要途径。本报告将讨论如何通过多级次的超分子组装实现手性分子、手性结构到发光体系的手性传递而产生和增强圆偏振发光，对圆偏振发光中的不对称因子的组装增强、限域增强效应以及如何利用等提出一些思考。
