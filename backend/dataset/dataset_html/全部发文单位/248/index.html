嘉宾简介：蔡双立，管理学博士，工商管理博士后，天津财经大学商学院企业管理系教授，博士生导师，服务管理学科方向负责人，天津财经大学期刊社社长，《现代财经-天津财经大学学报》主编，天津财经大学瑞讯金融研究院院长；青岛大学影像技术与数据科学研究院院长，《影像技术》主编，主持并完成国家自然基金面上项目1项，国家社科重点基金项目2项，省部级重点项目5项，出版专著6本，在国外和国内核心期刊上发表论文63篇以上，具有10年以上企业高层管理经验，30多年期刊管理和从业经验，在社会服务方面，为国内银行、上市公司50家企业提供过咨询和培训服务。现任中国感光学会常务理事，影像技术与数据科学专委会主任，天津市政协特聘专家，广西自治区政府金融开放特聘专家，天津市期刊学会副会长，曾任中国营销传媒控股战略研究院院长，《金融营销》杂志执行总编。
