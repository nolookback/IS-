报告人简介：

姜政，现任中国科学技术大学国家同步辐射实验室教授，北京同步辐射装置用户委员会委员。目前主要致力于合肥新同步辐射的光束线站设计和建设。主要工作：1）设计建造了上海光源的谱学研究平台；目前担任合肥先进光源合肥先进光源助理总工程师，光束线站总体主任（兼）。2）发展先进X射线谱学装置和方法，包括X射线发射谱、高分辨吸收谱、时间分辨谱学、原位谱学等；3）推动和建立了X射线谱学对能源材料从单原子到团簇的系统研究。迄今为止，共发表相关工作共计500多篇，H-index78，连续入选2020-2023 年度科睿唯安全球高被引科学家。

报告摘要：

同步辐射X射线谱学（SRXS）方法是随着同步辐射发展起来的独特技术，是研究能源材料局域原子结构和电子结构的一种重要方法，助力于阐明催化机理以及构建催化剂的构效关系。通过SRXS分析材料的局域原子、电子结构，并结合电镜、DFT计算等，可以明确提供关于中金属-金属键的数量以及它们与支撑载体的键合等信息。这种原子级的洞察力对于理解结构-活性关系、原子-原子协同作用和金属团簇的稳定性至关重要。报告对于能源材料的谱学实验方法进行了展示，重点介绍了课题组前期阐释低核位点构效关系，以及近期发展的高温原位SRXS实验方法揭示低核位点的形成机制，这些原位谱学方法为材料构效机理研究提供了强有力的支撑。
