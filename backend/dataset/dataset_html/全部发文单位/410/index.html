“ 资本为什么卷？——结构主义与精神分析视角下的《资本论》”学术讲座在马克思主义学院顺利举办
马克思主义学院　2024/5/7 17:56:00
  （钟沛基 2022280095）

2024年4月23日，南方医科大学马克思主义学院黄旺教授受邀为深大师生带来了一场题为“资本为什么卷？——结构主义与精神分析视角下的《资本论》”的精彩讲座。本次讲座由深圳大学马克思主义学院副院长张守奎教授主持，众多师生到场聆听。

   

在讲座中，黄旺教授首先指出，作为使用价值与价值统一体的商品是贯穿资本主义的底层逻辑。他认为，商品的使用价值是从生理物理学的角度把握劳动产品的特征，而价值则是从社会属性角度去把握商品的本质。这种理解方式有助于我们避免陷入唯心主义的误区，坚守历史唯物主义的立场。

随后，黄旺教授从结构主义精神分析的视角，对资本论进行了深入的解读。他提到阿尔都塞等学者在结构主义视角下对马克思的研究，并指出这些研究对于重新理解资本论具有重要意义。他强调，在理解资本论时，我们需要把握商品作为使用价值与价值的统一体，以及价值作为社会属性的核心地位。

讲座中，黄旺教授还分享了自己对利用结构主义精神分析理论资源重新理解资本论的思考。他认为，这种理论资源能够帮助我们更加深入地理解资本的逻辑，揭示资本主义社会的内在矛盾和问题。同时，他也提醒在座师生，在运用这些理论资源时，需要保持谨慎和批判的态度，避免陷入片面或极端的解读。

   

讲座最后，张守奎教授对黄旺教授的精彩讲述表示感谢并做总结发言，指出这是一场内涵丰富、具有启发性的学术讲座。此次讲座内容政治站位高、逻辑严谨并且见解独特，提升了师生们的专业素养，同时也营造出良好的学术氛围。在热烈的讨论氛围中，讲座圆满落下帷幕。
