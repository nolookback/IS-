题目：北京医院心内科团队介绍及超声相关项目汇报

主讲嘉宾：郭颖

时间： 2023年12月15日（周五）上午 9:00 – 11:00

地点： 深圳大学丽湖校区生物医学工程学院A2-517大会议室

主持人： 薛武峰

主讲嘉宾简介：

郭颖，主治医师，医学博士，毕业于北京协和医学院-阜外医院，现就职于北京医院心内科。

擅长冠心病超声心动图新技术的应用，人工智能与超声心动图跨学科合作，结构性心脏病学心脏病的诊断与治疗等。主持和参与多项国家级、省部级、院校级课题。主要侧重冠心病、人工智能、瓣膜病领域。执笔多项共识，包括《中国成人心力衰竭超声心动图规范化检查专家共识》、《中国成人心脏瓣膜病超声心动图规范化检查专家共识》、《新型冠状病毒肺炎相关心肌损伤的临床管理专家建议（第一版）》《新型冠状病毒感染相关心肌损伤、心肌炎和感染后状态管理专家共识（第二版）》，已发表论文30余篇，其中SCI 10余篇，共识4篇。申请多项发明专利。

 

报告简介（Abstract）：

介绍北京医院、心内科、超声心动图室临床及科研工作开展。对超声心动图室开展的一些临床和科研项目进行简要介绍和交流。
