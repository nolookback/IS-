  谭元植，男，1985年生，厦门大学化学化工学院教授，博士生导师，高层次引进青年人才。主要从事富碳纳米分子（如石墨烯分子、巴基碗、碳纳米环等）的合成、功能化、超分子组装及其相关性能研究。2012年获得教育部自然科学奖一等奖（第二完成人），2013年获得全国百篇优秀博士论文奖，2015年获得国家自然科学二等奖（第二完成人）。主要从事分子纳米碳（如石墨烯分子、巴基碗、碳纳米环等）的合成、功能化、超分子组装及其相关性能研究。以第一作者或通讯作者在Nature Materials，Nature Chemistry，Nature Communications，Science Advances, J. Am. Chem. Soc.，Angew. Chem. Int. Ed.等发表多篇论文。研究成果受到了国际重要期刊或网站如Nature Chemistry，Angew. Chem. Int. Ed.，Chemistry World，Nature China，NPG Asia Materials等专文进行关注和点评。
