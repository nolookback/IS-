【学术讲座】新一代无线通信及终端技术初探
电子与信息工程学院　2024/3/18 14:49:00
  （钟沛基 2022280095）

主讲嘉宾：庄宏成教授，中山大学

邀  请 人：黄磊教授

时间：2024年03月19 日（周二）上午10:30-11:30

地点：深圳大学致信楼电子与信息工程学院N710会议室

 

摘要：6G正在研发和即将开始标准化，从无线通信需求和发展趋势切入，阐述新一代无线通信的关键技术，并提出移动终端发展新模式-开放终端，介绍开放终端的核心技术。

 

 

个人简介：庄宏成，教授级高工，中山大学电子与通信工程学院教授，博士生导师，中山大学“百人计划”，鹏城实验室双聘博导。2002年中山大学无线电物理专业博士毕业，加入华为技术有限公司，一直从事无线通信技术的研发和标准化，包括3G、4G、5G和6G的相关技术，2022年加入中山大学。主持交付了多项华为战略项目和国家项目，发表论文70多篇，获得授权专利300多项，拥有多项3GPP标准基本专利和IEEE802标准基本专利。研究成果规模商用，在2014年、2018年、2021年、2022和2023年分别获得省部级技术发明一等奖1项、省部级科技进步二等奖1项、中国电子学会科技进步一等奖1项和省部级科技进步一等奖2项。
