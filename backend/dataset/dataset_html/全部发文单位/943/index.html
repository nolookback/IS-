主讲人介绍：

龚萍 博士，研究员，博士生导师。2008年获湖南大学分析化学博士，曾先后主持国家自然基金面上课题、国家自然基金青年基金、广东省自然基金、中国博后基金、深圳市基础研究计划等多个科研项目；多次参与973、863、国家自然科学基金等课题的攻关研究，参加了多次国际国内学术会议交流与研讨会。已在JACS、ASC nano、Advanced Materials，Advanced Functional Materials，Biomaterials 等国际期刊发表发表论文20余篇，申请专利50余项，授权PCT专利2件，授权中国专利34件。2004年曾获得湖南省科技进步二等奖，2009年获得深圳市高层次专业人才计划之“后备人才”称号。获得第十七届中国国际高新技术成果交易会CHTF优秀产品奖。2015年获深圳市自然科学奖二等奖。
