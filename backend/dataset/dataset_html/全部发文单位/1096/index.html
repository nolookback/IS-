讲座摘要：

传统的催化不对称合成往往依赖于活性官能团的转化。以碳氢键断裂为基元反应的催化不对称合成方法可为复杂分子的构建提供更为经济、直观的新策略，同时也是有机化学领域的一大难题。 我们课题组以氧化sp3碳氢键断裂作为基元反应，系统的研究了（1）不对称氧化碳氢官能化—碳碳键构筑、（2）氧化还原去外消旋化—碳氢键构筑以及（3）氧化动力学拆分—不对称氧化三种合成策略；分别应用这三种策略，实现了骨架和取代形式多样的醚、胺以及烷烃的不对称合成。这些方法在功能分子的不对称合成以及抗癌活性小分子的发现方面均展示出了良好的适用性。
