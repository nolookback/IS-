【学术讲座】视听联合体验质量评价
电子与信息工程学院　2024/7/15 10:22:00
  （钟沛基 2022280095）

主讲嘉宾：闵雄阔副教授，上海交通大学

邀 请 人：李元满助理教授

报告时间：2024年7月15日，下午4点

报告地点：致信楼N723


 

报告摘要/内容简介：

视觉和听觉是人类从外界感知信息的两大最重要的来源，多媒体信号也由视频和音频信号构成。当前多媒体质量评价技术大多聚焦于单一模态的信号，忽略了多种模态信号之间的相互影响。本报告将探讨与视听联合体验质量相关的若干问题，并介绍课题组在视听联合体验质量评价方面的研究成果。

 

 

 

报告嘉宾简介：

闵雄阔，上海交通大学电子工程系副教授。研究方向为多媒体质量评价，主持自然基金面上、青年等项目；入选博新计划、浦江人才计划；获中国电子学会优博、中国图像图形学会技术发明一等奖、中国电子学会技术发明一等奖等国内奖励；以及IEEE TMM最佳论文提名奖、IEEE ICME最佳学生论文奖等国际奖励；发表IEEE/ACM汇刊及CCF A类会议论文60余篇，谷歌学术引用6千余次。
