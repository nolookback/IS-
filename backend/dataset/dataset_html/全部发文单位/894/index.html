讲座题目：

《城市发展与政府边界》

（原名：城市增长与市场边界）


讲座简介：

　　　　城市增长和城市化进程中，有必要对增长的机制、动力和出现的规律做一些归纳与总结，找出哪些是“原因”，哪些是“结果”；哪些是市场自发的过程、哪些是政府的责任。对于中国城市化“下半场”，如何解决“上半场”出现的诸如“房价”、“土地财政”等突出问题，更好平衡市场与规则，是一个需要共识的探讨。
