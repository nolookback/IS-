【学术讲座】中国的现代化道路：不同于苏联模式的对西方现代性和资本主义现代化 （包括南方依附性现代化）的成功扬弃
马克思主义学院　2023/12/13 12:37:00
  （钟沛基 2022280095）

主讲人：曹天予 美国波士顿大学教授

时间：2023年12月15日(星期五)上午9:30

地点：汇文楼1600会议室

主讲人简介：

   曹天予，美国波士顿大学教授，剑桥大学博士。曾为牛津大学全灵学院访问学者和普林斯顿高研院高级研究员。代表作有《劳动产权与中国模式》《现代化、全球化与中国道路》 《二十世纪场论的 概念发展》《从流代数到量子色动力学》。他的研究兴趣包括：科学哲学和科学史，认识论，形而上学，以及社会和政治哲学，特别对现代性和后现代性相关的哲学问题有浓厚兴趣。

内容摘要：

   只有认清了西方对现代化的自我认知(理性和主体性的觉醒，其社会内容是市场经济和代议民 主);1950-70年代大为流行的现代化方案，其实质是西方与当时苏联的社会主义模式争夺第三世 界；1970年代起以高科技-网络-金融三位一体宰制全球经济的全球化，是资本主义现代化发展的必然 终局(当代的种种危机导致其出现种种变种)。我们才能恰当地理解社会主义现代化的中国道路所取得的成功的深远的历史意义。
