学术讲座（九）：仿生皮肤聚合物弹性体力学性能的实现-从合成到微结构定制

报告人简介：

王志刚，中国科学技术大学高分子科学与工程系教授。1991年毕业于南开大学化学系，1994年于南开大学获理学硕士学位，于中国科学院长春应用化学研究所获理学博士学位。曾入选中国科学院2004年“百人计划”，获得择优支持，并于2008年6月顺利通过终期评估。长期从事高分子物理与化学的基础研究 (聚合物及其共混物、填充体系的分相、结晶和流变学研究)，工程塑料的增强和增韧机理研究，以及高聚物/粘土纳米复合材料以及高聚物/碳纳米管复合材料制备和性能研究。迄今，已作为通讯作者或共同通讯作者在Macromolecules、ACS Sustainable Chem. Eng等期刊发表学术论文百余篇。

讲座摘要：

皮肤作为一种生物的防御系统，是人类和动物身上最大的器官。皮肤在单轴拉伸变形过程中呈现高度非线性和独特的两阶段应力应变曲线：第一阶段是低模量的“toe”区域，在这个阶段胶原纤维的褶皱被拉平；第二阶段是高模量的“tough”区域，在这个阶段胶原纤维在持续的拉伸中发生取向和相互滑动。我们首先采用原子转移自由基聚合联合卤素交换技术合成了Cell-g-PBA-b-PMMA瓶刷共聚物，再通过逐步循环拉伸处理Cell-g-PBA-b-PMMA瓶刷共聚物弹性体，就可以表现为类似真实皮肤的力学行为，这对未来仿生高分子材料的开发研究至关重要。依此，实现了以纤维素接枝两嵌段共聚物弹性体来模拟皮肤力学性能的现实目标。
