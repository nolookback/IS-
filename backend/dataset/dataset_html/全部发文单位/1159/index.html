复旦大学新闻学院朱春阳教授主讲“国家社会科学基金项目申报辅导”学术讲座
传播学院　2021/2/18 11:41:00
  （钟沛基 2022280095）

2021年2月8日上午，传播学院组织了一场关于国家社会科学基金项目申报辅导的学术讲座。本次学术活动是深圳大学“人文社会科学讲学计划”系列讲座之一，复旦大学新闻学院朱春阳教授受传播学院邀请担任本次讲座的主讲嘉宾。讲座采用腾讯会议线上的方式开展，吸引了来自深圳大学各个学院的老师们共同参加。

讲座正式开始前，传播学院副院长杨洸老师热情地向在线参会者介绍了朱春阳教授的基本情况。朱春阳教授先后入选国家万人计划哲学社会科学领军人才、教育部青年知名学者、中宣部文化名家暨四个一批等，目前担任国家社会科学基金重大项目首席专家，中国新闻史学会传媒经济与管理学会副会长、公共关系学会副会长等。

讲座伊始，朱教授向大家分享了他本人前后申报国家社会科学基金项目的经历，并将申报项目的经历形容为“探索的过程”。随后，朱教授围绕写项目申请书的逻辑起点、文献综述、研究设计、语言规范等方面展开详细阐述。

朱教授建议老师们申报项目时要选择自己熟悉的研究领域，并且非常熟悉这个领域的相关文献，了解同行的研究水准。在研究时做减法，形成逻辑主线，不研究不重要的部分。

朱教授特别强调文献综述的重要性，他认为文献综述是“别人对你提出问题的回应”，并强调有“问题意识”的文献综述才是有意义的。在项目申请书的研究设计部分，朱教授提出了几点建议。对核心概念要有权威的界定，整个理论框架是基于此界定来讨论问题的；核心内容阐述部分采用先介绍框架，再分几个子课题阐述的方式展开，等等。在项目申请书的语言使用方面，朱教授强调文字表述要具有逻辑性，要突出重点，简单直接，一目了然。最后，朱教授认为国家社会科学基金项目比较看重理论，建议大家基于理论来讨论现实社会问题。

听完朱教授的讲座后，在线的老师们针对自己在申报国家社会科学基金项目时遇到的问题咨询了朱教授，朱教授一一做了解答。至此，本次讲座在各位老师的答谢中圆满结束。
