【学术讲座】智能超材料物联网
电子与信息工程学院　2021/10/8 14:47:00
  （钟沛基 2022280095）

讲座题目：智能超材料物联网

主讲嘉宾：宋令阳 教授 北京大学

邀请人：张胜利 教授

讲座时间：2021年10月12日（周二下午4:00）

讲座地点：深圳大学沧海校区致信楼N710会议室

 

摘要：在即将到来的第六代移动通信(6G)中，海量的物联网传感器将被部署,从而实现对于环境状态的泛在感知。与传统的有源物联网相比，智能超材料物联网基于无源的超材料传感器，具有零能耗、极低成本、无需维护的特点，可以被大规模地铺设于结构体和建筑的表面或者内部，为智慧家居、工业、城市的决策提供感知数据。本报告将简要介绍智能超材料系统的研究背景及超材料传感器的原理和特点，分析智能超材料物联网设计面临的挑战，并阐述智能超材料物联网系统的实现中，传感器和信号处理智能算法设计的关键科学问题和阶段性研究进展。

 

嘉宾简介：宋令阳，英国约克大学博士、挪威奥斯陆大学博士后、美国哈佛大学博士后、英国飞利浦研究院高级研究员，现为北京大学博雅特聘教授、学科建设办公室副主任、电子学系副主任、信息与通信研究所所长。主要研究方向是无线通信网络、信号处理和机器学习。获得教育部自然科学一等奖、国家自然基金委杰出青年科学基金、首届国家973计划青年专题项目首席科学家、首届国家自然基金委优秀青年科学基金、中组部青年拔尖人才、中国青年科技奖、北京市五四青年奖章、IEEE通信协会亚太地区杰出青年研究奖、IEEE Communication Society Leonard G. Abraham Prize、IEEE Communications Society Heinrich Hertz Award等。曾被评为IEEE Fellow、Clarivate Analytics高被引科学家等。
