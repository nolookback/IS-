艺术学部“春之季”系列之学术讲座：庆祝中国共产党建党100周年——《黄河大合唱》 传奇与传承
艺术学部　2021/5/13 18:02:00
  （钟沛基 2022280095）

伴随着春暖花开，艺术学部将继续为广大师生带来高品质的学术讲座及精彩音乐会。本月敬请大家关注5月17日由《黄河大合唱》词作者光未然次子张安东先生和《黄河大合唱》传承志愿者黄平先生的艺术讲座，届时讲座将为大家分享《黄河大合唱》的传奇与传承。

 

　 讲座主题：《黄河大合唱》——百年来中国历史之文化经典

　　第一部分：《黄河大合唱》的传奇

　　第二部分：《黄河大合唱》的传承

 

   讲座时间：2021年5月17日（周一）下午14：30

 

　 讲座地点：师范学院B座音乐厅

 

主讲人简介：

张安东，《黄河大合唱》词作者光未然次子，工业产品设计师。1977年考入武汉大学中文系，1982年任《北京日报》/《北京晚报》记者编辑，1984年考入中国社会科学院研究生院新闻研究所读研。1989年初赴澳大利亚留学，研习绘画与设计，在悉尼从事广告设计和室内设计，1999年回国在北京做室内设计，2002年受聘于新加坡公司从事工业产品设计至今，多次获得德国红点奖等国际重大工业设计奖项。

2013年～2017年策划、编辑和校对张光年（光未然）旧作，古典文学理论译述专著《骈体语译[文心雕龙]》，并撰写跋文，由华中师范大学出版社在2017年4月以精装繁体直排本再版。

从2013年起，积极参与《黄河大合唱》的推广传承工作，与热爱《黄河大合唱》的朋友们共同建立“我爱《黄河大合唱》微信群”，以吸引众多海内外华人关注和参与《黄河大合唱》的传承。2019年，与黄平、黄炜等共同发起和组织纪念《黄河大合唱》诞生80周年系列活动，包括由海内外30多个团体共同参与的“五洲同声唱黄河”活动、海内外华人“重走《黄河大合唱》创作之路”活动，参加编辑《永远的<黄河大合唱>——<黄河大合唱>纵横谈续编》一书并撰文。与黄平、黄炜共同策划和运作2020年《永远的黄河大合唱》新书广州、武汉、老河口、北京、延安、太原和兰州七城联动发行暨捐赠活动。在延安和东莞等地举行“《黄河大合唱》——百年来中国历史之文化经典”讲座。与黄平、黄炜发起“2021年《黄河大合唱》讲座进高校活动”。

 

黄平，《黄河大合唱》传承志愿者。1999年，陪同《黄河大合唱》词作者光未然夫妇参加上海纪念《黄河大合唱》诞生60周年活动。2009年，参加由上海发起九省、市、自治区黄河流域共同纪念《黄河大合唱》诞生70周年演唱活动。2015年，策划和组织《黄河大合唱》诞生地寻访活动，撰写《〈黄河大合唱〉诞生地寻访记》文章，首次还原《黄河大合唱》创作过程。2019年，与《黄河大合唱》词作者光未然次子张安东共同策划和运作纪念《黄河大合唱》诞生八十周年系列活动，包括由海内外30多个团体共同参与的“五洲同声唱黄河”活动、海内外华人“重走《黄河大合唱》创作之路”活动，以及参与《永远的黄河大合唱》书籍的编辑工作。总策划和运作2020年《永远的黄河大合唱》新书广州、武汉、老河口、北京、延安、太原和兰州七城联动发行暨捐赠活动。与张安东在延安和东莞等地举行“《黄河大合唱》——百年来中国历史之文化经典”讲座。与张安东、黄炜发起“2021年《黄河大合唱》讲座进高校活动”。
