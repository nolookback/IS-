主讲人： 南兆旭

主讲人介绍：

南兆旭，深圳本土自然与历史研究者，主张“在地关怀”，关注、记录、保护深圳的自然生态，发起和组织了一系列自然保护活动。2019 年被深圳市人民政府聘为深圳国土空间总体规划专家顾问委员会委员。

图书作品：

《深圳记忆》（2008 年）、《解密深圳档案》（2010 年）、《深圳自然笔记》（2013 年）、《南寻深圳》（2014 年）、《山水相望》（2016 年）、《深圳自然博物百科》（2022年）

纪录片作品：

《深圳民间记忆》（2008 年）、《梦开始的地方》（2014 年）、《深圳自然笔记》（2020 年）
