报告人简介：

罗杰，教授，博士生导师。长期从事作物营养品质调控及改良研究。获“全国先进工作者”荣誉称号、获“国家杰出青年科学基金”资助、入选“万人计划”科技创新领军人才、“中青年科技创新领军人才”。主持国家自然科学基金重点项目、“863”重点项目、“973”课题等，以通讯作者在Cell、Nat Genet、Nat Plants、Nat Commun、Mol Plant、PNAS、Plant Cell、Trends Plant Sci等发表系列论文，多次应邀在国际学术会议上做主题及大会报告，是首位在“国际代谢组学大会”做主题报告的中国科学家。担任中国植物生理与植物分子生物学会常务理事、中国生化与分子生物学会理事、中国生化与分子生物学会教学专业委员会副主任委员。任《中国科学-生命科学》、《植物学报》等杂志编委。主讲的《分子生物学》入选首批“国家级精品资源共享课程”。指导的博士生1人获得“吴瑞奖”，2人获得“吴瑞奖”提名。
