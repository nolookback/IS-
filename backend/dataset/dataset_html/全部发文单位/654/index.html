主讲人：宋冠群博士

时间：11月4日上午11:00

地点：丽湖校区守正楼212会议室

报告人简介：

宋冠群，博士，2007年毕业于中国科学院生态环境研究中心，获博士学位。曾先后于香港浸会大学从事联合项目研究和在美国德克萨斯大学从事博士后研究工作。2009年任职于科学出版社从事期刊出版工作，曾入选“中国科学院文献情报和期刊出版领域引进优秀人才计划”。2014年加入英国皇家化学会。目前担任英国皇家化学会中国区出版总监，负责英国皇家化学会在中国大陆、香港、台湾地区的出版和拓展合作业务。

报告摘要：

1. RSC期刊介绍，更深入了解期刊的scope和出版标准 

2. 期刊审稿流程及出版服务，更清楚了解稿件的审理过程 

3. 论文写作，从编辑角度让作者了解论文写作的重要环节 

4. 中国科学院OA资助方案中RSC开放获取出版的申请流程
