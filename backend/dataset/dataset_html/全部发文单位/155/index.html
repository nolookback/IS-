【学术讲座】微塑料对微藻生态链的冲击现况和研发挑战
生命与海洋科学学院　2024/12/10 10:45:00
  （钟沛基 2022280095）

报告主题：微塑料对微藻生态链的冲击现况和研发挑战

主讲嘉宾：中国科学院宁波材料所 甄光明 教授 

报告主持：胡章立 教授

报告时间：2024年12月11日（星期三） 上午10:00 -11:00

报告地点：深圳大学丽湖校区 守慧楼A6-606会议室

个人简介：

甄光明教授，美国普渡大学农工硕士，明尼苏达大学化工硕士及博士候选，塔夫斯 (Tufts)大学生物化工博士。留美20年间任职多家生物技术及发酵生产研发公司，获得行业内多个荣誉如费城及三洲青年工程师奖章等。 2000年回归中国，任职世界最大的聚乳酸公司嘉吉陶氏(Cargill-Dow)合资公司及NatureWorks中国区首席代表，世界最大的乳酸及丙交酯公司Purac (普拉克) Corbion (科碧恩)及欧美第二大聚乳酸公司TotalEnergy-Corbion (道达尔能源科碧恩合资公司)大中华区总经理，引领中国聚乳酸行业。曾任香港中文大学、江南大学、南京工业大学等高校的兼任客座教授。

报告摘要：

全球每年生产约4亿吨塑料如PE、PP、PET、PS、PC、尼龙等，其废品逐渐崩解成微塑料及纳米塑料，进入生物及人体血液大脑内脏甚至胎盘影响健康。新兴的生物降解塑料如PLA（聚乳酸）、PHA、PGA、PBAT、及PBS等,比传统塑料降解快10-100倍，聚乳酸等并已在人体医学使用多年无问题。但目前有些研究论文因为误解或实验设计等问题,宣布聚乳酸有害，严重打击了我国降解塑料行业的发展。 本报告将对此提供资料分析及建议方案，希望以微藻食物链等证明降解塑料比传统塑料更安全，以缓解全球微塑料污染的环保问题。
