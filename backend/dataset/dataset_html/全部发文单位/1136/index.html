深圳大学马克思主义学院哲学社会科学论坛学术讲座：西南大学黄其洪教授主讲《西方马克思主义的前沿问题研究》
马克思主义学院　2021/3/30 16:39:00
  （钟沛基 2022280095）

       3月30日上午，深圳大学马克思主义学院哲学社会科学论坛学术讲座特邀请西南大学西方马克思主义研究所所长，西南大学国外马克思主义研究学科点负责人黄其洪教授为我院师生作题为《西方马克思主义的前沿问题研究》学术讲座。

黄其洪教授首先回顾并概述了20世纪西方马克思主义在主要国家的发展历程，代表人物及各个门派的学术观点，并结合当前国内开展西方马克思主义前沿问题研究的现状和进展，对西方马克思主义的前沿问题进行了归纳评述。内容包括马克思的全球化理论与世界体系的关系问题、马克思的生态思想与生态马克思主义的理论重构问题、马克思的金融理论与金融资本主义批判问题、马克思的机器思想与人工智能的反思问题、马克思的正义观与自由主义正义理论批判问题、马克思的美学思想与生活美学建构问题、马克思的劳动价值论与数字资本主义批判问题、马克思的法权思想与资本主义法权批判问题、马克思的意识形态理论与革命主体的重构问题、马克思的实践概念与实践智慧的复兴问题、媒介资本主义与符号批判问题、空间生产与城市批判、女权主义的马克思主义理论、微观权力批判与身体理论、对新冠疫情在资本主义世界大流行的哲学反思。

   黄其洪教授以敏锐的洞察力，开阔的视野和深厚的学养为师生带来了一场极具学术前瞻性，丰富理论内涵和现实意义的学术讲座。讲座结束前，黄其洪教授仔细记录并逐一解答了师生提出的问题，讲座在轻松愉悦的氛围中圆满结束。
