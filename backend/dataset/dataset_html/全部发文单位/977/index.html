艺术学部“秋之韵”系列——著名花腔女高音歌唱家常思思《声乐发声方法的基本构成》学术讲座。
艺术学部　2021/11/15 20:42:00
  （钟沛基 2022280095）

艺术学部音乐舞蹈学院邀请了“花腔精灵”女高音歌唱家常思思为大家带来一场高品质的声乐专业讲座。此次讲座精选了5首经典声乐作品，常思思老师以现场授课的形式系统地为大家讲解歌唱发声方法的基本构成。

歌唱家简介：

　 常思思，著名花腔女高音歌唱家，曾获第八届中国声乐金钟奖民族唱法金奖，第十三届全国青年歌手电视大奖赛民族唱法铜奖第一名。代表作品《春天的芭蕾》、《玛依拉变奏曲》等深受人们的喜爱，广为流传，是新一代民族歌唱家的代表人物之一。

 

时　　间： 2021年11月19日（周五）14：30

地　　点：师范学院B座音乐厅
