深圳大学马克思主义学院党史教育专题党课暨哲学社会科学论坛学术讲座：坚持正确党史观
马克思主义学院　2021/5/31 17:38:00
  （钟沛基 2022280095）

在中国共产党成立一百周年之际，5月26日下午，深圳大学马克思主义学院、深圳大学当代中国思想文化研究所举办党史教育专题党课暨马克思主义学院哲学社会科学论坛学术讲座，特邀请中国社会科学院马克思主义研究院副院长、中国社会科学院习近平新时代中国特色社会主义思想研究中心执行副主任龚云教授作《坚持正确党史观》专题党课和讲座。马克思主义学院师生党员参加了本次专题党课和讲座，本次专题党课和讲座由马克思主义学院党委书记王红教授主持。

 

龚云教授从三方面深入论述了如何坚持正确党史观：第一、要坚持以马克思主义唯物史观为方法论指导原则，用唯物史观指导党史的研究和宣传，向大众尤其是青年一代讲好党的故事；第二、要以大历史观来看待中国共产党建党百年以来所走过的辉煌历程；第三、要深刻领悟中国共产党“全心全意为人民服务”的立党宗旨，分清主流和支流，坚决抵制历史虚无主义，从而树立正确的党史观。

 龚云教授结合中国共产党的执政经验指出，中国共产党之所以能得到广大人民的拥护，在于中国共产党自成立之日起，就始终站在广大人民的立场，为大多数人谋利益，将实现共同富裕作为矢志不渝的追求；在于中国共产党始终尊重知识，尊重人才，始终重视解放生产、发展生产力，不断推动经济社会发展，不断增进人民生活水平。龚云教授特别指出，党的历史是最生动、最有说服力的教科书。党员教育最重要的是要让党员了解党的历史，更为细致深入地理解、认同党。

 

龚云教授的讲座视野宽广、内容丰富，既体现了历史的厚重感，又紧密结合现实，具有强烈的现实关怀，在总结党的百年辉煌的基础上，提出了树立正确的党史观的科学方法。讲座最后，王红书记对龚云教授的精彩讲述表示感谢并做总结发言，指出这不仅是一场内涵丰富的学术讲座，也是一堂荡涤思想的生动党课，希望全院师生以这次专题党课和讲座为契机，不断强化理论学习，提高工作能力和业务水平，以永不懈怠的精神状态和昂扬饱满的热情投入到学习和工作中去。
