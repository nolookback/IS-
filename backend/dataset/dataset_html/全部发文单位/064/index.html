学术讲座：智驾信任的心理机制和评估
机电与控制工程学院　2025/4/15 16:24:00
  （钟沛基 2022280095）

讲座题目：智驾信任的心理机制和评估

演讲嘉宾：周荣刚 教授

讲座时间：2025年4月17日（星期四）10：00

讲座地点：深圳大学沧海校区致信楼S512会议室

邀　请　人：张婷茹 副教授

讲座内容简介：

基于心理学的视角对智能驾驶中的人因工程问题，特别是人机信任相关的内容进行介绍。重点围绕智能驾驶过程中人的行为及信任的关联关系，面向驾驶安全讨论新型人车关系中人机双向信任的作用机制、评估方法和校准策略，为理解和优化人机共驾关系提供心理学与工程视角的理论基础与实践工具。

演讲嘉宾简介：

周荣刚，北京航空航天大学经济管理学院，管理科学与工程系教授，博士生导师。北航经管学院智能交互与体验研究中心负责人，现任中国人类工效学学会常务理事、智能交互与体验分会副主任委员兼秘书长，《人类工效学》编委，中华人因与工效学协会理事兼秘书长，中国心理学会工程心理学分会委员。
