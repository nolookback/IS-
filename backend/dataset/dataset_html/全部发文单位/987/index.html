报告题目：为高龄者规划家一样的设施

Planning the Facility for the Elderly as Home


特邀专家：松原茂樹

日本国立大阪大学大学院建筑与都市规划领域副教授

日本建筑学会环境行动研究小委员会委员

“城镇的居场所工作组”干事


讲座主持：吴向阳 深圳大学副教授

时间：11月16日 19:00-20:20

地点： 腾讯会议ID：145875809 　会议密码：160821

 

专家简介：

松原茂樹，日本大阪大学大学院建筑与都市规划领域副教授，日本建筑学会环境行动研究小委员会委员，“城镇的居场所工作组”干事。主要从事居住福利环境、建筑计划、环境行为学研究。主要论著有《城市的居场所》（共著，东洋书店），《空宅·空建筑转变为福利设施的再利用》（共著，学艺书店），《使用者本位的建筑设计——以住宅、地域设施、病院、学校为例》（共著，彰国社）。


内容概要：

高龄者都希望居住在自己家所在的社区直到人生的最后一刻。但在家中故去的高龄者比例却很低，一般都是在医院或者养老设施。当高龄者需要进入养老设施时要如何设置环境？高龄者在家中护理时，什么设施是需要优先考虑的？这个报告将讲述如何通过设计来维持如家一般的环境，以及如何考虑养老设施与社区的关系。
