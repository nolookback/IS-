专家简介

王德响 先生，山东枣庄人，现任深圳市住房和建设局住房改革发展处处长。

　 

工作经历：

1997年—2009年先后在深圳市规划国土局、深圳市国土房产局从事房地产管理工作；

2007—2008年在国家建设部住宅与房地产业司挂职锻炼；

2009年—2018年在深圳市住房和建设局从事住房保障和建筑市场管理等工作。

 

学术成果：

专著《中国保障性住房定价理论与实践》（人民出版社2014年7月）荣获深圳市第七届哲学社会科学优秀成果二等奖。

在国家核心期刊发表专业论文十余篇。

《建设部城乡建设重大课题研究丛书之——住房、住房制度改革和房地产市场专题研究》、《房屋登记办法释义》等专业书籍的编写成员。曾参与国家《城镇住房保障条例》、《城市房地产管理法》等 　 多部住房和房地产管理的立法调研。

主导的“加快住房保障制度改革”项目，获评深圳市委全面深化改革委员会2018年度“深圳市全面深化改革优秀项目”、市委政法委营造共建共治共享实践创新示范项目等。
