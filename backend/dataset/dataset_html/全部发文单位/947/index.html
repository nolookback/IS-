信息中心举办“数据驱动之上的教育理念与场景应用”学术讲座
信息中心　2021/12/3 8:34:00
  （钟沛基 2022280095）

2021年11月25日下午，信息中心邀请清华大学全球学校与学生发展评价研究中心副主任王刚博士，在致理楼L3-1115会议室举办“数据驱动之上的教育理念与场景应用”学术讲座，讲座由信息中心主任王志强教授主持，教务部副主任李振华、发展规划部、信息中心主要技术人员以及大数据安全专业的研究生共30余人参会。

讲座围绕科技赋能教育中的“桥梁”问题，给出一种拆解和链接“教育理念-数据科学-工程技术-用户体验”的四轴对齐方式，通过流程梳理工作，分析与记录师生全流程中的关键问题，并结合新时期高校“智慧校园”建设、教育大数据的场景应用、学生学情以及教学评价的具体案例给出落地路径。

王刚博士以高校迎新、中国大学生学习与发展追踪研究（CCSS）、南京大学教学立方过程性评估等项目为例，介绍了如何让大数据说话，如何进行评建指标分析，如何进行流程优化和关键优化。针对业务流程，从组织、功能技术、数据、资源等不同纬度来分析；自上而下，从未来到当前，从拆解办学理念入手，细化“智慧校园”需要落实的关键业务举措；通过战略与流程梳理，提供业务“最佳实践”，建立业务流程的“标准”，最终通过SIGS流程再造和一系列优化，建立了“智慧校园”总体架构。

王刚博士参与了多项国家主管部门及清华大学的教育评价指标体系设计及高教评估工作，如“双一流”评估、学科评估、国际化评估、创新人才评估、教学评估和教师评估等，曾担任国家发改委互联网教育系统技术及应用国家工程实验室专家组成员，工信部中国电子技术标准化研究院数据标准与数据治理评审委员会专家，入选“海聚工程”和“海英人才”，中国教育智库与教育治理50人论坛成员，中关村十大海归新星，中关村十大文化与创意产业人物。

讲座后，王刚博士与在场的听众进行了互动，并回答了听众多个饶有兴趣的问题。王志强主任在最后总结时指出，王刚博士的报告通过“数据驱动”建设应用系统的方法，深入细致，案例丰富，框架分析和流程再造，由浅入深，条理清晰，为我校今后的智慧校园和云教学资源平台建设提供了很好的开发思路，值得我中心技术人员学习与借鉴。
