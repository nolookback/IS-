讲座题目：万物互联时代的地球空间信息学

 

时间：2021年11月10日 8:30-10:00

地点： 本原讲堂（南区天桥粤海校区一侧）

 

主要内容：李德仁教授，中国科学院院士、中国工程院院士，2021年国家科学技术进步一等奖获得者，2021年全国十大“最美科技工作者”荣誉称号获得者。他是国际著名摄影测量与遥感学家，我国高精度高分辨率对地观测技术体系的开创者，40年来，他持续开展基础理论和重大技术创新，引领了我国测绘遥感产业跨越式发展，为我国从测绘遥感大国到测绘遥感强国作出了杰出贡献。2021年11月10日，李德仁院士将在深圳大学为我系本科生讲授《地理空间信息工程概论》课 ——《万物互联时代的地球空间信息学》，欢迎各位老师和同学们参加！
