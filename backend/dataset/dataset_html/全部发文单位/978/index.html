【学术讲座】轻量化图像质量增强网络
电子与信息工程学院　2021/11/15 16:28:00
  （钟沛基 2022280095）

报告题目：轻量化图像质量增强网络

嘉宾：金枝 副教授 中山大学

邀请人：李霞 教授

时间：2021年11月18日（周四）上午10:00

地点：深圳大学沧海校区N710会议室

主要内容：

图像质量增强作为智能系统的前端，其效果将直接影响后端的高层语义应用。与此同时，做为前端，增强网络本身不能引入过多的时间和算力的消耗，否则会降低后端高层语义应用在移动端的部署。基于此，我们课题组在轻量化图像质量增强方面进行了深入的研究，本次报告将介绍部分相关研究成果。欢迎各位领导、专家与会指导。

个人介绍：

金枝，中山大学“百人计划”副教授，硕士生导师，中国图象图形学学会CSIG多媒体专委会委员、交通视频专委会委员，CCF 多媒体委员会委员，主要从事图像视频处理、计算机视觉、三维重建等方面的研究和应用。本科由西安交大与英国利物浦大学联合培养，2016年博士毕业于英国利物浦大学，博士后分别在深圳大学和德国慕尼黑工业大学进行。近5年，在图像处理及计算机视觉领域顶级期刊及会议发表论文40篇，其中一作/通信论文27篇。作为第一发明人共申请国家专利8项，已通过PCT检索专利2项，已授权国家发明专利2项，软件著作专利1项。主持国家级科研项目4项（2020年国家自然科学基金面上项目、2020年中德中心交流项目、2017年国家自然科学基金青年基金和2018年国家博士后基金），并作为骨干成员参与多项科技部重点研发计划、国家自然科学基金面上项目。担任IEEE TIP、TCSVT、TMM和Information Fusion多个计算机视觉领域顶级期刊的审稿人。担任多媒体领域旗舰会议ICME2019的环节主席，ICIG2021大会委员会宣传主席。
