题目：数字创意产业及技术发展

时间：2024年7月9日14:30-15:30
地点：致真楼L6-901
邀请人：王可

报告人简介：王振中，中央广播电视总台技术局技术规划研究部副主任，教授级高工，长期从事超高清、5G传输、AI媒体应用方向的发展规划、标准研制和技术创新工作，参与多项国际、国家及行业技术标准规范的制定，曾获艾美奖-工程科学技术奖1项，中国影视学会科学技术奖一等奖1项、二等奖3项，获广电总局科技创新奖二等奖1项。

讲座摘要：数字创意作为我国重点推动的战略性新兴产业之一，与元宇宙、人工智能等技术紧密结合，具备产业规模大、涉及学科多、贯穿产业链端到端环节等特点，是加快发展新质生产力、促进产业深度转型升级的重要支撑。本次介绍了数字创意技术的概念，针对数字对象生产体系、数字世界构建体系、数字服务融合三大体系覆盖的具体技术方向进行详细阐述，为下一步推动相关产业发展和学科建设提出了新的思路。
