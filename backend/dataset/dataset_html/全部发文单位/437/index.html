主讲人简介：

孙劲松，武汉大学哲学学院暨国学院教授、博士生导师、国学院院长，湖北省国学研究会会长，湖北省佛教协会咨议委员会副主席，中国周易学会理事，南海佛学院兼职教授、研究生导师，贵阳孔学堂学术委员。已出版《周易程氏传译注》《心史》《净心与洗心》《禅廉文化》等著作4部，主编《黄梅禅文化与荆楚佛学丛书》全套8本，参编《中国哲学史》《中国哲学简史》等教材、著作多部，在《哲学研究》《宗教学研究》等各级学术刊物发表学术论文八十余篇。近年来亦致力于传统文化现代转化及普及工作，在《光明日报》《人民论坛》《学习时报》等报刊杂志发表相关文章多篇。
