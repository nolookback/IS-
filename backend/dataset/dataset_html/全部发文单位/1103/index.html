报告人简介：

曾杰，男，1980年9月出生于河南省商城县。分别在2002年、2008年于中国科学技术大学获学士、博士学位，师从侯建国院士；2008-2011年于美国圣路易斯华盛顿大学从事博士后研究，师从美籍华人科学家夏幼南教授；2011-2012年于美国圣路易斯华盛顿大学任研究助理教授；2012年至今于中国科学技术大学合肥微尺度物质科学国家研究中心任教授、博士生导师，研究方向为二氧化碳催化转化。入选国家杰出青年科学基金、国家“万人计划”科技创新领军人才，荣获中国青年科技奖特别奖、中国化学会-赢创化学创新奖、侯德榜化工科学技术青年奖等奖项。迄今为止，曾杰教授已在Nature Nanotechnol.、Nature Energy、Nature Commun.等高影响力学术期刊发表了168篇论文，总被引用13,000余次，H因子为62。申请中美专利共45项。出版书籍3部，其中以第一译者身份出版的《见微知著——纳米科学》一书荣获“全国优秀科普作品”等奖项。

讲座摘要：

CO2转化利用是实现碳中和目标的潜在解决方案之一。针对CO2加氢，曾杰教授基于合肥同步辐射光源搭建原位表征平台，探索CO2加氢机理，提出了“构筑对称破缺活性中心体用以活化高对称性小分子”的催化剂设计理念，创制出Co4N、Pt1/MoS2等催化剂用于CO2加氢制甲醇等液体燃料。进一步将该理念拓展到CH4分子的活化上，创制出Au1/BP等催化剂，实现CH4在温和条件下高选择性氧化制甲醇。
