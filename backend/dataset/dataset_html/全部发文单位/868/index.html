【学术讲座】OFDM雷达通信一体化信号波形设计方法
电子与信息工程学院　2022/6/7 9:52:00
  （钟沛基 2022280095）

演讲嘉宾：廖桂生 教授 博士生导师，西电杭州研究院院长

邀请人：  黄磊 教授

报告时间：2022年6月09日（周四）下午1:30-3:00

报告地点：沧海校区-致信楼-N710

 

报告摘要：现代作战中的电子设备众多，包括各种雷达、通信、侦察和干扰设备，这些设备装载在同一平台上的会造成装备资源冗余、电磁兼容性差、规模庞大、灵活性差、操作复杂等问题。近年来，电磁空间一体化概念、理论和技术得到了广泛关注，其中一体化波形设计与信号处理是其核心基础。本讲座集中讨论在雷达通信一体化波形设计与信号处理，针对正交频分多址（Orthogonal Frequency Division Multiplexing, OFDM）雷达通信一体化波形对通信调制信息敏感、功率资源利用率低、通信信道和目标散射特性误差的敏感的问题进行研究，提出了相应的解决方法。

 

报告人简介:廖桂生，男，1963年6月出生于广西桂林，1985年获广西大学数学专业理学士学位，分别于1990和1992年于西安电子科技大学获硕士和博士学位。现任西安电子科技大学教授、博士生导师，西电杭州研究院院长。长期致力于雷达信号处理技术研究，尤其在阵列信号处理、空时二维自适应信号处理、运动平台雷达地面运动目标检测、小卫星分布式雷达信号处理、合成孔径雷达信号处理、电磁空间一体化技术等方面开展了三十余年的研究。获2008年度国家杰出青年科学基金、2004年入选国家人事部颁发的首批“新世纪百千万人才工程国家级人选”，曾受聘教育部知名学者特聘教授，国家自然科学基金委创新研究群体负责人、教育部知名学者创新团队“雷达信号处理”负责人。
