【深圳大学40周年校庆千场学术活动（第882场）】“学术研究的苦与乐”学术讲座在传播学院举办
传播学院　2023/6/7 15:46:00
  （钟沛基 2022280095）

5月25日下午，暨南大学新闻与传播学院原党委书记、副院长，国内著名新闻史学者刘家林教授受邀为传播学院师生作了一场以“学术研究的苦与乐”为主题的学术讲座。本场讲座由传播学院黄春平教授主持。

 

（刘家林教授在讲座中）

刘家林，教授，博士生导师，国内著名新闻史学者，曾任暨南大学新闻与传播学院党委书记、副院长。其著作《中国新闻通史》《中外广告通史》被列为全国通用教材。曾倾十年之力写成《新中国新闻传播60年长编》，填补了新中国新闻事业史的学术空白。

在讲座中，刘老师从自身治学的经历出发与听众分享了他对新闻史研究的心得和思考，并结合自己几十年教学科研的经验，对在场师生提出了以下三点希望：

第一，做新闻史研究应当甘坐冷板凳。刘老师以自己为例，向同学们讲述他为做研究跑遍国内各大图书馆甚至远赴英伦和日本，寻求第一手资料的经历。由于新闻史既具有新闻学的特征，又具有历史学的特征，需要以新闻的视角“在故纸堆中做硬功夫”。学者面对史料往往是“单打独斗”，既要尽己所能占有最广大的史料，又要耐得住寂寞。“做研究要有‘上穷碧落下黄泉，动手动脚找东西’的精神”，刘老师说道。

第二，做新闻史研究要有雄心，努力写出“生龙活虎的新闻史”。刘老师认为，新闻史研究往往更需要站在一手史料和前人的肩膀上进行继承和发展，继承的是前人的精神和精准的史料，发展的是写史的视角和观点。这就要把人物放在历史中，写出“生龙活虎的新闻史”。刘老师在讲座中讲述了他过往研究马礼逊、梁启超、成舍我等近代著名新闻史人物时，实地考察故居、旧址等历史遗迹的经历，立足实际，合理想象，为史学研究注入更多色彩和活力。

第三，做新闻史研究要有国际视野，应努力创造新闻学研究的粤港澳大湾区学派，做出深圳特色。刘老师勉励学生在学习研究中要保持国际视野，应在阅读中文史料的基础上，加强外语学习，理解吸收外国成果，推进自身的研究，要借助深圳这片改革开放的前沿热土，形成自身的研究特色。

最后，黄春平教授对刘老师的分享做了总结，在场同学也表达了对刘家林老师的感谢。本次讲座在在场师生的掌声中顺利结束。
