学术讲座：中山大学田宇教授—是创造金字塔底层的财富还是创业减贫？
经济学院　2021/11/24 9:42:00
  （钟沛基 2022280095）

讲座题目：是创造金字塔底层的财富还是创业减贫？

讲座时间：2021年11月26日（周五）15:00

讲座地点：腾讯会议（会议ID：537 611 705）

主讲嘉宾：田宇教授  中山大学

主讲嘉宾简介：

　　田宇，1968 年4 月生，现为中山大学管理学院教授、工商管理博士生导师，主要研究物流与供应链管理、金字塔底层创业管理方向，享受国务院政府特殊津贴专家（2018 年）、“广东省特支计划”教学名师（2019 年）、“楚天学者”特聘教授（2011-2016 年）、宝钢优秀教师（2016 年）、教育部新世纪优秀人才（2012 年），广东省“千百十人才工程”省级培养对象（2011 年）。荣获省部级社科成果奖6 次，其中二等奖3 次（排名第一），获广东省、湖南省教学成果奖一等奖3 次（两次排名第一），研究成果报送中共中央办公厅1次、获国家最高人民法院院长周强同志签批1 次、获省级领导签批4 次。出版著作8 部，发表科研论文70 余篇，其中SSCI/SCI 收录论文17 篇（Q1区6 篇、Q2 区4 篇）、NSFC 管理学A 类期刊论文13 篇、NSFC 管理学B 类期刊论文9 篇，主持国际合作基金课题1 项、国家基金课题5 项，其中，国家自科面上基金结题评价优秀1 项。曾挂任湖南湘西土家族苗族自治州州长助理、吉首大学校长助理（2009-2010年）、聘任吉首大学商学院院长（2014-2018 年）、兼任中山大学教务处副处长（ 2011-2013 年） 、教育部物流管理与工程类专业教学指导委员会委员（2013-2017 年）。现担任广东省物流与供应链学会会长（省一级学会）、广东省高等学校物流管理与工程专业教学指导委员会副主任委员、教育部本科专业类教学质量国家标准审核专家组人文社科类管理学组专家。

讲座主要内容：

　　社会科学中多个学科都对贫困问题展开过大量研究，但如何从管理学视阈研究贫困问题却是近年来的一个新兴前沿研究主题。

　　本讲座从金字塔战略管理理论和创业减贫理论等两条研究线索出发，不仅一方面结合具体个案分析，阐述了两种不同的扶贫思路与路径，提出了制度创新是中国促进贫困地区创业的主要推动力的观点，同时还提出了贫困人口创业机会塑造观，进一步从理论上回答了贫困人口为什么可以创业的问题。

　　欢迎广大师生积极参加！
