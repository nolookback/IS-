学术讲座：多模态编码相位成像在超快测量中的研究与应用
机电与控制工程学院　2024/9/12 17:13:00
  （钟沛基 2022280095）

讲座题目：多模态编码相位成像在超快测量中的研究与应用

演讲嘉宾：徐英明 博士

讲座时间：9月 13 日（星期五） 11：00

讲座地点：深圳大学南校区致信楼S622会议室

邀 请 人：田宜彬 教授

讲座内容简介：

单次超快测量在极端物理现象的实时检测和超短脉冲的时空耦合测量方面具有重要应用，多模态编码相位成像作为一种无透镜相干衍射成像方法，可以实现时空复用的单次多帧高分辨超快成像。本报告主要介绍了编码相位成像在激光诱导损伤、光克尔效应、等离子体产生等超快现象的实时检测方面的测量应用，主要采用了高信噪比的分束编码平均相位成像方法，以及结合条纹相机的时间偏转相位成像方法。另外，报告介绍了编码相位成像在时空耦合测量方向的应用，包括采用相干调制成像结合FROG实现飞秒脉冲的时空耦合测量，以及结合电光晶体实现纳秒脉冲的时空测量。多模态编码相位成像，在单幅衍射光斑强度中可以区分不同波长、不同角度、不同相位分布的多个模态，作为无透镜相位成像方法，在X射线、极紫外、太赫兹等特殊波段具有重要应用前景，并且结合多模态成像方法，对宽带光谱阿秒脉冲的时空检测方向的研究具有重要意义。

演讲嘉宾简介：

徐英明，博士毕业于中国科学院上海光学精密机械研究所，目前在之江实验室进行博士后研究工作，主要从事单次相干衍射成像算法的研究工作，用于解决单次相位成像中收敛性低和信噪比低的问题，同时，用于大口径器件面型和缺陷的快速高精度测量。另外，从事基于多模态相干衍射成像的超快相位测量研究，包括对纳秒到飞秒动态场景的超快成像和超短脉冲的时空耦合测量研究。已经发表SCI论文8篇，申请专利9项，在研主持中国博士后科学基金面上项目和中国博士后基金特别资助项目。
