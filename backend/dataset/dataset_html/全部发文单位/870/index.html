学术讲座 | 城市形态演化信息提取与情景模拟
建筑与城市规划学院　2022/6/6 17:26:00
  （钟沛基 2022280095）


报告时间：6月7日下午2:15-4:00

报告地点：深圳大学建筑与城市规划学院C205


讲座嘉宾：

陈逸敏，中山大学地理科学与规划学院副教授、博士生导师。主要研究方向包括城市大数据分析与信息提取、城市演化模拟、城市化环境影响情景建模、城市可持续发展等。主持国家自然科学基金面上和青年基金、广东省自然科学基金杰出青年项目等，参与国家重点研发计划2项。获2021年科睿唯安高被引学者（交叉学科）。获广东省科技进步奖二等奖。

讲座内容：

本讲座将介绍基于多源大数据的非监督机器学习方法，归纳了城市建筑、人口、活动和功能等要素的空间组织模式与交互机制。结合IPCC提出的共享社会经济路径情景，提出了城市水平-垂直结构演化情景模拟模型，模拟了未来不同发展路径下的城市化格局及其环境影响。
