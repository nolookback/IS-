学术讲座（七十五）：生物可降解高分子材料及其产业化
化学与环境工程学院　2024/11/4 17:22:00
  （钟沛基 2022280095）

主讲人：陈学思院士
时间：11月5日下午14:40
地点：丽湖校区守信楼420会议室

 

报告人简介：

陈学思，1959年12月出生，中国科学院院士，中国科学院长春应用化学研究所研究员，所学委会常务副主任。第十四届全国人大代表，吉林省科协副主席，吉林省欧美同学会会长。长期从事高分子化学与物理研究，开发了高纯度的聚乳酸和聚氨基酸两大类可降解材料，实现了聚乳酸塑料和医用高分子材料及器件产业化，获得14件中国医疗器械III类注册证。同浙江海正生物材料股份有限公司合作实现了6万吨聚乳酸产业化，公司已成功上市。与普立思（芜湖）生物技术有限公司合作建立了第二条7.5万吨乳酸和5万吨聚乳酸全链条生产线。在长春市莲花山区设立院士科研产业园，分别成立了长春赛诺海思生物医用材料和赛诺美焱科技有限公司。作为负责人承担了国家“十四五”重点研发计划，基金委基础科学中心等项目。发表论文千余篇，他引5万余次，h因子121。授权中国发明专利300余项。
