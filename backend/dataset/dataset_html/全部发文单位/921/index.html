学术讲座（二十六）：G蛋白偶联受体研究和药物发现
生命与海洋科学学院　2022/1/5 9:55:00
  （钟沛基 2022280095）

时间：2022年1月7日（星期五）下午16：00

地点：深圳大学丽湖校区守慧楼501会议室

主持人：胡章立 教授

主讲人：杜洋 博士

香港中文大学(深圳)生命与健康科学学院/科比尔卡创新药物开发研究院

主讲人介绍：

杜洋，责任研究员、博士生导师、教育部知名学者奖励计划青年学者。2011年于中国科学技术大学取得博士学位。2011年末赴美国斯坦福大学医学院，师从2012年诺贝尔化学奖得主Brian Kobilka教授从事G蛋白偶联受体（GPCR）相关的博士后研究工作。期间获美国心脏协会全额博士后奖学金、斯坦福心血管研究所资助和密歇根大学安娜堡医学院药学系tenure-track助理教授职位等。主要研究方向是以重要药物靶点GPCR为对象，从事其与下游信号分子复合物的结构、功能和药物发现工作。迄今已发表50多篇高质量SCI论文，包括以第一或通讯作者在Cell、Nature、Cell Research、JACS、Nature Comm.等国际顶级期刊报道的一系列科研成果，并受邀为Science、Cell Research、Nature Comm.等多种国际顶尖科学杂志审稿，也担任国自然交叉科学部和教育部项目评阅人。杜博士已入选国家级(青年知名学者)、广东省(珠江青年拔尖人才)、深圳市(孔雀人才)、龙岗区(深龙英才)、大学(校长学者)等各级人才支持计划。

研究领域：

G蛋白偶联受体、受体生物学、基于结构的药物设计
