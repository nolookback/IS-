史桂芳，首都师范大学历史学院教授、博士生导师。国家社科基金重大项目首席专家、首都师范大学东亚历史研究中心主任、抗日战争史学会常务理事、中国日本史学会常务理事中华日本学会常务理事、中国中日关系史学会常务理事北京抗日战争史学会副会长。主要从事中国近现代史、抗日战争史、中日关系史等方面的教学与研究。主持并完成国家社科基金项目2项、北京市社科基金项目1项、中国社会科学院中日历史研究中心项目2项。正在主持国家社科基金项目1项，正在担任国家社科基金重大项目首席专家。独立出版《东亚联盟论研究》、《中国的对日战略与中日关系》、《日本"兴亚"论研究》等学术著作6部，合作出版学术专著4部，翻译出版《卢沟桥事件》、《张学良》等著作，在《历史研究》、《世界历史》、《当代中国史研究》、《中共党史研究》等国内外学术刊物上发表论文数十篇。
