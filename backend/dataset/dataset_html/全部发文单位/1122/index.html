学术讲座（二十四）：荧光识别信号调节与生物传感
化学与环境工程学院　2021/4/19 14:28:00
  （钟沛基 2022280095）

主讲人：仉华教授

时间: 4月20日下午15:00

地点: 丽湖校区守信楼（原B1栋）420会议室

报告人简介：

仉华，教授，博士生导师，河南师范大学化学化工学院副院长。国家优青、河南省优秀专家、河南省学术技术带头人、河南省中原千人特支人才计划-科技创新领军人次、河南省高校科技创新人才、河南省青年科技创新奖获得者，河南省模范教师等。研究工作中将“如何通过宏观识别信号反映微观的生物分子事件”作为主要解决的科学问题，以改变目前我国用于生化分析与生物成像的功能性分析试剂主要依靠进口的现状为着眼点，努力开展相关领域具有自主知识产权的创新性研究。以第一作者或通讯作者在ESI化学J. Am. Chem. Soc., Chem. Sci., Anal. Chem., Chem. Commun.等SCI学术期刊发表代表性论文30余篇，SCI 1区 25篇；建立的分析检测新技术、新试剂申请知识产权保护授权国家发明专利7项，获得美国、欧洲、日本专利各1项。在基础研究的基础上，积极从事社会服务事业，已转让专利一项，为新乡市锦源化工有限公司核心产品乙烯砜基染料合成提成技术支持，并已通过欧盟委员会隶属的欧洲化学品管理署签发的《REACH法规》；协助该公司组建、发展“河南省医用活性荧光染料工程技术研究中心”、“新乡市活性染料中间体工程技术研究中心”，为提高其自主科技创新提供源动力。
