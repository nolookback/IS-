“数字新闻认识论”学术讲座在传播学院举办
传播学院　2024/1/26 14:51:00
  （钟沛基 2022280095）

2023年12月29日上午，浙江大学求是特聘教授、博士生导师吴飞受邀为传播学院师生主讲了一场主题为“数字新闻认识论”的学术讲座。本场讲座由传播学院杨洸教授主持。

在讲座中，吴教授以新闻业变革为起点，从经验与规范、理性求真与感性关怀出发，围绕“数字时代新闻业合法性”“新闻认识论”与“新闻生产的事实核查”三部分内容展开叙述，认为后真相时代消解了客观、真实、专业主义等新闻业合法性根基，因而重新认识数字时代的新闻业态发展十分必要。

吴教授提出，“确证真信念”和“认知正义”是数字新闻的两个合法性来源。大众对于确证真信念的要求并未降低，同时在数字时代认知正义带来更加规范的要求。他表示，基于社会认识论的数字新闻认识论有助于新闻业寻找获取事实的最佳实践，满足经验层面的需要，也有助于从社会层面思考新闻业的发展意义。

最后，杨洸教授对吴老师的分享做了总结，在场师生也与吴老师进行了深入交流。
