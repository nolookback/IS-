学术讲座（十五）：我和木兰有个约会
生命与海洋科学学院　2021/11/11 9:02:00
  （钟沛基 2022280095）

时  间：2021年11月19日（周五）上午10：30-12：00

地  点：深圳大学丽湖校区守忠楼（A5）一楼报告厅

主讲人：朱开甫  教  授

主持人：张永夏  系主任

主讲人介绍：

朱开甫，华南农业大学客座教授，先后担任五家公司的执行董事、总经理，2003年创建神州木兰园，任董事长，现任国际木兰组委会副主席、国家林业局木兰科专家、广东省野生动植物保护协会理事、木兰产业国家创新联盟理事。

朱开甫一手创建的“神州木兰园”经过近二十年的发展，现已成为为世界上最大的木兰科种质资源保存及产业化基地。投资总额超过1亿元，总种植及示范面积6600亩，现已培育出213个种及品种，培育几十万株木兰科珍稀濒危树种。先后在广东、广西、海南、云南等十多个省建立了示范及迁地保护中心。现为（国家级）——“木兰植物保育基地”、“广东省木兰科种质资源库”并加入BGCI和MSI两大国际组织。培育的树种在中国南方绿化苗木博览会”上获得10个金奖。累计承担国家级项目5项，省部级项目12项，市级项目20余项。获得市级科技进步奖1项、省级科学技术奖1项、国际相关奖项3项。获得发明专利20余项、发表国内核心期刊文章16篇、SCI论文6篇、出版专著3部。

国家林业局，中华全国工商业联合会，中国光彩事业促进会联合为其颁发了“光彩事业国土绿化贡献奖”。这是广东省唯一获得此殊荣的个人。中国科学院华南植物园为了表彰朱开甫董事长对木兰做出贡献，把一木兰科新种定名为“开甫木莲”，广东电视台以朱开甫及神州木兰园为题材拍摄的“我和木兰有个约会”纪录片在国际上最具影响力的法国“欧洲电影节”上获得首映并荣获评审团大奖。电影节仅设四个纪录片奖项。这也是广东电视台成立以来第一次在国际上获得的大奖。
