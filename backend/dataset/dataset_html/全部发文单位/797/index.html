报告人简介：

魏涛教授1999 年毕业于香港中文大学化学系，之后留校攻读博士学位， 2003 年于香港中文大学获化学博士学位，2003 年-2005 年在德国巴斯夫公司任职，2005 年7-12月在美国明尼苏达大学做博士后研究，2006 年1 月起在香港中文大学任教。2008 年被聘为香港中文大学化学系助理教授，并于2012 年和2017 年分别晋升为副教授和教授。其团队主要研究领域包括：高分子、胶体、软物质的设计和合成及其相互作用表征，纳米粒子的合成与自组装，自组装理论与方法研究，Pickering 乳液制备和应用，生物骨材料和涂层应用等。
