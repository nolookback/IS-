学术讲座：暨南大学付业林副教授—基于多智能体的供应链金融中小企业信用评分鲁棒群决策方法

讲座题目：基于多智能体的供应链金融中小企业信用评分鲁棒群决策方法

讲座时间：2021年10月25日（周一）8：30-10：00

讲座地点：四方楼东303

主讲嘉宾：付业林副教授  暨南大学

主讲嘉宾简介：

　　付业林，男，博士，副教授。2017年11月于中国科学技术大学管理学院获管理学博士学位，2018年2月于香港城市大学商学院获哲学博士学位。2020年加入暨南大学智能科学与工程学院工业工程系。受邀担任中国管理现代化研究会管理与决策科学专业委员会理事、中国“双法”研究会服务科学与运作管理分会理事、广东省系统工程学会理事。研究领域包括供应链金融、ESG、能源与环境管理等。近年以第一作者/通讯作者发表30余篇SCI/SSCI匿名评审学术期刊论文。主持国家自然科学基金青年基金项目（一项）、广东省基础与应用基础研究基金粤深联合基金项目（一项）和中央高校基本科研业务费青年基金（自然科学类）项目（一项），以项目经理身份负责香港创新及科技基金项目（一项），联合主持香港大学-渣打慈善基金金融科技学院项目（一项）。

讲座主要内容：

　　作为缓解中小企业“融资难、融资贵”的有效手段之一，供应链金融近年来成为学术圈的热门研究课题。本研究关注供应链金融模式下中小企业信用评分问题，针对其多因素、复杂性、多主体和不确定性特征，以人工智能技术和现代决策理论为基础，对供应链金融模式下中小企业信用评分决策理论与方法进行系统分析与研究。主要内容包括：（1）供应链金融模式下中小企业信用评分智能决策模型；（2）基于多智能主体的供应链金融模式下中小企业信用评分鲁棒群决策方法。
