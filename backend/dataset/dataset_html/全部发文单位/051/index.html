讲座题目：摩擦耗能的声子机理与摩擦振动

讲座时间：2025年4月29日（星期二）15：00

讲座内容简介：

摩擦耗能和热量传递是自然界中最普遍的物理现象。据统计，全世界约1/3的一次性能源由摩擦过程消耗，研究摩擦耗能机理进而对摩擦进行主动控制是提高机械装备性能、解决当今世界能源危机的一条重要技术途径。当两个表面相互滑动时，摩擦会在界面处引起发热、振动、噪声等，这些现象主要与摩擦过程中的声子激励和声子输运有关。声子是固体晶格中原子振动的量子化，摩擦过程激发的声子频率、强度以及它与摩擦系统的相互作用对摩擦耗能具有重要影响。报告介绍课题组最近几年在摩擦激励声子方面的研究工作，介绍摩擦声子的发现及摩擦声子模型，研究成果对摩擦力及摩擦诱导振动的调控具有重要指导。

演讲嘉宾简介：

魏志勇，东南大学机械工程学院副教授，博士生导师，江苏省优秀博士论文获得者，唐仲英基金会“仲英青年学者”，是东南大学和美国加州大学伯克利分校联合培养博士（导师：东南大学陈云飞教授和加州大学伯克利分校Chris Dames教授），课题组主要研究方向为机械摩擦学与表界面科学，以第一作者或通讯作者在Physical Review Letters、Nano Letters、Journal of Applied Physics、ACS Applied Materials &Interfaces、Friction、Carbon等期刊发表学术论文30余篇，Google Scholar 引用1300余次，受邀担任摩擦学领域知名学术期刊Friction 青年编委、江苏省机械工程学会摩擦学分会常务委员，获2018年度国家自然科学二等奖（获奖名称：摩擦界面的声子传递理论与能量耗散模型）和2016年教育部自然科学一等奖（获奖名称：能量在多层膜界面的输运与耗散机理），是中国机械工程学会高级会员、教育部硕士/博士学位论文评审专家和国家自然科学基金通讯评审专家。
