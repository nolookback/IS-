报告主题： 我国真菌新农药创制与生防应用案例

主讲嘉宾：王中康 教授

报告主持：谢宁 副教授

报告时间：6月24日下午15:30

报告地点：深圳大学丽湖校区守慧楼A6-501

主讲嘉宾简介：

王中康，重庆大学教授、博士研究生导师。主要从事生物农药创制与植物病虫害绿色防控技术研究。现任国家生物农药与生物防治产业技术创新战略联盟常务理事，国家生物农药科技创新联盟专家委员会委员；曾任中国菌物学会理事，中国植物保护学会生物防治专业委员会副主任。先后主持国家自然科学基金项目，国家公益性行业专项课题、中美国际合作科研项目、国家农业成果转化项目以及国家烟草总局重大专项课题等科研项目10余项。主持起草的4个行业标准进入颁布实施阶段。研究成果获得重庆市科技进步一等奖和二等奖。
