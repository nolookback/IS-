基础医学院学术讲座-单细胞测序技术及其在科学研究中的应用
医学部　2021/10/9 19:45:00
  （钟沛基 2022280095）

主讲嘉宾： 梁羽 教授

时间： 　　2021年10月15日上午 10:15

地点： 　　深圳大学丽湖校区四方楼北303

主持人： 　郑多 教授

主讲内容简介：

单细胞分析技术是目前生物医学研究的前沿工具，目前的技术已经能对单个细胞进行DNA、RNA及蛋白等多种遗传-功能物质进行分析，比较全面的获得细胞层面的遗传物质变异、表达及相关功能的特性，单细胞测序已经在胚胎发育、肿瘤发生发展等研究领域成为首选的研究工具，并且已经在胚胎植入前诊断、肿瘤早期诊断及预后评估、核酸药物开发等多个领域得到实际应用。

主讲嘉宾简介：

梁羽，临床医学学士，基因组学博士，长期从事遗传学研究及基因检测相关研发工作。梁羽博士是国内最早从事NGS测序及遗传学产品开发的人员之一，在Science、Clin Chem等国际著名期刊上发表了多篇基因组医学论文，主持完成国家自然科学基金1项，参与完成973等多个项目。
