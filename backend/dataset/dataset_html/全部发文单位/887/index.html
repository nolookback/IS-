讲座简介：

我们熟悉设计工具是不是正“自异”于我们的日常生活，成为一个自我满足的体系? 今天城市设计师的重要任务就是创造性的重新建立两者之间的联系。“场景”正是这个过程的关键一环。 

讲座题目：正常 ·异常 ：“场景” 对于设计的回归

讲座时间：2022年05月09日，14：15-15：40

讲座地点：线下深大建规学院C205；线上腾讯会议312-515-311

专家简介：

张宁，德国斯图加特大学博士，作为ppas合伙人、中国区代表，中国建筑学会立体城市与复合建筑专委会委员。在带领ppas中国办公室其间，主持与参加了一系列重要的城市设计项目，多次获得以中国建筑学会创作奖银奖、意大利A‘DesignAward金奖、德国IconicAwards为代表设计奖项。
