03讲座概要

城中村承载着近1000万深圳人的居住与生活，是诸多新青年新市民来深圳的首选落脚地，为深圳的发展做出了巨大贡献。作为一种仍将长期存在的空间形态，城中村的高质量和可持续发展势在必行。近年来深圳稳步推进城中村改造已极大提升了城中村的居住环境、公共服务和基础设施质量，但对比现代化的城市社区而言，城中村在空间品质、生活环境、公服务水平、社区治理等方面均仍存在差距，进一步的改善提升迫在眉睫。

本讲座基于尊重与传承、融合与创新，结合多种规划手段综合促进城中村的韧性发展，旨在为满足居民日益增长的美好生活需求以成为更具安全感、获得感、幸福感的高品质人居环境的城中村做出积极实践探索。


04讲者简介

王嘉，深圳市城市规划设计研究院副总规划师，城市更新规划研究中心主任。教授级高级工程师，国家注册规划师，中国城市规划学会城市更新分会委员，中国建筑学会城市设计分会理事，中国城市科学研究会城市更新专业委员会 副主任委员，深圳市城市规划学会存量规划实施专业委员会 副主任委员。

主持、参与各类规划、研究几百项，具有丰富的实践和研究经验。工作期间完成项目荣获包括中规学科技进步奖、华夏建设科学技术奖在内的国家、省、市级各类奖项总计20余项，其中国家级奖项6项，省级奖项9项。

出版《深圳城市更新探索与实践》、《城乡规划编制技术手册》著作，参与多个城市更新技术规范的编制与标准的制定，在《城市规划学刊》、《规划师》及《城市建设》等核心期刊上发表多篇论文，包括我国城市更新演进历程、挑战与建议，多重视角下旧城改造项目地块合理容积率确定方法探析等。对深圳市城市更新背景下的政策制度、实施机制、规划编制、项目运作等方面具有独特的见解。
