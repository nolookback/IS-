电子与信息工程学院　2024/6/26 11:45:00

嘉　 宾：王鑫  深圳大学

时　 间：7月1日09:30

地  点：沧海校区致信楼N710

 

摘要：本次讲座旨在分享主讲人王鑫老师在机电双创俱乐部建设与管理方面的经验。在总体概览“智能驱动创新实践俱乐部”成立和发展的基础上，着重介绍AutoLeaders创新实践俱乐部的年度工作计划，具体包括从招新到竞赛组织、技术培训和企业参访等一系列活动；除此之外，对俱乐部的组织架构、经费来源、校企合作模式以及丰富的活动内容进行了详细阐述

 

主讲人简介：

 王鑫　机电学院自动化教学实验中心高级实验师、院办主任

2005年毕业于英国利物浦大学，软件工程专业，硕士

2010年至今，机电学院自动化教学实验中心高级实验师、院办主任

深圳市电磁控制重点实验室高级工程师
