主讲人：李芯芯 博士

　 　　南方科技大学海洋科学与工程系副教授、研究员

2013年获得德克萨斯农工大学化学海洋学博士学位。主要从事从河口向海沟各海洋生态系统有机碳的生物地球化学循环、海洋碳汇监测、评价和应用，及其与海水酸化、全球气候变化和碳中和的相互关系研究。在海洋科学知名学术期刊已发表30余篇论文和一篇国际海水营养盐标准报告。获深圳市孔雀计划海外高层次人才，南方科技大学“优秀书院导师”、“优秀共产党员”，“第六届南方科技大学青年教师教学竞赛特等奖”等荣誉称号。

 

此讲座由生命与海洋科学学院《海洋科学专题讲座》课程组织。

欢迎广大师生前来聆听！
