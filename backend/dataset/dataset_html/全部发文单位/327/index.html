“ 如何进入超越论生命的‘活的当下’”学术讲座在马克思主义学院顺利举办
马克思主义学院　2024/6/26 13:19:00
  （钟沛基 2022280095）

2024年6月14日下午，南京大学哲学博士，南昌大学人文学院教师陈员受邀为深大师生带来了一场题为“如何进入超越论生命的‘活的当下’”的精彩讲座。本次讲座由深圳大学马克思主义学院副院长张守奎教授主持，众多师生到场聆听。

　

在讲座中，陈员老师首先指出，目前学界对于胡塞尔的“活的当下”通常有一种本能化的理解，即将其视为一种先天生物的本能意向性。然而，这种理解未能深刻揭示“活的当下”的真正内涵和重要性。他强调，对于“活的当下”的理解，不应仅仅停留在本能层面，而应更加关注其作为一种超越论生命的存在状态。

接下来，陈员老师对超越论生命进行了具体论述。他指出，超越论生命的“活的当下”是一种持续再构造的生命存在，它自身虽非被构造，但持续地进行着构造活动。在这种构造活动中，活在当下是基点，为后续的所有构造提供了有效性的来源。陈老师进一步解释说，这种“活的当下”不仅是我们认识世界的起点，也是我们与世界建立联系的桥梁。

在探讨如何进入这种超越论生命的“活的当下”时，陈员老师提出了自己的见解。他认为，要进入这种状态，首先需要有清醒的自我意识，意识到“活的当下”的重要性。其次，需要通过现象学的分析和思考，深入理解“活的当下”的本质和意义。最后，需要通过实践和体验，将理论转化为实际行动，真正体验到“活的当下”的存在状态。

讲座最后，张守奎教授对陈员老师的精彩讲述表示感谢并做总结发言，指出这是一场内涵丰富、具有启发性的学术讲座。此次讲座内容政治站位高、逻辑严谨并且见解独特，提升了师生们的专业素养，同时也营造出良好的学术氛围。在热烈的讨论氛围中，讲座圆满落下帷幕。
