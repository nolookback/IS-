学术讲座——北欧五国学前教育政策创新研究
教育学部　2023/2/21 14:10:00
  （钟沛基 2022280095）

讲座主题：北欧五国学前教育政策创新研究

讲座时间：2023年2月27日（周一）下午16：00至17:30

讲座地点：国际会议厅A厅

主讲嘉宾：张兆芹

主讲嘉宾简介：

          　张兆芹，博士，教授，学科带头人，研究生导师，教育部“国培计划”专家。香港中文大学教育学院教育哲学博士，深圳大学教育学部教授，2001年深圳大学资助赴英国女王大学访问学者，2013年国家留学基金资助一年赴美国密歇根州立大学访问学者。兼任中国教育学会教育管理分会教育效能委员会委员；广东省教育学会教育评价专业委员会副理事长等。主要研究方向：教育管理及学校管理，学前教育及中小学教师教育，中小学学校发展和领导、学习型学校和学习共同体创建等。承担过国家、省、市、学校各级研究项目，出版专著7部，发表论文40多篇，研究成果曾获广东省教育厅“吴汉良”三等奖；获深圳市首届教育教学科研优秀成果三等奖，深圳市第二届教育教学科研优秀成果奖高校著作类一等奖。深圳大学政府奖后奖。深圳大学学术创新三等奖等。

讲座主要内容：

探讨北欧五国学前教育政策产生的相关背景，其政治、经济、文化及教育因素对学前教育政策的形成和发展的深入影响。该研究基于北欧五国以代表性政策目标对其主要内容进行分析，从北欧五国学前教育政策的制定背景、具体内容（学前教育准入、经费、课程及师资等）、实施成效及所取得的成就和发展趋势中总结出实际经验，并结合我国国情，为我国学前教育政策的制定提供了宝贵借鉴。
