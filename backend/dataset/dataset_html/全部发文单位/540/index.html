【学术讲座】基于深度学习的图像去雾、检测取证和质量评价
电子与信息工程学院　2023/12/28 17:32:00
  （钟沛基 2022280095）

讲座嘉宾：杨高波教授 湖南大学

邀 请 人：李斌教授

时      间： 1月12日下午16点-17点

地      点： 致信楼 N710

 

报告摘要/内容简介：近几年，深度学习技术在单幅图像去雾领域取得了巨大进展，但是其性能仍然在很大程度上受制于含雾图像和无雾图像数据集。对于同一个场景，收集对齐的含雾图像和无雾图像非常困难。本报告汇报我们在图像去雾、检测取证（判别自然无雾图像还是去雾图像）和图像去雾质量评价等方面的工作进展，期望以图像去雾检测和质量评价，启发和促进图像去雾。

 

报告嘉宾简介：杨高波，湖南大学信息科学与工程学院教授、博士生导师。2004年毕业于上海大学，获工学博士学位。同期，进入湖南大学工作。2005年晋升为副教授，2008年破格晋升为教授。先后入选湖南省青年骨干教师培养对象、湖南省新世纪121人才工程和教育部新世纪人秀人才支持计划、长沙市工业科技特派员、南京玄武紫金英才计划和江苏省创新创业人才计划。2010年8月至2011年8月，在英国萨里大学进行访问学者交流。主持国家973前期研究专项、国家重点研发计划项目子课题、GF创新特区重点项目子课题、国家自然科学基金（5项）和湖南省自然科学基金重点项目等。出版专著1本，译著2本。近几年，注重产学研合作和科研成果转化，应邀担任深圳爱协生科技股份有限公司高级技术顾问（2022-2026）。
