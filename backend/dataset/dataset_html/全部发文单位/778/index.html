数学与统计学院学术报告[2022] 092号

(高水平大学建设系列报告759)

报告题目：网络群集动力学与协同控制学术讲座暨国家自然科学基金申报辅导报告

报告人：曹进德院士  东南大学

报告时间：于2022年12月2日（周五）下午14:30开始

报告地点：汇星楼514 

报告内容简介：

本报告将首先介绍网络群集动力学与协同控制的研究背景、国内外研究最新进展、对本方向的未来进行展望；接着从个人角度谈谈国家自然科学基金申报体会，并与老师们进行交流。

专家简介：

东南大学首席教授、数学学院院长、理学部主任、江苏国家应用数学（东南大学）中心主任、江苏省网络群体智能重点实验室主任、教育部数学类专业教指委委员、中国工业与应用数学学会常务理事、江苏省工业与应用数学学会理事长、江苏省高校数学联盟理事长等。全国创新争先奖获得者，享受国务院政府特殊津贴。先后当选俄罗斯科学院院士、欧洲科学院院士、俄罗斯工程院院士、欧洲科学与艺术院院士、立陶宛科学院院士、非洲科学院院士、巴基斯坦科学院院士、国际系统与控制科学院院士和IEEE Fellow 等。长期从事复杂网络与复杂系统、神经动力学与优化和工程稳定性等研究，先后主持国家重点研发计划项目1项，国家自然科学基金项目9项（含重点项目1项），教育部博士点基金3项。
