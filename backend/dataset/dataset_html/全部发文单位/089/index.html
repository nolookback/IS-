【摘要】

  传统的超大规模多输入多输出（MIMO）系统中的基带信号处理算法通常依赖于集中式基带处理（CBP）架构，在这种架构中，所有天线的原始基带信号被汇集到一个中央计算单元进行进一步处理。然而，随着基站（BS）天线数量的增加，CBP架构遇到了两个瓶颈：1）天线与计算单元之间的前传通信带宽过大；2）计算单元的高计算复杂度。为了克服这些障碍，提出了分布式基带处理（DBP），将基站天线划分为多个独立的天线簇，每个天线簇连接到一个基带单元（BBU）。 BBU之间以星型网络或链式网络连接。在DBP架构下，BBU之间的通信带宽和BBU的计算能力都受到了极大的限制。因此，传统的信号处理任务，如信道估计、下行传输的预编码和上行传输的均衡，都需要重新设计，需要新的分布式信号处理技术。在本次演讲中，我们将介绍这些领域的最新进展，并指出未来研究的可行方向。
