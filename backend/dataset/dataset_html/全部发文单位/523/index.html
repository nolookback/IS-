主讲嘉宾：张潜翀 教授

时间：2024年1月15日 周一 下午16:00-17:00

张潜翀，中国科学院福建物质结构研究所研究员，硕士生导师。毕业于厦门大学化学化工学院，曾获教育部优秀研究生奖学金，入选中国科学院海西研究院“春苗”人才计划，已主持国家自然科学基金委重大研发计划培育项目1项、面上基金1项、青年科学基金项目1项、闽都创新实验室主任基金项目1项，中国科学院海西研究院“前瞻跨越”杰出青年基金项目1项、福建省科技厅面上项目1项，主要从事配位化学与分子电子学结合的相关实验和理论研究，擅长各种材料的单分子电导测试及数据分析，以及通过单分子电导的研究方法来表征微观尺度下分子的动态行为及结构信息，以第一作者或通讯作者身份在Angew. Chem. Int. Ed., Nano Lett., Coord. Chem. Rev., Sci. China Chem., Small.等国际高影响力期刊发表论文20篇。
