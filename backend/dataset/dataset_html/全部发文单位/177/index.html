主题：中国新文明理据一一兼谈学术论文选题

主讲人：项久雨  武汉大学弘毅特聘教授

时间：2024年12月3日（星期二）下午2:30

主讲人简介：

项久雨，中华人民共和国退伍军人事务部咨询委员会委员，武汉大学主流意识形态研究院院长、首席专家，武汉大学弘毅特聘教授，武汉大学学科负责人兼学术带头人，法学博士、政治学博士后、博士生导师，纽约哥伦比亚大学研究学者，享受国务院特殊津贴专家。主持国家社科基金重大项目4项（已完成3项）、重点项目1项，在《中国社会科学》等国内外期刊发表系列学术论文，分别荣获中宣部第四届、第六届、第七届“期刊主题宣传好文章”，荣获《中国社会科学》首届年度好文章，其中多篇论文被《新华文摘》全文转载。所撰写的咨询报告获中央主要领导同志重要批示或被中央办公厅等部门采纳多篇。在人民出版社等出版著作多部，科研成果获教育部第九届高等学校科学研究优秀成果奖 (人文社会科学 )一等奖，湖北省第十三届社会科学优秀成果一等奖，武汉大学2022年学术成果特等奖等奖励多项。
