“深大营销学术前沿”是我校市场营销系打造的品牌学术活动，将邀请国内外营销学者主讲，旨在分享前沿的学术思想、成果及方法。

 

题目：智慧营销：创新与案例

主讲人：赵占波（北京大学软件学院金融管理系教授）

时间：2020年5月14号（周五）上午10：15-11：40

地点：汇文楼（文科楼）H2-401教室

 

讲座内容：

随着5G时代的到来，智慧营销将迎来更广阔的空间。新技术融合下的智慧营销带来的远不止数据的采集、打通和分析，而是基于实际营销场景对消费者和业务的精准洞察，以此实现资源的精准匹配与价值的高效转化。把企业在触达目标消费者过程中产生的多元数据变为新的生产力，通过技术赋能形成预见未来的营销策略，建立实时交互的智慧营销体系，将成为把握时代机遇的重要战略。本讲座将聚焦智慧营销传播理论、方法与实践，为经管教师和学生的研究和营销实践提供全新思维。

 

主讲人简介：

赵占波，是北京大学软件学院金融管理系教授、教学指导委员会专职副主任、北京大数据研究院保险大数据研究中心主任以及科技部高新技术评审专家。赵老师的研究兴趣是互联网金融、保险、营销创新等，曾发表超过50篇论文，出版30余本译著/学术著作，包括《智慧营销》、《金融营销学》、《互联网保险》、《移动互联营销》等，并且翻译出版超过15个版本的菲利普·科特勒经典营销教材。曾于2010年获得IBM奖教金，2014年获得华为奖教金，2015年获得北京大学青年教师教学大赛优秀奖，2020年获得Emerald年度“杰出论文奖（Outstanding Paper）”。
