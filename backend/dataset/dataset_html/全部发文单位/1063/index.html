报告人简介：

徐明华教授，1999年7月博士毕业于中国科学院上海有机化学研究所，先后在美国弗吉尼亚大学化学系和乔治城大学医学研究中心从事博士后工作研究。曾任中国科学院上海有机化学研究所副研究员；中国科学院上海药物研究所研究员、课题组长、新药研究国家重点实验室研究员。2018年2月受邀加入南方科技大学，现为理学院化学系和深圳格拉布斯研究院双聘教授；是中科院“百人计划”、国家杰出青年科学基金获得者，上海市优秀学术带头人，中国科学院特聘骨干研究员，深圳市国家级领军人才，广东省“珠江人才计划”引进领军人才。2014年获上海市自然科学一等奖，2016年获国家自然科学二等奖。
