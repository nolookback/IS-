学术讲座：所见非所得：计算光学成像及其在半导体检测中的应用
机电与控制工程学院　2023/10/26 17:13:00
  （钟沛基 2022280095）

报告题目：所见非所得：计算光学成像及其在半导体检测中的应用

报告人：田宜彬，深圳大学机电与控制工程学院，特聘教授

时  间：2023年10月27日上午10:00

地  点：深圳大学南校区机电大楼S512会议室

报告简介：计算光学成像是融合光学器件、图像传感器和算法于一体的新一代成像技术。随着半导体技术的高速发展，传统光学成像系统难以满足半导体生产过程对检测的要求，近20年计算成像技术在半导体检测中得到了广泛应用。本次报告首先对半导体检测中常见的计算光学成像技术进行简要介绍，然后讨论两个实例：（1）利用计算光刻对光刻掩模板进行检测的WPI(Wafer-Plane Inspection)技术；（2）利用光学MEMS条纹投影3D重建进行芯片BGA封装和芯片连接器多模态检测的技术。

主讲人：田宜彬，美国伯克利加州大学视觉光学博士，深圳大学特聘教授。主要从事光学传感和计算成像、感知计算集成化智能传感器、多模式感知融合、人机交互、机器学习和检测设备等研究和开发工作。硅谷11年前沿技术研发和管理经验，在光电传感和成像行业龙头和开拓性企业任资深科学家和首席工程师等职，研发产品服务于苹果、微软、英特尔、台积电、三星和华为等20多家财富500强企业。作为技术和项目负责人，管理了10多个跨国光电成像和传感交叉学科项目，负责的项目累计研发经费超过1亿元。
