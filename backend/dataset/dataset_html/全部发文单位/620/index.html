北师大李敏谊教授受邀开设学术讲座
教育学部　2023/11/20 16:05:00
  （钟沛基 2022280095）

2023年11月9日晚，北京师范大学李敏谊教授应邀在国际会议厅D厅开设了主题为《与幼儿一起做研究—参与式视觉方法论与儿童的视角》讲座。学前教育系谢莎老师主持讲座，数十位学部师生认真聆听并进行交流。

李敏谊教授从儿童观、方法论和社会发展观的三个角度，围绕“为什么需要与幼儿一起做研究”、“儿童的视角+参与式研究+视觉方法论是什么”等问题，探讨了学前教育科学研究方法从 “针对儿童的研究”到 “与儿童一起研究”的转变。她强调了儿童的声音要被听见，也要有听众，幼小的儿童也是有决策权等科学的教育观与儿童观。  让师生进一步认识到在儿童研究中实地反思日记方法的重要意义。

      　

　 “童年不是为成人生活做准备，而是生活本身。”关于儿童研究，师生们都深刻认识到，我们还有很长的路要走。

    本次讲座在热烈的掌声中圆满结束。
