【学术讲座】时空细粒度人体动作分析研究
电子与信息工程学院　2022/7/4 9:35:00
  （钟沛基 2022280095）


报告人：刘胜蓝 、大连理工大学副教授

邀请人：邹文斌 副教授

讲座时间：2022年7月4日(周一)下午2:00-3:00

讲座地点：腾讯会议号：822-313-851

 

报告摘要：尽管目前主流的视频研究数据集很多，规模也很大，如视频识别：Kinetics, Moments in Time, UCF101等，视频分割：Breakfast, Epic Kitchens, Salads50等。但以上数据集都缺乏人体运动的特性（Kinetics等有部分运动特性，但不够专业，类别也很有限）。像UCF101选取适当的子集，只留下场景，准确率下降不多。本报告聚焦以运动为中心的人体动作分析，分析了多模态的细粒度动作识别、时序分割以及评估任务的挑战，并对相关的方法进行了介绍。此外，对多模态大类内方差的人肢体情感数据进行了介绍和分析。

 专家简介：
