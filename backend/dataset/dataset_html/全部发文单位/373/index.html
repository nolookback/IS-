深大人文论坛（第500期）暨深圳民族宗教研究基地学术讲座（第五场）：北宋辟佛论的思想史阐释
人文学院　2024/5/23 15:19:00
  （钟沛基 2022280095）

主讲人：龚隽 教授　（中山大学）

时 间：2024年5月27日　14：30

地 点：深圳大学粤海校区汇文楼国学院会议室 

 

主讲人简介：

龚隽，中山大学哲学系与中山大学（珠海）哲学系双聘教授，博导，中山大学佛学研究中心主任，兼任中华外国哲学史学会东方哲学专业委员会副主任等职。主要从事中国佛教思想史及中国哲学史研究，专著《作为“知识”的近代佛学史论——在东亚视域内的知识史论述》（合著）、《禅史钩沉：以问题为中心的思想史论述》等先后获得广东省哲学社会科学优秀成果奖，另在国内外发表论文40余篇。

 

 

内容简介：

本次讲座将以思想史的方式，重新考察了北宋时期的儒佛关系。以“辟佛”论为中心，对北宋儒佛关系史上几大重要却为学术界所忽略的问题进行论究，对以往学术史的相关讨论提出了较为重要的修正与补充。首先探讨欧阳修论史、修史思想中的辟佛论，揭示了北宋以史辟佛的类型。其次，以王安石的经子合观论为例，阐明王安石的新经学与佛学之关系。本次讲座内容还将对于北宋道学的辟佛论进行了分析，讨论道学辟佛论的思想源流与哲学方法，并特别指出道学没有完成对佛教哲学的有效批判，而使得宋代儒佛关系呈现出更为曲折与复杂的现象。
