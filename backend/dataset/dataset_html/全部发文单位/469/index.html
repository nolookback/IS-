生命与海洋科学学院　2024/3/26 9:57:00
  （钟沛基 2022280095）

报告题目：检测实验室智能化发展之路 

主讲嘉宾：靳保辉  研究员

报告主持：王潮岗  副教授 

报告时间：2024年3月27日（星期三）10:15-11:40 

报告地点：深圳大学丽湖校区四方楼 西205

报告人简介：

靳保辉，深圳海关食品检验检疫技术中心副主任，研究员。长期从事食品安全检验检测工作，在农药残留分析、食品真实性表征和食品快检技术方面承担国家和省部级科研项目20余项，获得省部级科研奖励11项，发表论文55篇，参编论著4部，制修订国家、行业标准13项。担任国家重点研发计划项目评审专家、国家科技奖励评审专家、广东省及深圳市科技评审专家、广东省食品安全专家委员会专家、质检总局风险会商专家、深圳市食品标准委员会委员。
