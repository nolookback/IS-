学术讲座：计算光学在激光核聚变中的应用

讲座题目：计算光学在激光核聚变中的应用

演讲嘉宾：刘 诚 教授

讲座时间：2024年10月16日（星期三）下午3:00

讲座地点：深圳大学沧海校区致信楼S622会议室

邀 请 人：田宜彬 教授

讲座内容简介：

自2022年8月起，美国NIF装置已经连续多次成功实现净能量增益大于1.0的惯性约束核聚变，使激光核聚变研究从可行性验证阶段步入技术提升阶段，引发了新一轮的前沿技术竞争。如何同时提升光束质量和能量是改善激光驱动器性能的核心问题，其传统的测量方法已经不能满足技术需求。计算光学方法具有结构简单、使用方便和精度高等诸多优点，在高功率激光在线测量领域有突出的优势；我国高功率激光装置的综合性能处于国际先进水平，计算光学成像方法的应用可以实时提供驱动器的核心参数，为激光核聚变相关研究提供新的诊断手段。

演讲嘉宾简介：

刘诚教授，中科院上海光机所研究员、博导、中国科学院 “百人计划”A 类人才、中科院重大专项总体技术专家，深圳大学访问教授。刘诚教授主要开展计算成像、光学检测和近场光学等研究，在层叠成像PIE和FPM相关领域的研究工作达到国际先进水平，并成功将其用于解决高功率激光束的近远场测量国际难题，为神光 II 国家激光装置研制了结构紧凑、精度高和速度快的在线检测系统，其综合性能超过日本和美国的同类测量设备。刘诚教授的研究工作获得多项国家自然科学基金、国防重大专项等支持；研究成果获中国仪器仪表学会科技进步一等奖、上海市科技进步一等奖、首届中国军民两用技术创新金奖等奖励；以第一作者和通讯作者发表 SCI 论文百余篇。
