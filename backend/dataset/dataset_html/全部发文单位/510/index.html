“技术迭代加速时代的适应性媒介制度实践”学术讲座在传播学院举办
传播学院　2024/1/19 14:30:00
  （钟沛基 2022280095）

2023年12月29日上午，苏州大学传媒学院执行院长、教育部“知名学者”特聘教授陈龙受邀为传播学院师生作了一场以“技术迭代加速时代的适应性媒介制度实践”为主题的学术讲座。本场讲座由传播学院杨洸教授主持。

在讲座中，陈教授从媒介化社会发展的轨迹出发，围绕“网络社会-平台化社会”“大数据-大模型”与“人人交互-人机交互”三部分内容展开叙述，向大家揭示在技术加速下媒介制度实践的体系与面向。他指出，技术迭代的发展下平台社会已经到来，当下平台社会展现出广泛连接、新型平台中心化、以算法为核心竞争力等系列特征。在平台社会中，用户为满足自身需求进行交互操作，从而实现价值的创造、分配与流通。

就大数据到大模型的技术迭代展现出的人工智能技术带来的新变局这一议题，陈老师认为今天的大模型不是工具，而是人类的伙伴，并为人类提供服务。人类从信息时代走向智能时代，再次刷新人们对传播，尤其是智能传播的认知，而新型交往沟通模式，将会逐渐过度到基于AGI技术，与IT文明发生联系。关于人社交互动走向人机互动这一变化趋势，他表示，算法将成为传播学领域重要的研究议题。算法的定义主要分为基于技术逻辑的定义和基于社会科学视角的定义两个层面，前者强调其生产和应用的专业性，后者则更注重社会建构与文化情境。

最后，杨洸教授对陈老师的分享做了总结，在场同学也表达了对陈龙老师的感谢。本次讲座在在场师生的掌声中顺利结束。
