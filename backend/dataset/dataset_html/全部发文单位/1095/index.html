深圳大学马克思主义学院哲学社会科学论坛学术讲座：中国共产党一百年的光辉历程和宝贵经验
马克思主义学院　2021/5/10 9:31:00
  （钟沛基 2022280095）

29日上午，深圳大学马克思主义学院哲学社会科学论坛学术讲座特邀请中国社会科学院世界历史研究所党委书记，副所长，二级研究员，中国社会科学院大学马克思主义学院教授，博士生导师罗文东教授为我院师生作题为《中国共产党一百年的光辉历程和宝贵经验》的线上学术讲座。讲座由深圳大学马克思主义学院副院长孙婷婷副教授主持。

罗文东教授首先从以下四个阶段回顾并梳理了中国共产党建党一百年以来的光辉历程：第一、新民主主义革命时期。从中国共产党的创立，到新中国的成立，中国共产党艰辛探索，奋勇开拓，风雨兼程。罗教授认为，只有中国共产党领导人民取得新民主主义革命的伟大胜利，完成由封建专制向人民民主专政的历史跨越，才能从根本上改变中国任人宰割的命运；第二、社会主义革命和社会主义建设时期。中国共产党在这一时期创造性地完成由新民主主义到社会主义的过渡，开启了在社会主义道路上进行现代化建设的历史征程；第三、改革开放和社会主义现代化建设的新阶段。罗教授强调，改革开放四十多年来，党和国家各项事业取得举世瞩目的伟大成就，为我国现代化建设积累了宝贵的经验；第四、新时代中国特色社会主义时期。进入新时代，党领导新的历史条件下的伟大斗争，统筹推进“五位一体”总体布局和“四个全面”战略布局，推动了强国大业的发展进程。

随后，罗文东教授从政治、理论和实践三个方面总结了中国共产党创造历史伟业的宝贵经验。罗教授强调，在新时代坚持和发展中国特色社会主义，必须要坚持人民至上的政治立场，保持党同人民群众的血肉联系，坚持马克思主义指导地位，不断推进理论创新和理论武装，坚持革命拼命强大的精神，以党的自我革命推动社会革命。

罗文东教授将历史与现实相紧密结合，对中国共产党一百年的光辉历程和宝贵经验进行了深入的总结和分析，讲座内容翔实、逻辑严明、思路清晰，注重归纳分析，引发广大师生的深入思考和热烈反响。
