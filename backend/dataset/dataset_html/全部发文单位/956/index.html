主讲人介绍：

杨琼 博士，教授级高级工程师,华南师范大学植物学硕士，中山大学环境生物学博士学位，现就职于广东内伶仃福田国家级自然保护区管理局。长期从事红树林保育、红树林湿地抗污除污能力及净化污水机理方面的研究，参与的“红树林湿地生态安全机制及应用研究”曾获深圳市自然科学奖。
