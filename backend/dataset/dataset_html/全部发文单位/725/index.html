【深圳大学40周年校庆800场学术活动 (第470场)】“‘延伸的心灵’和‘对话的撒播’——论作为书写的chatGPT”学术讲座在传播学院举办
传播学院　2023/4/13 17:50:00
  （钟沛基 2022280095）

2023年4月3日，复旦大学邓建国教授受邀为传播学院师生主讲了一场以“‘延伸的心灵’和‘对话的撒播’——论作为书写的chatGPT”为主题的学术讲座。本次讲座由传播学院李明伟教授主持。

2023年4月3日，复旦大学邓建国教授受邀为传播学院师生主讲了一场以“‘延伸的心灵’和‘对话的撒播’——论作为书写的chatGPT”为主题的学术讲座。本次讲座由传播学院李明伟教授主持。

邓建国老师现为复旦大学新闻学院教授，曾作为访问学者前往哥伦比亚大学进行访问学习。发表中英文论文70余篇，出版专著两部，译著多部，包括《对空言说》和《媒介即存有》，主持国家社科项目3项，并担任多份CSSCI学术刊物审稿人。邓老师通过本次讲座与大家分享其关于chatGPT的研究发现与学术思考。

讲座伊始，邓老师从彼得斯的传播思想史切入话题，并分别从语言和书写、chatGPT作为人－机共创的书写实验、chatGPT的书写特征以及作为“他者”的chatGPT这四个方面详尽地分享了其关于chatGPT的深入思考。邓老师指出，chatGPT这一强大的人工智能书写工具是人类作者的“延伸的心灵”。“写”将时间的媒介转换为空间的媒介，让思想变得可书写、可编辑。邓老师认为，作者与chatGPT的交流体现的是一种平等的人机互动关系，而沟通具有本体性作用，也是一个生产思想的过程，沟通方式会形塑内容生产，chatGPT将深刻影响人类作者的写作。

随后，邓老师指出，在面向全球用户的价值观、缺乏身体和基于书写的工作原理的语境下，chatGPT是以对话为表象的撒播。撒播的特征是若即若离、不偏不倚，这使得人机对话一直停留在表面与外部的信息交换。接着，邓老师进一步指出，人类与chatGPT的互动是我们与“他者”的互动，如何应对“他者”已成为沟通伦理的主要内容，我们需要珍惜这种差异，并从中获得收益和快乐。最后，邓老师引用柏拉图所述故事中的“pharmakon”一词作为分享结语，在希腊文中，“pharmakon”的含义具有两面性，即chatGPT“既是解药，也是毒药”。

在提问与探讨环节，邓建国老师就同学们提出的问题耐心地一一作答，并介绍了与chatGPT相关的研究怎样与理论结合，以及寻找理论的方法与技巧。

最后，李明伟教授对此次讲座分享进行总结发言，表示邓建国老师关于chatGPT的前沿研究为同学们开阔了眼界，提供了研究新思路。参会师生纷纷表达了对邓老师的衷心感谢，并上前交流、合影。至此，本次讲座圆满结束。
