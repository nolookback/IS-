报告人简介：　 

Yoshio Bando（板東義雄），前日本国立材料研究所（NIMS）MANA研究中心首席科学家以及首席运营官，澳大利亚卧龙岗大学讲席教授，日本东京大学客座教授，英国皇家化学学会会士、世界陶瓷科学院院士、美国陶瓷学会会士。Bando教授1975年博士毕业于大阪大学，1975-2017年在日本国立材料研究所（及其前身国立无机材料研究所）任研究员、高级电子束分析中心主任、MANA中心（日本文部省五大研究机构）首席科学家以及首席运营官等职，2017年至今任澳大利亚卧龙岗大学讲席教授。Bando教授研究方向主要为透射电子显微镜表征及机理研究以及多维无机纳米材料的合成和应用，是世界上顶尖透射电镜分析专家。其发现了一系列无机纳米结构，包括BN, MgO, AlN及其他无机材料的纳米管，至今仍然是全球领先的氮化硼纳米管科学家。
