报告简介（Abstract）：

　结构性心脏病是先天性或获得性心脏结构异常的统称，主要包括先天性心脏病和心肌病等，危害严重。先天性心脏病是我国最常见的高发病率、高致死率的出生缺陷，其发病率约为1.5%（全国幸存约200万患者），每年影响数十万人（或家庭）。心肌病在我国发病率约为1.6‰（全国约有200万患者），是30岁以下年轻人猝死的主要原因。结构性心脏病的诊断手段主要依靠超声影像或者CT影像技术，而治疗手段主要是心脏外科手术，其手术预后在整个治疗过程中也担任重要角色。本次报告将从以下三方面分享临床与人工智能的结合：1.介绍结构性心脏病及其现状，以及相关的临床问题和治疗方案等；2.介绍当前人工智能方法在结构性心脏病的研究与应用现状；3.介绍所在的团队亦即广东省人民医院心血管病人工智能与3D实验室在结构性心脏病的诊断、手术和预后上的系列智能化研究与应用。
