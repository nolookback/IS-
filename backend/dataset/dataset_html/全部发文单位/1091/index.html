Structure-disrupting Peptides: A New Paradigm for Drug Discovery

主讲人: 周耀旗教授

时间: 5月18日上午10:00

地点: 丽湖校区守信楼（原B1栋）420会议室

报告人简介：

周耀旗，1984年获得中国科技大学近代化学系学士学位，1990年获得美国纽约州立石溪大学化学物理博士学位，1994-2000年分别在北卡州立大学、哈佛大学进行博士后研究，2000年起担任纽约州立布法罗大学助理教授，2004年晋升为终身副教授，2006年获聘为印第安纳大学信息学院和医学院终身正教授，2013-2021年担任澳大利亚格里菲斯大学糖组学研究所正教授，2021年起被聘为广东省深圳湾实验室资深研究员。在结构生物信息学领域有着长期的工作经历，研究团队多次参加国际蛋白质结构预测和功能预测大赛，最高名次为第一名。到目前为止共发表论文200余篇，引用1万多次，H因子61。目前，周耀旗课题组主要从事蛋白质/RNA的序列、结构与功能关系方面的基础研究和生物分子检测、药物开发方面的应用研究，希望通过计算与实验相结合的方式，为数字医学及新药发现探索一条新路。

欢迎有兴趣的师生参加！
