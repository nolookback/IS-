【学术讲座】超宽带复杂电磁环境下的快速频谱感知技术研究
电子与信息工程学院　2022/6/30 15:51:00
  （钟沛基 2022280095）

题目：超宽带复杂电磁环境下的快速频谱感知技术研究

报告人：李林，西安电子科技大学副教授，博士生导师

报告时间：2022年7月4日（星期一）上午10:00-11:00

报告地点：在线腾讯会议：507-672-039，会议密码：518060

邀请人：李良群  教授

 

内容简介：航天测控技术是航天事业快速发展的重要保障。在实际卫星测控环境中，存在各种有意和无意电磁干扰，对航天测控链路造成了严重的电磁污染，威胁着航天测控的安全性、稳定性和可靠性。报告主要介绍解课题组近年来在航天测控链路干扰监测、干扰与信号识别、辐射源快速定位等方面取得的研究结果，包括相关非平稳信号处理、智能信息处理等理论与方法，以及超宽带复杂电磁环境下的信号快速检测、频谱感知与分类识别等工程实现和应用。

 

报告人：李林，副教授，博士生导师，中国电子学会高级会员，IEEE会员，美国《数学评论》特约评论员。主要研究领域为信号与信息处理、模式识别等理论与方法，以及航天测控链路探测、空间环境综合监测、外场电磁频谱感知等技术开发。曾担任香港浸会大学高级研究员，美国密苏里大学访问学者。在国内外核心期刊及会议发表论文80余篇，授权发明专利20余项，获得省部级以上科学技术奖3项，陕西省教育厅科学技术奖2项。获2022年度陕西省“科学家+工程师”团队建设——首席科学家称号，入选2019年江苏省双创博士、2018年昆山市双创领军人才。
