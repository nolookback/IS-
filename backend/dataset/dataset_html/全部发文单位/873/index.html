建筑与城市规划学院　2022/5/30 16:21:00

本课题目：城市更新中城市问题的解析与应对

主讲专家：王嘉

学术主持：高文秀 教授

时间：5月31日 14:15-16:00

地点：深圳大学建筑与城市规划学院C205

讲座简介：讲座阐述增量规划情形下深圳市城市更新规划体系，解析城市更新中各类规划需要应对的城市问题，并通过城市更新实践案例讲解解决城市问题的实施策略和过程。

专家简介：王嘉，深圳市城市规划设计研究院有限公司院副总规划师、城市更新规划研究中心主任。担任中国建筑学会城市设计分会理事、中国城市科学研究会城市更新专业委员会副主任委员、广东省国土空间规划专家库（第一批）入库专家、深圳市城市规划学会存量规划实施学会副主任委员等多个社会职务。主持完成了全国多地的总体规划、控制性详细规划、修建性详细规划、规划研究等各类规划项目和不同类型城市更新项目等百余项，其中6项获国家级奖项，9项获得省级奖项，8项获得市级奖项。出版了《深圳城市更新探索与实践》、《城乡规划编制技术手册》等书籍，参与编制多个城市更新技术方面的国家标准、团体标准、行业标准。在《城市规划学刊》、《规划师》及《城市建设》等核心期刊上发表多篇论文，对深圳市城市更新背景下的政策制度、实施机制、规划编制、项目运作等方面具有独特的见解。
