报告人简介：

张忠平，安徽大学教授，博士生导师，安徽省人民政府参事，国家杰出青年基金获得者、中科院“百人计划”学者。2000年在中国科技大学获得博士学位，2001-2005在新加坡国家材料工程研究院从事博士后研究，2005-2016任中国科学院智能机械研究所研究员、副所长。主持和承担过科技部重大研究计划、国际合作和国家重点基金等项目，目前主要从事分子生物影像学研究，在JACS, Angewante, Adv. Mater.和Anal. Chem.等期刊发表论文100余篇。获安徽省科学技术一等奖（两项）、安徽省青年科技奖和安徽省直机关优秀青年等荣誉。

讲座摘要：

荧光探针成像是研究活细胞内分子与细胞器动力学的有力工具，对揭示细胞生理病理学新机制具有重要意义，但面临的挑战是难以提供细胞内多个目标的实时、动态及关联信息。张忠平课题组致力于探针工程化设计及细胞成像方法研究，提出了荧光探针的功能模块化设计理念，实现了单一探针对细胞或细胞器内多重目标的同步定位、光谱响应区分、及超分辨成像；建立了基于单一探针对多重关联目标的同步成像分析方法，发现了多种生理学新机制；发展了一系列新型荧光寿命探针，揭示了细胞内脂质代谢机制及其对相关细胞器动力学的调控。上述研究加深了我们对细胞内分子及细胞器动力学的理解。
