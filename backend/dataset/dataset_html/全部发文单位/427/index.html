嘉宾：Peter Harrison 爱思唯尔全球高级副总裁

主讲人：Kai Wang，Jun Kee Cheng

时间：2024年04月18日（星期四）上午9:30-11:30

地点：深圳大学粤海校区本原楼118本原讲堂


讲座主题内容：

·　如何在高水平学术期刊上发表文章

·　同行评审流程

·　 学术道德问题

·　从作者、审稿人到编辑，从出版商角度如何任命编辑

 

嘉宾简介：

Peter Harrison先生在爱丁堡大学获得化学荣誉学位，并在澳大利亚国立大学获得博士学位后，于25 年前加入爱思唯尔。在此期间，他曾在多个学科领域担任过一系列科技与医学期刊出版和管理职务，并亲历了学术期刊出版从印刷到在线的变迁，以及随后的商业模式，文章/数据格式的演变。作为爱思唯尔全球期刊高级副总裁，他目前领导一支由来自中国、法国、德国、印度、美国、英国和荷兰等国的出版人与编辑组成的全球出版团队，全面负责物质科学领域期刊的管理和运营。物质科学是一个令人兴奋的多元化领域，市场和社会需求不断变化，内容和商业模式以及作者/读者的需求也在不断演化。彼得团队负责的期刊包括订阅/混合模式及开放获取模式期刊,也包括众多与全球顶尖学会/机构合作的期刊。

彼得目前居住在英国牛津，除2015-2018年曾在美国费城工作过三年外，他的大部分职业生涯都在那里度过。
