“中国思想与电影理论的演进”学术讲座在传播学院举办
传播学院　2024/10/17 11:26:00
  （钟沛基 2022280095）

9月20日，传播学院举办“中国思想与电影理论的演进”主题学术讲座，特邀谭政教授担任主讲人，王伟副教授担任主持人。

讲座伊始，王伟老师向现场师生介绍了谭政教授。谭教授是《电影艺术》主编、中国文联电影艺术中心研究员，曾担任中国电影家协会青联主席、华语青年影像论坛秘书长、上海国际电影节电影频道传媒关注单元评委、丝绸之路国际电影节“金丝路”传媒荣誉单元评委、北京大学生电影节主竞赛专家评委。

随后，谭政教授从世界电影理论发展概略开始，阐述了电影理论作为一种理性认识对电影艺术的重要性。他回顾了电影从被论证为艺术到商业性、艺术性并重的历程，并介绍了爱森斯坦、吉加·维尔托夫、巴赞等电影理论大师及其贡献。

在谈到中国电影理论的发展时，谭政教授表示，中国电影理论经历了从外在属性界定到内在本性探求、再到文化属性确认的演变过程。他介绍了20年代中国电影理论的草创时期、三四十年代电影批评的活跃与理论建树的不足、以及五六十年代电影理论的深化。谭教授提到，在七八十年代电影理论的繁荣时期，包括经典理论和现代理论在内的大量西方电影理论被引入中国，中国电影学界开始在理论层面上生根发芽，电影语言现代化、民族化的论证等成为热点。

谭政教授指出，21世纪以来，随着中国加入WTO，民族化的电影受到重视，电影史学也逐步建构。在如今的数字化媒介时代下，中国电影开始吸收其他学科，进入跨学科阶段，电影生产全流程被AI技术重构，电影理论进入后电影时期。

在马克思主义视阈下，谭政教授分析了百年间马克思主义对中国电影理论建构的影响。他指出，中国化马克思主义给中国电影理论输入了一种深广而厚重的社会革命论范式，坚持将开放的外来理论中国化并且传承和光大本民族艺术传统，回答了电影艺术为谁服务和如何服务等基本问题，重视电影理论与批评建设。此外，谭政教授还介绍了中国电影美学的建构及电影理论话语的生成。他通过“镜游”理论和电影理论的赋比兴等例子，展示了中国电影理论与中文理论的密切关系，以及中国电影理论从理论到美学到流派的全架构体系。

讲座结束后，现场师生表示受益匪浅，对谭政教授的精彩演讲表示感谢。本次讲座在掌声中顺利结束。
