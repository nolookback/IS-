讲座名称：基于多模态磁共振影像的脑可塑性研究

报告人：李琼玲，北京师范大学认知神经科学与学习国家重点实验室博士后

个人简介：

李琼玲，北京师范大学认知神经科学与学习国家重点实验室博士后，2021年博士毕业于北京航空航天大学，攻读博士学位期间曾在蒙特利尔神经病学研究所（MNI）交流学习一年。研究专长为基于神经影像的脑结构和功能网络计算方法及其应用基础研究，研究成果以第一作者身份发表在Brain、Cell reports、Human Brain Mapping 等神经科学和神经影像领域内权威期刊，在脑连接组学领域具有丰富的研究经验。主持国自然青年基金项目，并作为核心人员参与国自然重点项目和重大仪器等项目。


讲座摘要：

长期以来，科学家认为儿童的大脑具有高度可塑性，而成年人的脑细胞不断凋亡，导致记忆和学习能力下降，脑的结构和功能缺乏可塑性。然而，大量研究表明，不仅儿童脑具有高度可塑性，成年人甚至老年人的大脑也能发生可塑性变化。这种变化既可表现为脑结构的物理变化，也体现在大脑皮层功能的重映射和活动的改变。研究不同年龄段人脑的可塑性，不仅对儿童的健康发育有重要作用，还能为脑损伤患者的康复治疗以及神经精神类疾病的干预提供理论支持，具备重要的临床价值和社会意义。本次报告内容涉及：生命早期前1000天这一高度可塑性时期的大脑功能网络发育规律研究，成年人通过音乐训练引起的脑结构和功能网络变化，以及癫痫患者认知功能重组的网络机制研究。试图通过这些研究，阐明正常发育、学习训练及疾病状态下的脑可塑机制。
