学术讲座（十五）：高分子核酸、药物载体
化学与环境工程学院　2021/3/28 14:42:00
  （钟沛基 2022280095）

主讲人: 田华雨研究员

时间: 3月29日上午9:30

地点: 丽湖校区守信楼（原B1栋）420会议室

报告人简介：

田华雨，中国科学院长春应用化学研究所研究员、博士生导师、国家杰出青年基金获得者、中组部“万人计划”青年拔尖人才。1994-2000年在哈尔滨工业大学高分子材料专业攻读学士和硕士学位，2002-2005年在中国科学院长春应用化学研究所高分子物理与化学专业攻读博士学位，博士毕业后留所工作。先后在韩国KAIST、日本九州大学和美国犹他大学从事访问研究。主要从事生物医用高分子、基因/药物智能传输体系、纳米基因诊疗体系、纳米疫苗、生物制造关键材料的研发等领域的研究。在Prog Polym Sci、Sci Adv、JACS、Angew、Adv Mater等杂志发表论文150余篇，获权中国发明专利25项，所开发的细胞用基因转染试剂实现商品化。获中科院青促会优秀会员、 中国化学会高分子科学创新论文奖。担任《中国化学快报》、《中国科学：化学》青年编委、生物材料学会生物医用高分子材料分会副秘书长、中国医药生物技术协会纳米生物技术分会委员。

欢迎有兴趣的师生参加！
