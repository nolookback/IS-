学术讲座：深圳市统计局王文韵高级统计师—新发展格局下投资形势与投资统计的多维度解析——兼议经济形势判断的方法论
经济学院　2021/6/10 17:35:00
  （钟沛基 2022280095）


讲座题目：新发展格局下投资形势与投资统计的多维度解析——兼议经济形势判断的方法论

讲座时间：2021年6月11日10:00

讲座地点：汇文楼1500会议室

主讲嘉宾：王文韵  深圳市统计局

主讲嘉宾简介：

　　王文韵，高级统计师，曾任山西省统计局副总统计师,广东省四会市委副书记(分管:创新驱动、招商引智、高科技企业孵化器和高新技术产业培育与发展、先进装备制造业等工作),深圳市统计局工业贸易处处长、政策法规处处长、深圳市统计局综合协调处处长。现任:深圳市统计局投资和建筑业处处长。兼任:深圳市政府政策咨询（重大投资项目/行业和产业经济）委专家、深圳市应对气候变化与低碳经济研究中心特约研究员。

讲座主要内容：

　　本讲座围绕新发展阶段、新发展理念、新发展格局以及“十四五规划”、2035年远景目标纲要的要求，从怎么说（知识政策维度）、怎么样（形势状态维度）、怎么看（认识评价维度）、怎么办（落实推进维度）4方面对投资数据进行了深入分析研究，以期在优化投资结构、提高投资效率、增强投资后劲、扩大有效投资、保持投资合理增长，不断开拓投资新空间的实践力中发挥积极作用。

　　同时,以投资形势判断为例，从认知过程、思维视角、逻辑基点，分析框架、实践指引、表达赋能6个方面就经济形势判断的方法论进行了初步探讨。

　　欢迎广大师生参加！
