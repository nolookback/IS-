讲座名称：睡眠记忆的神经编解码和神经调控

讲座时间：2025年1月21日下午15：00

讲座地点：深圳大学沧海校区致理楼C座L3-133

主讲人：柳昀哲

 

主讲人简介：

柳昀哲，北京师范大学教授，北京脑科学与类脑研究所双聘研究员。研究专注于人类智能的计算神经机制，在认知地图领域取得国际领先成果，首创的神经编解码算法能在休息和睡眠期间解码记忆重现，为人类高效学习和决策揭示了神经计算机制。在心理学、精神病学等领域具有重要的贡献，对儿童脑智发育、心理与精神健康具有启示。
