【学术讲座】生物基药物运载体系在多种疾病治疗中的应用
生命与海洋科学学院　2024/3/28 16:01:00
  （钟沛基 2022280095）

报告题目：生物基药物运载体系在多种疾病治疗中的应用

主讲嘉宾：张海元  教授

报告主持：肖时锋  副教授

报告时间：2024年4月2日（星期二）9:30-11:00

报告地点：深圳大学丽湖校区守慧楼A6-503

张海元，2005年博士毕业于中科院长春应用化学研究所，2005-2013年先后在美国韦恩州立大学、伦斯勒理工学院和加州大学洛杉矶分校从事科研工作，2014-2023年就职于中科院长春应用化学研究所，任课题组长、研究员、博士生导师，2024年入职广州医科大学生物医学工程学院/附属第一医院。曾入选中科院“优秀学者”A类、吉林省B类国家级优秀学者和广州医科大学“南山学者（优秀学者）”。研究领域涉及纳米生物学、材料生物学和化学生物学等多学科交叉方向，着重于智能药物体系的构建与多种疾病的诊断与治疗。研究成果获中科院科技成果转化母基金二等奖和吉林省自然科学一等奖。先后主持科技部重点研发计划、基金委面上项目及多项省级重点项目；兼职中国毒理学会纳米毒理学专业委员会委员、中国生物物理学会材料生物学与智能诊疗技术分会委员会委员和国际纳米光电子学与生物光电子学联合中心学术委员会委员。获中国科学技术大学科教融合优秀导师奖；指导研究生多次获得光华奖学金。
