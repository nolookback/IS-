艺术学部“领航讲座”第九讲： 著名花腔女高音歌唱家常思思《声乐发声方法的基本构成》学术讲座
艺术学部　2021/11/18 10:57:00
  （钟沛基 2022280095）

讲座主题：声乐发声方法的基本构成

讲座时间：2021年11月19日（周五） 14:15-16:00

讲座地点：深圳大学师院音乐厅

主讲人简介：常思思，著名花腔女高音歌唱家，曾获第八届中国声乐金钟奖民族唱法金奖，第十三届全国青年歌手电视大奖赛民族唱法铜奖第一名。代表作品《春天的芭蕾》、《玛依拉变奏曲》等深受人们的喜爱，广为流传，是新一代民族歌唱家的代表人物之一。

讲座内容简介：艺术学部音乐舞蹈学院邀请了“花腔精灵”女高音歌唱家常思思为大家带来一场高品质的声乐专业讲座。此次讲座精选了5首经典声乐作品，常思思老师以现场授课的形式系统地为大家讲解歌唱发声方法的基本构成。

 

实体课堂报名方法及要求：

1.在内部网办事大厅“领航讲座”APP报名，报名时间：11月18日14:00-11月19日11:00，先报先得，报满即止。操作指南详见附件“领航讲座系统操作指南”，限2021级学生、2020级未完成领航讲座实体课堂要求的学生报名。

2.报名成功系统将分配座位号，请对号入座。未对号入座的同学视为未到课堂听讲。报名成功后可取消报名，取消报名规则如下：

（1）已报名成功的同学，在报名截止6个小时内可以取消报名，可再重报。

（2）已报名成功，但是在报名截止6小时以上取消，释放名额的，不做处理。

（3) 已报名成功，但是在报名截止6小时内取消的，取消下一讲的报名资格，一讲过后可继续报名。

（4）已报名成功，未按时听讲，取消接下来两讲报名资格，两讲过后可继续报名。

（5）已成功报名两次并有两次听讲记录的同学，提示“已报名两次，请留意在报名截止6小时内是否还有名额”。在报名截止6个小时内仍未报满的情况下可报名成功。

3.请同学们认真听讲，遵守课堂纪律，感谢配合！

欢迎感兴趣的2021级新生踊跃报名参加，期待“领航讲座”领航你的大学生活。
