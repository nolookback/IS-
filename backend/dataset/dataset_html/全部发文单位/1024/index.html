学术讲座（五十二）深圳“双区”背景下小学科学教育的着力点
化学与环境工程学院　2021/10/9 15:23:00
  （钟沛基 2022280095）

主讲人：姜玉晗

时间：2021年10月14日(周四)18:00-20:00

地点：深圳大学丽湖校区守信楼一楼报告厅

报告人简介：姜玉晗老师为深圳市首期“适应新型教与学模式先锋班”优秀学员，南山区优秀教师，南山区首批未来教师，近3年获广东省青年教师教学能力大赛二等奖、深圳市青年教师教学能力大赛一等奖、南山区青年教师教学能力大赛一等奖、深圳市学科融合与创客教育课例征集二等奖、南山区小学科学教学实践研究成果评选一等奖、深圳市青年教师基本功比赛一等奖，并获实用新型专利。

讲座摘要：深圳“双区”背景下小学科学教学理念及未来发展、教学设计要求、常用小学教育信息技术、部分公开课案例分享。
