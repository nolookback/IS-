学术讲座丨基于空间行为模拟的主体建模方法探索与应用
建筑与城市规划学院　2023/10/26 15:19:00
  （钟沛基 2022280095）

讲座题目：基于空间行为模拟的主体建模方法探索与应用

时间：11月2日19:00-20:30

讲座地点：C205报告厅

主持人：陈义勇


嘉宾介绍：戴冬晖，哈尔滨工业大学（深圳）建筑学院副院长、副教授、博士生导师，深圳市鹏城孔雀计划人才。研究领域为城市设计理论与方法，城市设计管理与实施、城市时空行为与空间规划设计。主持国家自然科学基金面上项目、十三五重点研发计划项目子课题、广东省自然科学基金面上项目、深圳市科创委稳定支撑计划面上项目各1项，主持、参与实践与研究课题50余项。

内容概要：主体建模方法作为一种新兴的数字化工具，为规划设计研究提供了“自下而上”开展分析的思路和方法。在宏观、中观、微观三个尺度下，主体建模方法通过对城市土地利用、设施布局、人群行为等各要素的建模分析，能够更好地理解城市的现状和未来趋势，从而制定更为科学的政策。在中微观尺度上，通过对活动者的行为模拟，主体建模方法有助于解读空间和人之间复杂的相互关系。结合社区生活圈、社区街道、住区公共空间三个不同尺度，本研究探索了基于空间行为模拟的主体模型构建方法，在此基础上结合案例开展应用，并总结了主体建模方法的优势、挑战及对未来研究的展望。
