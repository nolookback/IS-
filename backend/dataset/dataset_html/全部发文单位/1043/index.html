学术讲座（六）：华南地区植物群落调查方法

 

时  间：2021年9月12日（周日）15:00-17:30

地  点：广东省沙头角林场（深圳）会议室

主讲人：陈红锋  研究员

主持人：张永夏  系主任

 

主讲人介绍：

　　陈红锋，研究员，博士生导师。中国科学院卢嘉锡青年人才奖获得者，现任中国科学院华南植物园植物科学研究中心副主任，植物物种多样性保育研究组首席研究员。

　　长期从事植物分类、资源调查工作，熟悉华南地区植物种类和分布。近年来致力于珍稀濒危植物保护、园林植物资源利用与配置研究。主持完成广东省珍稀濒危植物第二次全国调查规划及总调查报告编撰、广东省特色林下经济植物资源发掘与利用、广州花城绿城水城建设植物选择与配置指引、广州市生态景观林带树种选择与配置研究等项目。主持国家自然科学基金项目、国家“十一五”科技支撑计划项目子课题和国家科技基础平台项目子专题，中国科学院先导项目子课题20多项。成果“华南珍稀濒危植物的野外回归研究与应用”“乡土植物在生态园林中应用的关键技术与产业化”“中国南海岛屿植物多样性研究及产业化”“广东省特色植物资源利用产业化关键技术研究与应用”分别获2012、2013、2016、2018年广东省科技进步一等奖，其他奖励5项；授权国家发明专利10多件；发表论文100多篇；主编《中国景观植物》、《东莞园林植物》、《南昆山植物》等及参编专著20多部；指导研究生30名。

　　主要学术兼职包括：中国科学院青年创新促进会会员、广东省植物学会秘书长、广东省园林学会理事、广东省环境科学学会自然保护与生态修复专业委员会委员、广东省林学会风景园林与城市林业专业委员会委员、广州市林学会理事、广州花城绿城水城建设战略专家等。

 

　  欢迎广大师生前来聆听！
