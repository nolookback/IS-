【学术讲座】隐私计算中的密码关键技术研究
电子与信息工程学院　2024/5/24 15:09:00
  （钟沛基 2022280095）

报告人：张鹏 副教授

邀请人：李岩山 副教授

时  间：5月27日周一上午10：00-11：00

地 点：汇星楼1504


报告人简介：主要从事信息安全领域的教学和科研工作，近年来重点关注公钥密码及应用、格密码与全同态加密、隐私保护机器学习等领域，主持本领域的国家自然科学基金项目1项，国防基础科研计划项目1项，深圳市科技计划项目3项，腾讯犀牛鸟项目1项，企业横向委托项目多项。近年来申请国家发明专利30余项，在本领域国内外权威期刊或会议发表高水平研究论文50余篇。

报告摘要：隐私计算是指在提供隐私保护的前提下，实现数据价值挖掘的技术体系。隐私计算能够在保证满足数据隐私安全的基础上，实现数据“价值”和“知识”的流动与共享，真正做到“数据可用不可见“。本报告从身份隐私与数据隐私两个方面，对所涉及的密码关键技术进行研究，主要包括多重签名技术及其在区块链、智能电网中的应用研究，以及全同态加密技术及其在隐私保护机器学习中的应用研究。
