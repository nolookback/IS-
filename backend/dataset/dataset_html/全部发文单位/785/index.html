学术讲座：深圳市新昊明科技有限公司林红总经理—浅谈跨境电商供应链管理的模式
经济学院　2022/11/21 11:24:00
  （钟沛基 2022280095）

讲座题目：浅谈跨境电商供应链管理的模式

讲座时间：2022年11月22日（周二）9:15-10：15

讲座地点：明德楼-303

主讲嘉宾：林红　　深圳市新昊明科技有限公司

主讲嘉宾简介：

　　林红，深圳大学物流管理学士学位，香港中文大学（深圳）供应链与物流管理高级管理人员理学硕士学位。从事跨境电商工作12年，具有丰富的跨境物流、供应链管理、数据挖掘与分析经验，实战功底深厚，属于跨境全能型人才。

讲座主要内容：

　　跨境电商，2022还能入局吗？在疫情席卷、经济低迷、通货膨胀的宏观经济市场背景下，跨境电商的生存环境愈发恶劣，迎来行业洗牌周期，跨境电商应该如何转型？报告会对跨境电商的含义、分类和现状进行简要的概述，阐述跨境电商供应链管理的含义与流程，以跨境电商头部平台“亚马逊”的实际运作为例，分析跨境电商供应链管理的模式、痛点以及如何优化。

　　欢迎广大师生参加！
