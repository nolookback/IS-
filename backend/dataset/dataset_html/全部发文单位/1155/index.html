广州大学陶冶教授主讲题为“1978年以来的中国电影金融史侧写”的学术讲座成功举办
传播学院　2021/3/12 15:31:00
  （钟沛基 2022280095）

3月11日下午，广州大学新闻与传播学院陶冶教授受邀来到深圳大学传播学院主讲题为“1978年以来的中国电影金融史侧写”的学术讲座。

陶冶，广州大学新闻与传播学院教授，博士生导师，广州大学粤港澳大湾区影视艺术研究院院长，电影学学科带头人。曾任浙江传媒学院文化创意学院副院长，浙江省中青年学科带头人，浙江省151人才，浙江省“之江青年学者”，全国广电“百优”理论人才。近年内，出版专著6部，发表论文近50篇，现主持国家级课题两项，主持完成省部级课题四项、市厅级课题两项。同时坚持在一线创作领域开展社会服务，曾担任电影《龙顶》执行制片人，监制电影《沿海出发》，兼任东方卫视《杨澜访谈录》策划，凤凰卫视《一虎一席谈》策划。

会议伊始，陶冶教授简单介绍了自己的学习背景以及学术研究方向。从影视产业发展以及改革开放四十年的历史进程的视角，陶冶教授提到中国电影的发展在相当长一段时间内是与其不同步的，特别是改革开放初期，经济领域的政策调整所带来的一系列产能释放和人民生活水平的提高，带来的却是电视机的进一步普及。以至于中国电影在经历了20世纪80年代一个短暂的创新高峰之后，紧接着便是中国电影票房和产量的双双大幅下滑。尽管美国在20世纪50年代也经历过这种“新媒体”阵痛，并直接导致了“新好莱坞运动”，从而使得好莱坞商业电影更趋向于能充分展现大银幕的大制作以使观众能将电影和电视区分开来。

接着，陶冶教授讲到，就电影本体而言，金融属性确是其与生俱来的。因此，改革开放四十年的中国电影史某种程度上亦是一部电影金融的历史。随后，陶冶教授就中国电影的融资模式演变进行了详细的讲解，其从国营体制的“无忧”融资模式转变为“市场拼盘”融资的发展模式再到阿里“全产业链”融资模式，从恶性循环的发展状态转变为良性循环的发展状态，使得中国电影的发展逐渐步入正轨且日渐繁荣。

最后，陶冶教授立足于当下，详细阐述在“互联网+金融”面前影视作品已然成为一种金融产品。至此，本次讲座在轻松愉悦的氛围中圆满结束。
