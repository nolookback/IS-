报告人简介：

陈振滨（Chin Ping Tan）教授是马来西亚博特拉大学（Universiti Putra Malaysia，UPM）食品科学与技术学院的教授，专注于油脂科学与技术领域，尤其在棕榈油、食品纳米技术、食品乳液和生物活性化合物提取等方面具有深厚的研究基础。

陈振滨教授于1998年获得马来西亚博特拉大学食品科学与技术学士一等荣誉学位，2001年获得同校食品加工博士学位。2002年至2004年，他在日本国家食品研究所担任JSPS博士后研究员。2008年晋升为副教授，2012年升任全职教授。2013年至2018年，他担任UPM食品技术系主任。

近五年主要成果主要包括：

1. 食品纳米技术应用：开发了多种纳米分散体系，用于提高生物活性化合物的稳定性和生物利用度。 例如，他研究了β-胡萝卜素纳米分散体的制备、特性和稳定性。 

2. 食品乳液研究：深入研究了食品乳液的物理化学性质，特别是橙汁饮料乳液中阿拉伯胶、黄原胶和橙油含量对乳液稳定性的影响。 

3. 生物活性化合物提取：优化了超声辅助提取技术，从木瓜籽中提取油脂，研究了油脂回收率、自由基清除抗氧化活性和氧化稳定性。 

4. 棕榈油研究：探讨了棕榈油在不同加工条件下的热稳定性和氧化稳定性，为棕榈油的工业应用提供了科学依据。

这些研究成果不仅丰富了食品科学领域的理论基础，也为食品工业的技术创新提供了重要支持。
