报告题目：认知智能的计算模型

报告内容简介：

感知和理解环境是动物最核心的认知功能。动物的认知能力集中体现在获取、表征、整合、存储、理解和运用信息的能力。脑科学在环路层次记录和解析了感知、记忆和决策等功能的神经编码，积累了丰富的实验数据。在脑活动和行为数据的约束下建立认知任务的计算模型，对揭示神经系统的复杂性、理解脑的工作机制具有重要意义。近年来，建立认知智能的计算模型、通过实现认知功能达到对脑的工作机制的理解越来越受重视。报告将介绍认知智能计算模型的一些进展，并讨论对类脑人工智能的启示。

报告嘉宾简介：

斯白露，北京师范大学系统科学学院教授。2007年获得德国不来梅大学理论神经物理专业博士学位。2008至2013年期间，先后在意大利国际高等研究院认知神经科学部和以色列魏茨曼研究院神经生物学系从事博士后工作。2013年12月至2018年9月任中国科学院沈阳自动化研究所机器人学国家重点实验室研究员。任北京脑网络组与类脑智能学会理事，中国神经科学学会计算神经科学和神经工程分会、中国人工智能学会智能交互专委会等专委会委员，国际理论物理中心、美国霍华德休斯医学研究所珍利亚农场研究园等研究机构的访问科学家。主要研究领域包括计算神经科学、类脑机器人。代表性成果包括记忆神经环路的计算理论、类脑导航系统等。
