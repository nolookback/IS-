“智能技术重塑虚拟现实影像与叙事”学术讲座在传播学院举办
传播学院　2024/10/17 11:35:00
  （钟沛基 2022280095）

2024年9月19日下午，北京师范大学艺术与传媒学院数字媒体系主任、数字创意媒体研究中心执行主任周雯教授受邀为传播学院师生作了一场以“智能技术重塑虚拟现实影像与叙事”为题的学术讲座。本场讲座由社会科学部战迪担任主持人。

讲座伊始，战迪老师向在座的师生介绍了本次讲座的主讲人——周雯教授。周雯教授是教育部高等学校动画、数字媒体专业教指委委员，中国高等教育学会影视传媒专业委员会副理事长，中国高校影视学会动画与数字媒体艺术专业委员会副主任委员，主要研究领域为数字影像、虚拟现实影像。

周雯教授围绕讲座主题，从虚拟现实艺术、AGI、AI与艺术三个层面展开其内容。周教授就“虚拟现实艺术”议题指出，随着技术的不断进步，影像的未来趋势是可交互式的影像。她以Open BCI galea为例，展示了脑电、肌电、皮电、光学心率等技术在虚拟现实中的应用，并借助“消失的法老”“风起洛阳”等案例展示了虚拟现实艺术在表现上的无限可能性。接着，周教授将话题转向“AI与AI时代”，并指出随着大语言模型（LLM）等技术的不断发展，AGI已经成为显性的存在，未来，我们将见证更加智能、更加自主的人工智能系统的出现。周教授介绍了大模型在资本投资领域的受欢迎程度，以及神经网络结合强化学习训练智能体的最新进展，并列举Midjourney、商汤科技、story-to- motion、英伟达发布矢量素描转动画等案例展示AI在内容生成方面的实际应用。

讲座最后，周雯教授指出，虚拟现实与人工智能的结合将为人类带来前所未有的体验和创新。同时，她也呼吁在场的师生要积极拥抱新技术，勇于探索未知领域。

讲座结束后，田少煦教授进行了总结发言。他认可了周雯教授以媒介为视角切入内容生成的深刻洞见，并表示这将为虚拟现实与人工智能领域的研究提供新思路和新方法。在座的师生纷纷表示，本次讲座让他们对虚拟现实与人工智能的结合有了更加深入的认识和理解。在师生的掌声中，本次讲座顺利结束。
