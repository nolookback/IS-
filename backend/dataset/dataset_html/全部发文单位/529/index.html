学术讲座：纳米多孔金属功能性应用中的关键构效关系
机电与控制工程学院　2024/1/7 11:53:00
  （钟沛基 2022280095）

讲座题目：纳米多孔金属功能性应用中的关键构效关系

演讲嘉宾：陈擎

讲座时间：1月8 日（星期一）11：00

讲座地点：深圳大学南校区机电大楼S601会议室

邀请人： 阮文清老师

讲座内容简介： 纳米多孔金属作为脱合金等制备方法的产物，因其高比表面积和自支撑特性，在力电耦合、催化、传感等领域得到广泛应用。过往的研究通常着眼于其结构尺度与表面积、反应活性等关系，忽视了其相对密度、结构尺度分布在功能应用中起到的关键作用。此报告将从随机均质介质传质和渗流理论出发，结合还原诱导分解这一新型制备方法，构建纳米多孔金属的关键构效关系，并以锂金属电池、电解槽等为例阐述其应用。

演讲嘉宾简介： 陈擎，香港科技大学机械与航空航天系和化学系双聘副教授，香港科大能源研究院副主任。课题组主要研究金属微结构在电化学界面的演化机理，并将其运用于电化学器件，突破包括电池、电解槽等的效率和稳定性瓶颈。工作曾发表于Nature Materials, Nature Communications等期刊，为纳米多孔金属演化机理、有机液流电池、碱性锌电池等领域的研究作出过突出贡献，并于2020年获得国家自然科学基金委港澳优青项目资助。
