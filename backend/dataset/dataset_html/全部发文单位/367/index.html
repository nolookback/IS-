“抗战时期外国档案文献对中共马克思主义中国化的认知”学术讲座在马克思主义学院顺利举办
马克思主义学院　2024/5/26 23:49:00
  （钟沛基 2022280095）

2024年5月20日，武汉大学人文社会科学资深教授，曾任武汉大学副校长、武汉大学中国边界与海洋研究院院长、国际问题研究院院长、中国第二次世界大战史学会会长胡德坤教授，为深大师生带来了一场题为“抗战时期外国档案文献对中共马克思主义中国化的认知”的精彩讲座。本次讲座由深圳大学改革开放史中心主任徐建华教授主持，众多师生到场聆听。

 

在讲座中，胡德坤教授首先指出中国共产党的百年史，是领导中国人民争取民族独立和解放、建设繁荣富强的新中国，使中国站起来、富起来、强起来的历史，在中国每个历史时期都创造了奇迹，是中国历史走向辉煌的百年。而抗日战争是近代以来中国抗击外敌入侵的第一次完全胜利，其根本原因是有了中国共产党。当前，我们从外国的档案文献来看抗战时期中共马克思主义中国化的理论与实践，重温中共在抗战中的丰功伟绩，能让我们更加增强历史自信。

接下来，胡德坤教授从两个方面选取史料档案来阐述自己的观点。第一，抗战时期美方对中共马克思主义中国化的认知。在这一部分，胡德坤教授从二战时美国的档案文献出发，通过对美国的观察专家、记者、军官、官员等记录的史料、访谈档案的分析，探求抗战时期美国对中国共产党的性质、马克思主义中国化的理论成就和实践创新的认知变迁；第二，抗战时期日方对中共马克思主义中国化的认知。接下来胡教授选取了二战时的敌国日本收集的资料、档案、文献，研究抗战时日方对中国共产党的地位作用、马克思主义中国化的理论创新和实践创新的认识。最后，胡教授总结道，抗战时期，两国都认为中国共产党是独立的马克思主义政党和领导抗日持久战的核心力量；中国共产党与时俱进、结合实际创造的理论，也就是马克思主义中国化的成果成为了抗战的指导思想，中国共产党已经成熟到可以将马克思主义中国化；中国共产党的理论和政策顺应民心民利，动员组织了广大群众投身抗战，未来是属于中国共产党的。可见，马克思主义中国化在当时就已经被国际社会知晓。

讲座最后，徐建华教授对胡德坤教授的精彩讲述表示感谢并做总结发言，指出这是一场内涵丰富、具有启发性的学术讲座。此次讲座内容政治站位高、逻辑严谨并且见解独特，提升了师生们的专业素养，同时也营造出良好的学术氛围。在热烈的讨论氛围中，讲座圆满落下帷幕。
