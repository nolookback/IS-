主讲人简介：

张伯伟，南京大学文学院教授，二级教授，兼任南京大学人文社会科学高级研究院特聘教授、南京大学域外汉籍研究所所长。曾任日本京都大学、韩国外国语大学、台湾大学、香港浸会大学客座教授。目前主要从事中国古代文学理论、文学批评史和域外汉籍的研究。著有《回向文学研究》《中国古代文学批评方法研究》《域外汉籍研究入门》《作为方法的汉文化圈》《东亚汉文学研究的方法与实践》等，主编有《朝鲜时代书目丛刊》《朝鲜时代女性诗文集全编》《日本世说新语注释集成》《全唐五代诗格汇考》等以及专业刊物《中国诗学》《域外汉籍研究集刊》。

 

内容简介：

　　东亚人文学没有理论和方法，这是百馀年前东亚知识界的共识，所以，一切理论和方法都是外来的。这一影响直到今天。欧美学者对此或惋惜、或批评、或嘲讽，这是首先需要加以回应的问题。其次，对理论的特质也需要作重新认识。理论不是僵硬的模式，不是抽象的教条，理论的本性是质疑，尤其是针对流行的、占主导倾向的思维框架的质疑。在以往的东亚研究中，就有四种常见的理论和方法值得反思，即中国中心论、影响研究、挑战—回应、内在发展。针对上述四种模式的缺陷，本文提出“作为方法的汉文化圈”，既将东亚汉文文献作为一个整体，又超越文献学的方法，使研究和观察、理论和经验相得益彰并互为保证，在与西方学术的对话中形成自身的理论和方法，从而建立起独立于而不孤立于、自立于而不自外于世界学术之林的中国学术。
