“影像生成新机制中的情感”学术讲座在传播学院举办
传播学院　2024/10/17 11:38:00
  （钟沛基 2022280095）

2024年9月20日，知名学者吴冠平教授受邀为传播学院师生做了一场以“影像生成新机制中的情感”为主题的学术讲座，王伟副教授担任本次讲座主持人。

首先，王伟老师向在座的师生介绍了吴冠平教授。吴冠平教授是北京电影学院中国电影文化研究院院长、《北京电影学院学报》主编。曾任北京电影学院电影学系主任，《电影艺术》杂志社社长、主编，华语青年电影周（CYGFF）创办人，中国电影家协会会员，中国电影评论学会常务理事，中国台港电影研究会副会长，中国高校影视学会理事，北京电影家协会理事等。曾担任第12、14、17、19届华语电影传媒大奖终评委；第18届、19届中国电影华表奖评委；第36届中国电影金鸡奖评委；第51届金马奖初评委、第55届金马奖终评委等。发表理论评论文章八十余万字，参与策划监制多部电影。

讲座中，吴教授指出，人工智能技术赋能之下，影像被赋予了更多的意义，包括影像技术、文化和情感三个维度。在技术层面，影像技术从传统的化学机械操作转变为算法驱动；在文化层面，影像从最初的仪式性图像转变为更广泛的影像实践，创造与观看的过程变得更加灵活多样；在情感层面，影像情感从公共领域转向私域，短视频等新媒体形式在不同圈层中引发着个性化的情感共鸣。

随后，吴教授介绍了胶片显影、电子成像、电脑合成和AI生成四种影像生成机制及对应的影像类型，这四种机制分别代表了影像技术的不同阶段和发展方向。在探讨影像生成机制的新维度时，吴教授强调了位置、叙事和真伪三个关键词。观看行为的位置变化、主体变化以及情感变化，都为影像的解读提供了新的视角；新的机制带来了新的叙事方式；真伪问题也成为体会影像生成机制的重要维度。

关于与影像有关的情感认知，吴教授引用了心理学家的观点，指出情感是由人体内部神经生化反应、动作行为表达和主观经验三者构成的。他强调，情感是文化建构的结果，具有解释成分和习得性。在探讨胶片影像机制和数字影像机制的情感时，吴教授分别做出解释：胶片影像具有实存之物的信仰机制、格式塔心理幻觉机制、黑匣子和不可触碰的观看机制；而数字影像则可以感知不可感知之物，具有可逆性和概念性作品的特点。

吴教授最后提出了生成影像机制带来的情感挑战：机器创造力与人类想象力的碰撞、定制的个性化与群体经验的冲突、真实伪造与伦理经验的矛盾以及交互参与和人际经验的平衡问题，值得我们深入思考和探讨。

本场讲座激发了师生对于影像生成新机制中情感的深刻思考。讲座在掌声中顺利结束。
