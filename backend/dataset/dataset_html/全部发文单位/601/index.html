【学术讲座】图挖掘技术与基因组学数据应用
电子与信息工程学院　2023/11/28 11:05:00
  （钟沛基 2022280095）


讲座嘉宾：马小科教授、博士生导师 西安电子科技大学

邀 请 人：欧阳乐 副教授

时  间：12月3日（周日）上午10:00-11:00

地  点：腾讯会议 会议号：330-714-027


 

报告摘要：图是描述自然与社会中复杂系统的有力工具，挖掘图模式对理解复杂系统结构和功能具有及其重要的意义。以下一代测序技术为代表的高通量深度测序技术产生了海量的生物医学数据，促进了生物医学网络的蓬勃发展。报告涉及生物网络图模式建模、挖掘算法研究，包括图聚类、时序网络链路预测、时序网络动态图聚类、多层网络挖掘等相关问题，同时研究图数据挖掘在生物医学方面的应用。

 

报告人：马小科，西安电子科技大学计算机科学与技术学院教授、博士生导师，先后入选国家级青年人才、陕西省杰青资助。分别于2005年、2008年和2012年获得理学学士、工学硕士与博士学位。2012-2015年在美国爱荷华大学从事博士后科研工作，2015年全职回国工作，2016年破格副教授，2019年破格教授。主要从事数据挖掘、机器学习、生物信息学等领域的研究工作。在国际期刊 PNAS、IEEE TKDE、IEEE CYB、Cell Stem Cell、Bioinformatics等发表SCI检索论文100多篇。主持国家自然科学基金重点项目(联合主持)、国家自然科学基金面上项目、青年基金、陕西省杰出青年基金、陕西省重点研发、省市自然科学基金等项目20余项，获中国电子学会科技技术奖二等奖、入选全球前2%顶尖科学家榜单等。
