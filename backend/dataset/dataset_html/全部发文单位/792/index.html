【学术讲座】基于集成学习的特征选择框架

报告摘要：

特征选择（feature selection）是机器学习和模式识别领域中一个重要的研究方向。与特征抽取（feature extraction）不同，原始特征分量具有特定且现实的几何、理化、生物学意义。尤其是针对高维小样本数据，有效的特征选择方法可以在降低特征维度的同时去除噪声的干扰，极大限度地保证分类、聚类、回归模型的预测准确性。传统的统计学习模型虽然具有可解释性，但预测能力较差。因此，采用分类、聚类、回归模型并依据其性能实现特征选择是这一领域的主流研究方向。然而，考虑到样本分布的复杂性，不同分类、聚类、回归模型的性能各不相同，如何从其中选取性能最优的学习模型以实现对特征分量的最优选择是一个关键问题。针对这一问题，我们提出了一种基于集成学习的特征选择框架，并将其用于图像处理、表达谱数据的差异表达分析、表达谱数据的预后生存分析、表达谱数据的类别自动发现等一系列问题。实验结果表明：所提基于集成学习的框架可以实现精确的特征选择；此外，在其对应的低维度特征空间中所建的学习模型均有较好的预测能力。
