【学术讲座】水下环境智能感知与理解
电子与信息工程学院　2024/6/21 16:21:00
  （钟沛基 2022280095）

嘉   宾：丛润民教授 山东大学

邀请人：邬文慧副教授

时   间：6月25日2:30PM

地  点：沧海校区致信楼N710


摘要：二十大报告对“发展海洋经济，保护海洋生态环境，加快建设海洋强国”的提出，进一步明确了视觉信息处理技术的新方向。水下图像作为水下信息的重要载体和呈现方式，在海洋探测、环境保护、海洋工程、水下考古等科研和工业领域中发挥着至关重要的作用。然而，由于客观成像环境和图像采集设备的限制，水下图像存在对比度低、细节模糊、颜色偏差等退化现象，严重制约水下视觉领域的发展。本报告将围绕水下环境感知与理解任务，具体介绍团队在水下图像增强、水下图像分割等方向的最新成果。
