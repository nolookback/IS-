报告主持：周海超 特聘研究员

报告时间：2025年4月16日（星期三）15：00

报告人简介：陈克林，国际湿地主任，原国家林业部保护司副司长，中国科学院东北地理与农业生态研究所研究员

我国最早从事湿地保护与研究工作者，担任国家湿地委员会委员、中国科学院湿地生态与环境重点实验室学术委员会委员、若尔盖高原泥炭湿地学术委员会委员、黑龙江省湿地旅游发展顾问、黑龙江省寒区湿地生态与环境研究重点实验室学术委员会委员、浙江省城市湿地与区域变化研究重点实验室学术委员会委员、北京湿地研究中心客座研究员、中国湿地保护协会常务理事（专家委员会委员）、《中国湿地百科全书》主编。

中国湿地保护领域的先行者和开拓者，创新并推广“湿地公园”、“湿地学校”的理念；具有国内、国际湿地保护的宏观视野，长期为国内地方政府及保护部门提供专业咨询与建议，并通过引进众多国际湿地保护项目，促进中国湿地保护与合理利用工作。
