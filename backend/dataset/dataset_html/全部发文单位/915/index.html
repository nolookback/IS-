嘉宾简介：

邹兵 先生，从事城市规划30年并长期深耕深圳，主持参与了上百项重大规划项目和重要课题研究，获得全国优秀城乡规划设计奖、全国优秀工程勘察设计奖、华夏建设科技奖等国家级奖项20多项（其中一等奖9项）；出版学术专著11部，发表论文近百篇，获得全国性优秀科技论文奖11次。‘


主要成就包括：

1、作为项目负责人连续主持了两轮深圳城市总体规划编制工作，也是主持第一部覆盖陆海全域的深圳国土空间总体规划（2020-2035年）的项目总负责人。

2、完成的2010版深圳总体规划在全国率先探索由增量空间扩张向存量空间优化转型的新型规划模式，被国家主管部门树立为“以转型规划引领城市转型”的范例，引领全国城市总体规划改革方向。

3、在全国率先开展针对存量土地二次开发的规划理论研究，主持最近10多年深圳城市更新和土地整备专项规划以及系列政策研究工作；撰写的《增量规划、存量规划与政策规划》论文被中国顶级学术期刊《城市规划》评选为“40年40篇”影响中国城乡规划进程的重要文献之一，对于城市规划学科理论创新产生深远影响。

4、参与划定全国第一条基本生态控制线以及后续动态维护工作，指导深圳生态保护红线划定和自然保护地规划研究工作，成为高密度超大城市实践生态文明的先行示范样板。在全国率先开展海洋空间战略规划研究，为深圳建设全球中心海洋城市、实现陆海统筹发展提供有力的空间支持。

5、被评为深圳市人民政府特殊津贴专家、全国优秀城市规划科技工作者和中国城市规划学会杰出学会工作者称号，2020年获得深圳市首届工程勘察设计大师称号，是全国城市规划著名专家、深圳城市规划学科带头人和行业领军人物，也是城市规划"深圳学派"的首先倡导者和主要推动者。
