【学术讲座】棉花的多重价值及其转基因体系构建与应用
生命与海洋科学学院　2024/11/1 11:11:00
  （钟沛基 2022280095）

报告题目：棉花的多重价值及其转基因体系构建与应用

主讲嘉宾：李付广  研究员

报告主持：刘宏涛  教授

报告时间：2024年11月4日（星期一）上午9:30

报告地点：深圳大学丽湖校区守慧楼A6-501

主讲嘉宾简介：

  二级研究员，博士生导师，国家杰出青年基金获得者，棉花生物育种与综合利用全国重点实验室主任，国家棉花

产业技术体系首席科学家。主要从事棉花遗传转化和分子改良研究工作，获国家科技进步二等奖2项、国家技术发明

二等奖1项。中国农业科学院棉花研究所原所长，现兼任郑州大学农学院院长。


报告简介：简要介绍棉花的纤油饲药等多重功能，棉花产业面临的问题，以及棉花从体细胞胚胎发生转基因技术体系

到种子顶尖干细胞转基因技术体系的迭代更新、转基因技术应用等。
