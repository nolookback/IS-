南京师范大学骆正林教授主讲题为“新闻发言人的素养、仪态与语言艺术”的学术讲座成功举办
传播学院　2021/3/15 14:37:00
  （钟沛基 2022280095）

3月11日下午，南京师范大学新闻与传播学院骆正林教授受邀来到传播学院为全院研究生主讲了一场题为“新闻发言人的素养、仪态与语言艺术”的学术讲座。本次讲座由传播学院副院长杨洸主持。

骆正林教授曾被评为江苏省社科英才（2019），南京师范大学“师德先进个人”“奕熙精英教师”（2019）。承担多项国家社科基金项目，目前为国家社科基金重大项目“我国青少年网络舆情的大数据预警体系与引导机制研究”首席专家。研究成果主要集中在舆论学、政治传播领域。长期耕耘“舆论传播与社会治理”研究，在舆论传播理论和实践领域进行了一系列深入的研究；近年关注“舆情大数据与网络空间治理”研究，研究过程更加注重“技术哲学”和“大数据手段”的结合；目前重点研究“青少年网络舆情”。

讲座伊始，骆正林教授向学生们介绍了新闻发言人的概念。他简要阐述了新闻发言人需要具备的基本素养：一是新闻发言人的政治素养，包括坚定的政治信念，规范的职业道德；二是业务素养，新闻发言人需要熟悉国家大政方针政策，熟悉新闻发布活动的整个流程，能够在团队协作的背景下开展工作。此外新闻发言人还需要具备媒介素养，心理素养、公关素养、语言素养等。

随后，骆正林教授列举了新闻发言人的不同案例。他通过国内新闻发言人的正反案例，全面解析了新闻发言人的仪表仪态。他指出新闻发言人的面部表情和眼神、肢体动作、穿着配饰、生活表演等对新闻发布会效果和舆论都有着重要的影响。

讲座最后，骆正林教授强调了新闻发言人的语言艺术的重要性。他认为，语言不当会引起舆论危机，并为学生们介绍了新闻发言人的语言技巧。至此，讲座在同学们的掌声中圆满结束。
