讲座题目：生活垃圾填埋场的功能性转变与碳中和

 

讲座时间：2022年5月5日9:00

 

讲座地点：腾讯会议（会议号585-9578-7496）

 

主讲嘉宾：陈瑞博士 副总经理 中兰环保

 

主讲嘉宾简介：

陈瑞，中兰环保副总经理，博士毕业于同济大学环境学院，中国科学院深圳先进技术研究院与中兰环保科技股份有限公司联合培养博士后，主要负责中兰环保技术研发、市场推广和品牌管理。工作期间，发表中英文学术论文多篇，获得发明及实用新型专利30余项，是中国城市环境卫生协会青年人才（第一批）及中国城市环境卫生协会生活垃圾卫生填埋及生态修复专业委员会副秘书长，广东省/海南省环卫行业专家，并任中共深圳市南山区第八届党代表。

中兰环保科技股份有限公司（简称“中兰环保”，300854.SZ）成立于2001年，总部位于深圳，现布局全国6大区域，2个研发中心，30余个分、子公司，是国内固体废弃物领域标杆企业之一。公司践行绿色低碳的理念，污染隔离、生态修复、综合运营及可再生资源利用四大业务业绩遍布全国31个省、自治区、直辖市及“一带一路”沿线国家。公司是国家高新技术企业，建设有广东省固废危废污染隔离防渗系统工程技术研究中心，是深圳市博士后创新实践基地，及中国城市环境卫生协会生活垃圾卫生填埋及生态修复专业委员会主任委员单位，参与编制10余项国家及行业标准，获得近200项专利、45项著作权。2021年9月16日，公司在深圳证券交易所上市，作为南山区第187家上市企业，在南山区187平方公里的热土上实现了上市公司全国密度第一。

 

讲座主要内容：

我们日常生活产生的垃圾都去哪了？垃圾填埋场又是什么？垃圾填埋场居然还可以变成4A级景区？本场讲座将为大家讲述“碳达峰、碳中和”大背景下填埋场与碳排放的关系，并对填埋行业发展中存在的问题、发展趋势等进行了分析，使学生充分了解了在城市化进程中垃圾填埋场所发挥的作用。
