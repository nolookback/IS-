学术讲座：从科技到商业——以新材料新能源、先进制造业为例
机电与控制工程学院　2025/4/6 17:26:00
  （钟沛基 2022280095）

讲座题目：从科技到商业——以新材料/新能源、先进制造业为例

演讲嘉宾：王懋

讲座时间：2025年4月7日（星期一）10:15

讲座地点：深圳大学沧海校区致信楼S630A会议室

邀 请 人：马将 教授

讲座内容简介：

科技创新作为“新质生产力”的重要抓手，如何将科研成果高效的商业化已经成为大家所关注的重要问题。科研院所的科研方向和选题应该如何选择？科研院所与企业合作应该采取何种形式？科研人员和老师在科研成果商业化过程中应该如何定位自己的角色？科研人员应该自己创业吗？科研人员应该如何与资本打交道？科技企业在成长过程中会碰到哪些难题？新材料、先进制造业及新能源产业的科技创新路径该如何设计？以上问题的探讨和交流有助于科研人员和老师明确自己在产学研转化过程中的定位，有助于科技界和企业界的高效互动沟通，最后也将对科技到商业的转化起到有益的推动作用。

演讲嘉宾简介：

王懋，硬科技领域知名投资人，北京大学EMBA，拥有从“管理咨询”到实业操盘再到“并购整合”和创业投资的完整丰富经历。对电子、汽车、高端装备、新材料、新能源等先进制造业有深刻理解，产业资源丰厚，善于“赋能”。立足于长期高质量价值的发现者和创造者，王懋先生目前在东方富海和富海中瑞专注于新材料、新能源、新一代信息技术、先进制造业等方向的创业投资和并购整合。部分投资案例包括：金力永磁（300748）、沃格光电（603773）、熵基科技（301330）、汇创达（300909）、思泉新材（301489）、元丰电控（创业板已经过会）、长联科技（301618）、好博窗控、腾盛精密、玉成有限、金磁海纳、仁仁医疗、商德陶瓷等。
