报告摘要：

眼球是人体中至关重要的复杂器官，它让我们能够观察到周围的世界。由于眼球内部结构的精细性和治疗所需的高精确度，眼科领域在医疗技术不断进步的今天，已经积累了大量的多模态影像资料和治疗视频数据。对这些数据进行深入的分析和理解，对于术前规划和术中辅助具有重要意义，它们可以为医生提供强有力的决策支持。本报告聚焦于眼科手术的智能辅助技术，旨在通过视觉分析提升医生在多目标识别、空间感知以及术中动作识别三方面的能力。具体来说，我们通过术前多模态影像实现血管和病灶的精准分割，辅助诊疗并提高对病变区域的空间感知能力；通过三维眼球重建为手术提供精确的规划指引，并增强对眼球结构的感知能力；以及通过手术视频的多层次理解，增强术中操作的智能感知能力。
