【深圳大学40周年校庆千场学术活动(第899场)】“媒体融合时代的跨媒体‘叙世’——以《哈利·波特》与‘魔法世界’为例”学术讲座在传播学院举办
传播学院　2023/6/7 15:36:00
  （钟沛基 2022280095）

5月29日下午，首都师范大学青年教师林品应邀来我校，主讲了题为“媒体融合时代的跨媒体‘叙世’——以《哈利·波特》与‘魔法世界’为例”的学术讲座。本次讲座由深圳大学区域国别与国际传播研究院副院长，传播学院副教授战迪主持。

林品，北京大学中文系博士，现为首都师范大学文学院教师，香港中文大学新闻与传播学院访问学者，中国科幻研究中心“起航学者”，主要从事文化理论与大众文化研究，著有《我的哈利·波特：哈7大猜想》《J.K.罗琳的幻想文学作品研究》（已签约，待出版）等书。同时也是Cosplay玩家，曾担任华纳兄弟公司&英国驻华大使馆“哈利·波特20周年”特别直播活动的主持人暨特邀Coser。

（图：讲座现场1）

林老师从亨利·詹金斯提出的“跨媒体叙事”理论与参与式文化理论出发，以《哈利·波特》系列为案例，探讨了经由媒介融合的文创产业链与媒介赋能的“产消者”如何共同运用魔法世界的“跨媒体数据库”，协同创作出21世纪以来最具有国际传播力和影响力的流行文化IP之一。

首先，林老师认为跨媒体叙事的关键要素是世界观的设定。《哈利·波特》系列巧妙地运用了古希腊自然哲学中的“四根说”和占星术，以“风土水火”的四大元素、黄道十二宫（Zodiac）等前科学的自然猜想构建出一个多元包容的奇幻魔法世界，并且在“现实化”和“奇幻化”之间维持了平衡，通过“并置”、“戏仿”和“变形”等手段，既持续引起读者的共鸣，又带来令人惊异的神秘色彩。

（图：讲座现场2）

随后，针对奇幻文学IP展开的跨媒体文创产业链条与互联网粉丝同人文化之间的有机互动，林老师在詹金斯理论的基础上，以“跨媒体数据库”的概念进一步揭示数码原住民与其消费对象之间的关系。通过《哈利·波特》电影拍摄取景地的“圣地巡礼”文化、学院文化、魁地奇运动文化和Cosplay文化等案例，林老师指出，原文本（例如《哈利·波特》小说）成为基础的设定数据库，系统性地散布在多个媒介平台、载体的社交游戏网站，与文化产品、衍生周边、粉丝同人文化作品等彼此呼应，共同营造出对“魔法世界”的强烈归属感。

（图：讲座现场3）

最后，林老师认为，这些由“产消者”积极参与的流行文化潮流折射出“魔法”与科技在当下社会存在的微妙关系，即处于后工业社会情境与新技术革命进程中的人类对“技术魔法化”的体验、想象、迷恋与焦虑。

本场学术讲座持续两个多小时。最后，战迪老师对林老师的精彩分享表达了感谢，并对本次主题讲座进行了简要总结。战迪老师从自身研究经历出发，表示无论是青年亚文化研究还是跨媒体叙事研究都需要不断对已有理论进行拓宽，根植于中国本土实践，从多元角度探索当代全新的文化样貌。
