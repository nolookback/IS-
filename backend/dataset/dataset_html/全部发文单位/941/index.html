化学与环境工程学院　2021/12/11 20:47:00
  （钟沛基 2022280095）

主讲人: 苏志教授

时间:12月12日上午10:00

地点: 丽湖校区守信楼（原B1栋）420会议室

报告人简介：

苏志，南京师范大学化学与材料科学学院教授，副院长，江苏省特聘教授，2005-2010年在南京大学配位化学国家重点实验室获理学博士学位；2010-2016年分别在美国伊利诺伊大学芝加哥分校、 美国伊利诺伊大学厄巴纳-香槟分校从事博士后、研究员，2017年回国加入南京师范大学化学与材料科学学院。已发表学术论文64篇，包括J. Am. Chem. Soc.，Chem. Sci., 等国际顶级期刊。其研究工作受到国内外同行的广泛关注，工作被Science， Chemistry World等推荐，并多次被Coord. Chem. Rev., Acc. Chem. Res., Chem. Soc. Rev.等综述正面引用报道。苏志教授主要的研究兴趣包括：1）新型多孔材料的开发应用；2）功能性抗癌药物的开发应用。
