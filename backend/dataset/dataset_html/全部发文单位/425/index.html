【学术讲座】为患有思觉失调年青人提供的生涯辅导：叙事生涯与表达艺术之创意结合
心理学院　2024/4/18 10:49:00
  （钟沛基 2022280095）

讲座名称：为患有思觉失调年青人提供的生涯辅导：叙事生涯与表达艺术之创意结合

讲座时间：2024年4月24日 星期三 下午13:30

讲座地点：深圳大学沧海校区致理楼C座L3-1201

主讲人：周德慧博士

讲座摘要：

在香港，政府和非牟利机构会为患有思觉失调的年青人提供辅助就业和职业培训，协助年青人获得一份正当职业和有规律的生活，促进复元。但是对于许多患有思觉失调的年青人特别是有大学学历的高功能年青人，在与精神疾病的抗争中， 他们不满现状，希望获得成功。叙事生涯规划是透过生涯建构晤谈和来访者共同构建其生涯设计。在过去三年里， 我们与精神科医生和职业治疗师紧密合作，为高功能患有思觉失调的年青人提供了叙事生涯个人辅导与表达艺术小组相结合的服务，探索他们的梦想、优势以及兴趣爱好，并且一起探讨生涯和复元的关系。报告以个案分享的方式介绍如何将叙事生涯个人辅导与表达艺术小组创意结合，同时也透过一间医院67位青年参加者的前后测和介入组与对照组的比对探讨该干预模式的有效性。

个人简介：

周德慧博士现为香港树仁大学辅导及心理学系副教授及副系主任（本科生课程）。她是香港树仁大学游戏治疗社会科学硕士课程总监及艺术与创意心理实验室工作室的主任。她是《亚太咨询与心理治疗杂志》主编和《香港心理健康杂志》的编辑。她的学术兴趣是心理健康领域。她开展了循证实践研究项目，为有特殊教育需要的学龄前儿童进行早期诊断和早期干预，并为精神分裂症患者的护理人员提供心理支援。近年来，她致力于为香港精神病青少年和戒毒青少年的叙事生涯设计及其相关实证的研究。她对隐喻认知和隐喻在心理治疗中的使用特别感兴趣。在临床实践中，她用游戏疗法、表达艺术疗法和叙事疗法来服务有需要的儿童、青少年和成人。

关键词：思觉失调 叙事生涯 生涯建构晤谈 表达艺术 复元
