主讲人简介：

张伯伟，南京大学文学院二级教授、博士生导师，兼任南京大学人文社会科学高级研究院特聘教授、南京大学域外汉籍研究所所长、韩国东方诗话学会理事、中国古代文学理论学会副会长、江苏省比较文学学会常务理事，曾任日本京都大学、韩国外国语大学、台湾大学、香港浸会大学客座教授。主要从事中国古代文学理论、文学批评史和域外汉籍的研究，著有《回向文学研究》《中国古代文学批评方法研究》《域外汉籍研究入门》《作为方法的汉文化圈》《东亚汉文学研究的方法与实践》等，主编有《朝鲜时代书目丛刊》《朝鲜时代女性诗文集全编》《日本世说新语注释集成》《全唐五代诗格汇考》等以及专业刊物《中国诗学》《域外汉籍研究集刊》。

 

内容简介：

我们问伟大的文学“能”带给我们什么，而不是“会”带给我们什么，就是指文学拥有一种可能性而非必然性，要把这种“可能”变成“必然”，需要读者的配合，而接受过文学高等教育的人，有理由成为更善于配合的读者。“伟大的文学能带给我们什么”，这个问题既没有标准答案，也没有终结答案。每个人应该去寻求自己的答案。当读者学会了用作家的身体、情感和心灵去观看文学，也就学会了改变自身感官的功能性习惯或惰性，从而更新了与这个世界的互动。要能理解“伟大的文学”，读者要努力将自己的精神层次提升到相应的高度。
