学术讲座：天职国际深圳分所钟斌—天职国际——“智慧财务应用分享”
经济学院　2022/12/1 15:27:00
  （钟沛基 2022280095）

讲座题目：天职国际——“智慧财务应用分享”

讲座时间：2022年12月3日（周六）10:00-12：00

讲座地点：腾讯会议（会议号：110-677-510）

主讲嘉宾：钟斌　　天职国际深圳分所

主讲嘉宾简介：

　　钟斌，清华大学MPA、香港城市大学EMBA学历；拥有近20年的投融资及管理咨询服务经验，服务领域包括战略咨询、风控咨询、财务标准化、财务数字化转型咨询、并购与融资咨询、人力资源咨询。服务过的大型企业集团有大唐集团、中广核集团、中国电子、华侨城集团、广东路桥集团、广东港航集团、深圳水务集团等。服务过的上市公司有桂冠电力、白云机场、深圳机场、深圳地铁等。现任天职国际管理咨询合伙人，天职国际深圳分所党总支书记，注册税务师(CTA)，国际注册内审师(CIA)，澳大利亚公共会计师(IPA Au)，英国财务会计师(IFA)，高级人力资源管理师，中国企业联合会全国中小企业管理咨询服务专家，深圳市财政局委任深圳市注协监督委员，福田区会计学会常务理事。

讲座主要内容：

　　企业财务数字化转型过程中，财务数字化转型是“面”、财务共享服务是“线”、财务机器人则是“点”。 财务机器人针对财务的业务内容和流程特点，以自动化替代财务手工操作，辅助财务人员完成交易量大、重复性高、易于标准化的基础业务，从而优化财务流程，提高业务处理效率和质量，减少财务岗位劳动强度，使资源分配在更多的增值业务上，促进财务转型。本次分享将用天职国际真实的财务机器人实例来分享财务机器人应用场景，并启发同学们思考与探索更广、更深的智能应用场景。

　　欢迎广大师生参加！
