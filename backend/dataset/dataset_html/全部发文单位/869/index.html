【学术讲座】变结构无人自主系统关键技术研究
电子与信息工程学院　2022/6/7 11:31:00
  （钟沛基 2022280095）

演讲嘉宾：鲁仁全 教授  博士生导师 国家杰出青年基金项目获得者

邀请人：   黄磊 教授

报告时间：2022年6月09日（周四）下午1:30-3:00

报告地点：沧海校区-致信楼-N710

 

报告摘要：针对特殊场景下的无人自主系统完成水陆空跨域飞行控制的难题，进行变姿态控制，减振控制与协同控制研究，实现无人载体完成水空，陆空，水陆两栖任务，研发了一系列无人自主载体，并在不同场景得到应用。

 

报告人简介：鲁仁全，博导，教育部高层次人才、万人计划领军人才、国家杰出青年基金项目获得者、科技部重点领域创新团队带头人、基金委创新群体负责人、科技部创新人才培养基地负责人、广东省自然科学基金研究团队带头人、广东省重点实验室主任、享受国务院特殊津贴、珠江学者特聘教授、国家百千万人才工程有突出贡献中青年专家、教育部新世纪优秀人才计划获得者，担任中国自动化学会控制理论专业委员会（TCCT）委员。获得国家自然科学基金杰出青年基金1项、主持863项目2项、主持NSFC-广东大数据科学中心项目1项（重点支持项目）、科技部重点领域创新团队项目1项、国家重点研发计划1项、中央财政支持地方高校改革发展资金人才培养和创新团队建设项目1项，广东省科技创新战略专项资金项目1项。荣获教育部自然科学一等奖（两项）、浙江省科学技术奖一等奖、教育部科学技术进步奖二等奖、中国纺织工业协会科学技术进步奖二等奖等奖励。主要研究工作包括网络化系统控制理论和应用、协同控制及量化反馈系统的鲁棒控制。至今为止，发表SCI论文140余篇，包括Automatica论文9篇，IEEE汇刊论文60余篇，SCI他引3400余次，高被引论文27篇，出版专著3部，授权发明专利近20项，申请发明专利60余项。
