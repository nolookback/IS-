学术讲座：大连海事大学靳志宏教授—集装箱码头物流作业组织与调度优化
经济学院　2021/12/28 9:20:00
  （钟沛基 2022280095）


讲座题目：集装箱码头物流作业组织与调度优化

讲座时间：2021年12月29日（周三）16：00

讲座地点：腾讯会议（会议号：424-300-665）

主讲嘉宾：靳志宏教授　　大连海事大学

主讲嘉宾简介：

　　靳志宏，东北大学工学学士、工学硕士、名古屋工业大学工学博士，先后任教于东北大学、名古屋工业大学、大连海事大学，历任大连海事大学交通运输管理学院院长、交通运输工程学院院长，二级教授，博士生导师。研究方向为物流系统优化、交通运输规划与管理。

　　主要学术兼职包括：教育部物流类专业教学指导委员会委员、辽宁省物流类专业教学指导委员会主任、《大连海事大学学报》主编等。先后获得辽宁省“百千万人才工程”百层次人才、辽宁省“海智百人计划”专家、“大连市归国留学创业英才”、“大连市领军人才”等称号。

讲座主要内容：

　　主要涵盖两个方面的内容，其一是聚焦于集装箱码头内部的物流作业组织与调度：基于混合流水线的集装箱码头内部作业组织与调度；其二是聚焦作为码头与内陆腹地集疏运系统交界处的码头闸口的物流作业组织与调度：基于移动互联的集装箱码头内外部资源运输组织协同。

　　欢迎广大师生参加！
