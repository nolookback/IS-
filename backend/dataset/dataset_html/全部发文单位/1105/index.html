厦门大学黄鸣奋教授主讲的“科幻电影创意与新媒体伦理”学术讲座在传播学院成功举办
传播学院　2021/4/27 14:32:00
  （钟沛基 2022280095）

4月22日上午，厦门大学黄鸣奋教授受邀来到传播学院，为学院研究生带来一场题为“科幻电影创意与新媒体伦理”的学术讲座。黄鸣奋老师是厦门大学人文学院中文系的教授。目前为南方科技大学人文社会科学学院科学与人类想象力中心高级访问学者，北京电影学院未来影像高精尖创新中心、中国作家协会网络文学研究院（杭州）特聘研究员。他先后从事中国古典文论、文艺心理学、海外汉学、地方文化、网络文学、新媒体艺术理论、科幻电影等研究，共主持国家社科基金课题7项（含重点2项），独立获得全国高校人文社科优秀成果奖3项、福建省社科优秀成果奖12项，出版个人专著28部、教材2种，发表论文400余篇，合计1000多万字。

黄鸣奋教授的讲座内容主要围绕“万物互联互通：新媒体开拓进取的途径”、“万物皆可追踪：新媒体大展身手的领域”、“批判蕴含机遇：新媒体走火入魔的可能”三部分展开。

在第一部分“万物互联互通：新媒体开拓进取的途径”中，黄鸣奋教授认为科幻电影是互联网时代的产物。科幻电影有三个特点：一是，以科技为参照系，不同于魔幻、玄幻等；二是，力求大大超越现有科技水平，不同于科普；三是，诉诸音像，讲究可视性、可听性，不同于科幻文学。紧接着，黄鸣奋教授从“科幻电影”、“新媒体”、“艺术创意”三者结合的问题切入，向现场的同学们讲述了科幻电影思维。

在第二部分“万物皆可追踪：新媒体大展身手的领域”中，黄鸣奋教授向大家介绍了互联网可能探索的领域。他指出可以从“星际通信”、“脑际通信”、“维际通信”三方面进行互联网探索。星际通信主要以追踪生物的运动为主。脑际通信以直接运用脑电波进行交流为特征，有别于各种以中介（如语言）为纽带的间接交流。脑际通信的本质就是追踪思想的运动。维际通信主要指超越现阶段四维时空限制的通信，它发生于多元宇宙或异度时空之中。从“万物皆可追踪”的角度看，维际通信的本质就是追踪似乎已经消失的运动。消失只是从我们的视野、听域或其他感受范围中消失，但只要物质不灭，能量可以转化，那运动就仍然存在，只不过不是存在我们当下的时间、空间中。

在第三部分“批判蕴含机遇：新媒体走火入魔的可能”中，黄鸣奋教授认为我们正在经历媒体匮乏向媒体富裕（甚至是媒体冗余）的时代转变。在媒体匮乏时代，科幻电影则从不同角度表现了人们对于更多、更新、更有效的媒体的渴望。在媒体冗余时代，科幻电影则从多种角度表现了人们对媒体负面影响的担忧与批判。黄鸣奋教授认为当今的科幻电影所起的作用并非验证某种科学原理，而是开拓创造、创新的道路，引导观众思考。

最后，现场的同学们就相关问题与黄鸣奋教授进行了交流。至此，本场讲座圆满结束。
