主讲人简介：

马天俊，中山大学哲学系教授，博士生导师，中山大学马克思主义哲学与现代化研究所副所长，主要学术研究领域是形而上学、知识论、修辞学，以及马克思思想。先后主持教育部重点研究基地重大项目、国家社会科学基金项目等多项，出版专著《真理的境遇》（1999）、《从生存的观点看》（2008初版，2020再版），在《哲学研究》《吉林大学学报》等国内重点刊物发表学术论文50余篇。
