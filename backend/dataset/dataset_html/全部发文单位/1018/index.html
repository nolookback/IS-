学术讲座（五十三）：含N、S有机荧光材料的设计、绿色合成与检测应用
化学与环境工程学院　2021/10/13 17:10:00
  （钟沛基 2022280095）

主讲人: 汪朝阳教授

时间: 10月15日下午2：30

地点: 丽湖校区守信楼（原B1栋）420会议室

报告人简介：

汪朝阳，教授，博导，九三学社成员，广东省高校“千百十工程”省级人才，华南师范大学化学学院化学系主任。北京师范大学本科、硕士，华南理工大学博士、博士后。主要从事有机合成、绿色化学及功能材料等方向的研究，先后主持完成多项国家和省级科研和教学项目，相关研究成果以通讯作者在Angew. Chem. Int. Ed.、Green Chem.、Org. Lett.、J. Org. Chem.、Macro. Rapid Commun.等期刊发表，授权发明专利20余件，获得广东省科学技术奖一等奖（排名第二）。作为首批国家一流、省一流线下课程《有机化学》主持人，主编《有机化学》、《有机化学学习指导》、《绿色化学通用教程》、《化学史人文教程》等多部教材。

讲座摘要：

本次报告将介绍该课题组近年来在含N、S等杂原子有机荧光材料的设计、绿色合成与检测应用等方面的进展。

欢迎有兴趣的师生参加！
