【学术讲座】基于下视成像声纳的超分辨水声成像技术初探
电子与信息工程学院　2022/6/29 11:13:00
  （钟沛基 2022280095）

报告人：王俊、国防科技大学副研究员

报告时间：2022年7月2日（周六）晚上19:00-22:00

报告地点：致信楼N710，腾讯会议：161-205-891，会议密码：220702

邀请人：孙维泽 助理教授

报告摘要：

超分辨成像在光学和雷达领域是前沿热点，而水声成像领域的超分辨技术研究则刚刚起步。受复杂海洋环境等因素影响，现有成像声纳分辨率提升面临技术瓶颈问题，急需新兴成像理论和算法突破，基于稀疏贝叶斯学习的超分辨图像增强方法是一类典型的超分辨成像算法。本报告首先阐述超分辨的内涵和基本方法，进一步综合分析当前超分辨水声算法的研究现状，最后，介绍本单位在下视合成孔径超分辨水声成像算法方面的研究进展和技术难点。

  

 

报告人：王俊，博士, 现为国防科技大学气象海洋学院深海科学技术实验室副研究员。2010年本科毕业于国防科技大学光电学院光信息科学与技术专业，2016年博士毕业于国防科技大学光电学院光学工程专业，2013年赴澳大利亚新南威尔士大学联合培养一年。长期从事光纤水声探测技术研究，现主要研究方向为高分辨水声成像技术和超分辨水声成像算法。主持科研项目3项，作为核心骨干参与国家、军队级重要科研项目9项，作为总指挥兼副首席科学家组织国家海洋声学调查航次任务1项。近年来，发表SCI\EI论文十余篇，授权发明专利6项，实用新型专利1项，软件著作权2项，获省部级科技进步二等奖2项。
