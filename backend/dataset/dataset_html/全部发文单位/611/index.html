【学术讲座】社交媒体内容真实性鉴别
电子与信息工程学院　2023/11/22 18:13:00
  （钟沛基 2022280095）

报告嘉宾：廖鑫，湖南大学教授

邀 请  人：李斌    教授

时      间： 11.24下午3：30-4:30

报告地点：致信楼N723



报告摘要：数字图像/视频作为对客观事实最直接的记录方式，在社交媒体、新闻报道、司法证据等应用中得到广泛普及。然而，多媒体编辑软件的飞速发展, 使得数字图像/视频篡改越来越便捷，特别是以Deepfakes为代表的AI换脸技术进入公众视野，正在颠覆人们“眼见为实”的传统观念，带来严重的公共信任危机。本报告将围绕多媒体取证与安全展开讨论，试图为社交媒体内容的真实性保障提供更有效的检测手段。

报告人简介：廖鑫，湖南大学教授，博导，岳麓学者，国家自然基金委重点项目负责人，入选全球前2%顶尖科学家榜单。现为网络空间安全系主任，大数据研究与应用湖南省重点实验室副主任，湖南省研究生拔尖创新人才培养基地主任。北邮本硕博，信息安全博士，中科院软件所博士后，香港大学研究学者，美国马里兰大学公派访问学者。主要研究方向为多媒体取证与安全，近五年在国内外重要期刊会议发表学术论文60余篇，其中8篇入选ESI高被引论文，3篇入选ESI热点论文。主持国家自然科学基金委重点/面上/青年项目、国家重点研发计划课题等20余项科研项目。主编出版专著《数字图像取证》。获IEEE国际信号处理挑战赛冠军，CCF推荐A类会议ACM MM最佳论文奖，IEEE Trans CSVT最佳论文提名奖，中国计算机学会技术发明二等奖，中国产学研合作创新成果一等奖，全国信息安全竞赛一等奖优秀指导教师，湖南省计算机学会优秀成果奖一等奖。

 担任1区TOP期刊IEEE Signal Processing Magazine编委，亚太信号与信息处理学会APSIPA媒体安全与取证专委会秘书长，中国电子学会电子数据取证专家委员会委员，中国电子学会网络空间安全专家委员会委员，中国图象图形学学会数字媒体取证与安全专委会委员，国家网络安全产业园区特聘专家，湖南商用密码产业示范基地特聘专家，湖南广电网络安全特聘专家，IEEE Senior Member。
