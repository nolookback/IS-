学术讲座丨规划里的市政工程与市政工程里的规划
建筑与城市规划学院　2021/12/14 21:40:00
  （钟沛基 2022280095）

讲座题目：

规划里的市政工程与市政工程里的规划

 

讲座概要：

近日，一则《深圳精准调控供水压力，全民节水》的文章刷爆了深圳人的朋友圈，让全社会体验到市政基础设施建设的重要性，以及市政基础设施与整个城市发展之间的关系。完善的市政设施促进城市的高速发展，反过来又将限制城市的发展，甚至反噬城市发展的成果。“以水定城、以水定地、以水定人、以水定产”，就是深刻反映了这种辩证的关系。延展至整个城市规划的框架里，也反映了城市规划与市政工程规划之间的支撑与反馈的关系。

 

演讲嘉宾：

傅世明，深圳市城市规划设计研究院市政工程设计院副院长，给排水专业副总工程师，高级工程师，注册公用设备工程师（给排水），广东省城乡规划行业专家库专家、广东省建设科技与标准化协会专家委员会委员，深圳市勘察设计协会给排水专业委员会委员。

扎根深圳，从事市政工程规划和设计工作20年，曾参与深圳市后海片区、前海片区、深圳湾口岸片区等重点区域的市政工程设计工作，并主持蛇口海上世界片区、香蜜湖片区等重点区域的市政工程详细规划工作。参与编制省级规范一项，在《中国给排水》、《给水排水》等核心期刊发表论文多篇，参与项目获广东省科技进步特等奖一项，主持项目获中国土木工程詹天佑奖一项，及深圳市优秀工程勘察设计一二等奖多项。
