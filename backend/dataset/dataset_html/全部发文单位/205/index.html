讲座题目：智能绿色、引领未来—船海领域国际热点研究方向

演讲嘉宾：张卫东 讲席教授（上海交通大学）

讲座时间：11月 14 日（星期四） 下午15：00

讲座地点：深圳大学沧海校区致信楼S630-A会议室

邀 请 人：白小山 特聘研究员

讲座内容简介：

海洋蕴藏着丰富的石油、渔业和矿产资源，是远未充分开发的资源宝库、世界经济竞争的重要领域。因此国家在新世纪制定了建设海洋强国的宏伟目标。开发深海资源与发展海洋经济、改善海洋环境与保护海洋生态、维护海洋权益与保卫国家安全必须具有强大的船舶海洋装备技术与产业的支撑能力。报告结合上海交通大学/海南大学IPAC海上无人系统团队近年来围绕智能与绿色船舶开展的研究工作，介绍了船海领域国际热点研究方向，分析了领域的前沿技术，展望了未来的技术发展前景。

演讲嘉宾简介：

张卫东，在浙江大学获得学士、硕士和博士学位，上海交通大学讲席教授、国家杰青、教育部知名学者、享受国务院政府特殊津贴专家、国家重大专项首席、德国洪堡学者、爱思维尔高被引学者，现任上海高校船舶自动化工程研究中心主任，教育部海洋智能系统工程研究中心主任。研究领域包括智能控制理论和人工智能理论，及其在海上无人系统中的应用。出版1本英文专著；发表SCI论文200多篇，Google引用1.2万余次；申请国家发明专利62项。
