报告人简介：

吕莹爽博士于2016年在美国伊利诺伊大学香槟分校取得食品科学与人类营养学博士学位。博士后出站后，先后就职于美国Kerry，Pharmavite研发总部，全职担任研究型科学家一职。在美期间，本人研究兴趣广泛。曾多次在北美IFT食品大会上进行学术汇报，并获得第76届IFT碳水化合物组研究论文决赛第一名。此外作为国际农业荣誉学会Gamma Sigma Delta会员，积极参与行业内期刊评审工作。

具体科研经历包括：硕士期间致力于食品过敏原在加工过程中的变化研究及检测。博士期间着重于碳水化合物的结晶以及热稳定性的研究，并掌握了相关的物理、化学分析手段，建立了一套对食品材料稳定性研究的基本方法。工作中利用食品化学背景，帮助公司解决生产以及鉴定中的技术难题，并参与专利的撰写。尤其是近年来作为感官评定小组的负责人，参与了风味以及新产品的改进及研发。
