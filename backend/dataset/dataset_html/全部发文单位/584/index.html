【学术讲座】真菌几丁质触发的植物免疫信号转导
生命与海洋科学学院　2023/12/5 10:11:00
  （钟沛基 2022280095）

报告题目：真菌几丁质触发的植物免疫信号转导

报告人：李剑峰 教授

主持人：刘宏涛 教授

报告时间：2023年12月12日（星期二）下午14：30

报告地点：深圳大学丽湖校区守慧楼A6-501

个人简介：

李剑峰，国家杰出青年科学基金获得者，中山大学生命科学学院教授，博士生导师，生物科学与技术系主任。1995-2005年先后在中山大学获得学士、硕士和博士学位。其后在美国田纳西大学、哈佛医学院从事博士后研究，2014年9月受聘于中山大学。先后获得国家海外高层次人才引进计划青年项目、国家优秀青年科学基金、国家杰出青年科学基金资助。长期致力于植物免疫信号转导的分子机制和植物基因编辑技术研究，作为通讯作者（含共同）在Cell Host Microbe （2篇）、Nature Plants（4篇）、Plant Cell（2篇）、Molecular Plant（2篇）、Nature Communications、Trends in Plant Science等SCI期刊发表论文35篇。

内容简介：

植物通过细胞膜上的模式识别受体（PRRs）感知胞外空间中存在的病原菌保守的标签性分子（MAMPs），进而激活胞内免疫信号转导，使植物产生对一大类病原菌的广谱抗性。本次报告将介绍模式植物拟南芥中真菌细胞壁组分几丁质触发的植物免疫信号转导机制，重点聚焦几丁质PRR受体复合物激活和调控几丁质信号转导的分子机制，以及几丁质信号通路与细菌鞭毛、植物细胞因子信号通路之间的协同调控。
