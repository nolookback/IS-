学术讲座：教育信息化新进展及如何开展教育研究
教育学部　2021/3/18 17:14:00
  （钟沛基 2022280095）

主题: 教育信息化新进展及如何开展教育研究

主讲人：许林

主持人：曹晓明

时间：2021年3月19日 8:30—10:30

地点: 粤海校区师范学院B座未来教室 B303

 

主讲人简介：

许林，研究员，《中国电化教育》杂志社（中文核心、CSSCI源刊）社长/主编，中国教育技术协会常务理事、期刊专业委员会会长，中央电化教育馆学术委员会委员，长期从事现代教育技术及多媒体资源建设与应用研究。出版《现代教育技术》等专著3部，作为负责人组织了全国教育科学规划项目教育部重点课题“基于信息技术环境下学与教的理论与实践”及全国教育科学规划重点课题“信息化进程中的教育技术发展研究”等课题研究；并组织编制了《交互式电子白板教学功能规范》和《交互式电子白板教学资源通用文件格式》两项部颁标准；有丰富的教育信息化研究与实践经验。
