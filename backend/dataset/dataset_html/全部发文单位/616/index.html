报告人简介：

何自开，哈尔滨工业大学（深圳）理学院副教授，博士生导师。2009年本科毕业于中国科学技术大学；2013年在香港中文大学获得博士学位；随后加入香港科技大学唐本忠院士团队，开展博士后研究；2016年9月加入哈工大（深圳）开展独立研究。长期从事有机光电功能材料的设计、开发和应用研究。研究内容涵盖分子结构设计、有机合成、器件制备、光电性能表征和理论计算等多领域交叉：从分子到器件、从理论到实验、从电学到光学，均取得了一定研究成果。已经在Nat. Rev. Mater.、Nat. Commun.、Chem、J. Am. Chem. Soc.、Adv. Mater.、Angew. Chem. 等期刊发表论文40余篇, 其中第一作者（通讯作者）论文20篇, 其中7篇入选ESI高被引论文。相关工作相继被AAAS, C&EN, 新华社等予以亮点报道。

讲座摘要：

现代分子光化学对电子激发态的系统研究所建立的概念、理论和方法极大地拓展了人们对物质认识的深度，加深了对自然界中光合作用和生命过程的理解。同时，也为利用太阳能、保护环境、开发新反应、寻求新材料提供了重要的理论基础，在新能源、新材料和信息处理等高技术领域中发挥着越来越重要的作用。目前，光化学领域在研究对象上，正由分子层次向分子以上层次发展，如超分子体系、凝聚态体系、分子聚集体等，涉及多维度、多尺度、以及复杂化学体系的研究。在研究技术手段上，超快、超高分辨技术的运用使得领域发展从稳态走向瞬态，从宏观走向微观，更加准确深刻地认识光化学机制，掌握光化学反应、过程及功能的规律认知和精准调控。有机固体发光体的光物理过程正是目前光化学领域备受关注的研究前沿之一，是分子聚集体层次上的系统性科学问题。报告人将围绕新型有机固体发光体，开展发光材料的设计、探索和应用研究。从聚集体角度研究其电子激发态（单线态、三线态）的复杂光物理过程，阐述相关的结构与能学等相关变化规律及调控策略。
