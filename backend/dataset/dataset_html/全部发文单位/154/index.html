讲座名称：我国临床与咨询心理学的发展：历史与现状

讲座时间：2024年12月16号下午3点

讲座地点：深圳大学沧海校区致理楼C座L3-1201

主讲人：钱铭怡教授

主讲人简介：

钱铭怡，博士，北京大学心理与认知科学学院教授（退休），香港中文大学（深圳）学生健康辅导中心名誉主任，人文学院应用心理学客座兼职教授；中国心理学会理事；中国心理学会临床心理学注册工作委员会副主任委员；注册督导师（D-06-061）；教育部普通高等学校学生心理健康教育专家指导委员会委员；中国心理学会会士，世界心理治疗学会会士。

讲座简介：

临床与咨询心理学在我国的发展并非一帆风顺，本讲座将从历史和现状两个方面对此领域的情况做一概要的介绍。讲座内容涉及：（1）1949建国前后-文化革命的情况简单介绍；（2）改革开放以后的大致发展状况；（3）目前国内在此方面的发展；（4）中国心理学会临床心理学注册工作委员会（注册系统）的建立及其工作等。

欢迎感兴趣的师生参加！
