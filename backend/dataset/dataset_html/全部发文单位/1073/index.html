附：龚云教授简介

龚云，男，汉族，1971年9月出生，二级研究员，历史学博士，博士生导师，中国社会科学院马克思主义研究院副院长，中国社会科学院中国特色社会主义理论体系研究中心副主任，中国社会科学院习近平新时代中国特色社会主义思想研究中心执行副主任，世界社会主义研究中心副主任，中国政治学会常务理事，中华人民共和国国史学会常务理事，中国无神论学会副理事长，国家百千万人才工程入选者,享受国务院政府特殊津贴，国家有突出贡献的中青年专家,国家马工程重大课题首席专家，中央马工程重点教材《中国近现代史纲要》（2018年修订版）和《中国近现代史纲要辅导读本》主要成员。长期研究中国特色社会主义理论与实践、党史党建，独立出版8本著作，在《人民日报》、《光明日报》、《求是》、《马克思主义研究》等报刊发表学术论文百余篇。
