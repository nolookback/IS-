学术讲座：深圳稳健医疗，集团CIO邓建辉—零售行业供应链趋势洞察：数字浪潮起，变革喻新机
经济学院　2021/5/25 8:59:00
  （钟沛基 2022280095）


讲座题目：零售行业供应链趋势洞察：数字浪潮起，变革喻新机

讲座时间：2021年5月25日14:15-15：40

讲座地点：汇紫楼Ｂ206

主讲嘉宾：邓建辉　　稳健医疗，集团CIO

主讲嘉宾简介：

　　邓建辉，香港中文大学（深圳）供应链与物流管理高级管理人员理学硕士，目前稳健医疗，集团CIO。具有18年数字化转型从业经验，零售业科技创新先行者，零售企业数字化转型领军人，率先实现数字化中台体系。擅长时尚零售业务顶层设计、业务咨询、技术研发，也是一直努力建立跨学科思维模型新复合型从业者。

长期研究领域：

1.科技

长期专注于新技术领域研究，具备互联网,云计算、大数据、人工智能、物联网，数字化转型实战落地经验。

2.业务

长期推动大型企业商业变革管理咨询工作, 具备企业商业模式战略规划，企业顶层设计、企业结构治理设计、品牌策划、组织变革、物流规划，供应链等业务运营落地变革成功案例。

3.学习

长期学习及研究社会学、经济学、哲学、科学、数学及统计学。

讲座主要内容：

　　邓建辉的讲座主要介绍以下三方面的内容,即:

　　第一， 零售行业供应链趋势:

　　通过对行业内外的驱动因素和前沿观察进行分析，回归底层商业逻辑，对供应链行业进行全面、系统的思考，从商业模式的视角解读新经济下的生态供应链，介绍供应链洞察行业趋势，探索行业机会，把控行业脉络。

　　第二， 零售行业供应链数字化解决方案:

　　创新科技点燃变革新引擎, 在数字经济大发展的背景下，物联网、大数据、 云计算、人工智能、 区块链、机器视觉、实时计算、柔性自动化等技术将呈爆发趋势。加速了行业的智能化演进与模式革新，通过本段介绍供应链数字化解决方案。

　　第三， 分享对学习供应链管理的理解。

　　欢迎广大师生参加！
