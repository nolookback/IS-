报告人简介：

施苏华，博士，教授，博士生导师。中山大学逸仙学者计划领军学者，美国植物学会终身通讯会员，国际生物地理学会杰出会士。1982年毕业于江西大学生物系，1990年毕业于中山大学，获植物学博士学位并留校任教，1995年晋升为教授。长期从事植物演化研究，在物种形成、趋同演化和红树植物气候变化适应等领域作出重要贡献。发表Nature，Nat. Ecol. Evol.，Natl. Sci. Rev., Global Change Biol.等SCI期刊论文140余篇。曾获国家杰出青年基金资助，国家自然科学二等奖，教育部自然科学一等奖和求是杰出青年学者奖等荣誉。担任Mol. Ecol.、Evolution等国际期刊编委。
