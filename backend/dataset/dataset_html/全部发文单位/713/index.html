艺术学部“春之季”系列——张若迪的民族情怀”学术讲座圆满结束
艺术学部　2023/6/5 15:49:00
  （钟沛基 2022280095）

2023 年5月24号上午，著名作曲家、指挥家张若迪副教授的专题讲座在深圳大学音乐厅圆满举行。此次《张若迪的民族情怀》专题讲座为深圳大学40 周年校庆学术活动的第747场，由艺术学部音乐舞蹈学院主办。艺术学部副主任兼音乐舞蹈学院院长刘琨教授、音乐系中青年教师出席本次专题讲座，音乐系主任杨婷老师主持此次活动。

讲座开始前，音乐系杨婷主任介绍主讲人简介及本次讲座主题内容，随后刘琨教授赠予张若迪副教授深圳大学纪念礼品。

图1 刘琨院长赠予张若迪副教授深大纪念品

 

图2 杨婷主任介绍作曲家

 

此次讲座张若迪副教授以其在深圳大剧院录制的《张若迪的民族情怀·第四套原创作品音乐会》视频为例，分享其创作背景、创作手法及创作理念。

 

图3 张若迪副教授专题讲座

 

讲座共展示了张若迪副教授倾注多年心力创作的《新时代交响序曲——丰碑》《月下独酌》《牡丹花语》《第二交响曲“中国颂”》等4部交响作品。

其中《丰碑》以“不屈”“我能”作为创作的关键词，通过宏大叙事，描绘了脱贫攻坚战的波澜壮阔，叙述出14亿人民携手同心迸发出改变命运的磅礴伟力。

《月下独酌——为男高音、琵琶与管弦乐队而作》立意源自李白诗中意境。张若迪副教授表示：“读他的诗，与他沟通，感悟他的大愁和孤独，我希望通过音乐带大家进入李白的精神世界。”

《牡丹花语》以“花中之王”——牡丹为叙事对象。张若迪副教授认为：“牡丹花开，国色天香牡丹的精神特质，在中华民族的审美与精神象征方面意义非凡，寄以此作彰显国人的民族自信，祝福祖国繁荣昌盛，国泰民安。

第二交响曲《中国颂》是本场讲座的压台之作。张若迪副教授坦言，这部作品光构思就花费数十年，因为一次登山，被峭壁上六个鲜红大字“江山如此多娇”触动，激发创作冲动。在酝酿22年后，这部最初计划写成单乐章交响诗的作品转为四乐章的交响曲——第一乐章《致敬五千年》、第二乐章《苦难与坚韧》、第三乐章《大河上下》、第四乐章《还看今朝》。从四个乐章的小标题便能感受到，作曲家力图站在一定的文化高度上，怀着一颗赤子之心，抒发着对祖国的无限敬仰和无限深情。

张若迪教授的作品大气恢弘，民族风格鲜明，作曲技法结合今古，融汇中西。创作思想建立在“弘扬中华传统文化，赞美华夏灿烂文明，传承先人民族气质，歌颂爱国主义精神”这一哲学与美学意识基础之上。表达出其信奉“文化自信”的鲜明态度。讲座结束之时，其提及：“外来的交响乐队只是一个碗，用这个碗来装我们自己的食物，中国的老百姓就爱吃了。”故一直以来，他在坚守在博大精深的中国传统文化中寻找灵感，讲述中国故事，写老百姓听得懂的交响乐。
