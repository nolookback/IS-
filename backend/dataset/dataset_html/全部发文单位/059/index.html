报告题目：脑内胆固醇对阿尔兹海默症患者的认知能力影响

主讲嘉宾：Scott Hansen 博士

报告时间：2025年4月18日（星期五）下午14:00

报告地点：深圳大学丽湖校区守慧楼A6-501

Hansen实验室专注于研究兴奋性细胞在其天然三维状态下的信号传导，以更好地理解⼈类疾病的分⼦基础。通过研究这些分子机制，旨在阐明疾病的发展过程并开发出预防或减轻疾病影响的治疗方法。相关工作发表在PNAS，Nature Communication等期刊。
