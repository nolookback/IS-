主讲嘉宾：王建勋 教授

王建勋教授：加州大学圣迭戈分校（UCSD）免疫学博士、博士后，兼任北京中医药大学二级教授、博士生导师、博士后导师，深圳北京中医药大学研究院细胞与基因治疗研究所所长，表观遗传学专家，逆病毒、慢病毒等基因载体的国际顶级专家，也是目前国内唯一掌握逆病毒载体工业化生产技术的科学家。

师从美国科学院院士MichaelG.Rosenfeld 院士、Christ -opherK.Glass院士。在Nature等世界顶级杂志发表多篇论文，主要探索利用T细胞表观遗传性状改变来增强T细胞识别并杀伤肿瘤的能力，实现CAR-T治疗实体肿瘤的突破。
