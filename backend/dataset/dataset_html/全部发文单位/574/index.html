【学术讲座】马克思主义理论学科发展面临的主要问题和破解路径
马克思主义学院　2023/12/11 9:07:00
  （钟沛基 2022280095）

主讲人：燕连福 西安交通大学教授

时间：2023年12月11日（星期一）下午2:30

地点：汇文楼1600会议室 

主讲人简介：

燕连福，教授，博士生导师，西安交通大学马克思主义学院院长。近年来，主持教育部哲学社会科学重大攻关项目1项、国家社科基金项目1项、国家乡村振兴局重大招标项目8项，及省部级以上课题20多项。兼任第八届国务院学科评议组马克思主义理论组成员，教育部高等学校思想政治理论课教学指导委员会委员、教育部“研究生思想政治理论课”分教学指导委员会副主任，陕西高校思想品德教学研究会会长，陕西大学生形势与政策教育信息中心主任等职务。在《人民日报》《光明日报》《经济日报》《哲学研究》《马克思主义研究》《马克思主义与现实》《红旗文稿》等权威报纸和期刊上共发表120多篇学术文章，其中CSSCI收录60多篇。入选教育部“全国思想政治教育中青年杰出人才支持计划”（思政杰青）、“全国高校思想政治理论课影响力人物”等。获国家级教学成果奖、陕西省优秀教学成果奖、陕西高校人文社会科学优秀成果奖、陕西省哲学社会科学优秀成果奖等省级以上奖项20余项。
