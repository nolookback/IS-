【学术讲座】基于深度学习和无人机遥感的野生动物监测

  （钟沛基 2022280095）

主要内容：

报告主题：基于深度学习和无人机遥感的野生动物监测

主讲嘉宾：邱方 教授

报告主持：胡忠文副教授

报告时间：2024年3月22日（星期五）14:30

报告地点：深圳大学粤海校区建筑与城市规划学院C205


 

邱方,美国达拉斯得克萨斯大学（UTD）地理空间信息科学系教授。他2000年获美国南卡罗莱纳大学地理学遥感方向博士学位，1993年获中科院资源与环境信息系统国家重点实验室硕士学位，1990年华东师范大学地理学士学位。邱方教授主要研究方向为地理信息科学(GIS)，遥感，人工智能与深度学习,空间分析与建模等。他开发了一系列基于曲线的创新算法用于高光谱，高空分辨率，激光雷达，无人机热红外等多种新型传感器所收集的遥感大数据的分析及其融合，其研究成果被应用于城市变化，人口预测，环境监测，健康与灾害，野生动物保护等各领域。邱方教授曾主持并参与了美国国家科学基金委(NSF),宇航局(NASA),环保局(EPA)和疾病控制中心(CDC)等资助的多项研究项目。其研究团队有多项研究成果获得最优论文或研究奖，其中包括美国摄影测量与遥感协会（ASPRS）年度遥感最佳科学论文奖。邱方教授曾担任国际华人地理信息系统协会（CPGIS）主席，美国地理学会空间分析与建模专业委员会副主席，美国人口统计局研究顾问，以及UTD地理空间信息科学系主任（2013-2023）。

 

报告内容摘要： 野生动物监测传统上依赖于对动物栖息地的实地视觉调查，这种方法耗时耗力，空间和时间覆盖范围非常有限，不仅对动物具有侵入性，而且对观察者也很危险。无人机和传感器技术的最新进展为野生动物监测提供了令人兴奋的新机遇。无人机遥感具有成本低，灵活性高，空间覆盖范围大，对动物的侵入性较小，及对观察者的危险性较低的特点。近年来，我们在尼泊尔的Chitwan国家公园利用热红外传感器进行了多次无人机遥感观测，显示出热红外传感器在动物监测上巨大潜力，即使动物隐藏在植被冠层下， 也可以被探测到。我们利用一个卷积神经网络（CNN）深度学习分类器，对获取的热红外像图进行了野生动物的存在检测，以及动物种类的标记、训练、验证和分类。该深度学习分类器的野生动物存在检测与种类识别准确率都超过了97%。通过无人机云台捕捉到的GPS坐标和俯仰、偏航和翻滚的附加信息，可以将检测结果投影到地图上，准确描绘动物分布状况并估算其在景观中的占据率和密度。基于深度学习和无人机遥感的野生动物监测的使用，将会大大提高环境保护和国家公园管理的效率。
