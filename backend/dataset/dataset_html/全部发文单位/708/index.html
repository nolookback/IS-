【深圳大学40周年校庆千场学术活动(第959场)】“转型新风口：构建省市县一体化传播体系”学术讲座在传播学院举办
传播学院　2023/6/14 14:31:00
  （钟沛基 2022280095）

6月6日下午，浙江日报报业集团社长姜军为学院师生主讲了一场以“转型新风口：构建省市县一体化传播体系”为题的学术讲座。本次讲座由传播学院副院长杨洸教授主持。

姜军，工商管理硕士，高级编辑，浙江日报报业集团社长、党委书记。曾任杭州市江干区委常委，西藏那曲县委常委、常务副县长，杭州市委宣传部副部长，富阳市委书记，浙江广播电视集团总编辑，浙江省文化产业投资集团有限公司董事长等职。曾获教育援藏先进个人、全国县(市)科技进步工作先进个人、全国绿化奖章等荣誉。西泠印社社员。

讲座中，姜社长首先向大家介绍《浙江日报》的历史，74年来，每个时代的浙报人都秉持着开拓创新的精神，为践行新闻理想而努力奋斗，许多名篇从中诞生，例如：1958年爱国卫生运动轰轰烈烈之际，日报最早报道毛主席视察基层《毛主席视察小营巷》；1978年初面对文革流毒仍未肃清、一些地方还在“割资本主义的尾巴”，发表《砍掉四千株桃树的错误必须纠正》，支持农民发展多种经营，助推十一届三中全会出台重要政策；改革开放之初的1983年，以整版文章《企业家之歌》为当时备受争议的步鑫生的企业改革张目；2003年至2007年习近平同志担任中共浙江省委书记期间，《浙江日报》开辟“之江新语”专栏，连续发表232篇体现总书记早年省域治国理政思想的短论；等等。

接下来姜社长主要分享了浙江日报报业集团在媒体转型过程中的创新与坚守。

第一是浙报集团的开拓创新。早期在媒体形态创新上，浙报集团创办了《钱江晚报》，并创造了连续十年的晚报现象。面对竞争日益激烈的市场环境，集团提出且秉承着“传媒控制资本，资本壮大传媒”的发展理念，在国内报业率先进军资本市场，实现媒体经营性资产整体上市（600633）。新媒体时代到来后，积极打造“浙江新闻”等移动客户端，自主研发融媒体技术平台“媒立方”，助力媒体融合深化改革。去年6月起，浙报集团举全集团之力打造省级重大新闻传播平台，成立“传播大脑”科技公司，并整合了“浙江新闻”、“天目新闻”和“小时新闻”三个新闻客户端，推出“潮新闻”客户端，这是其媒体深度融合的又一重大举措。在浙报集团牵头下，浙报集团的“天目云”与浙江广电“新蓝云”整合并推出贯通省市县的“天目蓝云”，力图将11家市级党报与90家县级融媒体中心连成一张网，构建“省市县一体化传播体系”，目前已有70个市县参与进来。“我们不能走常规商业平台的路，而是要走出能与之一较高低、具有差异化的路。”姜社长说道，“浙报集团在创新方面一直走在前列，在其他人观望时先动，赶上先机，若是落在后面就永远只能是模仿。”

第二是党媒姓党要走在前。在媒体融合创新的同时，姜社长强调，党报的历史、地位与价值是无可替代的，浙报集团一直坚持要把党报办好，在政治上敏锐精准，维护好党报的权威性、连续性与专业性，敢于为民发声。

在讨论环节，新闻系教师对浙报集团在媒体融合的先进做法表示钦慕，并提问交流。辜晓进老师感叹浙报集团是上市公司反哺主业的成功范例，其带来的资本、受众与技术极大推动了报业的发展。杨洸老师概括了浙报集团的两个特点“豪”与“拼”，既能有充裕的资金投入，又能有充满干劲的拼搏精神，并坦言这本应也是深圳的特点，在媒体转型方面深圳报业还有许多需要向行业范例学习之处。
