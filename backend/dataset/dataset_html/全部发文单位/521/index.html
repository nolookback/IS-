孔祥建，厦门大学化学化工学院教授，博士生导师。 2003年7月毕业于聊城大学获理学学士，2009年毕业于厦门大学获得无机化学专业博士学位。博士毕业后在厦门大学化学化工学院工作至今，2014年9月到2015年9月期间在美国芝加哥大学做访问学者。 主要从事高核稀土团簇的合成策略、组装规律和磁光效应研究。累积在J. Am. Chem. Soc.、Angew.Chem. Int. Ed.、Matter、Natl. Sci. Rev.、Nature. Commun.、Acc. Chem. Res.等期刊发表SCI论文120余篇，被引7000余次。曾获教育部高等学校科学研究优秀成果一等奖（2020年，排名第二）、中国化学会青年化学奖（2012年）、全国百篇优博（2011年）、教育部青年知名学者（2016年）、国家优秀青年科学基金（2014年）和福建省运盛青年科技奖（2017）等奖励与项目支持。目前担任《结构化学》和《Polyoxometalates》期刊编委, 《中国科学化学》、《高等学校化学学报》和《中国化学快报》青年编委。
