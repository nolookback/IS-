学术讲座：新型绿色能源存储材料的设计
机电与控制工程学院　2024/12/31 14:01:00
  （钟沛基 2022280095）

讲座题目：新型绿色能源存储材料的设计

演讲嘉宾：朱国银 教授

讲座时间：2025年1月6日（星期一）10:00

讲座地点：深圳大学沧海校区致信楼S630B会议室

邀 请 人：阮文清 老师

讲座内容简介：

随着全球对可持续能源的需求增加，新能源存储技术在绿色能源系统中日益重要。传统的储能设备存在能量密度低和环境污染等问题，因此，超级电容器、锂离子电池和锌离子电池等新型能源存储器件成为研究的重点。这些设备面临着提升能量密度、循环稳定性和高温性能等挑战。本次报告着眼于创新能源材料的开发，通过优化电极材料、电解液和界面设计，特别是在锌离子电池中引入3D打印技术，成功提升了其稳定性和性能。此项研究不仅为新能源存储技术的进步提供了支持，也为绿色能源的普及和可持续发展贡献了新的解决方案。
