“甲方乙方：获得科研项目的关键思维”学术讲座在传播学院举办
传播学院　2024/1/26 14:56:00
  （钟沛基 2022280095）

2023年12月21日上午，重庆大学弘深学者、智能传播与治理研究中心主任曾润喜教授受邀为传播学院师生主讲了一场题为“甲方乙方：获得科研项目的关键思维”的学术讲座。本场讲座是传播学院2024年国家级项目申报辅导系列讲座之一，由传播学院副院长杨洸教授主持。

曾润喜教授将申报课题与是否获得国家立项形象比喻为甲方与乙方的关系，从选题、文献综述以及申报书的写作技巧几个方面，分享了他申报国家社会科学基金项目的经验。在选题方面，曾教授认为要综合考虑国家政策方针、理论问题、个人研究旨趣、研究基础等因素，强调选题要有亮点，要具学术价值，并具体介绍了社会科学基金项目和自然科学基金项目的区别以及选题与课题指南的关系。曾教授强调在撰写过程要明确文献梳理的目的，还详细讲解了国家社会科学基金项目书的写作规范与技巧。

最后，杨洸教授对本场讲座进行了总结发言，对曾教授十分细致且毫无保留的分享表达了衷心的感谢。在场师生向曾教授请教了有关申报项目的各种问题，曾教授都给予了详细和个性化的指导，师生都表示受益匪浅，深受启发。
