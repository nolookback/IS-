学术讲座：严苛环境下柔软密封界面的理论与共性关键技术
机电与控制工程学院　2021/1/19 9:11:00
  （钟沛基 2022280095）

报告主题：《严苛环境下柔软密封界面的理论与共性关键技术》

报告人：谭桂斌  　高级工程师、博士

报告地点：深圳大学沧海校区致信楼 S601

会议时间：2021年1月21日　 下午14:00-16:00

主持人：范  雪  博士

　　　

（一） 报告内容简介：

高性能密封、轴承、液压、传感器等的技术门槛高、创新性强、根植性深、难以轻易模仿，需要长期投入与积累，是制造业中的“硬科技”，对于“国之重器”的性能、质量和可靠性具有“锚定”作用。

“十三五”期间，“国家工业强基专项工程”项目《高端橡塑密封元件研发检测服务平台实施方案》（编号：TC150B5C0-42），谭桂斌高级工程师负责了课题一《橡塑密封可靠性技术研究》、担任了课题组长，期间主持的广东省自然基金项目《橡胶密封系统的摩擦损伤与失效演化特性研究》、“民机专项”子课题核心任务《密封圈动密封磨损寿命试验》与《液压作动器密封性能试验验证技术及方法研究》等关键共性技术、“卡脖子”技术研发，提升了高参数橡塑密封的性能与安全可靠性，有关研究成果打破了国外技术垄断；相关成果参编和出版的行业著作2部，主导编制2项可靠性技术标准，组织和召开了2016~2019年国际橡塑密封技术交流会。此次学术交流，将分享相关的表界面技术及应用进展。

（二）报告人简介：

谭桂斌，男，湖南郴州人，高级工程师、工学博士，2020年入选广东工业大学“青百人才”，精密电子制造技术与装备国家重点实验室固定人员；2009年至2015年，在中国石油大学（北京）机械与储运工程学院—“橡胶摩擦学团队”攻读博士学位（硕博连读），博士论文《油气管道内检测装备橡胶摩擦学特性研究》获2016年中国石油大学（北京）优秀博士学位论文奖；2015年6月博士毕业进入国机集团广州机械科学研究院，2015年至2020年在国家橡塑密封工程技术研究中心、广东省工业摩擦学企业重点实验室从事了应用基础研究（其中，2018年7月，获得中国机械工业集团严格评审的高级工程师技术职称）。目前，在机械表界面科学与摩擦学方向，已经发表学术论文30余篇，授权国家发明专利10余件，兼任“中国机械工程学会摩擦学分会青工委副主任委员”、“中国机械工程学会摩擦学分会工业摩擦学工作委员会副秘书长”、“广东省机械工程学会摩擦学分会副秘书长”等。
