报告人简介：

冷发光，湖南人，享受国务院政府特殊津贴。

清华大学土木系材料学专业毕业，博士。

2002年至今，在中国建筑科学研究院建筑材料研究所（建研建材有限公司）工作，任党支部书记、董事长、总经理、总工程师。

目前兼任全国混凝土标准化技术委员会秘书长（对口国际ISO/TC71）、全国水泥制品标准化技术委员会副主任委员；全国高性能混凝土推广应用技术指导组办公室主任兼材料及生产专业组组长；中国混凝土与水泥制品协会混凝土工程质量及标准化分会执行理事长；中国砂石协会专家委员会副主任委员等。

长安大学、湖南大学博士生导师。中国建筑科学研究院、郑州大学、武汉大学、重庆大学、防灾科技学院、华北水电大学等硕士生导师。

长期致力于高性能混凝土、混凝土耐久性、混凝土质量控制、混凝土标准化和固废再生利用等研究和工程技术服务工作。

获得省部级科技奖十余项。在国内外公开发表学术论文和著作百余篇（册）。主持和参与完成省部级和国家级课题二十多项；主编和参与国家行业等标准五十多本。主持马来西亚槟城二桥混凝土耐久性技术研究等重大工程技术咨询项目多项。培养研究生二十多名。
