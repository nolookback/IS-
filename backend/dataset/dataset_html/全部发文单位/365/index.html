02嘉宾介绍

 

刘迎宾

    在深规院从事各层次城市规划及设计领域实践工作十年以上，长期跟踪深圳蓝绿空间体系的演变和发展过程，不断探索规划领域的创新型实践，参与了包括深圳市“山海连城计划”城市设计及技术服务、深圳市海洋发展规划、深圳市总体城市设计、深圳市城市设计数字化管理平台、南山区城市意象设计、武汉汉江湾城市设计、厦门新机场片区城市设计等重大影响力项目；在综合规划、城市设计、宜居城市研究等方面积累了丰富经验，获得国优一等奖、深圳金牛奖等十多个奖项。
