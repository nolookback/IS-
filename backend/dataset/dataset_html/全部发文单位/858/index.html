学术讲座：东南大学凌振教授—面向深度网站指纹的流量特征整形防御技术
大数据系统计算技术国家工程实验室　2022/6/17 11:16:00
  （钟沛基 2022280095）

讲座题目：面向深度网站指纹的流量特征整形防御技术

讲座时间：6月18日 16:00-17:00

讲座地点：腾讯会议 （会议号：994-216-870）

主讲嘉宾：凌振（东南大学教授）

主持人：罗成文 副教授


讲座摘要：

       随着用户对隐私保护问题的日益关注，世界各地成千上万的用户普遍采用匿名通信系统来保护他们的通信隐私。然而，这种具有高度匿名性的网络系统容易遭受基于网站指纹（Website Fingerprinting，WF）的流量分析攻击而失去匿名性。该攻击允许攻击者窃听受害者与匿名通信系统之间的加密网络流量，并抽取流量特征进而识别受害者正在访问的目标Web站点。近年来，研究者们提出了基于深度学习模型的WF攻击技术，利用深度学习模型抽取高维度特征，以获得比基于传统机器学习模型WF攻击技术更好的性能，使现有防御技术失效。为了有效地防御此类基于深度学习模型的WF攻击技术，我们提出了一种基于遗传算法的流量特征整形防御技术。通过对原始流量中填充少量有效伪元数据包，在不影响用户正常通信的前提下使得攻击者无法识别出用户正在访问的Web站点，达到保护用户通信隐私的目的。此防御方法利用了当前主流的深度指纹攻击中设计的卷积神经网络来提取流量中的高维度特征向量，根据目标保护站点与其他站点的流量特征向量间的距离，设计了变异方向控制机制和适应度函数以指导目标保护站点流量变异的方向，进而填充更少的伪元数据包，以减少带宽消耗。最终针对每个网站都找到一条成功逃逸深度学习模型的伪元数据包填充模式，并在封闭世界和开放世界验证了此防御方法的有效性和高效性。经过大量实验表明，在开放世界我们在只引入8.1%的带宽开销的情况下能将当前WF攻击的正确率显著降低到0.4%以下。


个人简介：

       凌振博士，教授，博导，国家优秀青年基金获得者，东南大学青年首席教授，于2005年和2014年分别获得南京工程学院计算机科学与技术学士学位和东南大学计算机应用技术博士学位。研究方向为网络安全、匿名网络、移动终端安全及物联网安全。2008年8月至2009年11月，在香港城市大学计算机科学系任研究助理参与网络安全相关研究工作，2011年8月至2013年8月访问加拿大维多利亚大学计算机系并从事相关研究工作。2014年任东南大学计算机学科与工程学院讲师，2017年和2021年分别破格晋升为副教授和教授并工作至今。近年来凌振博士发表学术论文60余篇，包括重要国际期刊，如IEEE/ACM ToN、IEEE TIFS、IEEE TPDS、IEEETC、IEEE TDSC等，以及重要国际会议，如ACM CCS、IEEE INFOCOM、IEEE ICDCS、ESORICS等。先后主持国家自然科学基金青年、面上和优秀青年基金项目，国家重点研发计划课题，江苏省自然科学基金青年基金项目和优秀青年基金项目等。此外，多次受邀在重要国内外学术会议作特邀报告，如国际著名黑客会议Black Hat上演讲2次等，相关研究成果还受到中央电视台科教频道CCTV10《走近科学》栏目采访在第二届国家网络安全宣传周系列节目中进行报道。获得2014年ACM中国优秀博士论文奖、2015年CCF优秀博士论文奖、2016及2020年黑客极棒竞赛（GeekPwn）优胜奖等奖项。
