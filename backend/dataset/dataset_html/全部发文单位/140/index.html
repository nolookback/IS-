学术讲座：人工智能时代科技人文命运共同体的构建
外国语学院　2024/12/16 11:29:00
  （钟沛基 2022280095）
讲座题目：人工智能时代科技人文命运共同体的构建

讲座时间：2024年12月27日(周五) 上午 9:30-11:30

讲座地点：汇文楼1200会议室

主讲人：王宁

主讲人简介：王宁，上海交通大学人文艺术研究院院长，文科资深教授；清华大学外国语言文学系知名学者特聘教授。2000年获国务院特殊津贴，2010年当选为拉丁美洲科学院院士，2011年入选教育部“知名学者”，2013年当选为欧洲科学院外籍院士。此外，还担任国家社会科学基金中国文学组会评专家，教育部知名学者奖励计划会评专家，中组部万人计划哲学社会科学领军人才和国家级名师会评专家等。学术兼职包括中国比较文学学会学术顾问，中国中外文艺理论学会副会长等。王宁教授著述甚丰，对现代性理论、后现代主义、全球化与文化问题、世界文学、翻译学等领域的研究尤其精深，除出版五部英文专著和二十多部中文专著以及500余篇中文论文外，还在四十多种国际权威刊物或文集发表英文论文150余篇，收录SSCI或A&HCI数据库论文110余篇，部分论文被译成意大利文、西班牙文、葡萄牙文、日文、德文、韩文、俄文、塞尔维亚文、法文、阿拉伯文等。多年来蝉联爱思唯尔中国高被引学者，2024年再度入选斯坦福-爱思唯尔发布的全球2%顶尖科学家（文学研究）年度影响力榜单和终身影响力榜单。

内容简介：在高科技日新月异的时代，随着越来越多的工作由人工智能来做，原有的人力之过剩已成为一个社会问题。人工智能的发展使得一些大学不惜裁撤一些学科专业，致使一些专业的毕业生在就业市场上丧失竞争力，甚至难以找到适当的工作。这显然与建设一个和谐和共同富裕的小康社会的治国理念不相吻合，更无法实现在国际上构建人类命运共同体的伟大设想。本文作者既热情拥抱科技同时又坚守人文立场，认为对于我们高校教师而言，我们的主要任务依然是培养掌握多学科知识并具有较强实践能力的高水平复合型人才，而人文学科的主要研究对象正好就是人，这样看来，构建人类命运共同体便应首先考虑科技与人文的通融以便率先构建一个科技人文共同发展的命运共同体。在作者看来，科技与人文不存在绝对的对立和对话关系，而是一种对立统一的辨证关系。作为两种不同的思维方式，人文思维与科学思维不同的方面主要在于，它所诉诸的并非是一致性，而更是个体的独特性和差异性。但这二者的对话和共融是可能的，也是十分必要的。特别是我们在构建人类命运共同的进程中，首先要构建这样一个科技人文命运共同体。
