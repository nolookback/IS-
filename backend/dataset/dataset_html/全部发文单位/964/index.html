学术讲座：深圳市新昊明科技有限公司林红—跨境电商供应链运作流程
经济学院　2021/11/24 9:38:00
  （钟沛基 2022280095）

讲座题目：跨境电商供应链运作流程

讲座时间：2021年11月25日（周四）19:00-20：05

讲座地点：四方楼北106

主讲嘉宾：林红  深圳市新昊明科技有限公司

主讲嘉宾简介：

　　林红，深圳大学物流管理学士，香港中文大学（深圳）供应链与物流管理高级管理人员理学硕士。从事跨境电商工作11年，具有丰富的跨境物流，供应链管理，数据挖掘与分析经验，实战功底深厚，属于跨境全能型人才。

讲座主要内容：

　　电子商务的概念，跨境电商供应链管理的阐述与流程分析，电子商务对供应链管理的影响，跨境电商现状与未来发展趋势，跨境电商的分类与优缺点分析，跨境电商的操作流程分析，具体分析跨境电商平台“亚马逊”的特点，跨境电商行业的岗位职能分析。

　　欢迎广大师生积极参加！
