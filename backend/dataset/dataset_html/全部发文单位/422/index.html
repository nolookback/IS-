  （钟沛基 2022280095）

报告主题：数智洞察——新一代人工智能技术在决策分析及学术研究中应用

主讲嘉宾：康哲 教授

报告主持：刘琼 教授

报告时间：2024年4月23日 (星期二) 下午14：00 -16:30

报告地点：深圳大学丽湖校区守慧楼（A6）-501

报告人简介：

深圳市智通国际创新研究中心创办人。深圳市高新技术产业政策专家顾问、政协咨询委员会委员。在高新技术产业领域从事投资管理、政府产业规划、企业运营工作20多年，研究方向是基于大数据、AI技术、GIS技术的产业大数据分析、产业发展规划、产业空间规划和公共政策制定应用。曾担任深圳市专家委秘书处副秘书长，作为深圳市两区两级政府的高新技术产业政策专家顾问，受各级政府部门委托，先后负责执笔起草编制《智慧深圳规划纲要》《智慧深圳建设方案》《深圳市航空航天产业发展规划》《中国制造2025深圳市行动计划》《深圳市新一代人工智能发展行动计划》《深圳市信息化“十三五”规划研究》《深圳市推动软件产业高质量发展的若干措施》《深圳市推动现代农业产业高质量发展的意见及若干措施》等一系列政府政策。也承担了深圳市委决策咨询委员会、深圳市府政研室、政协关于《深圳智能经济产业生态研究》、《深圳智能经济先导区发展策略》等若干深圳市年度重点课题研究。
