重庆大学新闻学院郭小安教授主讲题为“新媒体环境下舆论学知识体系的重构”的学术讲座成功举办
传播学院　2021/6/16 15:30:00
  （钟沛基 2022280095）

5月21日下午，重庆大学新闻学院郭小安教授受邀来到传播学院，主讲题为“新媒体环境下舆论学知识体系的重构”的学术讲座。本次讲座由传播学院副院长杨洸老师主持。

郭小安教授的主要研究领域为政治传播、公共舆论。他在《新闻与传播研究》、《传播与社会学刊》、《国际新闻界》、《现代传播》、《新闻大学》等刊物发表学术文章50余篇。此外，郭教授入选了第四批国家“万人计划”青年拔尖人才，并荣获了第六届新闻传播学学会杰出青年奖。

讲座伊始，郭小安教授向现场的老师与同学们推荐了和舆论学有关的核心阅读书目。在对书目的简要介绍和讲解中，郭小安教授分享了他对“乌合之众”、“公共领域”、“信息茧房”及理性与情感的关系等看法。

接着，郭教授通过案例阐述了舆论的内涵。他将舆论定义为特定群体或机构围绕公共议题以各种方法开展的关注、表达与聚集活动，并从舆论学的经典范式转向新媒体环境下的舆论学这个角度出发对舆论的内涵及其主体、客体、本体、数量以及质量等组成要素进行了梳理。他指出舆论的核心要素呈现主体多元性、客体泛公共性、本体多样性、数量多变性、质量多维性等特性。

最后，基于以上分析，郭小安教授讲解了重思后的舆论学知识体系。他将新媒体环境下舆论学研究的核心议题划分为三个维度：关注、表达与行为。同时，他指出围绕此核心议题可以展开多模态话语分析、群体心态分析、行为层面的结构化数量分析及舆论的质量分析。

讲座现场气氛热烈，现场的同学们就郭教授所讲内容提出了许多问题，郭教授耐心地一一作答。此外，郭教授与在座的老师们就讲座相关主题进行了更深层次的讨论。至此，本次讲座在大家热烈的掌声中圆满结束。
