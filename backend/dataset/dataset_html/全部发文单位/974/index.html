学术讲座：华南理工大学牛保庄教授—如何做好管理科学研究与社会服务
经济学院　2021/11/16 9:47:00
  （钟沛基 2022280095）

讲座题目：如何做好管理科学研究与社会服务

讲座时间：2021年11月19日（周五）10:40-12：00

讲座地点：明德楼402

主讲嘉宾：牛保庄教授  华南理工大学

主讲嘉宾简介：

　　牛保庄，华南理工大学教授、博导。国家杰青，教育部重要人才计划入选者，国家优青，广东省政协委员。在中国跨境供应链管理领域做出系列研究，已有11篇论文发表于国际顶级期刊，包括MSOM 2篇，POM 6篇，TRB 3篇，另有60余篇文章发表在Risk Analysis, EJOR, Omega等知名期刊。成果曾获第八届高等学校科学研究优秀成果（人文社会科学）二等奖、中国管理学青年奖，中国信息经济学优秀成果奖、广东省哲学社会科学优秀成果一等奖、二等奖等奖项。培养的博士生应届获聘海外名校教职，为华南地区首次。

讲座主要内容：

　　以最近发表的几篇UTD论文以及做过的教学、企业咨询和社会服务为例，对竞争与合作博弈研究如何与社会应用相结合进行经验总结。具体而言，针对中国合同制造企业转型升级过程中的若干供应链特征刻画、结构选择、结果解释、企业和政府政策咨询进行案例剖析，提炼出对管理学科国际顶级期刊上对供应链竞合博弈的论文质量要求，探讨学者服务社会的若干方式和途径。

　　欢迎广大师生积极参加！
