学术报告：海洋立体网络信息体系发展初探

主讲嘉宾：乔永杰 研究员 中国电科电科院

邀请人：张胜利 教授

报告时间：2021年12月15日（周三）下午4:00

报告地点：深圳大学沧海校区致信楼N710会议室

摘要：围绕海洋强国战略发展，分析了海洋信息化的三个挑战。按照海洋产业化发展要求，阐述了物质、能量、信息三位一体的海洋立体化信息网络发展理念。突出概括了水下感知通信的瓶颈，结合海洋治理，进行了案例阐述。

嘉宾简介：乔永杰，中国电科电科院研究员，长期从事情报指挥信息系统专业研究，参加与主持国家重点研发计划、预研及大型信息系统研发等多个项目，获得省部级科技奖励五项。

欢迎各位师生积极参加！
