嘉宾简介：


  1. 谢庆国

    谢庆国教授，中国科学技术大学电子工程与信息科学系、华中科技大学生命科学与技术学院、武汉光电国家研究中心教授，博士生导师。

    谢庆国教授专注于正电子发射断层成像（Positron Emission Tomography, PET）基础研究与仪器研制，历经理论创新、技术突破和系统创制的三阶段，开创了全数字PET这一重要国际新方向。作为发明人申请专利383件，已获授权168件，在等本领域权威期刊发表论文70余篇。

     

  2. 邱才明

    邱才明教授，国家重大人才工程A类，现担任华中科技大学电子信息与通信学院院长。

邱才明教授因在超宽带通信领域中的杰出贡献当选IEEE Fellow (2014 Class)。先后在大数据、物联网以及人工智能领域出版了英文学术专著3部，待出版专著1部，奠定了随机大数据理论及其在物联网以及人工智能等领域应用的理论框架。发表超过200篇学术论文，引用7100余次，获得8项美国及欧洲发明专利，参与15项3GPP与IEEE标准提案工作。承担国内外人工智能、物联网以及工业大数据等领域科研项目20多项。
