学术讲座：人工智能、虚拟现实技术结合问题导向学习法（PBL）在新加坡 高等教育中的实践与应用
机电与控制工程学院　2024/10/29 9:39:00
  （钟沛基 2022280095）

讲座题目：人工智能、虚拟现实技术结合问题导向学习法（PBL）在新加坡高等教育中的实践与应用

演讲嘉宾：姜力军 博士

讲座时间：10月 31 日（星期四） 10:00

讲座地点：深圳大学沧海校区致信楼S601会议室

邀 请 人：曹广忠 教授

讲座内容简介：

2023年，以ChatGPT为首的人工智能技术使教育工作者进入一种措手不及的状态。随着大型语言模型(Large Language Model)的成熟，衍生许多令人惊艳的全新应用，并带来了新的问题。生成式AI(Generative AI)有能力产生高品质的文字和语音、影片甚或程式设计，将重塑人类未来的工作、学习与生活。面对科技和社会的多重挑战，高等教育工作者面临新的压力和机遇。讲座将根据新加坡高等教育界在过去二十年全方位展开问题导向学习法（PBL）实践中积累的经验，结合近年来人工智能、虚拟现实技术最新成果，探索如何提高教师的工作效率，如何提高教学技能，如何提高学生的学习兴趣和主动性，如何利用新技术对问题导向教学法进行改进和提高等问题。

演讲嘉宾简介：

姜力军博士，IEEE高级会员，新加坡-中国科学技术交流促进协会（SCAAST）终身会员。先后任职于新加坡南洋理工大学（NTU）、新加坡国立大学（NUS）、新加坡国家科技局（A*Star）、新加坡国立共和理工学院(RP)，曾任IEEE新加坡教育协会主席、IEEE新加坡工业电子协会秘书长、国立新加坡理工学院技术中心主任、新加坡国家科技局研究员，已在国内外会议、期刊上发表论文、书籍和专利100余篇，并有4项科技产品授权执照。有20多年的科研、教学研究和国际教师的培训的经验。曾应邀在中国大陆、中国香港和中国台湾、美国、欧洲、日本、澳洲、新西兰、东南亚等地的高校和国际会议演讲授课，被新加坡海峡时报、联合早报、联合晚报、“今日报”(Today)、“我报”(Mypaper)亚洲新闻电视台(ChannelNewsAsia)、新传媒电视8波道(Channel8)、5波道(Channel5)、FM938Live(938电台)等国际媒体报道，荣获新加坡工程师设计(IESDesign，2011)金奖、李显龙智能国家科技奖（2022）、陈嘉庚发明奖（2011）、纽约国际广告科技创意奖（2012）等奖项。
