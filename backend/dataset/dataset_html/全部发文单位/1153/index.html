讲座摘要：

纳米材料在物理，化学，生物，医药等领域应用前景广阔，纳米材料的合成加工受到广泛的关注。报告主要介绍课题组近几年在纳米制造及其应用方面的研究成果，包括极端环境驱动的纳米材料，如纳米颗粒，二维材料等的超快速组装，及其在能源，催化，新型器件等方面的应用；用于研究表界面科学的冷冻电镜技术；以及新兴的钾离子能源存储。
