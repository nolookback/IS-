“新冠疫情中的虚假信息研究”学术讲座在传播学院举办
传播学院　2023/1/30 16:42:00
  （钟沛基 2022280095）

2022年12月15日美国密歇根州立大学彭泰权副教授受邀为传播学院研究生主讲了一场以《新冠疫情中的虚假信息研究：基于社会网络视角理解信念系统》为题的学术讲座。本次讲座由传播学院杨洸副院长主持。

彭泰权老师的主要研究方向包括计算社会科学、健康传播、政治传播和受众分析，并在相关研究领域发表了一系列有社会影响力的研究成果。彭老师通过本次讲座与大家分享其科研团队的最新研究成果。

讲座伊始，彭老师阐述了研究的源起。他指出虚假新闻一直都存在，新冠疫情期间人们正处于充满高度不确定性的特殊时期，这是研究虚假新闻的一个契机。随后，彭老师提出了三个问题：“虚假信息有何影响”“如果有负面影响的话哪些特定人群更容易受到影响”以及“如果特定人群更容易受到负面影响，那么我们能否自愈”来启发大家思考虚假信息的相关话题。紧接着彭老师从研究问题、研究设计和数据分析这3个方面详细讲解了自己关于虚假信息的研究。

在提问与探讨环节，彭老师耐心回答了同学们提出的相关问题，并与杨洸副院长共同讨论了虚假信息研究中的发现和思考，提出信念系统在其他研究领域运用的可能性。

最后，杨洸副院长对此次讲座进行总结发言，表示彭老师的无私分享和细致讲解能够对同学们未来的学术研究予以启发。参会同学纷纷表达了对彭老师的衷心感谢。至此，本次讲座圆满结束。
