学术讲座（二十八）：细胞增殖动力学的建立与发展
生命与海洋科学学院　2022/5/23 16:40:00
  （钟沛基 2022280095）

时间：2022年5月24日（周二）19:00-21:00 

地点：丽湖校区四方楼南109

主讲人：周际 董事长

　 　　 华瑞同康生物技术（深圳）有限公司 

主讲人介绍：

周际 董事长1993年毕业于武汉大学国际经济管理系国际金融专业，1997-2000年在瑞典卡洛琳斯卡医学院临床肿瘤中心胸苷激酶1（TK1）研究团队从事学习和研究工作，2002年8月至今，担任华瑞同康生物技术（深圳）有限公司董事长、总经理。华瑞同康生物技术（深圳）有限公司是一家承载了瑞典价值医学理念的创新型国家高新技术企业，致力于重大疾病风险评估技术研究与产品开发，是深圳市高新技术企业、国家级高新技术企业。华瑞同康成功研制了体可问血清TK1和组化TK1检测产品，在细胞增殖领域是全球首个获批临床应用的检测试剂盒，可提供从肿瘤早期风险预警、肿瘤预后评估、精准治疗动态监测、到复发/转移风险评估的肿瘤全程动态检测解决方案。

周际董事长目前兼任中国健康促进基金会肿瘤早筛细胞质胸苷激酶检测技术发展专项基金 副主任委员会、中国健康管理协会 理事、广东省食品药品审评认证技术协会 常务理事、深圳市生命科学与生物技术协会 常务理事等职务，发表学术论文50多篇，申请或授权国内和国际专利23项。
