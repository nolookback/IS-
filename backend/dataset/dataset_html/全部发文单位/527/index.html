【学术讲座】新型选择器技术及其神经形态应用

报告摘要/内容简介：

传统冯-诺依曼架构在处理数据密集型任务时面临“存储墙”问题，基于忆阻器等新型存储器的存内计算架构具有更高的能效比。其中，交叉阵列式存内计算芯片具有低功耗、低成本、CMOS兼容等优势，但潜行路径电流限制了其可扩展性。为解决这一问题，我们探索了忆阻器和选择器串联的1S1R交叉阵列，围绕选择器这一新型器件开展了一系列研究。

研究结果表明，基于金属绝缘体转变的VO2选择器具有光热效应，可用于光电传感但是开关比不足，限制了1S1R阵列的规模。金属丝选择器具有极高的开关比（>107），但银电极选择器在高温下自发团簇，铜电极选择器则需要很高的电铸电压且容易击穿。为实现热稳定性好且无电铸的选择器，我们首次提出采用铜银合金作为电极的方案，在此基础上实现的CuAg/SiO2/CuAg选择器具有高开启电流和无电铸等特性,并在400°C下展现出优良的热稳定性。基于合金电极选择器的64×64 1S1R交叉阵列能显著抑制潜行路径电流，提升突触计算的准确率。此外，我们还基于选择器实现了紧凑的LIF神经元，为突触-神经元的同质集成提供了一条新的技术路线，在高密度存储和神经形态计算领域有广泛应用前景。

 

 

报告嘉宾简介：

赵亮，现任浙江大学信息与电子工程学院百人计划研究员、集成电路先导技术研究所所长、睿科微电子首席科学家。本科毕业于清华大学微纳电子系，博士毕业于斯坦福大学电子工程系，长期从事先进存储器技术研发，在RRAM领域有超过15年技术积累。曾在多家知名存储器企业任研发骨干和高级管理职务，作为负责人成功研制国产化RRAM成套工艺和技术方案，进入中芯国际40/28/22nm技术路线图并大规模量产。目前，作为负责人主持浙江省重点研发计划项目1项、张江国家实验室重大项目课题1项。近年来，他的主要研究方向包括新型存储器和神经形态计算，取得的主要成果包括：实现了国际首创的基于嵌入式RRAM的新型显示驱动芯片，发明了平行激活多层字线的3D NAND神经网络计算新范式，发明了新型合金电极选择器并实现了大规模1S1R阵列和LIF神经元行为模拟。迄今为止，他已在Nature Communications，Nano Letters，EDL，APL等期刊和IEDM，VLSI，DAC，HPCA等顶级会议发表论文50余篇，获发明专利授权10余项。曾获评浙江大学启真优秀青年学者和2023年“中国芯”优秀技术创新产品奖，入选了浙江省海外引才计划创新长期项目。
