讲座内容简介：

（讲座1）英国伦敦大学学院（UCL）是国际著名大学，世界排名前十，在生物医学工程领域的成就享誉世界。本次讲座主讲人将介绍英国伦敦大学学院（UCL）的骨科生物工程和植入科学的学科研究进展，重点介绍基础研究到临床转化的过程，并提出应对骨科医学关键问题的创新解决方案。主要内容包括（1）人工关节的仿生设计与生物摩擦学和生物力学研究；（2）生物材料的创新与其在骨骼组织修复应用；（3）临床康复科学与工程；（4）人体生物力学；（5）临床组织工程与再生医药的研究。

（讲座2）随着人口老龄化加剧、慢性疾病发病率上升以及新兴医疗技术不断涌现，传统医疗器械已难以满足多样化、精准化的医疗需求，这些促使医疗器械领域积极寻求创新突破，产学研合作模式应运而生。产学研合作整合了高校与科研机构的前沿研究成果、企业的生产与市场推广能力以及医疗机构的临床实践经验与需求反馈。本次讲座将针对生物医用材料、植介入器械、家用诊疗器械领域，分享研究团队在产学研方面的实践、探索与思考。

（讲座3）在现代医疗领域，随着运动损伤病例的增多以及人们对术后功能恢复期望的提升，对于高性能医疗器械的需求愈发迫切。传统治疗手段在某些复杂伤病如韧带损伤的修复上存在局限性，难以完全恢复关节的稳定性与功能。在此背景下，医工融合成为推动医疗器械创新发展的关键路径，以人工韧带为典型代表的生物材料在医疗器械领域的应用得到广泛关注。本次讲座通过介绍医工融合视角下人工韧带研发相关内容，包括其临床意义、预后改进策略、“刚柔并济” 策略下的 PET 编织人工韧带研发成果，以及机器学习在工艺参数优化方面的应用，阐述从生物材料到医疗器械的转变及医工交叉到融合的过程。

（讲座4）3D打印个性化骨缺损植入体在匹配骨骼解剖结构，调节生物力学平衡方面具有独特优势，比传统的修复材料表现出更佳的骨整合及软组织结合能力。本次讲座基于组织器官解析与构建技术，解析骨骼系统的解剖形态、力学特征以及骨-植入物的相互作用，介绍自主设计的具有仿生微结构的3D打印个性化植入体，并展示临床前研究及临床试验研究的结果，为骨修复过程降低并发症的发生率，促进缺损骨骼的形态修复与功能重建提供参考。

演讲嘉宾简介：

Chaozong Liu（刘朝宗），1997年获吉林大学仿生学博士学位，2004年获英国纽卡斯尔大学骨科科学博士学位，牛津大学博士后。现任英国伦敦大学学院（UCL）外科与介入医学学部骨科生物医学工程专业教授、UCL 生物医学工程与人工关节科学研究中心主任、UCL 肌肉骨骼科学硕士课程负责人、英国皇家国立骨科医院客座研究员。研究方向主要包括：生物材料创新研究、骨科疾病治疗、骨骼组织修复与再生的高效医疗器械的研发等。刘教授围绕生物材料的创新和仿生技术原理，依托UCL骨科与肌肉骨骼科学研究所并结合英国皇家骨科医院的骨外科优势，开展人工关节的仿生设计、临床组织工程与再生医药、关节疾病的早期诊断与介入治疗，以及关节植入体表面修饰对提高其在体功能和服务寿命的作用与影响研究，获得来自英国、欧盟、中国等政府以及医疗企业的支持与资助。授权中国、美国和欧盟等专利多项，发表学术论文240余篇。
