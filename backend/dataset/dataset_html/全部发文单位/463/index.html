报告人简介：

李晓玺，现任华南理工大学食品科学与工程学院副院长、教授、博士生导师。主要从事淀粉功能化修饰及营养组分的活性保持及控制释放领域的科研工作。先后主持了国家自然科学基金区域创新发展联合重点项目、国家重点研发计划课题、武器装备预研基金等多项国家级、省部级科研项目。相关研究成果发表SCI收录论文100余篇，主编专著1部；获授权发明专利30余项；获得国家技术发明二等奖、广东省科学技术进步二等奖、中国专利优秀奖、第38届日内瓦国际发明展金奖、广东省教学成果一等奖等奖励。
