报告主题：阿兹海默症的系统生物学研究

主讲嘉宾：徐 鹰 教授

报告主持：倪嘉缵   院士

报告时间：2024年4月9日 (星期二) 下午14：30 -15:30

报告地点：深圳大学丽湖校区守慧楼（A6）-501

线 上：腾讯会议（801-616-994）

  阿兹海默症（AD）是一种复杂的、中枢神经系统的退行性疾病，有多种细胞及临床水平的特异表现，包括（1）认知功能障碍, (2) 脑神经细胞大量凋亡，（3）β-淀粉样蛋白在胞外聚集， （4） tau蛋白在胞内聚集，（5）胞内碱性化、及胞外酸性化、（6）神经细胞过度激活、但激活幅度下降，（7） 慢性发炎和氧化压力，及（8）一系列代谢重编程等。目前对于AD的起因及驱动力，还没有一个清晰的认识；世界上还没有可以有效减缓AD进程的药物。在这一报告中，我将讨论我们实验室通过AD大数据分析及在化学水平建模，来解析AD发生、发展的可能驱动力及主要机理的工作。这一模型将涵盖大部分AD的特异表现，包括以上列举的种种。

报告人简介：

徐鹰，男，教授，博士生导师，南方科技大学医学院讲席教授。计算系统生物 生物信息学者，优秀学者讲座教授，中组部千人计划讲座教授，AAAS Fellow, IEEE Fellow。徐鹰教授于1993年加入美国橡树岭国家实验室，参与过人类基因组测序计划，主要负责在基因组中搜寻编码蛋白基因的计算方法及软件实施，此外还从事过蛋白质三级结构预测的计算方法研发，以及微生物基因组的统计分析、代谢系统建模和功能预测。自2003年起任美国佐治亚大学“校董事会及佐治亚科协著名学者”讲座教授，同时任佐治亚大学生物信息研究所首任所长。

2008年至今，徐鹰教授主要从事肿瘤系统生物学研究，重点关注肿瘤代谢。徐鹰教授共发表学术论文近400篇，著作5部，总引用次数近20000次，H-Index=71，受邀于研究机构、大学和会议进行了超过300次演讲和报告。
