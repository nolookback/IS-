学术讲座：压电微机电--研究与进展
机电与控制工程学院　2021/9/7 9:33:00
  （钟沛基 2022280095）

报告人：董蜀湘/深圳大学特聘教授

报告地点：深圳大学致信楼S630（深圳大学机电大楼S630）

报告时间：2021年9月9日14:30

报告简介：不同于传统电磁原理的微机电，压电微机电涉及压电、铁电、磁电功能材料与以及它们的器件集成，属于材料、物理、电子、与工程跨学科综合性研究领域。本讲座将介绍我们在压电微机电领域里开展的开拓性研究工作，包括我们创立的压电超材料理论、压电微机电的“磁－弹－电”耦合等效电路方法；介绍电微马达工作原理、磁电耦合工作原理、以及我们在压电微机电领域的重要发现和重要进步，包括：电压增益、电流-电压转换效应；发明了世界上最小的压电微马达，研发出低温驱动器、高温压电马达与高温驱动器，具有微纳米分辨率的系列直线压电马达，压电-电磁双机理马达，以及压电微机电在空间领域的重要应用，（成功应用于墨子号卫星激光通讯（2018））。

 

【董蜀湘教授简介】1993年获清华大学材料科学博士学位后，留清华工作，历任清华大学材料系讲师、副教授；1997-1999年，董蜀湘访问新加坡国立大学；2000年至2008，先后受聘为美国宾州大学、美国弗吉尼亚理工大学任副研究员、研究科学家，研究教授。2008年5月至2021年3月，从美国Virginia Tech大学回国受聘于北京大学工学院材料科学与工程系特聘研究员、教授、北京大学终身教授。目前为深圳大学高等研究院特聘教授、博导。董教授在国际重要刊物上Science Advances, EES, Advanced Materials 等发表SCI文章200篇，SCI引用次数9400余次，拥有4项美国发明专利和30项中国发明专利。董教授曾获美国宾州大学2000年材料科学进步一等奖，曾获美国宾州大学2000年材料科学进步一等奖，2014年-2018年入选Elsevier物理与天文领域中国高被引学者和最具世界影响力的中国学者，2021年入选全球前2%顶尖科学家榜单。
