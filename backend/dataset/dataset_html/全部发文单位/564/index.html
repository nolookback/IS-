讲座题目：Dynamic Neural Representations in Human Episodic Memory

刘婧博士现为香港理工大学应用社会科学系研究助理教授。刘婧博士毕业于北京师范大学认知神经科学与学习国家重点实验室，其论文获评北京市优秀博士论文。随后在香港大学心理学系从事博士后研究，并获得香港研究资助局博士后奖学金。刘婧博士擅长使用颅内脑电/头皮脑电/脑磁图/核磁影像、睡眠标靶记忆激活等多种研究方法考察人脑记忆的动态神经表征以及基于睡眠的记忆调控。她以第一作者身份在PNAS, Science Advances和PLOS Biology杂志上发表文章。


讲座摘要：

记忆是人脑最高级的认知功能之一。经典的记忆模型指出记忆一般包含编码、保持、巩固和提取等不同阶段。早期研究认为记忆保持是通过部分参与编码的神经元的持续激活来实现，而记忆提取则是编码阶段神经表征模式的复现。这些研究都着重强调了编码阶段的记忆神经表征模式在保持和提取过程中的再激活。然而，近年来越来越多的研究证据提示，不论是在清醒状态下的记忆编码过程中，还是在随后的睡眠状态下的记忆巩固过程中，大脑不是简单地重复编码初期的神经表征模式，其对记忆的表征会随时间不断的发生变化。刘婧博士将在本报告中详细介绍不同记忆阶段人脑对情景记忆表征的动态变化及其在长时记忆形成中的作用。
