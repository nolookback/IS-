【学术讲座】自适应测量分配边缘多目标贝叶斯滤波器
电子与信息工程学院　2024/6/24 9:41:00
  （钟沛基 2022280095）

报告嘉宾：刘宗香 教授

邀 请 人：  张鹏副教授

时     间：   6月24日周一上午10：00-11：00

地     点：   汇星楼1504


报告人简介：刘宗香，教授，主要从事多传感器信息融合的教学和科研工作，近年来重点关注智能信息处理技术、多目标跟踪技术和多源信息融合技术的理论研究和应用研究。先后参加了“多传感器组网空情信息系统”、“多源空情综合应用系统”的研究工作，负责项目核心软件的开发与研制。主持国家自然科学基金面上项目2项，国防预研基金项目1项，深圳市科技计划项目2项。获军队科技进步一等奖2项、二等奖1项、三等奖8项。近年来在本领域国内外权威期刊或会议发表高水平研究论文80余篇。

报告摘要：多目标跟踪是用传感器的测量以获取各个目标在不同时刻的状态。在存在目标运动不确定、数据关联不确定、漏检、杂波和噪声的情况下，多目标跟踪问题是一个极具挑战的问题。本次报告主要介绍自己在多目标跟踪领域的研究成果：自适应测量分配边缘多目标贝叶斯滤波器。涉及测量分配边缘多目标贝叶斯滤波器，自适应的逻辑基轨迹起始技术和自适应测量分配边缘贝叶斯滤波器的算法实现。
