讲座摘要：

仿生材料通过模仿天然生命系统的独特结构或特殊性质来设计和合成工程材料，从而实现材料从结构到功能的系统性优化。在生物医学领域，对免疫系统的仿生能帮助人们认识免疫调节的基本规律，进而阐明免疫系统如何调控其组成单元的生理功能来维系人类健康。本次报告从中性粒细胞胞外诱捕网（简称NETs）的生理功能谈起，讲述仿生材料的构筑对于认识NETs的生理功能的重要意义。我们通过自下而上的方式构建NETs仿生材料“微网”，阐明“微网”组成-结构-性质-抗菌功能之间的基本构效关系。同时，当NETs发生浓度、成分、或结构的异常变化时，常造成自身功能失效并诱发多种免疫失调疾病。NETs的这类异常病变也可通过“微网”来仿生，从而建立NETs异常与其自身功能失效的关系模型，并为干预NETs介导的免疫疾病提供新思路。
