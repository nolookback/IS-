学术讲座（四十七）：IOP出版社论文投稿讲座：如何撰写及发表高质量论文
化学与环境工程学院　2021/5/25 10:00:00
  （钟沛基 2022280095）

主讲人: 郭晴霞博士

时间: 5月27日上午10:00

地点: 丽湖校区守信楼（原B1栋）420会议室

报告人简介：

郭晴霞，英国物理学会（IOP）出版社的出版人，在学术出版行业有8年从业经历，先后在Springer Nature，SAGE Publishing任职，熟知编辑出版流程，在学术期刊的出版以及发展方面有丰富的经验。现担任英国物理学会（IOP）出版社中国区出版人，负责英国物理学会在中国的期刊发展支持，加强英国物理学会与中国科研学术机构的合作，为中国的科研人员提供高质量的服务。

讲座摘要：

IOP出版社出版人将为大家介绍英国物理学会出版社的基本情况，并从国际期刊的视角为您详解编辑处理稿件流程，剖析论文的撰写、投稿以及发表每一个流程，以及投稿关键要点，帮助大家提期刊投稿成功率，同时分享论文发表后影响力提升的途径，主要包含以下内容：

• 英国物理学会（IOP）期刊介绍

• 如何选择适合的国际期刊？

• 什么样的文章会吸引编辑的目光？

• 学术论文写作过程中有哪些误区？

• 如何能使您的论文更加清晰易读，更易发表？

• 论文发表后提升影响力的途径？
