学术讲座——海洋“碳中和”及系统模型构建
生命与海洋科学学院　2024/11/21 15:31:00
  （钟沛基 2022280095）

时间：2024年11月26日 10:15

地点：丽湖校区 四方楼 西205

主讲人：姚泓名 博士

　 　　中国科学院深圳先进技术研究院 副研究员

姚泓名博士2019年于美国德州农工大学系统获得博士学位，入选深圳高层次人才C类。中国环境协会、中国药理学会海洋药物药理专业委员会委员，美国湖沼与海洋学会(ASLO)、美国河口海岸研究联合会(CERF)会员。主要从事近岸海域碳循环过程及相关机理的研究，包括多界面碳通量模型、海洋酸化、多源信息融合监测等。主持国家至地方层级科研项目共6项，发表包括Limnology and Oceanography、Water Research在内的海洋学领域主流学术期刊共计12篇,参与英文专著2部。研发的河口碳通量模型获得了美国航空航天局（NASA）的亮点报导。

 

 

此讲座由生命与海洋科学学院《海洋科学专题讲座》课程组织。

欢迎广大师生前来聆听！
