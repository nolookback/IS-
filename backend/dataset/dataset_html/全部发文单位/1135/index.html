孟涛，西南交通大学生物工程系教授，博士生导师，“微纳米生物技术”四川省技能人才创新工作室带头人，四川省学术与技术带头人后备人选。致力于生物催化、药物递送和功能材料研究。曾在美国哈佛大学工程与应用科学学院、国家生化工程研究中心、国家生物反应器重点实验室、四川大学从事研究工作。主持国家自然科学基金、省科技厅重点研发项目、企业合作课题10余项。近5年来以第一或通讯作者在美国化学会，英国皇家化学会/物理学会旗下权威期刊在内的学术杂志发表科研论文30余篇，出版专著1部等。受邀为多个国际学术刊物审稿。研究成果已申请/授权国家发明专利25/15项，并开展工业生物技术转化。中国化工学会会员，《日用化学工业》杂志编委，四川省科技青年联合会理事。
