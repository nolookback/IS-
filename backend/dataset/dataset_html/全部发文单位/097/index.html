学术讲座（七）：生物医用材料和护理及植介入类医疗器械的热点和难点问题
化学与环境工程学院　2025/2/28 17:13:00
  （钟沛基 2022280095）

主讲人：殷敬华研究员
时间：3月3日上午9:30
地点：丽湖校区守信楼420会议室 

报告人简介：

殷敬华，中国科学院长春应用化学研究所研究员、博士生导师。曾任威高集团有限公司总工程师、研究院院长。主要研究方向为生物医用材料和医疗器械。迄今共发表SCI论文300余篇，编著和主编《现代高分子物理学》和《高分子材料的反应加工》专著2部；获授权专利80多件，有50余件实现转化，实现直接经济效益100多亿元；创建了威高集团上海研究院。主要奖励和荣誉有：以第一获奖人先后获得国家技术发明奖二等奖和国家科技进步奖二等奖各一项，中国科学院、吉林省和山东省科技进步奖一等奖三项和威海市科学技术最高奖。
