【学术讲座】5G移动终端MIMO天线和毫米波相控阵天线
电子与信息工程学院　2024/6/1 0:06:00
  （钟沛基 2022280095）

讲座嘉宾：邓长江 副教授 北京理工大学集成电路与电子学院

邀请人：张晓 副教授

讲座时间：2024年6月2日20:00-21:00

讲座地点：深圳大学粤海校区致信楼N308会议室

 

讲座摘要：

移动通信的飞速发展驱动着手机移动终端天线技术的持续创新。在过去几十年，移动终端天线的形态发生了巨大变化，经历了从单个单元到多个单元和众多单元，从单频段到多频段和不连续的多频段，从固定单波束到可变多波束。为了满足大带宽和高容量的要求，MIMO技术和毫米波相控阵技术被应用于5G移动终端天线之中。本报告将介绍课题组在5G移动终端MIMO天线和毫米波相控阵天线设计方面的最新进展。其中，MIMO天线设计重点研究天线间的解耦，将介绍集总元件解耦和方向图分集解耦技术；毫米波相控阵天线设计重点研究端射双极化阵列，将介绍小净空和宽频带毫米波终端天线设计。


嘉宾简介：

邓长江，北京理工大学集成电路与电子学院副教授，博士生导师，IEEE 高级会员，毫米波与太赫兹技术北京市重点实验室成员，2011年在北京邮电大学获得学士学位，2016年在清华大学获得博士学位。研究方向为移动终端天线、毫米波相控阵天线、智能超表面等。以第一作者或通讯作者发表SCI论文40余篇，EI论文30余篇，学术引用1500余次，获国家发明专利授权16项，出版学术专著1部。承担国家自然科学基金，华为产学研合作项目等。担任IET Microw. Antennas Propag.副主编。曾获得清华大学一等博士学位论文、中国电子学会首届优秀博士学位论文、ICMMT会议最佳学生论文、EMTS会议和ACES-      China会议青年科学家奖、IEEE Trans. Antennas Propag.优秀审稿人。
