中国社会科学院新闻与传播研究所朱鸿军教授主讲题为“媒体融合的国家深意与主导路径暨国家社科基金项目申报辅导”的学术讲座
传播学院　2021/1/25 16:19:00
  （钟沛基 2022280095）

2020年12月24日，中国社会科学院新闻与传播研究所朱鸿军教授受邀来到深圳大学传播学院，为师生开展题为“媒体融合的国家深意与主导路径暨国家社科基金项目申报辅导”的学术讲座。

朱鸿军，博士、教授、博士生导师。中国社会科学院新闻与传播研究所研究员，《新闻与传播研究》执行主编。中国社科院创新工程首席研究员，中国社科院新媒体研究中心副秘书长，中国社科院大学新闻传播学院长聘教授。国家社科基金重大项目首席专家，国家新闻记者培训教材编写项目负责人，中国新闻史学会媒介法规与伦理研究会副会长，中国新闻史学会传媒经济研究会副会长，中国新闻史学会网络传播研究会常务理事。台湾政治大学、香港城市大学访问学者。主要研究方向为媒介融合、媒体版权、新媒体经济、学术论文知识生产等。

讲座伊始，杨洸副院长简要介绍了朱鸿军教授在媒体融合方面取得的成就并高度赞扬了他扎根实践、与时俱进的学术精神。接着，朱鸿军教授便从媒体融合的起源谈起，勾勒出一条完整的研究路径，并对学术前沿展开进一步探讨。朱教授指出，原本的媒介理论适用于传统媒体事件，在社交媒体的影响下，原有的理论解释在当下有很大的不适应性。由于新兴的商业媒体与传统媒体之间有很大的跨度，因此需要找一个过渡地带——传统媒体和新兴媒体融合的地带。

随后，朱教授还探讨了媒体融合的意义，他指出媒体融合不能简单地从行业里看，还需要站到国家层面进行分析。同时，媒体融合是系统的结构性转型，而不是局部的小范围转型。媒体是民意最重要的载体和传播者，在现实语境下，互联网开始崛起使得媒体在一定程度上起到了引导舆论的作用，如何正确引导媒体舆论影响党长期执政的安全性，切实推进治理体系、治理能力的现代化，这便要求政府需要有强大的信息搜集和处理能力，同时还需要有强大的社会动员能力。媒体融合不仅有理论意义，在国家政策的支持下，还具有很强的现实意义。渐进式的推动媒体融合，有利于迎合民意，进而把党的主张上升为国家意志。

而后，朱教授结合自身的学术路径进一步强调了调研的重要性。在调研的过程中我们可以学会如何在坚持自己观点的同时不与对方产生抵触，进而用自己的观点影响别人，由此得以顺利接洽。基于此，朱教授还强调到调研不仅有利于加强人和人之间的沟通和交流，实现思想的变通，而且也能进一步关切现实、体察民意、培养严谨的治学作风。同时，朱教授还提出学术研究要长期沉淀、不断积累，不能一蹴而就、投机取巧。

最后，朱教授还提到国家社科基金项目申报的相关问题，并从选题、前期成果、参考文献等角度给在场的师生做出详尽的指导。本次讲座在朱教授与在场师生的互动和讨论声中圆满地画上了句号。
