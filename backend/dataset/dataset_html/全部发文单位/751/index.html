学术讲座：空间分布孔径雷达目标探测技术
电子与信息工程学院　2023/2/16 15:13:00
  （钟沛基 2022280095）


演讲嘉宾：廖桂生教授、博士生导师，西电杭州研究院院长。

邀 请  人：黄磊教授

讲座时间：2月18日周六上午10：00-11：00

地      点：致信楼N710



摘要：

分布式（FBS）卫星系统是由物理上互不相连、共同协作完成同一航天任务的多颗卫星编队构成的空间系统，本报告简要讨论了由分布式卫星系统构成的分布孔径雷达技术工作原理方法、优势及存在的技术难点，重点研究了降低减低分布超大稀疏孔径雷达天线栅瓣的阵列及波形优化设计方法、阵列误差在线估计以及长时间相干积累补偿方法，展示了分布孔径雷达目标探测技术的优势。

 

 

报告人简介：廖桂生，男，1963年6月出生于广西桂林，1985年获广西大学数学专业理学士学位，分别于1990和1992年于西安电子科技大学获硕士和博士学位。现任西安电子科技大学教授、博士生导师，西电杭州研究院院长。

长期致力于雷达信号处理技术研究，尤其在阵列信号处理、空时二维自适应信号处理、运动平台雷达地面运动目标检测、小卫星分布式雷达信号处理、合成孔径雷达信号处理、智能天线技术等等方面开展了三十余年的研究。 获2008年度国家杰出青年科学基金、2004年入选国家人事部颁发的首批 “新世纪百千万人才工程国家级人选”，曾受聘教育部知名学者特聘教授，国家自然科学基金委优秀创新群体负责人、教育部知名学者创新团队“雷达信号处理”负责人。
