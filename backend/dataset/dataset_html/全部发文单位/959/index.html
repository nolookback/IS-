【基础医学院学术讲座】 合成生物学的发展与挑战
医学部　2021/11/25 11:11:00
  （钟沛基 2022280095）

主讲嘉宾：黄卫人 研究员

时间：       2021年11月26日上午 10:15

地点：       深圳大学丽湖校区四方楼北303

主持人：   郑　多 教授

主讲内容简介：

合成生物学是一门综合学科，意在以传统生物学获得的知识与材料为基础，利用系统生物学的手段对其加以定量的解析，在工程学以及计算机的指导下设计新的生物系统或对原有生物系统进行深度改造。

课程首先简单介绍合成生物学的理念、原则及发展历史，然后从合成生物学元件、线路和系统几个层次介绍合成生物学的发展方向，接下来重点介绍合成生物学的应用领域，最后简单说明合成生物学面临的挑战和机遇。

主讲嘉宾简介：

黄卫人，国家重点研发计划首席科学家，深圳大学第一附属医院（深圳市第二人民医院）研究员，国家地方联合肿瘤基因组临床应用关键技术工程实验室副主任，广东省泌尿生殖肿瘤系统与合成生物学重点实验室副主任，中国生物工程协会合成生物学专委会委员、中华医学会泌尿外科分会基础学组委员。

主持国家重点研发计划等科技项目10多项，在Nature Methods、NatureCommunications、GenomeBiology、Advance science、ACS SyntheticBiology等刊物发表学术论文100多篇，获授权国家发明专利14项，获深圳市自然科学奖二等奖1项（第一完成人），获中华医学奖青年科技奖1项（第一完成人）。
