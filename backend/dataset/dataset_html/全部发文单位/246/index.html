讲座名称：Generation, Functions, Computational Models, and Clinical/Subclinical Applications of Visual Attention




个人简介： 张喜淋，华南师范大学心理学院研究员，博士生导师，国家自然科学基金优青项目获得者，目前担任脑影像期刊Frontiers in Neuroimage for Cognitive Neuroscience副主编、eLife编委、国内心理学期刊《心理科学进展》和《心理科学》编委、中国认知科学学会认知与类脑计算专委会、中国认知科学学会意识科学专委会和中国心理学会普通心理和实验心理专委会委员。主要利用行为、脑成像、计算模型和人工神经网络研究视知觉、注意、意识、学习及工作记忆的神经机制。相关成果以第一或通讯作者发表在Neuron, PNAS, Current Biology, PLoS Biology, Cell Reports等学术期刊上。曾获广东省第九届哲学社会科学优秀成果二等奖、中国神经科学学会（CNS）葛兰素史克明日之星奖、美国心理科学学会（APS）学术新星奖、教育部高等学校科学研究优秀成果奖（科学技术）自然科学一等奖（第三完成人）。
