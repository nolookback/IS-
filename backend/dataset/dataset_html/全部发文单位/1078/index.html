学术讲座（四十五）：聚集诱导延迟荧光与多通道电输运

报告人简介：

赵祖金，理学博士，教授，博导。入选国家“万人计划”青年拔尖人才，广东省自然科学杰出青年基金获得者。英国皇家化学会会士、中国化学会高级会员。2018-2020年爱思唯尔高被引科学家。2003年毕业于浙江大学化学系，获理学学士学位。2008年获浙江大学化学博士学位。2008-2010年在香港科技大学唐本忠教授课题组进行博士后研究。2010-2013年在杭州师范大学材料与化学化工学院工作。2013年进入华南理工大学发光材料与器件国家重点实验室工作。主要从事有机光电功能材料与聚集诱导发光分子的开发及在OLED器件，单分子器件和生物成像与诊疗等方面的应用研究。已公开发表SCI收录的学术论文260余篇，包括Nat. Commun.、Angew. Chem. Int. Ed.、Adv. Mater.、Adv. Funct. Mater.、Adv. Sci.、ACS Nano、Nano Lett.等高影响力杂志，他引8000余次，h因子56。获授权发明专利28项。

讲座摘要：

通过分子扭曲结构调控、建立和集成独特光电机制，开发了具有高发光效率、高激子利用率和高效电输运特性的有机光电功能材料，在有机发光二极管（OLED）、单分子电输运器件等方面表现出独特性能。（1）制备了新型聚集诱导延迟荧光材料，在保证高固态发光效率的同时提高了材料的激子利用率，抑制了高电压下的激子湮灭。实现了高效率、低滚降非掺杂OLED器件的性能突破，解决了热延迟荧光材料效率滚降严重的难题；（2）建立了具有空间共轭特性的新型分子体系，为多维度能量和电荷输运材料提供了高效的构建基元。利用空间共轭骨架制备了新型单分子导线，提出了集成空间共轭通道和价健共轭通道的多通道电输运机制，构建了高性能单分子器件。
