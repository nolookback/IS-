深圳民族宗教研究基地学术讲座第十一场 神圣者与天命神圣原型的流变——以《黑神话：孙悟空》为例
人文学院　2024/11/14 10:09:00
  （钟沛基 2022280095）

主讲人：李向平 教授　（华东师范大学）

时 间：2024年11月22日　14：15

地 点：深圳大学粤海校区汇文楼H3-201

 

主讲人简介：

李向平，山东大学讲席教授，华东师范大学二级教授。国家社科基金规划项目会评专家，中国宗教学会、中国社会学会常务理事，国家社科基金重大项目《中国民间信仰研究》、《中国特色宗教社会学话语体系和本土知识结构研究》首席专家。主要研究领域：宗教社会学、宗教伦理学、道德社会学、中国思想史、宗教文化心态。出版学术专著二十余种，在《世界宗教研究》《社会学研究》等刊物发表专业论文200多篇，共计500多万字。

 

内容简介：

神圣、神圣者作为中国宗教学的基本范畴，似能突破西方“宗教”概念所形成的话语困境，进行文明互鉴和宗教学层面的跨学科研究。《黑神话：孙悟空》中的“天命人”现象，即是源自于传统中国天命原型的一个流变，体现为天命人即神圣者对神圣原型的一种激活，直接呈现为当下文化消费过程中的社会心态。
