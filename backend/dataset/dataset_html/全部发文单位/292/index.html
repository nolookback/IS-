嘉 宾 ：周家贤 教授 澳门城市大学

邀请人：张勇 教授

地 点 ：深圳大学汇星楼1502

 

摘要：

研究文章是學術和科研工作中常見的重要成果表達形式。高質量的研究文章不僅可以有效地傳播研究成果,也能提升作者的學術影響力。本次讲座交流旨在分享優秀研究文章的寫作技巧,包括:確定明確的研究目的和研究問題,構建論證框架;合理組織文章結構,包括文獻回顧、研究方法、結果分析等關鍵部分;運用生動形象的語言表述,提高文章的可讀性和吸引力;注重文章邏輯性和嚴謹性,確保論證過程合理有力;撰寫高質量的摘要和關鍵詞,方便讀者快速了解文章內容;合理使用圖表,以可視化的方式輔助論點闡述;審慎處理引用和參考文獻,確保文章的學術誠信。

 

嘉宾个人简介：

周家贤教授现任澳门城市大学亚太商业研究中心主任，曾担任多个学院管理职位（副院长及执行副院长），拥有领导及管理商学、旅游与管理学院丰富经验。他在进入学术领域之前，周教授拥有10多年在多个高科技公司担任高级管理人员经验。

周教授的研究领域涉及可持续发展、新能源、创新和运营管理等方面。他拥有2项专利，编辑了4本书，发表了120多篇学术和会议论文，其中超过80篇为SSCI/SCI收录期刊论文，41篇一区，17篇高被引文章以及2篇会议论文荣获2019年最佳论文奖。从2020年至2024年4月，他的论文共被引用4524次，h-index=38以及i10指数=73。
