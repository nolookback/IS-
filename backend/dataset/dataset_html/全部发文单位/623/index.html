“‘两个结合’的重大意义”学术讲座在马克思主义学院成功举办
马克思主义学院　2023/11/20 10:52:00
  （钟沛基 2022280095）

2023年11月18日下午，中国社科院金融研究所党委书记兼副所长、中国特色社会主义理论体系研究中心副主任、中国社会科学院习近平新时代中国特色社会主义思想研究中心执行副主任龚云，在深圳大学粤海校区汇文楼1600会议室为深大师生带来了一场题为“‘两个结合’的重大意义”的精彩讲座。深圳大学马克思主义学院党委副书记王萍、马克思主义中国化研究方向学科带头人王双印教授出席本次讲座。讲座由深圳大学马克思主义学院副院长张守奎教授主持。

龚云教授首先强调了大历史观之于把握问题的重要性。他指出，无论将来从事什么工作都要有宏观的历史视野，体现历史的纵深性，要深度了解历史，进而引出本次讲座的主题即为什么要提出“两个结合”以及习近平总书记提出“两个结合”的重大意义。龚云教授从三个方面对其进行深入探讨。一是要通过“两个结合”提出的宏阔历史背景把握文化无形的力量和重要性。龚云教授强调习近平总书记提出将马克思主义基本原理与中华优秀传统文化相结合，实际上就是要求把马克思主义放到中华民族五千多年的历史长河中去理解和定位。他认为，马克思主义只有变成中国人的思维方式，变成中华文化的内在构成，才能真正大众化，才能深入到老百姓的内心世界。二是要着眼于中华民族伟大复兴战略全局把握“两个结合”。龚云教授指出，民族复兴的本质就是中华文明的复兴。习近平总书记提出第二个结合，实际上就是站在中华文明复兴的高度呼吁把马克思主义与中华优秀传统文化相结合。马克思主义是一个伟大的认识工具，不应该停留于少数人，而要使其变成大多数人掌握的工具，变成他们的习俗和习惯。三是要明确“两个结合”的前提是不放弃马克思主义基本原理。理论创新的前提是“守正”，“两个结合”是中国共产党对马克思主义的“创新”，但这与中国共产党始终坚持马克思主义基本原理为基础是密不可分的。他指出，第一个结合体现的是马克思主义理论与中国革命、改革和建设时期的具体实际的关系，第二个结合体现的是马克思主义理论与中国文化土壤、中国人的风俗习惯的关系。最后，龚云教授分别从马克思主义发展史、社会主义发展史、中华民族发展史三个方面对“两个结合”的意义进行了总结。

    讲座最后，龚云教授与学院师生进行了热烈的学术交流讨论。张守奎教授对龚云教授的报告进行了总结，他指出这是一场既有深度又有高度的讲座，使我们受益匪浅。近现代史基本问题研究方向的徐贝贝同学就如何把鲜活的史料融入论文的文本写作提出自己的困惑，龚云教授指出史料是动态的，要通过研究证明共产党领导的正确性，即通过证明理论正确反过来推动理论发展；思想政治教育方向的周雨柔同学就如何理解“两个结合”与马克思主义大众化的关系和如何推进马克思主义大众化两个问题提出自己的疑惑，龚云教授指出第二个结合隐含了解决马克思主义大众化的问题，同时强调当前的思政课教学手段应服从于目的，大众化应代表大多数人的利益，解决大多数人的问题。

整场讲座反响热烈，进一步提升了师生们的专业素养，同时也营造出良好的学术氛围。在热烈的讨论氛围中，讲座圆满落下帷幕。
