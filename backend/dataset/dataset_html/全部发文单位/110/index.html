“人机传播对人际传播的影响”学术讲座在传播学院举办
传播学院　2025/1/8 10:26:00
  （钟沛基 2022280095）

2025年1月3日下午，一场主题为“人机传播对人际传播的影响”的学术讲座在传播学院致艺楼L7-813会议室举办。香港中文大学新闻与传播学院梁海副教授应邀担任讲座主讲嘉宾，传播学院副院长杨洸教授主持讲座，传播学院众多师生参加了此次讲座。

讲座伊始，梁海副教授指出当下人机交互领域的研究热度持续高涨，尤其是ChatGPT出现后，引发了研究热潮。他回溯过往，社交媒体研究因computer social science的兴起而产生重大变革，如今人机传播研究也呈现出多元趋势。一方面，主流研究多从人际传播理论切入，将ChatGPT等视为新型媒体，探究其对人类社会诸多方面的影响；另一方面，部分研究也将ChatGPT视为工具，用于提升效率、辅助研究，其提供的大量数据也成为研究新素材。

梁海副教授进一步深入探讨人机传播的本质影响。他强调了“human-AI coevolution”这一核心概念。他指出，机器与人类在交互中相互影响，ChatGPT依据人类反应训练，而人类在与机器频繁交流中，思维与表达也可能发生改变。通过对比人—人、人—机交流的实验数据，发现人在与ChatGPT对话时，语言生成模式会趋近机器，不过在长期使用过程中也存在一定的反弹现象，表明人类在一定程度上会保持自身的独立性与反思能力。

在互动环节，师生们积极参与讨论。其中，针对ChatGPT在心理咨询等领域的应用，梁海副教授从专业角度分析了其目前存在的局限性，如准确率不足和法律责任界定模糊等问题。他指出，培养专业心理咨询师仍是解决相关问题的重要途径。同时，对于不同聊天机器人及大模型的差异，他表示尽管存在一定区别，但在与人类的本质差异上具有共性。此外，他还提醒师生们“机器使人变懒”，不可过分依赖机器。

讲座结束后，现场师生反响热烈。此次讲座为师生们提供了全新的学术视角，有力推动了人机传播领域的学术交流与思想碰撞，引发了师生们对人机关系及传播研究方向的深入思考，在学术探索的道路上迈出了重要一步，也让人对该领域未来的研究充满期待。
