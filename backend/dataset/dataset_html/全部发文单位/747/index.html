学术讲座：超声调控玻璃形成微观机理的研究
机电与控制工程学院　2023/2/15 19:26:00
  （钟沛基 2022280095）

讲座题目：超声调控玻璃形成微观机理的研究

演讲嘉宾：尚宝双 特聘研究员

讲座时间：3月1日（星期三）下午 14：30－16：00

讲座地点：深圳大学南校区机电大楼S601会议室

邀请人：马将 老师

讲座内容简介：

　　超声作为一种新型的加工制造模式被广泛用于金属玻璃的焊接与热塑成型中。研究表明，金属玻璃在超声作用下表现出了良好的热塑性与可加工性，但相应玻璃形成能力往往变得恶化，使得超声作用下金属玻璃容易发生晶化及脆化等行为。因此，如何利用超声调控金属玻璃形成能力和理解其微观机理，具有重要的科学意义和广阔的应用前景。在本次报告中，报告人将介绍其在超声调控玻璃形成方面的工作进展，从理论分析的角度讨论超声作用对玻璃形成能力影响的微观机理，以及给出利用超声作用提高玻璃形成能力的可行性分析和具体地实现策略，为实验上更好地利用超声调控玻璃形成提供理论指导与参考。

演讲嘉宾简介：

　　尚宝双，男，松山湖材料实验室特聘研究员。博士毕业于中科院力学所，先后于北京计算科学研究中心、法国Liphy实验室从事博士后研究工作； 现为松山湖材料实验室卓越青年学者并入选广东省高层次人才计划青年人才项目；目前在PRL，PNAS，Acta Mater.等杂志发表论文二十余篇。主要利用分子动力学模拟从事金属玻璃及过冷液体的基础理论研究工作，研究方向为: 金属玻璃塑性变形及过冷液体玻璃形成微观机制研究。
