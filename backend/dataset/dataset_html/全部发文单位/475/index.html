深圳民族宗教研究基地学术讲座第二场：为什么总要追求完美？——基于神圣社会学的理论视角
人文学院　2024/3/25 10:17:00
  （钟沛基 2022280095）

主讲人：李向平 教授　（华东师范大学）

时 间：2024年3月29日　16：00

地 点：深圳大学粤海校区国学院会议室

 

主讲人简介：

李向平，山东大学犹太教与跨宗教研究中心教授，华东师范大学社会学系二级教授；中国社会学会常务理事，中国社会学会宗教社会学分会主任，中国宗教学学会理事。主要研究领域：宗教社会学、道德社会学、心态与社会变迁、神圣社会学。

 

内容简介：

中国道德的神圣起源，建构了天人之际、神人之间、家国同构以及个体与社会间的总体结果，提供了中国文化理想的完美原型。从“天生德于予”到“为天地立心”，从“宇宙便是吾心”到“不为圣人、便是禽兽”……，在很大程度上提供并强化了人们追求理想的道德动力，最终形成以完美为道德资质的神圣心态。千百年来，这种始终要追求完美的心态影响极其深远，甚至塑造了当代国民心态及其思维方式。
