报告人简介

张益茬博士，现任法国工程技术大学UTBM机械设计工程系教授(MCF,HDR）, 在法国科学研究中心（CNRS）交叉科学实验室ICB-COMM主要研究面向增材制造的设计与工艺规划基础理论，开发通用的增材制造集成设计与工艺规划专家系统。先后参与多项欧盟第7框架及地平线H2020研究项目，发表学术期刊与会议论文60余篇。2018年8月，被国际生产工程研究院（CIRP）授予青年研究(Research Affiliate)，2020年8月，晋升为通讯会员(Associate Member), 2021年4月获CIRP青年学术奖章（CIRP Taylor Medal）。
