【学术讲座】语音伪造与鉴别技术研究进展
电子与信息工程学院　2023/10/31 11:14:00
  （钟沛基 2022280095）

讲座嘉宾：武执政副教授 香港中文大学（深圳）

邀 请 人：李斌教授

时  间 ：11月2日上午10点-11点

地  点 ：致信楼N710

 

 

报告摘要/内容简介：近几年，深度学习技术在语音生成领域取得了巨大进展。最新的技术能够只用几秒的声音来克隆目标说话人的语音。虽然声音克隆技术有很多潜在的应用价值，但是语音伪造也给虚假信息、电信诈骗等方面带来了潜在的威胁。该报告将概述语音伪造技术的最新进展，并汇报鉴别技术的最新发展，特别是物理攻击检测方面的最新进展。

 

 

 

报告嘉宾简介：武执政博士现任香港中文大学（深圳）副教授。入选国家级青年人才，连续多次入选斯坦福大学“全球前2％顶尖科学家”、爱思唯尔“中国高被引学者”榜单。他于2015年获得南洋理工大学博士学位，并在Meta（原Facebook）、京东、苹果、爱丁堡大学、微软亚洲研究院等机构从事学术研究和技术领导工作。武博士带领开发了语音合成开源系统Merlin，发起并组织了第一届声纹识别欺骗检测国际评测（ASVspoof Challenge）、第一届语音转换国际评测 (VCC: VC Challenge)，组织了2019年语音合成国际评测（Blizzard Challenge 2019）。曾获得INTERSPEECH 2016最佳学生论文奖、2012年亚太信号与信息处理协会年度峰会最佳论文奖。武博士现为IEEE语音与语言处理技术委员会委员，语音领域权威期刊IEEE/ACM Transactions on Audio, Speech and Language Processing的Associate Editor, IEEE Spoken Language Technology Workshop 2024的大会主席。
