主讲人：董彦辉 博士

　 　　自然资源部第二海洋研究所海底科学实验室 副研究员

董彦辉博士2003年于中国地质大学（武汉）获得学士学位，2008年于中国科学院广州地球化学研究所获得博士学位。主要从事海底资源与成矿作用相关研究工作，专长为矿物学及地球化学，目前专注于海底锰的地球化学研究。长期从事多金属结核、富钴结壳的海上调查工作，曾获国家海洋局大洋科考先进个人。曾主持国家自然科学基金青年基金、面上项目、大洋十三五资源与环境专项课题，做为研究骨干参与多项重点研发计划的研究工作，累计合作发表论文60余篇。

此讲座由生命与海洋科学学院《海洋科学专题讲座》课程组织。

欢迎广大师生前来聆听！
