嘉宾简介

章俊华，毕业于北京林业大学园林学院，后留学日本。先后在日本千叶大学自然科学研究科取得硕士及博士学位，并于1998年回国任中国清华大学建筑学院副教授， 2004年赴日本任教千叶大学园艺学部。2006年4月以来，担当千叶大学研究生院园艺学研究科准教授、园艺学研究科博士生导师。中华人民共和国建设部风景园林专家委员会委员， R-Land 北京源树景观规划设计事务所首席设计师等。2006、2007年期间担任日本造园作品选集发行委员会委员，日本造园学会国际委员会委员，主要研究方向 ----“传统造园手法的空间表现及特征”，“以中日韩为中心的东亚地区环境文化学的比较研究”等。
