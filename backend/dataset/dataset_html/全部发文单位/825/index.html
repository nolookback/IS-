报告人简介：

肖桂山教授，俄罗斯工程院外籍院士，大连理工大学化工学院特聘教授，毒理药理学学科带头人，博士生导师。现任中国科技部癌症防控科技攻关专家组成员，中国医师协会临床精准医疗专业委员会首届委员。美国《Pancreas》（胰腺）杂志副主编，《Biomarker Research》资深副主编；科技部国际重大合作研究计划首席科学家，2016 年获国家科技进步突出奖及年度人物。主要研究领域为：“细胞代谢生物学”“癌症系统生物学”及“生物药物学”方向。在人类疾病（尤其是恶性肿瘤）早期诊断和新的治疗药物研发上已经取得了6项国际发明专利，并在精准诊断和新抗肿瘤药物研究领域展显出巨大的应用前景，在胰腺癌及乳腺癌领域的部分研究成果已经实现了临床转化。
