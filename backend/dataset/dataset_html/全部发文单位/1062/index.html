报告摘要：This lecture has particular relevance to research students, post-docs, and academic staff in science and engineering. The starting point of this lecture is that one has a set good results from recent research. This lecture will outline how to construct a paper – how to choose a title, how to write an abstract, how to write conclusions, how to presents results, when to start writing the paper, and many other things (e.g., references, which journal).

报告人简介： Asoke K. Nandi教授，英国皇家工程院院士，英国剑桥大学（三一学院）博士，美国电气研究所和电子工程师协会会士（IEEE fellow）。他曾在英国牛津大学，伦敦帝国理工学院，斯特拉斯克莱德，利物浦以及芬兰等担任教授职位。2013年起，在布鲁内尔大学担任电子与计算机工程系的系主任。
