题目：中国医学科学院阜外医院结构性心脏病团队介绍及超声相关项目汇报

主讲嘉宾：李慧

时间： 2023年12月15日（周五）上午 9:00 – 11:00

地点： 深圳大学丽湖校区生物医学工程学院A2-517大会议室

主持人： 薛武峰

主讲嘉宾简介：

李慧，副主任医师，医学博士，毕业于四川大学华西临床医学院，现就职于中国医学科学院阜外医院。担任海峡两岸医药卫生交流协会超声医学分会委员，中国心胸血管麻醉学会超声分会委员，中国超声心动图学会青年委员。从事超声心动图诊断及术中经食道超声监测工作11年，擅长各种先天性心脏病、瓣膜病、心肌病、心脏肿瘤及复杂疑难疾病的心脏超声诊断。持和参与多项国家级、省部级、院校级课题。研究方向主要为心力衰竭相关基础研究，成人心脏瓣膜病、先天性心脏病、人工智能领域。第一作者或通讯作者已发表论文20余篇，其中SCI 9篇。申请多项发明专利。

 

报告简介（Abstract）：

介绍阜外医院、结构性心脏病团队，超声影像中心临床及科研工作开展。对超声影像中心开展的一些临床和科研项目进行简要介绍和交流。
