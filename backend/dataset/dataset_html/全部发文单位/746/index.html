学术讲座：通过畸变晶格应力场设计多功能高熵合金
机电与控制工程学院　2023/2/23 7:52:00
  （钟沛基 2022280095）

讲座题目：通过畸变晶格应力场设计多功能高熵合金

 演讲嘉宾：杨勇教授 香港城市大学

 讲座时间：2月24日（星期五）下午 14：30－16：00

 讲座地点：深圳大学南校区机电大楼S601会议室

 邀请人：马将 老师

 讲座内容简介：

高熵合金由于其优异的机械、化学和物理性能，近年来引起了学术界和工程界极大的研究兴趣。在本次报告中，我会讨论我们组最近的一些研究工作，包括理论，实验和模拟，来试图解决一些本领域内的基础问题 (例如多组元合金的熵设计和晶格畸变)。通过大量的实验和模拟，我们发现通过仔细混合不同尺寸的元素，可以开发出多功能高熵合金，这些合金可以表现出独特的机械/物理性能，例如超强的强度、出色的弹性应变极限、超低机械耗散、显著的恒弹(elinvar)效应、优越的塑性等等。通过这些已有的发现，我们发现这些多组元合金的多功能性不仅可以归因于纳米尺度的化学波动，还可以归因于一个不均匀(或者异质)晶格应力场的存在，这两者互相耦合但都是传统合金所缺乏的。

 演讲嘉宾简介：

杨勇教授2001年获得北京大学学士学位，2003年获得香港科技大学硕士学位，2007年获得普林斯顿大学博士学位。现就职于香港城市大学，在工程学院机械工程系、材料科学与工程系等多个院系担任教授。他目前的研究兴趣包括纳米力学、先进合金（例如块体金属玻璃、高熵合金）和低维合金/陶瓷的设计、增材制造（例如粘合剂喷射 3D 打印）。研究成果发表在Nature、Nature Materials、Nature Communications、Science Advances、PNAS、Advanced Materials、Materials Today、Nano Letters等材料领域的期刊上。
