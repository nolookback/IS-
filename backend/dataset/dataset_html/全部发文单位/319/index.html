学术讲座：纳米多孔金属能源材料
机电与控制工程学院　2024/7/8 9:55:00
  （钟沛基 2022280095）

讲座题目：纳米多孔金属能源材料

演讲嘉宾：丁轶 教授

讲座时间： 7月9日（星期二）10 点

讲座地点：深圳大学南校区机电大楼S630A会议室

邀请人： 任帅老师

讲座内容简介：

  基于“脱合金”法制得的纳米多孔金属及相关复合材料，兼具低维纳米材料的高比表面和传统金属材料良好的导电导热及机械性能，是一类新颖的宏观尺度纳米结构材料。借助于先进的结构表征和表面功能化技术，我们将重点讨论这类材料独特的表界面特性以及它们在高效能源转化(催化、燃料电池)方面的应用潜力。

演讲嘉宾简介：

  丁轶，天津理工大学教授，1998和2001年毕业于中国科学技术大学并获得学士和硕士学位，2005年博士毕业于美国约翰霍普金斯大学；2005-2014年在山东大学工作，2015年调入天津理工大学参与筹建新能源材料与低碳技术研究院，2021年至今任院长。主要从事纳米多孔金属材料应用基础研究。入选英国皇家化学会会士、国家特支计划科技创新领军人才、国家百千万人才并获国家杰出青年科学基金资助；承担国家级科研项目20余项；发表学术论文120余篇，授权发明专利50余项，被引用19000余次，个人H指数78。
