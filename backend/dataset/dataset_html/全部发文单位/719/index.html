【深圳大学40周年校庆800场学术活动(第640场)】“像人类学家一样思考——田野调查方法谈”学术讲座在传播学院举办
传播学院　2023/4/26 15:03:00
  （钟沛基 2022280095）

2023年4月23日上午，云南大学郭建斌教授受邀为学院师生主讲了一场以“像人类学家一样思考——田野调查方法谈”为主题的学术讲座。本场讲座由传播学院李辉老师主持。

郭建斌老师现为云南大学民族学与社会学学院教授，博士生导师，复旦大学信息与传播研究中心研究员。承担过国家社科基金项目、教育部人文社会学科研究基地重点项目多项，在《新闻与传播研究》等刊物上发表论文30余篇，出版专著、参著、译著10余部。代表性成果有《在场：流动电影与当代中国社会建构》等。郭老师通过本次讲座与大家分享他对田野调查方法的研究心得与学术思考。

讲座伊始，郭老师从“田野调查（field work）方法的概念意义和田野调查应该去哪里”这一问题出发，分别从田野研究的进入、在“田野”中做什么、田野笔记的用处和田野研究论文写作这四个方面详细介绍了田野调查方法，结合他在“独乡”与滇川藏“大三角”地区三十余年的田野调查研究成果，为在场师生分享了他生动具象的思考。

郭老师指出，传统人类学研究的核心在于探寻“远方的文化之谜”。关于“本文化”与“异文化”之分的问题，郭老师认为，所有超越了文化边界的文化，均具有一定“异文化”的意义。跨越文化边界即为“跨文化”。而想要进入、融入田野，在行动上应与当地人“同吃同住同劳动”；在思想上，进行“主位”思考，站在当地人的立场上思考问题；郭老师还特别指出学习当地人的语言也是一种很好的调查方式。

随后，郭老师指出，在田野中要聆听与众不同的田野声音。“参与观察”是田野调查的核心，其重点在于“参与”，还要调动嗅觉、味觉、触觉等感官，获得切身体验，在观察体验之后再进行提问。郭老师强调，田野笔记可以驱动思考、整理研究思路并触发研究问题，帮助研究者回到当时的此情此景，在论文写作中，自我的田野感受是不能忽视的。

最后，郭老师通过“媒介的权力网络”这一操作化的概念和自身田野调查故事讲述如何凭借经验性材料启发研究论文的写作思路。郭老师提醒大家，人类学是人学，其核心在于对“底层”的人文关怀，这也是像人类学家一样思考最为重要的方面。

最后，李辉老师对本次讲座分享进行总结发言。他对郭建斌老师关于田野调查的研究分享为同学们拓宽视野、提供了宝贵的研究思路表达感谢。在场师生纷纷表达了对郭老师的感谢，本次讲座在掌声中圆满结束。
