报告人简介：

肖志刚，男，辽宁省特聘教授、粮食工程领域学术头雁，博士生导师。美国堪萨斯州立大学访问学者、密苏里大学高访学者。担任中国粮油学会食品分会副会长、东北稻米加工技术创新联盟专家委副主任、辽宁省高等学校科技创新团队首席专家等职。在植物蛋白和淀粉挤压加工及生物制造研究方面，积累了丰富的实践经验，是高水分植物组织蛋白食品的积极倡导者，在技术和产品开发利用方面有着一定贡献。主持国际合作、国家自然科学基金等各类科研项目42项，取得国际先进水平科技成果6项，发表高水平研究论文300余篇，出版论著及教材11部，授权发明专利32项（含2项国际专利），获辽宁省科技进步二等奖、黑龙江农垦总局科学技术一等奖等科技奖励12项。赴美国、韩国、土耳其及巴基斯等国参加IFT、AACC等学术会议并作专题报告10余次，培养硕、博研究生和博士后及国外高端学者117名。
