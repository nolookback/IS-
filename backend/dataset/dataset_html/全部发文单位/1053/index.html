主讲嘉宾简介：

宋海波教授，华西医院麻醉科围术期床旁超声专业组组长，麻醉学博士，硕士研究生导师，主任医师

学术任职：中国医药教育协会超声医学专业委员会麻醉超声学组主任委员；四川省计算机学会智能医学分会秘书长；中华医学会麻醉学分会超声学组学术秘书，超声医学专科能力建设项目专家委员会委员，中国医药生物技术协会心血管外科技术与工程分会常务委员。

主要研究领域：主要从事围术期智能医学平台建设及经食管超声模拟教学培训。主持国家级及省级项目3项，其中1项为国家自然科学基金面上项目，发表论文共10余篇，其中SCI共6篇，有4项国家发明专利项，2项实用新型专利，2015年获得省部级四川省科学技术进步奖三等奖，2016年超声可视化技术在围术期的创新应用及推广获四川省医学科技奖一等奖，2008年获临床创新奖一等奖。
