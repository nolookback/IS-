学术讲座（二）：离子液体强化分离新过程及应用
化学与环境工程学院　2023/3/7 16:34:00
  （钟沛基 2022280095）

主讲人：董海峰研究员

时间：3月8日下午14:00

地点：丽湖校区守信楼（原B1栋）420会议室

报告人简介：

董海峰，中科院过程工程所研究员，博士生导师，惠州市绿色能源与新材料研究院执行院长。入选中国科学院青年创新促进会会员、中国科学院关键技术人才，曾获美国李氏基金资助赴美访学。主持国家自然科学基金、中科院国际合作项目、国家重大研发计划子课题、国家科技支撑计划子课题等多项。在Chem. Eng. J., Green Chem.，Chem. Rev.等国际学术刊物上发表SCI论文72篇，申请发明专利48项，参与撰写学术专著4个章节；获得中国科学院科技促进发展奖、中国产学研合作创新成果优秀奖和中国化工学会科学技术奖一等奖等。

讲座摘要：

报告人主要致力于离子液体绿色化工新过程、新材料开发以及新型仪器装备研制工作，设计合成了系列离子液体，解决了离子液体规模制备纯化难和长周期使用后原位再生的关键技术难题；率先开展了离子液体流动-传递规律研究，提出了离子液体放大规律预测新模型，解决了基于传统分子介质放大规律不适用于离子液体体系的难题；研发成功了离子液体强化气体分离回收成套技术，并在企业推广应用，建成了多套中试和工业示范装置。
