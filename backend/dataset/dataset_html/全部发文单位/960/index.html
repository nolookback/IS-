讲座题目：数据驱动之上的教育理念与场景应用

主讲嘉宾：王刚博士（清华大学）

讲座时间：2021年11月25日（周四）下午15:30

讲座地点：深圳大学沧海校区致理楼L3-1115会议室

主讲嘉宾简介：

王刚，清华大学全球学校与学生发展评价研究中心副主任，平方创想创始人，方略研究院院长。国家高层次人才，入选“海聚工程”和“海英人才”，北京市特聘专家，中国教育智库与教育治理50人论坛成员，中关村十大海归新星，中关村十大文化与创意产业人物。

参与多项国家主管部门及清华大学的教育评价指标体系设计及高教评估工作，如双一流评估、学科评估、国际化评估、创新人才评估、教学评估和教师评估等。曾担任国家发改委互联网教育系统技术及应用国家工程实验室专家组成员，工信部中国电子技术标准化研究院数据标准与数据治理评审委员会专家，现代教育技术杂志编委。王刚获得美国北卡罗来纳大学管理学博士，统计和运筹学硕士。本科就读于清华大学基科班，获得数学与物理及经济学学位。

讲座主要内容：

本讲座讨论科技赋能教育中的“桥梁”问题，给出一种拆解和链接“教育理念-数据科学-工程技术-用户体验”的四轴对齐方式，并结合新时期高校智慧校园建设、教育大数据的场景应用、学生学情以及教学评价的具体案例给出落地路径。

欢迎感兴趣的师生参加！
