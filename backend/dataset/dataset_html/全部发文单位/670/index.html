学术讲座：非晶合金及其复合材料的多尺度形变
机电与控制工程学院　2023/10/26 17:21:00
  （钟沛基 2022280095）

讲座题目：非晶合金及其复合材料的多尺度形变

演讲嘉宾：董杰 副研究员

讲座时间：11月 6日（星期一）14 点

讲座地点：深圳大学南校区机电大楼S601会议室

邀请人：马将 老师

讲座内容简介：由于无序的独特原子结构，非晶合金及其复合材料展现出高强度、大弹性和高韧性等优异力学特性，极具工程应用前景和科学吸引力。但是，在不同空间尺度上，二者皆有着复杂无序的原子和组织结构，其均匀的宏观变形中隐藏着复杂的局域形变，对宏观力学性能具有决定性影响。寻找这种隐藏的局域形变，在理论和实验上都具有一定的挑战性。这里，本报告将主要介绍基于高能射线技术的应变测量方法，揭示非晶合金及其复合材料的多尺度局域形变图像和内在机制，助于其宏观力学行为的理解及性能调控。

演讲嘉宾简介： 董杰，松山湖材料实验室副研究员。2009年于山东大学力学系获学士学位，2016于中科院力学所所获硕士学位，2021于中科院物理所获博士学位，博士导师为白海洋和孙保安研究员，研究方向主要为非晶合金的多尺度力学行为及其结构起源和物理本质。2021年加入松山湖材料实验中子科学平台从事非晶复合材料的相变和增韧机制研究，合作导师为马东研究员。迄今在Nat. Phys., The innovation, J. Mater. Sci. Tech. 等期刊发表学术论文18篇（一作7篇），申请发明专利13项目（一作2项）。
