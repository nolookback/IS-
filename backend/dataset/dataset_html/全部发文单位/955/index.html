学术讲座：用极简技术让课堂有点酷
信息中心　2021/11/29 16:14:00
  （钟沛基 2022280095）

讲座题目：用极简技术让课堂有点酷

主讲嘉宾：焦建利教授（华南师范大学）及团队

讲座时间：2021年11月30日（周二）下午14:30

讲座地点：深圳大学沧海校区致理楼L1-508智慧教室

主讲嘉宾简介：

焦建利，博士，华南师范大学教授，博士生导师，未来教育研究中心主任。技术发烧友和英语发烧友，教育自媒体人，利用业余时间，经营个人学术博客“教育技术自留地”二十年，发表博文2000余篇，访客来自海内外。共发表论文180余篇，出版、翻译著作15余部，主持、参与省部级以上和国际合作项目15项，主要包括瑞典、美国等国际合作研究项目、国家和省部级研究项目。

讲座主要内容：

今天，大学的老师，如果不能用手机让学生参与到课堂教学之中，学生就会用手机忽略老师的存在。

在这样一个技术无处不在的时代，大学的课堂教学究竟怎样更加有趣、有效、高效？

在2个多小时的工作坊中，焦建利教授及其助手将会手把手，着眼于当前高校课堂教学的难题与困境，结合智慧教室和极简教育技术，使得每一位参与工作坊的老师，快速掌握课堂互动与参与式教学的策略与方法，从而使得课堂有点酷。

欢迎感兴趣的师生参加！
