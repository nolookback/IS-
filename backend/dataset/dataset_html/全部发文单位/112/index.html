学术讲座（一）：高性能水凝胶材料与器件

报告人简介：　 

吴子良，浙江大学求是特聘教授，高分子科学与工程学系副主任。2003年毕业于浙江大学，2006年于华东理工大学获硕士学位，2010年于北海道大学获博士学位，之后分别在多伦多大学、居里研究所、北海道大学从事博士后研究工作。2013年入职浙江大学高分子科学与工程学系，2015年入选国家海外高层次人才引进计划（青年项目），2023年获国家杰出青年科学基金资助。主要研究方向为高性能水凝胶材料与器件，包括玻璃态水凝胶制备及结构-性能关系、水凝胶可控变形与软驱动器设计。作为通讯或共同通讯作者在 Nat. Commun.、Adv. Mater.、J. Am. Chem. Soc.等期刊发表论文140余篇。

报告摘要：　 

发展软驱动器、软机器人等软体机器具有重大科学意义与应用价值。高分子水凝胶因其独特的智能响应性、与生物组织的相似性，成为构筑软体机器的理想材料。本次报告将汇报我们在高性能水凝胶设计制备、变形驱动功能调控等方面的研究进展。从研究凝胶黏弹行为入手，通过分子设计在网络中形成稠密缔合结构创制玻璃态水凝胶，进而阐明水凝胶玻璃态这一反常现象的分子机制。继而，基于屈曲变形双稳态特征，通过凝胶结构设计、溶胀动力学调控，实现了多稳态变形并拓展了构型控制方法。在突破力学性能、驱动功能的基础上，从材料向凝胶器件拓展，制备出高性能水凝胶驱动器及多模态水凝胶机器人，发现了动态协同新机制。
