主讲人：刘岗研究员

时间：3月15日下午14:00

地点：丽湖校区守信楼420会议室

报告人简介：

刘岗，现为中国科学院金属研究所研究员、所长。2003年本科毕业于吉林大学材料物理专业，2009年在中国科学院金属研究所获材料学博士学位，随后留所工作。长期致力于太阳能驱动光催化材料等研究，发表SCI论文200余篇，论文被SCI引用3.3万次，获授权专利36项。作为负责人承担了包括国家重点研发计划项目、国家973计划项目课题、国家自然科学基金委杰出青年科学基金、优秀青年基金、重点以及重点国际合作研究项目等项目十余项。曾获包括国家自然科学奖二等奖（第1完成人）、科学探索奖（新基石科学基金会）、中国青年科技奖、中国科学院青年科学家奖、国家级教学成果奖二等奖（4/15）等十余项奖励。入选国家“万人计划”青年拔尖人才及科技创新领军人才，全球高被引学者（2017-2023年），为英国皇家化学学会会士。兼任中国材料研究学会青年工作委员会，以及先进陶瓷分委员会副主任、中国可再生能源学会光化学专委会副主任，Wiley出版集团MetalMat、EcoEnergy副主编。
