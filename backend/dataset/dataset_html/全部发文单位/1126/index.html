报告人简介：

朱斌博士， 2020年加入东南大学能源和环境学院，任太阳能技术研究中心和储能联合研究中心首席科学家。1998-2018在瑞典皇家工学院任教授级高级研究员。研究方向主要包括纳米复合和能源材料/技术, 燃料电池和应用, 新一代能源技术：半导体基燃料电池和固体电池以及可再生能源等。1996以来主持瑞典国家国际合作基金委(STINT Fellow)、瑞典国家创新局、瑞典国家能源局、瑞典国家研究理事会和欧盟先进材料和燃料电池项目以及欧盟-中国研究网络www.nanocofc.com。多次在瑞典国家组织的国际专家评审为国际领先的研究地位。2013年入选湖北省百人计划，在新型燃料电池和绿色能源研究连续取得好的成果，多次在大学官网和国家自然科学基金委官网报导。在材料和能源等国际顶级期刊发表论文400多篇，主编Wiley出版社的固体氧化物燃料电池（从传统电解质到无电解质器件）一书，引用1万多次，H-因子52 （谷歌学术）, 引领国际燃料电池发展的新方向。发明与经典燃料电池不同原理的单部件燃料电池，Nature Nanotechnology选择作为研究亮点：“Three in One”三合一进行报道。开发了大量新型半导体离子功能材料，发展先进半导体离子燃料电池，申报并授权一批专利。曾获2018年可持续能源技术创新国际奖。朱斌带领的团队在该领域的研究获得中国国家自然科学基金委(NSFC)多个项目资助。连续6年（2014-）进入中国高被引学者榜单（能源类）。曾担任美国燃料电池联合公司董事和低温SOFC首席专家，瑞典全球燃料电池CTO十多年。现在为欧洲和中国多所大学和研究机构的特聘和讲座教授以及南京纳米科技和先进燃料电池公司董事会主席。曾担任多个国际研究机构的项目评审委员，包括欧盟、英国、瑞典、芬兰、波兰、加拿大、美国等。
