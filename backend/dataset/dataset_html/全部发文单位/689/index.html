刘学波，博士，教授，博士生导师，现任西北农林科技大学校长助理、国际合作交流处处长。入选国家“万人计划”科技创新领军人才、教育部“新世纪优秀人才”支持计划、科技部“中青年科技创新领军人才”计划，荣获中国食品科学技术学会科技创新奖杰出青年奖，享受国务院政府特殊津贴。担任国务院学位委员会第八届食品科学与工程学科评议组成员，中国食品科学技术学会常务理事、青年工作委员会副主任委员，陕西省食品科学技术学会副理事长，陕西省营养学会副理事长。主要开展食品分子营养与健康领域的研究工作。主持国家及省部级重大科研项目20 余项，以通讯作者在Cell Metabolism、Nature Communications 等期刊上发表SCI 论文150 余篇。
