讲座摘要：

试图呈现用作钠离子电池碳负极的理想碳结构模型——筛分型碳。讲缘起，从精确分离氧氮的碳分子筛谈起，对溶剂分子的筛分效应和钠离子的脱溶剂效应，是提高钠电池库伦效率的关键；论进展，订制筛分型碳，提出低电位平台的产生及延展策略——辅以界面工程，库伦效率可以够高，平台容量可以够大；话未来，筛分型碳为高比能碳负极带来无限遐想，不仅仅是钠电池…
