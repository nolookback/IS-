讲座题目：国家自然科学基金项目申请书设计要点与撰写注意事项

讲座时间：2023年3月5日（周日）9:00-11：30

讲座地点：明德楼1101会议室

主讲嘉宾：卞亦文教授 　　上海大学悉尼工商学院

主讲嘉宾简介：

　　卞亦文，上海大学悉尼工商学院教授、博导、副院长，入选国家级重大人才计划青年学者、爱思唯尔中国高被引学者（2020-2021）；主要研究方向包括平台运营管理与供应链管理。在国内外重要核心期刊发表论文60余篇，包括POM、DS、JR、NRL、EJOR与《管理科学学报》等。主持国家级与省部级科研项目7项，含国家自然科学基金重点项目1项；参与国家科技支撑计划项目等国家级重要项目3项；参与制定“电子商务模式规范”国家标准1项；获得省部级领导批示10篇，其中3篇获得中办采纳、中央主要领导人批示。担任中国“双法”研究会理事、管理科学与工程学会理事以及多个二级分会副理事长与秘书长等职务；担任4个国内外知名期刊编委。先后获得多项省部级科研奖励与荣誉称号，包括上海市优秀决策咨询成果二等奖、上海市教学成果一等奖与二等奖等。

　　欢迎广大师生参加！
