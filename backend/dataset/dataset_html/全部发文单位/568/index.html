和展望

时间：2023年12月14日 10:15

主讲人：庄晓珊 高工

深圳市海洋发展研究促进中心（深圳市海洋监测预报中心） 海洋预报员

庄晓珊高工自2014年硕士毕业于深圳大学生命科学学院，一直从事海洋监测、海洋预警预报和海洋科普等业务技术工作。近几年，主持或参与完成市(厅)级、省部级项目10余项，主要解决了海洋浮标自动监测、数据管理分析、海洋预警预报和赤潮监控预警等方面的关键技术问题，取得显著技术成果。主持或参与完成制定《海洋环境监测浮标运维管理技术规程》海洋行业标准和《海洋灾害预警信号发布规范》、《海洋灾害隐患排查技术导则》两个地方标准，以第一作者在核心期刊发表了多篇学术论文。

 

此讲座由生命与海洋科学学院《海洋科学专题讲座》课程组织。

欢迎广大师生前来聆听！
