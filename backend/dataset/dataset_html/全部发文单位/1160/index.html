武汉大学强月新教授主讲的“国家社会科学基金申报辅导”学术讲座顺利举办
传播学院　2021/2/18 11:26:00
  （钟沛基 2022280095）

2021年2月5日上午，武汉大学强月新教授受传播学院邀请主讲了一场关于国家社会科学基金申报辅导的讲座，此次讲座是深圳大学“人文社会科学讲学计划”系列学术讲座之一，采用线上的方式开展，吸引了传播学院和其他院系的老师们共同参加。强月新教授是武汉大学新闻与传播学院院长、教育部高等学校新闻学学科教学指导委员会副主任委员、中国高等教育学会新闻学与传播学专业理事会副理事长、湖北省新闻传播教育专业理事会理事长。先后主持过国家社会科学基金重大项目、重点项目、一般项目、教育部基地重大项目、省社科基金重点项目10多项，曾荣获国家级优秀教学成果二等奖、教育部优秀社科成果二等奖、省社科二等奖等多项奖励。与此同时，强教授在进入武汉大学工作前，曾在新华社广西分社、湖北分社从事新闻工作，是在业界和学界双栖的学者。

讲座伊始，传播学院副院长杨洸向在线的老师们介绍了强月新教授，并期待强教授“传经送宝”，解答老师们在项目申报过程中的困惑，助力老师们的课题申报。

强教授主要围绕项目选题和研究设计两大模块进行详细阐述。强教授建议项目申报者前期要仔细研究课题指南，关注国家提出的重大理论、方针政策，关注国家、区域或行业发展中的重点、焦点、难点问题，关注学科研究前沿动态与根本问题，并结合自己的研究专长，选择适合自己的研究选题。强教授指出研究选题要遵循四大原则：价值性、创新性、准确性、可行性。与此同时，选题要具备“问题意识”，比如关注国家关切的问题，实践亟需破解的难题，理论亟待突破的问题，突出有特色的问题。通过文献综述了解研究领域目前的研究水准，前人研究存在的问题，并将自己的研究与前人的研究做对比，从而帮助确定选题与证明本课题的研究价值。在研究设计部分，需要回答三个问题：1、研究什么？2、为什么研究？3、怎样开展研究？明确研究的问题，通过缩小问题的内容范围，对研究问题进行界定，清楚明确地陈述研究问题及其所具有的理论和实践意义。构建概念框架，使用适合研究具体问题的研究方法。强教授还列举了众多例子说明在国家社科基金项目申报过程应该注意的事项。

随后，在线聆听的诸位老师针对自己在国家社科基金项目申报过程中遇到的新问题等困惑咨询了强教授。强教授耐心地一一进行了解答。至此，本次讲座在老师们的答谢中圆满结束。
