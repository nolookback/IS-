主题：深圳，自然生态的本底与机遇

时间：2024年5月28日（周二）19:00-21:00　

地点：深圳大学丽湖校区四方楼南112

主讲人： 南兆旭 

深圳自然与历史研究者。深圳城市规划委员会委员。

主张“在地关怀”，关注、记录、保护深圳的自然生态，发起和组织了一系列自然保护活动，出版了一系列关于深圳的自然图书。

图书作品：

《深圳记忆》（2008 年）、《解密深圳档案》（2010 年）、《深圳自然笔记》（2013 年）、《南寻深圳》（2014 年）、《山水相望》（2016 年）、《南科大自然笔记》（2022年）、《深圳自然博物百科》（2022年）

纪录片作品：

《深圳民间记忆》（2008 年）、《梦开始的地方》（2014 年）、《深圳自然笔记》（2020 年）、《野性都市》（2022年）
