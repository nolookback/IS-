报告摘要：

高动态范围(High Dynamic Range, HDR)是我国4K超高清产业链中承上启下的关键环节。发展HDR技术不仅能满足人民日益增长的美好生活需求，还能促进我国消费电子市场升级，带动我国4K超高清产业快速发展。然而，HDR素材资源稀缺的短板极大制约了相关产业的发展，催生了对动态范围扩展算法的强烈需求。动态范围扩展算法旨在对低动态范围(Low Dynamic Range, LDR)图像的动态范围进行扩展，恢复在过曝、欠曝区域丢失的细节，拓展亮度信息灰阶，从而带给观众更加愉悦的视觉体验。现有的LDR图像数学模型无法准确模拟现有成像设备的成像特性，LDR图像过曝区域信息增益方法也尚不明确，最终导致动态范围扩展算法存在噪声敏感、过曝区域信息增益差及算法鲁棒性差等问题。这些瓶颈和难点问题制约着动态范围扩展算法产业落地的前景，也进一步影响了HDR技术的推广和应用。通过拓展相机响应函数数据集及研究LDR图像成像各个阶段噪声特性的方式，优化现有的LDR图像数学模型，完善动态范围扩展算法的理论基础；通过噪声热力图及注意力机制研究，进一步解决算法噪声敏感的缺陷；更进一步地，基于掩膜调制技术的过曝区域信息增益模型，利用过曝区域掩膜和高维语义特征，通过特征调制的方式，实现过曝区域的信息增益，突破现有算法的瓶颈问题；最后，利用生成式对抗网络强大的图像翻译功能，提升算法对真实图像的鲁棒性。

嘉宾简介:

刘康淋，鹏城实验室研究员（“鹏城孔雀计划特聘C类”）；主要从事计算机视觉、深度学习、生成式对抗网络（GANs）、神经渲染（Neural Rendering）、图像增强等领域相关研究。
