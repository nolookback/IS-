【学术讲座】染色质折叠的物理机制与机器学习
生命与海洋科学学院　2025/5/29 16:46:00
  （钟沛基 2022280095）

报告题目：染色质折叠的物理机制与机器学习

主讲嘉宾：黄恺 特聘研究员

报告主持：刘宏涛 特聘教授

报告时间：2025年5月30日（星期五）10：00

报告地点：深圳大学丽湖校区守慧楼A6-606

个人简介：

2009年本科毕业于清华大学工程物理系，2015年获威斯康星大学麦迪逊分校材料学博士学位，之后在美国西北大学生物医学工程系开展博士后研究工作。于2020年加入深圳湾实验室任特聘研究员，从事四维基因组、生物相分离、仿生智能体系设计等领域的理论计算。以第一作者或共同通讯作者身份在Nature、Cell Research、Science Advances、Nature Communications、JACS、ACS Nano、Cell Reports等知名学术期刊发表多篇论文。在染色质高级结构方面的研究成果被人类基因组计划前负责人Francis Collins评价为“人类三维基因组的新图像”。

报告摘要： 

作为遗传信息的物理载体，染色质的高级结构在基因表达、DNA复制等生命过程中发挥着关键的调控作用。与蛋白质不同，染色质的折叠具有显著的热力学非平衡特性。其高度复杂且动态的构象为实验表征带来了巨大挑战，高分子理论与模拟因而成为不可或缺的研究手段。本次报告将介绍理论计算在染色质折叠领域的最新进展，从结构模型、动力学模拟和机器学习三个方面展开。在染色质高级结构方面，我们提出了三维森林模型，成功解释了实验观测到的染色质反常高分子标度性质。在折叠机制方面，我们通过计算机模拟揭示了黏连蛋白两个亚型在染色质折叠过程中的互作规则。在环挤压机制外，我们提出了动态相分离驱动染色质多体互作从而调控基因表达的高阶催化模型。最后，我们利用机器学习预测基因表达调控中的顺式调控元件，展示人工智能在解码生命语言中的巨大潜力。
