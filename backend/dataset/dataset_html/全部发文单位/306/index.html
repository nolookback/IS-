学术讲座（五十五）：二维与三维液滴操控技术在功能器件中的应用
化学与环境工程学院　2024/7/16 15:23:00
  （钟沛基 2022280095）

主讲人：冯文骞研究员
时间：7月19日下午15:00
地点：丽湖校区守信楼420会议室

 

报告人简介：

冯文骞，四川大学高分子科学与工程学院研究员。于德国海德堡大学获得博士学位，德国马普医学所与美国劳伦斯伯克利国家实验室从事博士后研究工作。2020年9月加入四川大学，组建聚合物功能微器件研究课题组，组着力于界面材料的功能化设计，研究液滴在固-液界面、液-液界面的操控方式，实现功能微器件在化学与生物分子的高通量筛选与检测、植入式芯片等领域的应用。课题组成立后，以独立通讯在Nature Communications、Advanced Materials、Angew. Chem.等期刊发表论文多篇。

讲座摘要：

具有区域功能化的薄膜器件在生物诊断技术、微化学合成分析以及微电子器件等领域均具有广泛的应用。报告人将介绍利用固体表面的图案化技术以及液-液界面的组装技术，实现对液滴在二维与三维空间的操纵，以此制备具有界面区域功能化的微型器件，并将该类器件向化学与生物检测集成芯片的方向推进。
