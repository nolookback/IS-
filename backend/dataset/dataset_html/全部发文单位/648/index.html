讲座摘要：

固体氧化物燃料电池/电解池（SOFC/SOEC）作为一种高效的能量转换与储存装置，其商业化发展面临的一大挑战就是热机械不稳定性——电池在热循环中容易发生开裂、分层、破损等热机械损伤。其中一大原因是常用经典钴基钙钛矿空气电极较高的热膨胀系数（TEC）远远大于电解质，这可能导致电池空气电极与电解质这一关键功能界面处发生分层或断裂，因此解决热膨胀不匹配的问题是推进SOFC/SOEC生产应用的迫切需求。针对钙钛矿材料本身存在的与电解质不匹配的热膨胀特性，导致在工作条件下空气极材料热稳定性较差的问题。本工作将含有稀土元素的负热膨胀材料复合钙钛矿氧化物作为空气极，研究其相结构，热膨胀变化以及电化学性能，开发出一种活性高、热匹配性好的新型空气极材料。我们选取了负热膨胀（NTE）材料Y2W3O12（YWO），与ORR活性较高但稳定性差的钙钛矿空气极进行复合。NTE材料的加入，使复合材料的热膨胀系数（TEC）显著降低，可以达到与电解质的TEC值高度匹配的程度。本工作还重点研究了复合材料的相结构和反应机理，探究其优异ORR催化活性的形成原因，从而找到一条设计高活性、高稳定性材料的新方案。该工作成果克服了固体氧化物电池电极-电解质界面热膨胀系数不匹配的难题，实现了燃料电池空气极与其他电池组件之间的完全热机械兼容，也为高性能、高稳定性SOFC/SOEC空气极开发提供了一种全新的思路。
