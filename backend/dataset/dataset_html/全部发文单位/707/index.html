【深圳大学40周年校庆千场学术活动 (第987场)】“数据：平台资本主义与人的异化之间的媒介——读《现实的中介化建构》”学术讲座在传播学院举办
传播学院　2023/6/29 14:30:00
  （钟沛基 2022280095）

2023年6月13日晚，南京财经大学新闻学院副教授刘泱育受邀为传播学院师生作了一场以“数据：平台资本主义与人的异化之间的媒介——读《现实的中介化建构》”为主题的学术讲座。本场讲座由传播学院李明伟教授主持。

刘泱育，现任南京财经大学新闻学院副教授，研究生导师。获江苏省“333”工程高层次人才、江苏省“青蓝工程”优秀青年骨干教师、南京财经大学首批青年拔尖人才等荣誉称号。主持多项国家级课题，在《新闻与传播研究》、《国际新闻界》等中文权威期刊发表论文多篇。

在伦敦政治经济学院 (LSE) 媒体与传播系访问期间，刘泱育老师与媒介化研究的领军人物尼克·库尔德利 (Nick Couldry)教授合作，借此契机，他翻译了库尔德利所著的《现实的中介化建构》一书。本次讲座上，刘老师从译者视角出发，分享了他关于此书的心得和思考。

讲座伊始，刘泱育老师以何道宽教授的译著《理解媒介》中的经典观点“媒介即信息”作为引入，援引了库尔德利“因为数据，我们从使用媒介，转变为媒介使用我们”的观点，提出数据是理解当下的媒介环境最为重要的切入点。通过梳理库尔德利学术生涯的不同研究阶段，刘老师发现库尔德利的学术研究主题在2010-2013年之间发生了“数据转向”。循着曼海姆（Karl Mannheim）的“知识社会学”、伯格（Peter L. Berger）和卢克曼（Thomas Luckmann）的“语言社会学”脉络，刘老师提出，可以将库尔德利的研究命名为“数据社会学”。随后，刘老师将其研究放入批判理论的视野下加以考察，分别从空间、时间、自我、集体、秩序五个层面深入剖析了数据化时代日常的媒介实践对人的异化。

最后，刘老师与大家分享了自己的媒介断连经验和一些实用的读书方法，勉励大家在数据化时代要增强主体性，以好书充实自己，做“think differently”的人。李明伟教授对此次讲座分享做总结发言，并与刘老师一起就参会同学们的提问展开对话，同学们听完两位老师的回答纷纷表示受益匪浅。至此，本次讲座圆满结束。
