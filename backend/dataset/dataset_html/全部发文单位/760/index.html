“算法、外卖和田野”学术讲座在传播学院举办
传播学院　2022/12/14 12:50:00
  （钟沛基 2022280095）

11月30日，中国社会科学院新闻与传播研究所助理研究员孙萍受邀为传播学院师生带来一场以“算法、外卖和田野”为主题的学术讲座。孙萍老师是研究算法与劳动方向的学术新秀，发表了一系列有社会影响力的研究成果。本次讲座由传播学院丁未教授主持。

孙萍老师以“算法技术如何中介和影响平台化的外卖劳动？个体劳动者如何感知算法技术？”等问题为中心，从字节化与平台经济下的“陌生相逢”、外卖场里的“游戏人生”、算法区隔与社群文化的分裂、当人工智能变成“人工智障”这几个方面详实地分享了自己对外卖与算法的研究发现和思考。

　 在提问与探讨环节，孙萍老师与丁未教授就田野调查的经历以及过程中遇到的困难展开了深刻且风趣的对话，并对大家关心的一些问题作出了针对性答复，并介绍了西方学界关于平台算法的研究新趋势。

　 传播学院丁未教授最后进行了总结发言。她对孙萍老师生动、细致的讲授表示了肯定与感谢，认为孙萍老师关于算法与平台领域的独到见解为同学们拓宽了思维和眼界。参会师生也纷纷表达了对孙萍老师的衷心感谢。至此，本次讲座圆满结束。
