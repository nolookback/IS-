【学术讲座】植物细胞内免疫受体的酶学和离子通道活性研究
生命与海洋科学学院　2024/11/27 10:04:00
  （钟沛基 2022280095）

报告题目：植物细胞内免疫受体的酶学和离子通道活性研究

主讲嘉宾：万里 研究员

报告主持：刘宏涛 教授

报告时间：2024年12月2日（星期一）上午9：30

报告地点：深圳大学丽湖校区守慧楼A6-501

主讲嘉宾简介：

   

    万里，研究员，博士生导师，中国科学院分子植物科学卓越创新中心。2014年博士毕业于澳大利亚昆士兰大学，2015年至2020年在美国北卡罗莱纳大学教堂山分校从事博士后研究。2020年5月起，任中国科学院分子植物科学卓越创新中心课题组长/研究员。

万里课题组专注于植物免疫机制的研究和广谱抗性农作物的创制。在植物细胞内免疫受体NLR工作的分子机理方面取得重要进展，率先揭示了植物NLR受体的两个重要生化功能：NAD+降解酶活性和钙离子通道活性，并实现了NLR介导的ETI抗性的人工激活。相关成果以第一作者（含共同）发表在Science（2014，2019），以通讯作者（含共同）发表在Nature (In press),Science (2021，2024)、PNAS(2023)、Science Advances(2023)、JIPB（2024）等学术期刊。

报告摘要:

    植物细胞内的免疫受体NLR特异性识别病原菌分泌的效应蛋白而激活ETI免疫反应。ETI免疫反应强烈且能赋予植物对病原菌有效的抗性，是农业上抗病育种的重要靶标。含有TIR结构域的NLR作为感知病原菌的sensor被激活后作为NAD+降解酶产生信号小分子。在TIR-NLR下游，ADR1 和NRG1作为helper NLR执行免疫功能。我们发现TIR-NLR产生小分子2’cADPR作为信号小分子前体激活ADR1介导抗性。2’cADPR处理植物即可诱导类似于ETI的免疫反应。因此我们的研究实现了植物ETI免疫反应的人工激活，且2’cADPR有望开发成为新型“生物农药”。此外，我们的研究还揭示了TIR-NLR激活NRG1介导细胞死亡的分子机理。这些研究极大推进了人们对植物免疫系统的认知。
