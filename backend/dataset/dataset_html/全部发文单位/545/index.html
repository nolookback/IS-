讲座题目：多材料3D/4D打印: 从水凝胶到陶瓷

演讲嘉宾：葛琦教授

讲座时间：2023年12月27日（星期三）10点

讲座地点：深圳大学粤海校区汇星楼1010（增材制造研究所）

邀请人：陈张伟 教授

讲座内容简介：

4D打印作为一种新型的增材制造技术，通过将智能材料打印成三维结构，并借助外界刺激使得三维结构的形状随着第四个维度时间发生改变。将智能材料与弹性基底复合多材料打印，可以拓展4D打印结构的设计空间，实现比单材料4D打印结构更为丰富功能和更为强大的功能。但是，多材料增材制造在装备、材料、设计上仍有具体的发展空间。本次报告将主要围绕多材料增材制造装备、材料、设计开发上的取得进展展开，并介绍其在4D打印方面的最新应用。

 演讲嘉宾简介：
