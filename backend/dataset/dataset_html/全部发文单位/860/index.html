学术讲座：中山大学夏南新教授—国际金融市场波动非线性因果性检验
经济学院　2022/6/13 9:17:00
  （钟沛基 2022280095）

讲座题目：国际金融市场波动非线性因果性检验

讲座时间：2022年6月16日（周四）19:30-21：00

讲座地点：腾讯会议（会议号：314-912-140）

主讲嘉宾：夏南新教授　　中山大学

主讲嘉宾简介：

　　夏南新，江西省南昌市人，经济学博士，依次任教于中山大学岭南学院、马克思主义学院，教授职称，博士生导师，博士后导师，享受校特殊一线岗位人才保健。1997年9月-2000年6月,厦门大学统计学专业博士研究生，师承“中国GDP之父”钱伯海统计学泰斗。2014年当选中国统计学会常务理事；2016年、2020年连续当选广东省统计学会副会长；2019年当选中国数量经济学会常务理事。主持2005年国家社科基金一般项目，并以题为《关于防范和遏制地下经济对国家经济安全影响的对策建议》的研究报告，被中央宣传部全国哲学社科工作办编入2009年第12期国家社科基金项目《成果要报》，上报给中央政治局委员，中央书记处书记，国务院总理、副总理。2009年12月，中央宣传部全国哲学社科工作办向广东省委宣传部、中山大学党委发来《关于中山大学夏南新同志研究成果受到有关领导和部门重视的通报》。主持2015年国家社科基金重点项目“现代地下经济实证研究”。在理论经济、应用经济两个一级学科中，这是广东省唯一获得立项的重点项目。

讲座主要内容：

　　针对人民币、欧元、日元兑美元的官方汇率，选取了金融危机前后各三年的数据参照对比分析，计算三者汇率的均值方程的残差之间的交叉相关（Cross-correlation function，缩写为CCF），利用Yin-Wong Cheung, Lilian K. Ng检验统计量和Hong Y.检验统计量，验证三种汇率的标准残差的方差之间的非线性因果关系。在中外文献中，该文率先详尽地利用了检验统计量验证变量序列之间的非线性因果关系。在Hong检验中引入了截断核函数，使得对低阶时滞项赋予了较大权重，从而准确地刻画近期波动对当前波动影响更大的特征。
