报告人简介：

张华，1992和1995年分别获南京大学学士和硕士学位，1998年获北京大学博士学位。1999和2001年分别赴比利时鲁汶大学和美国西北大学从事博士后研究。2003和2015年分别在美国NanoInk 公司和新加坡生物工程与纳米技术研究院工作。2006年加入新加坡南洋理工大学材料科学与工程学院任助理教授，分别于2011、2013年晋升为副教授、教授。2019年加盟香港城市大学化学系，现任胡晓明讲座教授（纳米材料）。 张华教授课题组目前的研究聚焦在纳米材料相工程（PEN）、精细多级结构的可控外延生长等；主要包括以下几个方面：超薄二维纳米材料（如金属纳米片、金属硫族化合物、石墨烯、金属有机骨架、共价有机框架等）、新晶相的金属和半导体纳米材料、新型无定形纳米材料，及其多功能纳米复合材料的制备，以及在催化、清洁能源、光电器件、纳米与生物传感、环境水污染处理等方面的应用研究。
