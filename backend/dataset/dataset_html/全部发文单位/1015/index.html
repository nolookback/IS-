学术讲座：智能机器人及其应用研究
计算机与软件学院　2021/10/15 11:14:00
  （钟沛基 2022280095）

时间：2021年10月18日15：30 -17：30

地点：计算机与软件学院大楼（致腾楼）938

讲座摘要

智能机器人是《中国制造2025》和《新一代人工智能发展规划》的核心领域，是新一代人工智能的重要载体，已上升为国家战略，习近平总书记指出，机器人是“制造业皇冠顶端的明珠”，智能机器人的发展已经成为衡量一个国家科技水平的重要标志。图书盘点是图书馆诸多业务中的第一要务，这个问题长期以来一直依赖于人工干预，是图书馆工作中最耗时、最繁杂的一项业务。陈力军教授团队率先研发了全球首台“智能图书盘点机器人”，实现了高效、安全、精准的全自动化图书盘点，彻底解决了图书盘查困难的痛点问题，目前智能图书盘点机器人已在全国数十个省市30余家图书馆落地应用，累计帮助上千万人次寻找图书。研究成果可广泛应用于大型仓储管理、零售业务追踪、军需物资盘库、危险化学品监管和战略物资储备等诸多领域。

主讲人简介

陈力军，男，南京大学计算机软件新技术国家重点实验室教授，博士生导师，南京大学智能机器人研究院院长。科学研究始终以国家重大战略需求为导向，坚持以智能机器人为核心的“基础研究→创造技术→成果转化”科研实践，致力于移动机器人、物联网感知、计算机视觉和人工智能等领域的研究，在IEEE/ACM Transactions on Networking、IEEE Transactions on Mobile Computing和USENIX NSDI、USENIX ATC等国际一流期刊和一流会议上发表多篇学术论文。成功研发全球首台“智能图书盘点机器人”——南大图客，各项核心技术均处于国际先进水平，彻底解决了千百年来图书馆找书难的问题，在海内外引起了广泛社会反响，受到央视（CCTV）、江苏卫视和湖南卫视等媒体的专题报道。目前，研究成果已在南京大学、武汉大学、中国农业大学、香港中文大学（深圳）、同济大学和天津中新生态城图书档案馆等30余家图书馆落地应用，并成功延伸至大型仓储管理等其它领域。荣获“第46届日内瓦国际发明展”最高奖项——特别金奖，“第22届中国国际工业博览会高校展区优秀展品”最高奖项——特等奖，相关国家标准“图书盘点机器人通用技术条件”（编号：2020102880）通过立项。
