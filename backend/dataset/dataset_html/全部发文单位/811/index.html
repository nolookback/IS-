学术讲座——AI Plankton 爱浮游
生命与海洋科学学院　2022/10/25 15:46:00
  （钟沛基 2022280095）

时  间：2022年11月2日 19:00

地  点：丽湖校区 四方楼 北415

主讲人：李剑平 博士

 中国科学院深圳先进技术研究院正高级工程师

 

 李剑平，中国科学院深圳先进技术研究院正高级工程师，中国科学院大学博士生导师，香港浸会大学物理学博士。李剑平于香港浸会大学高级光学仪器研究实验室取得博士学位，后继续受聘于香港浸会大学物理系，先后担任项目助理、讲师、研究助理教授。2016年加入中国科学院深圳先进技术研究院，任光电工程技术中心副研究员。2017年入选中国科学院BR计划、深圳市海外高层次人才、地方级领军人才、南山区领航人才。李博士的研究兴趣和领域是创新光学方法、先进光电仪器、机器视觉与机器学习。先后主持和参与了科技部国家重点研发计划、国家自然科学基金、中国科学院、香港大学教育资助局(RGC)、深圳市科技创新委等研究项目14项。带领团队研制了水下浮游生物成像仪、走航式浮游植物成像流式细胞仪、海水叶绿素a、COD、BOD传感器等多种海洋观测探测仪器。在IEEE Journal of Oceanic Engineering，Optics Letters, Optics Express、Applied Optics等知名期刊发表论文多篇，申请和授权中国发明专利和实用新型专利38项，在Ocean Optics, International Ocean Color Sciences, ICCV，ECCV等知名国际会议做主旨报告、口头报告多次。李博士是国际电子电气工程协会IEEE高级会员、美国光学学会Optica会员、国际光电工程师协会SPIE会员、国际海洋技术学会MTS会员、中国仪器仪表学会高级会员、中国海洋与湖沼学会海洋观测分会理事、深圳市人工智能学会会员、广东省自然资源厅赤潮专家库专家。

此讲座由生命与海洋科学学院《海洋科学专题讲座》课程组织。
