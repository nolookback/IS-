学术讲座：深圳市新南山控股（集团）股份有限公司卢忠宝—战略管理的理论、方法与实践
经济学院　2021/10/26 9:47:00

讲座题目：战略管理的理论、方法与实践

讲座时间：2021年10月29日（周五）16:00-17:30

讲座地点：四方楼北211

主讲嘉宾：卢忠宝  深圳市新南山控股（集团）股份有限公司

主讲嘉宾简介：

1、华中科技大学西方经济学博士，出版专著一部，核心期刊发表论文多篇，并在大型国有企业集团20多年宏观经济、产业经济、区域经济、企业管理等领域研究工作经验。

2、10年以上企业经营管理经验，多年担任上市公司高管，并兼任华南建材、赤湾胜宝旺、新都工业园等南山控股旗下多家企业的董事长，长期从事战略管理、投资及信息化管理工作。

3、有15年投资、并购及资本运作工作经验，参与并主导多个重大资产重组项目，在股权投资、产业基金、产业园区投资及开发等领域有着丰富的实践经验，对不动产投资及REITS等领域有丰富的理论和实践工作经验。

讲座主要内容：

1、现代企业的组织架构特点及委托代理理论概述，现代企业的战略管理理论流派、主要观点及评析。

2、以战略地图、平衡计分卡为主要工具的战略管理的体系构建与工具运用。

3、以房地产、物流企业为例，介绍如何运用战略地图、平衡计分卡等工具对公司战略进行制定、描述和管理，以及战略地图工具在投资决策方面的应用。

　　欢迎广大师生参加！
