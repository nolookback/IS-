学术讲座：数据驱动的非晶形成能力研究与电子实验记录本的开发应用
机电与控制工程学院　2023/11/20 15:52:00
  （钟沛基 2022280095）

讲座题目：数据驱动的非晶形成能力研究与电子实验记录本的开发应用

演讲嘉宾：李明星 副研究员

讲座时间：11 月22 日（星期三）15 点

讲座地点：深圳大学南校区机电大楼S601会议室

邀请人：马将 老师

讲座内容简介：

  非晶与高熵合金是典型的多组元材料体系，给予了我们无限的造梦空间但也存在巨大的挑战。在成分复杂和工艺敏感的层层迷雾中，如何寻找材料的构效关系是新材料设计的关键。我们以材料基因工程为基本理念，设计开发了高通量实验方法，以快速、集成化的方式产生高度可比的实验大数据，发掘出可量化非晶形成的实用判据。结合分子动力学模拟，研究了原子团簇特征对非晶形成的影响，为开发高性能非晶合金奠定了理论和实验基础。我们也认识到数据对复杂体系的研究十分重要，为此我们以电子实验记录本为工具，建立了从个人实验到集体数据的优化流程，并结合机器学习推出了非晶合金预测模型。本报告将介绍数据驱动的研究案例，以及电子记录本的开发过程和使用方法。

演讲嘉宾简介：李明星，中国科学院物理研究所副研究员，博士生导师。目前主要从事非晶、高熵合金等多组元合金的高通量实验和数据驱动的交叉研究，以此探索材料的化学组分、微观结构和物理性能的关系。相关成果发表在Nature，Nature Materials，Advanced Materials等期刊，入选2019年中国科学十大进展，2021年中科院数据科学十大进展。先后获得“博新计划”，国家优秀青年基金等项目资助。
