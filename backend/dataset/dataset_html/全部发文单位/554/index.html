【学术讲座】浅谈嫦娥工程中图形图像智能计算
电子与信息工程学院　2023/12/15 15:58:00
  （钟沛基 2022280095）

讲座嘉宾：蔡占川 教授 澳门科技大学

邀 请 人：李斌 教授

时    间：12月18日下午2点15

地      点：致信楼N710

 

报告摘要/内容简介：利用中国嫦娥系列探月卫星获取的探月图形图像数据，介绍嫦娥工程中图形图像的智能计算问题。包括利用嫦娥一号卫星激光测高数据来构建高精度的月球数字高程模型的智能计算问题；采用嫦娥二号卫星四个频段月表微波辐射亮温数据构建全月亮温图的智能计算问题；使用嫦娥三号卫星CCD立体相机获取的影像数据拼接得到月面巡视区的大视场全景图像智能计算问题。对嫦娥系列探月卫星获取的探月图形图像数据的智能计算与应用进行有意义的探索。

 

 

报告嘉宾简介：蔡占川教授现为澳门科技大学计算机科学与工程学院助理院长、教授、博士生导师；中国科协海智计划特聘专家、澳门工业与应用数学学会副会长；主要研究领域为：智能信息处理、计算机图形图形处理、多媒体信息安全等。已在IEEE Transactions on Pattern Analysis and Machine Intelligence、IEEE Transactions on Image Processing、IEEE Transactions on Industrial Informatics及中国科学等发表学术论文百余篇；授权美国等国际发明专利逾10项；主持国家自然科学基金与澳门科技发展基金联合项目、澳门科技发展基金项目、国家高新技术研究计划863项目子课题等项目多项。曾获国家教学成果奖二等奖、澳门自然科学奖及技术发明奖、澳门科技大学“中银学术研究优秀奖”
