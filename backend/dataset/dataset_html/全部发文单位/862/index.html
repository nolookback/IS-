【学术讲座】面向超导量子计算的大规模微波测控系统
电子与信息工程学院　2022/6/14 9:23:00
  （钟沛基 2022280095）

报告人：钟有鹏 副研究员（南方科技大学）

邀请人：姜梅 副教授

报告时间：2022年06月15日 　　 下午14:30-15:30

报告地点：深圳大学沧海校区致信楼N710


报告摘要：量子计算是当今科技和产业界最热门的话题之一，其中以约瑟夫森结为核心元件的超导量子电路，在近年来的量子计算竞争舞台上大放异彩，获得各大研究机构和科技巨头如IBM，谷歌，阿里巴巴，腾讯，华为等的关注与持续投入。用于量子比特控制和测量的微波电子系统是超导量子计算体系的核心之一。随着超导量子处理器朝着大规模，容错量子计算方向发展，其微波测控系统也相应的面临新的挑战。本报告将介绍超导量子电路的基本原理和近期进展，并探讨面向超导量子计算的，基于FPGA和高速DA/AD电路的大规模微波测控系统架构。

报告人简介：

钟有鹏于2013年获得浙江大学学士学位，2019年获得芝加哥大学博士学位。2019-2020年留在芝加哥大学继续从事博士后研究，于2020年11月加入南方科技大学量子科学与工程研究院，主要研究超导量子计算，目前任副研究员，硕导，博导。2021年获得国家海外高层次人才青年项目和深圳市国家级领军人才支持。共发表SCI论文24篇，包括3篇Nature、Science；其中5篇一作，包括1篇Nature，1篇Nature Physics，1篇Nature Communications，2篇Physical Review Letters。
