学术讲座--海洋原位观测技术在致灾生物聚集爆发监测预警的应用
生命与海洋科学学院　2023/11/28 16:43:00
  （钟沛基 2022280095）

报告题目：海洋原位观测技术在致灾生物聚集爆发监测预警的应用

报告人：蔡中华 研究员

主持人：张俊彬 教授

报告时间：2023年12月5日（星期二）下午15：00

报告地点：深圳大学丽湖校区守慧楼A6-501


报告人简介：

蔡中华 清华大学深圳国际研究生院海洋工程院研究员，博士生导师。长期从事海洋生态的研究，近年来主要围绕海洋生物原位观测，微生物生态及藻菌关系等方面开展了大量的研究。针对海洋生物观测的难题，组织研发了国际先进的浮游生物原位成像仪，并在海洋生态调查、核电冷源预警和渔业资源评估等方面进行了广泛的应用。在海洋生态领域，围绕富营养化生态修复和藻菌关系等，从微生物生态系统发生，到微生物对藻类生态调节，藻类的规模化培养以及贝类控制富营养化等方面都积累了较多的研究经验。主持、参加并完成多项国家和省部级课题，获得多项奖励，在国内外期刊发表研究论文上百篇，申请并获得发明专利多项。
