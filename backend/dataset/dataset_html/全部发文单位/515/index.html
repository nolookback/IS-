【学术讲座】我国候鸟迁徙与禽流感主动监测
生命与海洋科学学院　2024/1/18 15:04:00
  （钟沛基 2022280095）

报告主题：我国候鸟迁徙与禽流感主动监测

报告时间：2024年01月16日（星期二）上午9:30 -10:30

报告地点：福田红树林保护区管理局101会议室

主讲嘉宾：张国钢研究员 全国鸟类环志中心

报告主持：周海超副教授

线上会议：腾讯会议号558-434-393

现任中国林业科学研究院森林生态环境与自然保护研究所研究员，全国鸟类环志中心副主任，兼任中国鸟类学会委员，空军后勤部鸟撞防范专家，东北亚禽流感专家组成员等。主持东北亚环境合作机制下韩国亚太经济和社会委员会《东北亚候鸟栖息环境保护和恢复》国际项目，以及亚洲开发银行“同一世界 同一健康”理念下《湄公河流域生物安全防护》国际培训项目；主持科技部、国家自然科学基金项目和国家林业和草原局等50余个国家和省部级项目。作为主要人员编写《云南夜间迁徙鸟类研究》、《中国大陆野生鸟类迁徙动态与禽流感》和中国工程院重大咨询项目《中国养殖业可持续发展战略研究（动物疫病防控卷）》等专著。

目前主要围绕濒危鸟类保护、鸟类迁徙生态学和鸟类禽流感生态学传播机制等方面开展了多方面的研究。研究成果揭示了鸟类迁徙中重要的时空转换节点在病毒重组变异和扩散中起着“催化剂”的关键作用；发现了2020-2022年期间全球快速传播的H5N8亚型高致病性禽流感病毒源于欧洲。现已获得授权国家发明专利10余项，先后在国际和国内期刊发表论文120余篇。
