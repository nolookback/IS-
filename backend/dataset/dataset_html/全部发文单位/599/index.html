主讲人简介：

刘放桐，著名哲学家，复旦大学文科资深教授。1957年初入中国人民大学哲学系作西方哲学史专业副博士研究生，1961年初毕业后到复旦大学哲学系任教授、硕士生导师、博士生导师至今，培养过博士生50余名。曾担任国务院学位委员会哲学学科评议组成员（第三、四届），第一届高等学校哲学学科教学指导委员会副主任，全国高等教育自学考试指导委员会哲学专业委员会委员，中国现代外国哲学学会副理事长、顾问，复旦大学杜威与美国哲学研究中心主任。被国内十余所大学聘请为兼职教授（或名誉教授、顾问、学术委员）。主要代表性著作有：《新编现代西方哲学》《马克思主义与西方哲学的现当代走向》《西方近现代过渡时期的哲学》《探索、争鸣和超越：马克思主义哲学和现代西方思潮比较研究》等。
