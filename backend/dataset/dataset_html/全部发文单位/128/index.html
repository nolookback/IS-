学术讲座（九十六）：手性烯烃配体促进的不对称卡宾转移反应
化学与环境工程学院　2024/12/20 19:23:00
  （钟沛基 2022280095）

主讲人：徐明华教授
时间：12月23日下午16:00
地点：丽湖校区守信楼420会议室

报告人简介：

徐明华，中科院百人计划、国家杰出青年科学基金获得者。博士毕业于中国科学院上海有机化学研究所，曾任中国科学院上海药物研究所研究员、课题组长、博士生导师，新药研究国家重点实验室研究员。2018年受邀加入南方科技大学，现任化学系和深圳格拉布斯研究院双聘教授。在国内外学术期刊发表论文150多篇，申请专利27项（已授权20项）。曾获中国化学会青年化学奖(2005)，首届药明康德生命化学研究奖(2007)，中国科学院朱李月华优秀教师奖(2012)，中科院优秀研究生指导教师(2013)，上海市优秀学术带头人(2014)，中国科学院特聘骨干研究员(2015)，深圳市国家级领军人才(2018)，广东省“珠江人才计划”领军人才(2019)，前沿有机化学Asian Core Program Lectureship Award（日本、韩国、新加坡、泰国、台湾），以及法国斯特拉斯堡大学邀请教授，日本九州大学“Global 30”计划“Part-time Lecturer”，北海道大学Hokkaido Summer Institute邀请教授，台湾国科会化学促进中心访问教授，日本化学会Asian Special Speaker，英国皇家化学会会士等荣誉。2014年获上海市自然科学奖一等奖，2016年获国家自然科学奖二等奖。

主要从事有机不对称合成及手性药物方面的研究，研究方向包括：新型手性配体/催化剂的设计及不对称反应研究；生物活性分子的高效合成和小分子药物发现；基于疾病诊断的小分子荧光探针研究。

欢迎有兴趣的师生参加！
