外国语学院　2024/11/17 19:28:00
  （钟沛基 2022280095）
讲座题目：AI时代外语人才与家国情怀
座谈题目：国家社科基金辅导座谈会

讲座时间：2024年11月29日(周五) 下午 14:00-16:00

讲座地点：汇文楼1200会议室

主讲人：陈众议

主讲人简介：陈众议，中国社会科学院学部委员，国家文科一级教授，西班牙皇家学院外籍院士。主要从事西班牙语文学与文艺学研究，发表专著和文集二十余种，论文和书评二百余篇，间或涉足文学创作。目前正参与全国人大教科文卫立法等相关工作。


内容简介：随着AI推动的第四次工业革命浪潮汹涌而至，外语学科面临危机。这似乎是不争的事实。首先是招生难，其次是就业难，二者相辅相成。但我不是来火上浇油的，恰恰相反，我要说外语学科的新黄金时代即将来临。何也？且听我顺着家国情怀或者人类情感这个话题慢慢道来。
