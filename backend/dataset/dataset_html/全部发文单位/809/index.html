报告人简介：

帅志刚，国家杰出青年基金获得者，教育部知名学者特聘教授。1983年中山大学物理系获学士学位，1989年复旦大学物理系获博士学位，导师是孙鑫院士。比利时蒙斯大学博士后。2000年入选中国科学院百人计划，加入中科院化学所工作，2008年调入清华大学，2022年入职香港中文大学（深圳）。取得如下学术成果：（1）理论预言高分子电致发光内量子效率可以超过传统观念的25%极限，得到广泛的实验验证；（2）率先将密度矩阵重正化群(DMRG)的算法引入量子化学，并应用于高分子光电材料，解决了关于激子束缚能和激发态排序的争端。近年来进一步发展了含时DMRG算法，已经成为了处理复杂体系量子动力学的有效新方法；（3）提出了量子核隧穿的电荷传输模型，解决了有机/高分子半导体材料中导电机理的争端，在此基础上发展了电子迁移率的多尺度计算方法和程序； (4) 开发了计算化学程序MOMAP，目前已经实现了商业化应用。共发表440余篇论文，被引用25000余次。先后当选为国际量子分子科学研究院院士、欧洲科学院外籍院士、比利时皇家科学院外籍院士和世界理论与计算化学家协会(WATOC)理事。2012年获中国化学会阿克苏-诺贝尔化学科学奖，2018年法国化学会法-中化学讲座奖，2020年获得北京市自然科学奖一等奖(排名第一)。

讲座摘要：

高分子的导电机制长期以来存在争议，尤其是在低温下电导随电压的非线性行为无法用传统的理论去解释。我们从有机/高分子材料的柔性骨架所带来的电荷局域化出发，提出了电荷传输的量子核隧穿模型，不仅可以合理地解释实验，同时所预言的负同位素效应得到了实验证实，据此所开发的电荷迁移率计算程序更加合理，被广泛地应用到材料设计。我们将进一步介绍设计同时具有发光和高迁移性的分子设计理论模型，所预言了若干同时具有高发光和高迁移率的材料，并再实验中得到证实。最后，我们将介绍一种基于量子计算机的分子光谱计算方法。
