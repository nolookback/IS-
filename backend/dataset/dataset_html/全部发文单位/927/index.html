讲座摘要：

表面增强拉曼光谱（SERS）可提供物质指纹结构信息，且具有极高的检测灵敏度，然而，只有在粗糙的Au、Ag、Cu等材料表面才具有较高的SERS增强效应，极大地限制了实际应用。针对此瓶颈，我们通过构筑不同核壳纳米结构，发展了“借力”策略、壳层隔绝纳米粒子增强拉曼光谱（SHINERS）及其卫星结构策略，突破传统SERS对材料与形貌的限制，提升拉曼光谱的检测灵敏度和空间分辨率。成功实现模型单晶、实际纳米催化剂甚至单原子催化剂表面重要能源/催化反应过程的原位拉曼光谱研究，捕获了不同表面反应中间物种（OOH*、OH*、O2-等）的直接光谱证据，从分子水平阐明了氧化原、氢氧化、CO2还原等反应的机理与构效关系，以及界面水分子结构对催化机制的调控作用。除此之外，我们将壳层隔绝模式拓展至其它等离激元增强光谱，发展了具有高灵敏的壳层隔绝纳米粒子增强荧光/磷光/和频等系列谱学技术。
