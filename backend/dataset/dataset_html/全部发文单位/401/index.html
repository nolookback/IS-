报告摘要   
线性信息融合是诸多估计与学习模型和算法的核心架构，如经典的高斯混合/有限混合模型、(交互)多模型、贝叶斯平均模型、线性意见池等，乃至基本神经网络模型核心也是线性信息融合架构。本报告围绕报告人在单目标概率密度PDF和多目标概率假设密度PHD算术均值（Arithmetic Average，AA）融合方向的一些研究工作，介绍保守融合基本概念、技术要点和AA融合方法的提出与发展过程，重点分析AA融合方法所具有的容错性、保守性、保模性、混合分布闭合性，给出AA融合方法的一些主要统计与信息学属性，并基于最大最小原理推导次优融合权重。指出在处理随机频繁漏检、密集新生目标、不完全重合视场和大规模传感网以及快速/在线计算等方面的优越性能，着重指出不同类型的随机有限集滤波器（包括PHD/CPHD/MB/PMBM/GLMB/LMB）有效线性融合实现的统一理论框架：（标签）PHD-AA，分析了（标签）PHD-AA的有效性和物理意义，并指出计算标签密度距离的难点。基于此PHD-AA统一框架，首次给出了异构（不同随机点过程）随机有限集滤波器融合协同算法模型，实现了异构密度信息融合突破。最后结合实际组网需求介绍当前AA融合所面临的挑战、迫切需要解决的难点问题和国际最新研究成果。
