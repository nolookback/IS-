讲座名称：矩阵推理任务中题目内认知策略切换的识别


个人简介： 詹沛达，博士，副教授；现任浙江师范大学心理学院党委委员，应用心理学专硕学位点负责人;浙江省高校领军人才培养计划青年优秀人才，浙江省哲学社会科学领军人才，浙江省之江社科青年学者。二级学科为心理测量学，聚焦于问题解决等行为过程数据的心理测量模型构建。担任《心理科学》、PLoS One和BMC Psychology编委；在Psychometrika、BRM、BJMSP、Intelligence、JEBS、EPM和《心理学报》等专业期刊发表学术论文60余篇，谷歌学术H指数18。主持国自然、教育部人文社科和浙江省哲社规划课题等项目。 


 讲座摘要： 识别问题解决过程中的认知策略有助于研究者了解个体的高级认知过程和不同策略的适用情景。目前的认知策略识别方法局限于题目水平，只能揭示个体在不同题目间的认知策略切换(cognitive strategy switching, CSS)。尽管人们已经意识到个体在解决单一问题过程中可能存在的题目内认知策略切换，但尚缺乏相应的识别方法。对此，本研究提出了一种使用基于眼动的隐马尔可夫模型来识别题目内认知策略切换的分析范式，以揭示每名参与者在解决每道题目过程中的认知策略切换。本研究通过一则矩阵推理测验中的眼动过程数据探究并展示了新范式的价值。结果表明，在问题解决过程中：(1)参与者主要使用两种认知策略——建构匹配和作答消除；(2)参与者更大概率从建构匹配策略切换到作答消除策略；(3)题目难度越高则认知策略切换越频繁；(4)策略切换越频繁则参与者在矩阵区和元计划上耗费的时间越少，但增加了题目作答总时间。此外，随着测验的进行，题目内认知策略切换的执行呈现出三种不同模式，不同模式下的参与者的智力水平和题目作答时间存在显著差异。总之，本研究提出的新范式能有效识别矩阵推理任务中的认知策略切换。
