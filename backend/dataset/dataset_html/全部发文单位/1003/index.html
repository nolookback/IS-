信息中心　2021/10/22 7:50:00

讲座题目：教育大数据与数据挖掘方法

主讲嘉宾：傅向华教授（深圳技术大学）

讲座时间：2021年10月27日（周三）下午15:00

讲座地点：深圳大学沧海校区致理楼C座L3-1115会议室

主讲嘉宾简介：

傅向华，现任深圳技术大学大数据与互联网学院副院长，教授。2005年12月于西安交通大学获计算机科学与技术博士学位，2005年12月至2018年10月，在深圳大学信息工程学院/计算机与软件学院工作，历任计算机学科讲师、副教授、教授，曾担任深圳大学计算机与软件学院网络工程系主任、计算机系副主任，大数据应用与技术研究所文本大数据研究团队负责人；2018年11月，转入深圳技术大学大数据与互联网学院。已主持国家自然科学基金面上项目“面向大规模社会媒体的动态舆情内容安全监测关键技术研究”1项，广东省项目2项，深圳市项目4项；主持完成包括华为公司在内的数据挖掘相关企业横向项目多项，在国内外重要期刊以及国际会议上发表60余篇，参与编写教材5本,出版译著1本，参与完成省级精品课程、省级优质资源课程多门，获省级教学成果一等奖1次，深圳市科技进步奖2次，申请专利14项，软件著作权7项。主要从事数据挖掘、自然语言处理、机器学习、信息检索等领域的相关研究。

 

讲座主要内容：

主要包括教育大数据相关技术、典型数据挖掘方法以及教育大数据分析应用等内容。首先从现代科学方法的起源引入，举例说明以人的计算为基础的数据分析在现代文明发展中的重要作用，然后介绍以计算机为基础的大数据技术和教育大数据；在此基础上，介绍数据挖掘的一般处理过程和典型数据挖掘方法，如Boosting决策树、朴素Bayes、神经网络等分类方法以及关联规则、聚类方法；最后以教育大数据中的具体业务为例，分析数据挖掘方法在教育大数据中的应用思路。

欢迎感兴趣的师生参加！
