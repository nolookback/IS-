建筑与城市规划学院　2022/3/25 16:03:00

讲座题目：

《因应城市化和规划实施需要的深圳土地产权管理创新与实践》

 

讲座简介：

围绕深圳从原特区内城市化、原特区外城市化到深度城市化的三个发展阶段，分别介绍每个阶段如何通过土地产权管理的创新和实践支撑规划实施，并推动城市高快速发展。

 

特邀专家：施源

教授级高级规划师。就职于深圳市规划囯土发展研究中心，从事深圳规划与土地政策研究工作。

 

课程名称：城市案例专题研究

讲座主持：陈燕萍 教授

讲座助理：蓝瑾

 

讲座时间：3月31日（周四） 14：15-16：40

腾讯会议号：47343913941

 

讲座纲要：

一、深圳城市发展史是一部生动的土地改革史

二、以1992年土地统征推进原特区内城市化

三、以2005年土地统转推进原特区外城市化

四、以征转地历史用地制度创新推进深度城市化
