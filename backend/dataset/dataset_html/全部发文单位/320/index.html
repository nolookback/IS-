“后人类的当代性与未来性：科幻作为方法”学术讲座在人文学院顺利举办
人文学院　2024/7/4 14:53:00
  （钟沛基 2022280095）

2024年6月27日下午，深大人文论坛（第505期）在深圳大学人文学院国学院会议室顺利举行。韦尔斯利学院东亚系主任、中国文学研究讲席教授、复旦大学校级顾问教授、（美国）中国现代文学与比较文学学会会长、美国现代语言协会(MLA)理事、美国国家人文中学项目评审宋明炜教授受邀为师生带来题为“后人类的当代性与未来性：科幻作为方法”的学术讲座。本场讲座由深圳大学人文学院江玉琴教授主持，众多师生到场聆听。

讲座伊始，宋明炜教授结合自己的科幻研究，简要回顾了21世纪中国科幻文学发展的过程。宋明炜教授提及科幻作家韩松在2010年“新世纪十年文学：现状与未来”国际研讨会上作的发言使中国科幻浮出水面，从此在中国文学界拥有一席之地。在此之前，中国科幻已经经过了10年的强势发展形成了独居特色的创作风格。宋明炜教授还强调道，科幻并不是类型文学。科幻出现在2010年的新世纪大会上，改变了我们看待文学的方式，使我们重新审视文学与世界的关系。

接着，围绕着“《狂人日记》是科幻小说吗？”这一问题，宋明炜教授辨析了现代性与当代性的关系。宋明炜教授指出现代与当代并不是必然的线性关系、先后关系。在西方殖民世界观强势的主体与客体意识投射下，现代社会中性别、种族、时间、技术都被放置在单一指向的结构中。科幻撕开了单向箭头下的世界，瓦解了现代性中的单一性。当代意识与科幻打破惯性思维的当代性书写不谋而合。宋明炜教授进一步说到科幻文学是对鲁迅打破原有束缚的革命精神的继承；五四期间章太炎提出的“齐物论”也可以帮助我们理解后人类思想。以科幻为方法，我们可以重读现代文学与现代思想。

随后，宋明炜教授围绕“我们能否将刘慈欣视为一位超越二项性的作者？能否将刘慈欣的创作视为超越二项性的文本？”的问题，提出性别是当前文学的有待突破的天花板。通过分析双翅目《太阳系片场：宇宙尽头的茶馆》，宋明炜教授提出科幻文学中性别超越二项性是超越现代二项性的重要表现，科幻的意义在于为一个性别转变为众多不同的性别提供想象文本。

讲座接近尾声，宋明炜教授与参会嘉宾与深大师生就建立当代性的时间意识、当代科幻中性别的超前意蕴、中国科幻的末日意识等方面进行了热烈讨论。本次讲座拓宽了大家研究科幻的学术视野，激发了大家对科幻当代性的深刻思考。活动在师生热烈的掌声中圆满结束。
