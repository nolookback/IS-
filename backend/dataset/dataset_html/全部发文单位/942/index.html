讲者介绍：

纪啸林，肃木丁建筑事务所创始合伙人、主创设计师。

1984年生于山东。2007年毕业于重庆大学建筑城规学院城市规划专业，获工学学士学位。2013年于香港中文大学建筑学院荣誉毕业，获建筑学硕士学位。

2016年，纪啸林与丁樯成立肃木丁，随后带领团队在城市设计、高层建筑、城市公共文化建筑、高铁枢纽和机场航站楼领域，迅速成长为国内炙手可热的新锐团队。

纪啸林的代表作有：深圳湾超级总部基地城市设计，深圳金融文化中心（在建），TCL深超总先进半导体总部大厦（在建）、深汕高铁站综合交通枢纽和深圳机场T4航站楼等。
