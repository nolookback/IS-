深圳民族宗教研究基地学术讲座第十二场 新文化运动中施莱尔马赫的影子：宗教的情感化及其后果
人文学院　2024/11/14 10:10:00
  （钟沛基 2022280095）

主讲人：孙尚扬 教授　（北京大学）

时 间：2024年11月28日　14：15

地 点：深圳大学粤海校区汇文楼人文学院大会议室1100

 

主讲人简介：

孙尚扬，北京大学哲学系宗教学系教授、博士生导师，曾在美国天主教大学、耶鲁大学、比利时鲁汶大学、香港中文大学等高校访学，现任中国宗教学会理事、中国社会学会宗教社会学学术委员会委员。研究领域包括中西文化交流史、中国基督教史、宗教社会学等，主持国家社科基金项目、教育部项目多项，曾数次获校级、省部级和国家级奖励。代表作有《明末天主教与儒学的互动》、《1840年前的中国基督教》（合著），《宗教社会学》，另有译著六部，在《哲学研究》《世界宗教研究》《学术月刊》《道风》等国内外刊物发表中英文论文八十余篇。

 

内容简介：

讲座以1916年至1922年初中国知识分子讨论宗教问题的著述为对象，探讨施莱尔马赫的宗教哲学在新文化运动中的影响。新文化运动为中国社会知识界带来了一种决定性的变化，即向现代社会知识的转型，当时的知识分子对宗教问题的关切也包含了对现代民族国家的探寻。1916年前后围绕孔教问题展开的激烈争论表明，新文化运动中一些知识分子由反对建立国教、主张政教分离转向反对宗教本身、主张取消宗教，可归因于当时引入中国的来自不同国家的启蒙理念。
