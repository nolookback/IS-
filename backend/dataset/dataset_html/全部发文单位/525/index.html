主讲人简介：

彭定光，哲学博士，湖南师范大学二级教授，博士生导师，湖南省伦理学学会副会长，主要研究方向是伦理学基础理论、政治伦理学及德育理论，主持完成国家哲学社会科学基金重点项目1项、一般项目1项、教育部人文社会科学重点研究基地重大项目1项、省级项目6项等课题的研究，参与了多项国家重大招标项目和其他项目的研究，在《哲学研究》、《道德与文明》、《马克思主义与现实》等刊物上发表学术论文110余篇，出版了《政治伦理的现代建构》等专著4部，参与撰写《现代公共管理伦理导论》等著作15部。
