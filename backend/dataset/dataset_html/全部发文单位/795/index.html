学术讲座丨人机结合的空间认知
建筑与城市规划学院　2022/11/4 11:41:00
  （钟沛基 2022280095）

讲座概要：

围绕建筑的全生命周期，简要介绍了团队在大数据行为分析、算法辅助设计等研究方向中的研究实践与最新成果。在此基础上，阐述了包含广义的认知主体和认知活动的“人机结合的空间认知”的概念，其中既包含了在建成空间使用过程中的一般意义的“空间认知”，也包含了建筑创作中的思维活动——“设计认知”，而机器智能正在这类研究中扮演着越来越重要的角色。


主讲人：黄蔚欣

清华大学建筑学院长聘副教授，日本京都大学博士，瑞士苏黎世理工大学(ETH)访问学者。全国高等学校建筑学专业教育指导委员会建筑数字技术教学工作委员会副主任委员，中国建筑学会环境行为学术委员会常务委员兼副秘书长，中国建筑学会计算性设计学术委员会委员，中国建筑学会建筑结构分会新材料及新型结构专业委员会委员，中国城市科学研究会大数据专业委员会委员，曾任亚洲计算机辅助建筑设计学会（CAADRIA）委员。

研究方向：数字建筑设计，基于结构分析的建筑形态生成，人机结合的空间认知。承担和参加多项国家自然科学基金、国家科技支撑项目的研究，发表中、英论文80余篇。

 

主持人：吴向阳

深圳大学 副教授
