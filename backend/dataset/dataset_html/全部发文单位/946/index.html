学术讲座丨基于距离的空间分析方法以及在城市数据分析中的应用
建筑与城市规划学院　2021/12/6 9:04:00
  （钟沛基 2022280095）

讲座简介：

空间是地理学的核心概念。距离是描述空间物体关系的基础概念。我们将重点介绍两种基于距离的空间分析方法：1)如何考虑用距离刻画空间物体的几何形态差异；2)如何在地理加权回归中处理多尺度的时空非平稳性。同时，也将介绍他们在建筑物匹配和房价时空变异性相关的城市数据分析中的应用，以期为城市的规划设计提供科学依据。

专家简介：

周煜，香港中文大学未来城市研究所城市信息学研究计划(Urban Informatics Programme)荣誉研究员。主要研究方向为时间序列分析为主的应用数学、空间分析为主的计量地理学以及城市数据分析的跨学科研究。

报告地点：深圳大学建筑与城市规划学院A103

报告时间：2021年12月7日 19:00~20:30
