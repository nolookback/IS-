【学术讲座】单比特雷达成像与对抗

报告摘要：

单比特量化在简化系统、节省算力、降低成本、提高效率方面优势显著，尤其是在需要海量数据处理的宽带成像雷达及其对抗系统中具有极大的应用潜力。在时域上，单比特的非线性量化过程损失了绝对幅度，但信号的频率、相位、相对幅度得以保留，为高质量信号处理提供了可能；在频谱中，单比特量化突变引入了丰富的谐波分量，为多样化的对抗提供了更高的自由度。

在单比特量化中引入单频阈值，从成像的角度而言，能够为幅度非线性失真的校正提供参考，同时优化频谱结构，保障单比特成像质量；从对抗的需求出发，单频阈值能够对实现频谱解构与能量重分配，从而为干扰对抗提供更多的策略选择。此外，单频阈值模型具备极强的泛化表征能力，既能为传统量化理论提供全新视角，又能为单比特量化提供更多可能。

  

 

报告人：赵博，1986年11月生，现为深圳大学特聘副研究员，广东省工程技术中心副主任，深圳市后备级人才，硕士生导师。现已在国内外重要学术期刊和会议上发表论文30余篇，现任IEEE TGRS、TSP、TAES等学术期刊杂志的审稿人。申请国家发明专利30余项。主要研究方向包括雷达系统设计、雷达信号处理、电子对抗、单比特压缩采样与重建等。在国际上率先突破了高性能单比特雷达成像技术，在实测成像处理中获得了成功应用。主持国家级重点项目2项、国家自然科学基金面上项目1项、青年基金1项、博士后基金1项，作为技术负责人参与国家自然科学基金重点项目1项、杰出青年基金1项、国防科技创新特区项目等课题。
