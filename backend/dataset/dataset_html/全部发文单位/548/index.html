学术讲座：国家自然科学基金伴随我成长
机电与控制工程学院　2023/12/22 11:10:00
  （钟沛基 2022280095）

讲座题目：国家自然科学基金伴随我成长

演讲嘉宾：邓子辰 教授（西北工业大学-航空学院）

讲座时间：2023年12月24日（星期日）上午9点

讲座地点：深圳大学沧海校区致信楼S512会议室

邀请人：赖志慧 老师，方仕童 老师

讲座内容简介：

本报告从国家自然科学基金申请人和评审人的视角分享基金申请经验，明确对基金项目申请的正确认识，介绍基金申请状况（尤其是力学学科），重点分享杰青、优青等人才项目的申请经验，介绍具体内容的完成方法和完成重点，期望提振申请人信心，帮助申请人更好的完成各项申请准备工作。

 演讲嘉宾简介：

邓子辰，西北工业大学航空学院、极端力学研究院教授，“复杂系统动力学与控制”工信部重点实验室主任，教育部“知名学者特聘教授”。

近年来主要从事计算力学与控制理论交叉学科及其哈密顿动力系统的辛数值方法研究，并有效地将研究成果应用于航天系统动力学、柔性多体系统动力学、先进复合材料优化设计等领域。先后主持40余项国家及省部级科研项目，包括主持13项国家自然科学基金（重点2项，其它11项），发表论文320余篇，出版学术专著5部，获省部级一等奖等科技奖励5次。

培养的研究生成长为国家杰青1名、国家级青年人才6名；获全国优博提名奖1人、中国力学学会优博1人、陕西省优博4人。以第一完成人获国家级教学成果二等奖1项、陕西省教学成果奖特等奖1项。

主要学术兼职包括：国务院学科评议组力学学科成员，教育部力学专业指导委员会委员，国际计算力学协会理事，中国力学学会动力学与控制专业委员会副主任，中国力学学会计算力学专业委员会副主任，《固体力学学报》、《应用数学和力学》副主编，国内外9个学术刊物的编委。
