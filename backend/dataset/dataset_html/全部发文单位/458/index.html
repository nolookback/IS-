【学术讲座】星载SAR技术研究进展及发展趋势
电子与信息工程学院　2024/4/7 11:20:00
  （钟沛基 2022280095）

嘉  宾：闫伟高级工程师，中国航天科技集团

邀 请人：李文廷 助理教授

时 间 ：4月10日 上午10:30-11:30

地 点 ：沧海校区致信楼N710


报告摘要：

星载高分辨率SAR是目前国内外发展的热点，可广泛应用于应急减灾、国土资源调查、农作物物候期监测、森林生物量反演等方面。本次报告将围绕国内外星载SAR卫星的研究进展、星载SAR在轨应用情况、星载SAR天线设计等三个方面进行介绍，以及本单位在星载SAR产品及应用方面的研究工作，并就未来典型星载SAR的天线需求开展交流。

报告人简介：

闫伟，男，西安空间无线电技术研究所高级工程师，于2007年和2013年在西北工业大学获电子信息工程学士学位和电子科学技术博士学位。长期从事星载微波遥感规划论证与星载SAR研制，为科工局有效载荷与卫星应用组秘书组成员，航天科技集团微波遥感专家组成员。在星载SAR系统设计、关键技术、系统仿真、卫星产品研制及卫星试验等方面从事了大量的研究工作，有着丰富的经验，作为主任设计师承担国家级型号项目三项，有多个星载SAR在轨产品研制经历，主持国家/省部级项目5项，发表学术期刊及会议论文20余篇，授权国家发明专利10余项。曾获2019年高分年会青年基金论文奖。
