关闭窗口｜打印张贴版    
华中科技大学特聘研究员陈力娜主讲题为“大文博时代文旅和科技、艺术的融合及创新表达”的学术讲座成功举办
传播学院　2021/5/28 16:35:00
  （钟沛基 2022280095）

5月20日，华中科技大学数字光影技术湖北省工程研究中心特聘研究员陈力娜受邀来到传播学院主讲题为“大文博时代文旅和科技、艺术的融合及创新表达”的学术讲座。本次讲座由传播学院田少煦教授主持。

陈力娜，上海风语筑文化科技股份有限公司首席文旅战略官，华中科技大学数字光影技术湖北省工程研究中心特聘研究员。从事文商旅娱博行业十余年，在城市数字化体验空间、数字产品及服务、数字艺术矩阵等方面拥有丰富的营销战略以及管理经验，在国内多个数字创意论坛和文博峰会上作主题演讲。

讲座伊始，陈老师简要介绍了中国的文博旅，并指出新冠疫情背景下，文博旅出现了一系列新的特征。同时，陈老师通过列举一系列数据指出，2021年国内旅游行业相较于疫情前期有所增长。与此同时，国家在政策层面进一步支持数字文化产业的发展，鼓励发展沉浸式产业，这为文博旅的发展打下了坚实基础。

接着，陈老师讲解了全国及中国沉浸产业雏形，并表示沉浸产业分为商旅活动、创意感官、演艺演出、教育创作、主题娱乐、复合业态、互动叙事、惊悚挑战、万物沉浸、休闲商业、实景游戏、综合现实这十二大市场。此外，陈老师以“风语筑”为例，为在座同学深入讲解了中国数字艺术产业沉浸式战略赋能的相关要点。

随后，陈老师对城市数字化体验空间做了介绍。陈老师表示，这类空间主要展示城市及地方历史文化、发展历程、建设成就和未来规划愿景，主要以城市馆、园区馆等为载体。此外，陈老师展示了“2020迪拜世博会中国馆”、“石家庄城市馆”、“上海奉贤区规划资源展示馆”、“天水工业博物馆”、“沉浸式网红党建中心”等相关体验空间。在数字艺术赋能新媒体展览上，陈力娜老师着重介绍了“上海西岸油罐艺术中心”、“广东珠江琶醒文化创意艺术区”、“生命之森与未来游乐园”三个新媒体展览，并向同学们介绍了“中国首个建筑模型博物馆”以及“中国首个沉浸式网红直播空间”。

紧接着，陈老师对文旅品牌“放语空”进行了讲解，“放语空”是中国第一个乡宿文创综合体，“放语空”突破了文旅农家乐模式和单一模式，致力打造文旅3.0。最后，陈老师对2020年推出的“雷电所”进行了介绍，试图通过跨学科的合作，探索艺术、科技、自然的边界，以及技术与人类认识和感官的内在联系。

讲座现场气氛热烈，同学们就陈老师所讲内容提出了许多问题，陈老师一一作答，并与同学们进行了更深层次的讨论。至此，讲座在同学们的掌声中圆满结束。
