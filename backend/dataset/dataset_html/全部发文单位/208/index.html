学术讲座：俄底修斯后裔们的冒险：科恩论西方海洋文学
外国语学院　2024/11/11 9:35:00
  （钟沛基 2022280095）
 
讲座题目：俄底修斯后裔们的冒险：科恩论西方海洋文学

讲座时间：2024年11月23日(周六) 上午 9:30-11:30

讲座地点：汇文楼1200会议室

主讲人：朱国华

主讲人简介：朱国华，曾就读于华东师范大学、南京大学中文系，并在复旦大学博士后流动站工作。曾在杜克大学、巴黎高师和海德堡大学访学。先后入选教育部新世纪优秀人才支持计划、上海市浦江人才计划、教育部知名学者特聘教授。历任华东师大中文系系主任、国际汉语文化学院院长。目前兼任《文艺理论研究》主编，中国文艺理论学会会长，教育部高等学校中文学科教学指导委员会成员，并获国家“万人计划”教学名师称号。论文论著曾五获上海市哲学社会科学优秀成果奖，以及教育部高等学校科学研究优秀成果奖二、三等奖，并获宝钢优秀教师特等奖。从1986年开始发表学术论文，著有《文学与权力：文学合法性的批判性考察》、《乌合的思想》和《权力的文化逻辑》等。
撰稿：申杨硕　审核：翁冰莹
