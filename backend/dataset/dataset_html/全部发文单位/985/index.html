  （钟沛基 2022280095）

题目： 从Food-mucus相互作用理解食物成分的健康效应

主讲嘉宾： 杨晓泉 教授

时间： 　　2021年11月10日下午 15:00-17:00

地点：      深圳大学丽湖校区A7-551会议室

主持人：   吴序栎 博士

 报告摘要：

人体黏蛋白（mucin）是人体分泌的大分子胶体物质，其形成的粘液层（mucus layer）在人体中起着复杂的物理及免疫保护屏障作用，可阻止各种微生物侵染及非消化食物大分子透过，同时也控制着营养成分、水及风味成分的扩散及食品质构的感知。研究Food-mucus相互作用是理解食物成分健康效应及食物风味/质构感知的关键，可进一步加深我们对食物成分健康效应的理解。

主讲嘉宾简介：

杨晓泉，博士，华南理工大学教授、博士生导师。长期从事植物蛋白质营养与功能的科学研究及产品开发工作。主持 “863计划”、“十二五”科技支撑及国家自然科学基金、公益性行业科技专项多项国家级科研项目，包括“特殊病理时期营养功能食品专用配料的研究”、“抗极端环境蛋白质加工技术研究与产业化”、“特殊人群食品（中老年、婴儿）品质改良专用蛋白配料关键技术研究”等项目。近5年在此领域发表SCI学术论文100余篇，论文总引用超过4000次，并入选爱思唯尔（Elsevier）2020年“中国高被引学者”。获得植物蛋白相关授权专利15项。相关技术在企业成功实现产业转化，研究成果获2011年和2015年国家科技进步二等奖（均排名第2），获得2019年“泰山产业领军人才”称号。目前担任小麦与玉米深加工国家工程实验室副主任，广东省天然产物利用与产品安全重点实验室副主任。学术兼职主要包括广东省营养学会常务理事，中国粮油学会食品分会常务理事，中国食品科学技术学会功能食品分会常务理事以及多个国际期刊审稿人。
