讲座摘要：

吸附技术在污染控制和资源化中具有广阔应用前景，研发高效吸附剂是高效能吸附处理技术的关键，报告人从碳纳米管衍生吸附剂设计、限域水增强吸附行为、双网络凝胶吸附剂构建及能量场辅助增强吸附等方面，介绍吸附材料设计开发及应用的研究现状、问题及展望。
