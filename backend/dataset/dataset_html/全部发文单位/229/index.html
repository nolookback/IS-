讲座预告 | 我院张洁婷副教授访学回国学术讲座


讲座名称：抑郁的诊断与干预

讲座时间：2024年11月8日上午10：00

讲座地点：深圳大学致理楼C座L3-1201

主讲人：张洁婷副教授

 

讲座简介：

结合访学经历，讲述两方向的研究与思考（1）比较各量表计分方法在抑郁诊断的准确性；（2）轻度抑郁症药物和非药物干预效果。

 

主讲人简介：

    深圳市高层次后备级人才，兼任广东省心理学会正念专委会副主任。在2008年和2014年6月于华南师范大学心理学院依次获心理学学士和博士学位，硕博期间师从张敏强教授，博士联合培养期间在美国宾州州立大学方法研究中心，从事潜变量模型的方法学研究。目前研究领域主要是心理健康测评与干预、童年逆境、攻击性和攻击行为的风险评估。主持国家自然科学基金、国家社科基金、省市级等科研项目5项，在国内外核心期刊发表的主要研究论文40余篇。
