讲座题目：城市设计作为基础的理论

专家简介：

海尔穆特･博特是德国斯图加特大学教授，博士生导师。曾任德国斯图加特大学建筑与城市规划学院院长兼斯图加特大学城市设计与规划研究所(SI)所长，斯图加特大学国际文化中心主任IZKT、科隆应用技术大学城市设计学教授、埃及德国大学建筑与城市规划学院(GUC)创始院长，是International Cittaslow Organaisation 学术委员会成员。近年来以可持续城市规划、大学城规划发展、文化与科技互惠为研究方向，著有《可持续城市规划：可持续社区理念》、《大学城—大学与城市》、《家庭、邻里和基础设施的转变—斯图加特2030年未来图景”》等书籍。同时担任浙江大学、东南大学、韩国国立首尔大学客座教授。博特教授作为独立城市规划师，曾在多次德国、日本、欧洲城市设计竞赛中获奖，主持斯图加特“玫瑰山”片区规划、海尔步隆市火车南站城区规划等项目实践。
