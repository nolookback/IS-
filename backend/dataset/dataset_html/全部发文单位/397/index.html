主讲人简介：

刘永连，历史学博士，现为暨南大学文学院中外关系研究所教授、博士生导师、所长，国家人文社科基金重大招标项目首席专家。长期从事中外关系史研究，先后承担中国社会科学院重大项目、教育部社科基金项目、国家社科基金项目、国家社科基金重大项目等各级科研项目13项。出版著作《突厥丧葬风俗研究》、《16世纪前的中国与世界》等15部，在《文史》、《世界历史》等海内外学术期刊发表论文120余篇。2013年获国家新闻出版总局第三届中国出版政府奖出版奖，2016年获第九届中国社会科学院科研成果一等奖。兼任中国海外交通史研究会副会长、中国中外关系史学会副秘书长、中国朝鲜史研究会副秘书长、广东珠江文化研究会副会长等，及学术刊物《中国与域外》顾问委员会委员、《广州大典》研究中心学术委员会委员等。

  

内容简介：

　　主要介绍自17世纪起流行于朝鲜半岛的“八学士”意象，分析其在姓氏籍贯、始祖身份、东迁原因等方面记载存在的问题，探究这些问题背后朝鲜王朝整个国度打造“中华正宗”的政治动机及其渗入民族肌体的文化改造运动。由此梳理出一种奇特的历史迹象：朝鲜半岛既因慕华心里而注重继承中国传统文化，又在政治和现实层面上对中国态度疏离，渐行渐远。
