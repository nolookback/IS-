学术讲座：深圳市怡亚通供应链股份有限公司姜月总经理助理—供应链企业的转型与变革
经济学院　2022/11/21 11:26:00
  （钟沛基 2022280095）

讲座题目：供应链企业的转型与变革

讲座时间：2022年11月22日（周二）10:25-11：40

讲座地点：明德楼-303

主讲嘉宾：姜月　　深圳市怡亚通供应链股份有限公司

主讲嘉宾简介：

　　姜月，本科、硕士毕业于深圳大学经济学院，现就职怡亚通供应链股份有限公司，企业总经理助理。

　　拥有数据分析经验：参与过大型公司数字化项目，深入理解主要数据可视化展现形式，针对实际场景梳理数据信息，推动公司数据可视化项目，帮助公司管理者进行日常工作决策分析。

　　从事项目管理：针对要开展的项目进行前期尽调、收集并整理好有关资料，制定初步的项目可行性研究报告，为决策层提供建议。协同合同洽谈、申报立项报告等工作，最终推动项目落地。

讲座主要内容：

　　讲座主要介绍以下三方面的内容,即:

　　第一， 供应链行业的趋势。

　　结合企业本身的发展，对整个供应链行业的发展变化及趋势进行分析，同时通过讲解商业模式，解读现阶段供应链企业的发展规划重心所在。

　　第二， 供应链企业的转型。

　　在大数据、定制化需求增多的市场背景下，传统的TOB供应链如何进行转型：如何通过数字化实现精细化管理；如何通过创造增值服务提升企业价值；如何从B端转型到C端，直面消费者进行营销。本段将以怡亚通为例，讲述供应链企业的转型与变革。

　　第三， 分享对供应链管理的理解。

　　欢迎广大师生参加！
