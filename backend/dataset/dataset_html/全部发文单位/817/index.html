报告人简介：

严加宝教授、博导、天津大学建筑工程学院土木系副主任、土木工程实验中心主任。2012年获新加坡国立大学博士学位。长期致力于组合结构、寒区及极地工程防灾减灾、高性能组合结构及新材料、结构非线性分析等领域研究。截止目前已发表SCI一作/通讯论文110余篇, 发表建筑结构学报等中文论文10余篇，出版专著/教材4部，谷歌学术h指数27，引用2000余次，SCI他引1100余次。主持国家自然基金2项、省部级及横向课题20余项。研究成果被国内多部标准、美国钢结构协会（AISC）规范、新加坡规范所采纳。担任中国钢结构协会理事、中国钢结构协会钢-混组合结构分会理事、中国建筑金属结构协会检测鉴定加固改造分会委员、天津市钢结构协会副秘书长等职。担任SCI期刊《Buildings》编委、《EER》期刊青年编委等职。指导研究生获国家奖学金、天津市优秀学生等称号30余次。获得天津市高层次人才（青年）、天津大学北洋青年学者、天津大学我心目中好导师等荣誉，多次获得天津大学优秀硕士论文指导教师。入选全球Top2%顶尖科学家职业生涯学者及多次年度学者，获得韩国钢结构协会-浦项制铁最佳论文奖（Best paper award）、广东省钢结构协会科技进步一等奖等荣誉。
