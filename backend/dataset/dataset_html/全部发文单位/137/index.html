“传播学研究的方向和方法：争议、现状与趋势”学术讲座顺利举办
传播学院　2024/12/17 10:44:00
  （钟沛基 2022280095）

2024年12月2日，《传播学研究的方向和方法：争议、现状与趋势》主题讲座在传播学院致艺楼813会议室举办。本次讲座由美国纽约州立大学传播系教授洪浚浩主讲，传播学院副院长杨洸教授担任讲座主持人。

讲座正式开始前，杨洸教授向在座师生介绍了洪浚浩教授。洪教授是联合国全球媒体和信息倡议项目专家，曾师从议程设置理论创始人麦库姆斯等教授，先后任国际中华传播学会会长和全美中国研究联合会会长等，多年被聘为哈佛大学费正清中心研究员及英国OS世界著名大学排名评审专家等。

洪教授以“传播学是一门什么样的学科？”的问题作为讲座的切入点与同学们进行了交流。他指出，不同于其他学科，传播学的研究兼具社会科学及人文科学两个方向，这一划分与传播学的两大源流——美国和欧洲密切相关。随后，洪教授围绕量化和质化两种主要研究方法的争议和共同点展开了详细讨论。他强调，尽管这两种方法存在诸多争议，但它们都以与传播相关的社会现象为研究对象，不同的是，社会科学方向的量化研究旨在寻求因果关系，探究事物规律；而人文科学方向的质化研究则更注重挖掘社会现象背后的原因和意义。

洪教授还指出，传播学作为一门独立学科，至少应具备三点：一是形成独有的理论体系；二是具有代表性的研究内容；三是具有科学的研究方法。同时，他也与师生共同探讨了这两种主要研究方法的短处，给师生在研究方法的具体应用提供启发。

讲座最后，洪教授耐心解答了师生们提出的问题，并就研究方法实操、传播学理论创新等相关问题进行了讨论。
