第十二届深圳学术年会“深圳经济特区40周年回顾与前瞻”系列学术讲座“推进全面从严治党创新深圳特色治党机制”在深圳大学开讲
城市治理研究院　2021/1/30 11:41:00
  （钟沛基 2022280095）

2021年1月26日下午，由中共深圳市委宣传部、深圳市社会科学联合会与深圳报业集团共同主办，深圳大学城市治理研究院、当代中国政治研究所、党内法规研究中心联合承办的第十二届深圳学术年会“深圳经济特区40周年回顾与前瞻”系列学术讲座在深圳大学粤海校区汇文楼1400会议室举行。

本次讲座由深圳大学城市治理研究院执行院长陈文教授主持，由教育部知名学者、中山大学政治与公共事务管理学院肖滨教授主讲“推进全面从严治党创新深圳特色治党机制”专题讲座之“百年大党与中国现代化国家建设”。

首先，肖滨教授以构建理论分析框架为起点，强调从“现代性”的角度重新考察现代国家的建设：“现代国家”大致可以从“军事财政国家”“国族建设”和“公民权建设”三个维度进行分析。从现代国家建设的角度看，政治文明是在诸多复杂的紧张关系中寻求平衡的过程，例如，政权建设中，如何使权力有限且有为有效；国族建设中，如何既整合于一体，又尊重多元；等等。

其次，中国共产党在中国现代国家建构中究竟扮演了怎样的角色？对此，肖教授指出，“中国共产党是中国现代国家建构的主导者和推动者”，并强调三个方面：第一，中国共产党构建了现代国家的统治体系、治理体系；第二，中国共产党推动了中国现代国家的国防建设；第三，中国共产党推动了公民权利的建设与满足。

再次，肖教授对中国共产党的组织属性做出深入分析。一是回答“中国共产党不是什么”的问题，比如中国共产党不是选举型政党、不是竞争性政党、不是部分性政党，而是马克思主义政党；二是回答“中国共产党是什么”的问题，肖教授重点围绕“一个核心、两个先锋队、三个代表”进行阐释，“一个核心”即中国共产党是中国特色社会主义事业的领导核心，“两个先锋队”即中国共产党是中国人民和中华民族的先锋队，“三个代表”即中国共产党始终代表中国先进生产力的发展要求、中国先进文化的前进方向以及中国最广大人民的根本利益。

最后，肖教授就中西方政治文明建设的未来发展进行了拓展分析。他指出，国内有学者用“政党政治”和“人民政治”的概念将中西方现代政治文明截然分开，这种观点是值得商榷的。实际上，自1949年以来，我国在现代国家建设中所展现出来的政治文明虽有着明显的独特性，但与西方政治文明对比也不是截然对立的。在努力建设社会主义政治文明的过程中，中国既要保持自身的独特性与自主性，也要与各种政治文明保持交流对话，坚持做到兼容并蓄、求同存异，努力为人类政治文明发展做贡献。

在互动环节，与会师生围绕党领导国家现代化发展、基层社会治理等问题与肖滨教授进行了深入学术探讨。陈文教授在学术总结中强调，肖滨教授从党的百年奋斗历程来透视中国的现代化国家建设，是立足于中国自身历史发展、充分重视中国独特政治实践的研究，具有十分重要的理论意义。今年是中国共产党建党一百周年，深圳大学城市治理研究院、当代中国政治研究所、党内法规研究中心将会继续发挥所长，坚持从政治学理论、当代中国政治以及执政党建设等学科方向，深耕党的建设、党的领导与中国现代国家建设等主题的研究，积极为党的百年诞辰献礼。

据悉，本次讲座承办方——深圳大学城市治理研究院是直属深圳大学的独立科研实体机构，主要下设“当代中国政治研究所”（广东省高校人文社会科学重点研究基地）、“党内法规研究中心”（广东省高校人文社会科学重点研究基地）、“创新型城市建设与治理研究中心”（广东省协同创新国家级重点培育平台）、“深圳大学廉政研究院”（深圳市廉政研究基地）等省市级科研平台，全面支撑深圳大学的政治学学科建设。长期以来，研究院秉持“顶天立地”和“没有调研就没有研究权”的科研方法，深耕于政治学理论、中国政治、比较政治和城市政治等四大研究方向，形成了研究问题的前沿性、研究方法的实证性、研究活动的开放性、研究项目的多元性、研究体制的创新性等特色，不仅形成了一系列享誉国内外学术界的研究成果，还积极为各级党委政府提供高质量的咨政服务。
