【学术讲座】轻量化智能多机器人协同SLAM系统构建
电子与信息工程学院　2024/5/20 14:55:00
  （钟沛基 2022280095）

报告嘉宾：陈昌川副教授，重庆邮电大学

邀 请 人：李斌教授

报告时间：2024年5月23日9:00

报告地点：致信楼N609


报告摘要：移动机器人不仅可以满足传统行业转型过程中的智能化需求，而且与人工智能及工业互联网等领域相叠加，可以创造出许多高质量、具有商业价值的新场景。视觉多机协同即时定位与地图构建(SLAM)主要以相机作为传感器，并通过多机器人合作实现定位与建图。然而，在面对复杂环境时前端计算量过大，易导致整体系统精度不理想。环境感知过程中由于计算资源受限易使单体移动机器人跟踪丢失，经典“集中-分布式”系统构建的稀疏点云地图无法对周围环境有进一步的理解进而执行更高层次的任务。在报告中介绍SLAM轻量化多机器人协同三维重建系统设计及相应实验分析，完成系统集成部署，进行系统性能测试，最后进行总结与展望。
