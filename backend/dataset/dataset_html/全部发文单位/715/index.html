“春之季”系列讲座——歌剧表演的灵魂支柱——人物塑造学术讲座成功举办
艺术学部　2023/5/9 16:53:00
  （钟沛基 2022280095）

2023年4月17日下午，艺术学部“春之季”系列讲座《歌剧表演的灵魂支柱——人物塑造》在汇德楼音乐厅成功举办。此次讲座由旅德女中音歌唱家、南方科技大学艺术中心毕宝仪教授主讲，音乐系杨婷主任及音乐系多位教师出席本场讲座。

此次活动由声乐教研室主任黄戈副教授主持，讲座之前进行了毕宝仪教授艺术学部客座教授聘书颁发仪式。讲座中，毕宝仪教授首先从“人物塑造是戏剧灵魂”这一个概念引入，介绍了人物塑造的理念。期间，毕宝仪教授着重讲述了声音人物、舞台形象、艺术内涵的塑造——从声音的角度体会用什么样的声音诠释人物；从舞台的角度思考如何去寻找歌剧中人物的内心；从内涵的角度思考综合艺术修养和文化素养的问题等。她提出，在人物塑造的过程中一定要秉持时代感和使命感，需要蕴含传承性与创新性。

图二：毕宝仪教授与学生进行交流互动

随后，毕宝仪教授结合自身教学经验介绍了人物塑造的具体路径。最后，毕宝仪教授总结到，歌剧的表演需要一个长期探索的过程。我们需要在不断探索中发现人物特征，创造有灵魂的经典人物形象，从而支撑并实现歌剧思想和情感的表达与传递。整个讲座内容深入浅出，令在场听众受益匪浅，讲座在一片热情的掌声中圆满落幕。
