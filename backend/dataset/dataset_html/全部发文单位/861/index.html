题目一：能源化学过程相关基础与应用研究

题目二：电催化前沿

主讲人：徐维林研究员

时间：6月16日下午2：30

地点：腾讯会议（ID：572755490） 

报告人简介：

徐维林，中科院长春应用化学研究所研究员。本科毕业于吉林大学，博士毕业于中科院长春应用化学研究所。先后在美国康奈尔大学/加州大学伯克利分校/劳伦斯伯克利国家实验室从事能源催化方面的博士后研究。回国后获中科院百人计划、国家基金委优秀青年科学基金和杰出青年基金等项目资助。目前主要研究方向为能源过程及材料相关的基础与应用研究。已在Nat. Mater., Nat. Commun., Chem. Rev., PNAS, JACS, Angew. Chem., Adv. Mater. 等国际学术期刊发表学术论文70余篇。

讲座摘要:

在本报告中，报告人将介绍其研究团队在能源化学过程相关的基础与应用两方面的最新研究进展。在基础研究方面，主要介绍在单分子单粒子水平的纳米催化研究中揭示的一系列用常规催化方法无法获得的全新催化信息，包括单纳米粒子水平的催化动力学信息和热力学信息，比如表面不同类型原子的催化特性以及单纳米粒子表面活性模式时空上的动态变化等；在应用研究方面，主要介绍为能源器件燃料电池开发的几种高性能、低成本的可代替常规铂基催化剂的新型功能材料。以上这些工作将有助于实现高效催化材料的理性设计和制备。
