【学术讲座】基于3D打印折纸致动器的水陆两栖软体机器人
电子与信息工程学院　2023/11/22 18:27:00
  （钟沛基 2022280095）

讲座嘉宾：杨扬教授 南京信息工程大学

邀　请  人：黄海明 副教授

时      　间：2023年11月24日上午10：30

地       点：致信楼N723

 

报告摘要：本报告将介绍团队近期研制的一种新型仿生水陆两栖软体机器人，它集成了四种类型的折纸致动器:Z 型，扭转塔，腹鳍型和卷/开型折纸致动器。对青蛙游泳、弹涂鱼爬行和青蛙狩猎的运动机理进行了观察和分析。结果表明，Miura和Kresling折纸的运动轨迹可以通过巧妙的设计来适应动物的四肢、关节和器官。我们的研究结果证明了这些致动器在产生所需运动方面的有效性，并为未来应用3D打印折纸致动器开发具有仿生特征的软体机器人研究提供了见解。

个人简介：

南京信息工程大学自动化学院副教授，机械电子工程系系主任，江苏省智能气象探测机器人工程研究中心副主任。现为IEEE Senior Member，中国人工智能学会(CAAI)智能机器人专业委员会委员，《中国机械工程学报：增材制造前沿(英文)》）首届青年编委。2011年和2013年获哈尔滨工业大学本科及硕士学位，2017年9月获香港大学博士学位。目前主要研究方向是软体及仿生机器人、人工肌肉、智能材料及柔性传感器。到目前为止已经以第一作者或通讯作者在机器人领域顶尖刊物Soft Robotics发表论文4篇(其中第一作者3篇，通讯作者1篇)，以共同一作在材料领域著名杂志Materials Today发表论文1篇，目前已发表45篇国际期刊及会议论文，Google Scholar被引2173次，h因子为19。
