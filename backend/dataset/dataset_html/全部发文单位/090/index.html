讲座题目：当“口腔医学”遇上“机电”——医工交叉创新探索

讲座地点：深圳大学沧海校区致信楼S630-B会议室

邀 请 人：梁雄 研究员

讲座内容简介：

1、简要介绍口腔医学；

钟婉金，深圳市儿童医院口腔科副主任医师、硕士生导师，2008年硕士毕业于中山大学光华口腔医学院，曾于四川大学华西口腔医院正畸科、上海交通大学附属第九人民医院颅颌面科正颌正畸中心进修。主要研究方向为儿童错颌畸形早期矫治。现任广东省儿童口腔医学专业委员会委员、广东省医院协会口腔数字化材料技术管理专业委员会委员、广东省基层卫生协会第二届口腔医学专委会委员、深圳市医师协会口腔医师分会理事、深圳市医师协会儿童颜面管理专业委员会常务理事、深圳市口腔医学会口腔正畸专业委员会委员、深圳市口腔医学会儿童口腔医学专业委员会委员、深圳市女医师协会口腔医学专业委员会委员
