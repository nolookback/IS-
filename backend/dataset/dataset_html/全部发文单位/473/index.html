【学术讲座】被动微波遥感探测及其天线设计
电子与信息工程学院　2024/3/25 10:49:00
  （钟沛基 2022280095）

主讲嘉宾： 陈良兵副教授，中山大学

邀 请 人： 张龙 助理教授

时   间：　04月01日 10:30-11:30

地    点:　致信楼N710



演讲摘要：被动微波遥感探测与雷达遥感探测是微波遥感探测的两个分支。被动微波遥感探测近年来在海洋遥感、大气遥感、极地遥感以及目标探测领域的应用越来越广泛，其遥感探测体制也从传统实孔径扫描成像发展到了具有高空间分辨率与宽视场瞬时成像的综合孔径成像。本次报告将围绕被动微波遥感、被动微波目标探测、被动微波系统天线设计等三个方面介绍其内涵、技术特点、关键技术以及我们在综合孔径被动微波遥感与探测方面的研究工作，并就几种典型应用中的天线设计需求开展交流。

个人简介：

陈良兵，男，中山大学“百人计划”副教授，于2005年和2010年在华中科技大学获电子信息工程学士学位和信息与通信工程博士学位。长期从事被动微波遥感与探测方面的研究，在微波辐射测量理论、关键技术、仿真平台、样机设计及外场实验等方面从事了大量基础研究工作并积累了丰富经验，参与开发了一维毫米波综合孔径测量系统，主持开发了国内第一套二维实时毫米波综合孔径无源成像系统，联合主持开发了50-60GHz镜像综合孔径海洋大气探测系统样机等。国家自然科学基金面上项目所提出的原创性方法“基于阵列平移复用的镜像综合孔径辐射测量方法”入选国家自科基金委19个“鼓励探索、突出原创”典型案例。主持了863项目、国家自然科学青年基金、国家自然科学基金面上项目、探索预研项目子课题、重点实验室基金等项目10余项，在IEEE TGRS、IEEE GRSL、IEEE IGARSS等发表SCI/EI论文20余篇。
