【学术讲座】植物盐碱胁迫应答反应

报告题目：植物盐碱胁迫应答反应

主讲嘉宾：郭岩 教授

报告主持：刘宏涛 教授

报告时间：2024年12月13日（星期五）下午15:00

报告地点：深圳大学丽湖校区守慧楼A6-501

主讲嘉宾简介：

 

 郭岩，博士，中国农业大学教授，生物学院院长。1988年毕业于北京农业大学，获农学学士学位；1999年于德国科隆大学科隆马普植物育种所获博士学位。1999-2003年在美国亚利桑那大学植物系从事植物抗逆分子生物学博士后研究，2004年被聘为北京生命科学研究所研究员，2009起在中国农业大学生物学院工作。主要研究方向：植物感受和响应盐碱胁迫的信号转导分子机理。近5年在Plant Cell、Developmental Cell、PNAS、Molecular Plant、Nature Communications 等杂志发表论文。

报告简介：

我国约有1亿公顷的盐碱地，同时由于自然和人为因素的影响，盐碱地面积仍在不断增加。土壤的盐碱化极大地影响了农业、林业和草业等植（作）物生存、生长和产量。据估算，由于盐碱等环境胁迫造成的我国主要农作物减产每年高达总产量的15%-20%以上。通过培育适合盐碱化土地种植的耐盐/盐碱作物新品种是解决盐碱化耕地的重要出路。经过科学家们长期的努力发现盐碱胁迫对植物造成的影响主要包括三个方面：离子胁迫、渗透胁迫和由此引起的次生伤害，尤其是氧化伤害。盐碱胁迫会造成植物吸水困难，引起渗透胁迫；离子毒害会影响植物细胞代谢, 进而造成营养的不均衡以及氧化胁迫。随着研究的不断深入，人们发现植物应答盐碱胁迫是一个复杂的过程，植物耐盐碱能力的形成需要许多生物学过程在不同调控层次上共同参与调控。近年来科学家们利用拟南芥、水稻、小麦和玉米等，在植物感知和应答盐碱的分子机制方面取得了一系列成果，特别是盐碱胁迫下植物体内离子稳态维持方面取得了较大的进展，发现了离子胁迫信号感知和应答的关键信号通路（例如Salt overly sensitive途径）以及重要调控因子。但是由于植物的耐盐碱能力是多基因控制的数量性状，目前关于作物逆境应答机制的研究仍不够系统和深入，制约了作物抗逆性改良育种分子设计，这需要进一步深入解析植物盐碱胁迫信号感受和作用机制，为作物耐盐碱品种培育的分子设计提供理论支撑。
