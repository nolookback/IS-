主讲人简介：

甘文标教授，美国纽约大学医学院终身教授，1986年毕业于清华大学激光物理系,1995年在哥伦比亚大学获得神经生物学博士学位。甘教授在国际上首次研发出了经颅双光子活体显微成像技术，该技术能够实现长时程活体观察动物的神经元突触变化和神经元电活动情况。利用这个平台就大脑皮层神经突触可塑性进行研究，通过对学习和记忆过程中以及疾病状态下的树突棘可塑性进行研究，从而研究在学习和记忆过程中以及疾病状态下（睡眠障碍、阿尔茨海默症、自闭症、慢性疼痛、抑郁症和精神分裂症）的树突棘可塑性的变化。

甘教授的研究证明了大脑皮层部分树突棘能稳定存在于整个生命过程，并能作为长期信息储存的结构基础；在学习记忆、睡眠功能、小胶质细胞功能领域做了一系列开创性工作；在阿尔茨海默症的基础研究与新药研发方面也做了多年的工作。目前，甘教授在国际期刊Nature、Science、Cell、Nature Neuroscience等杂志上发表近百篇高水准文章，其论文引用次数高达约一万七千次。
