学术讲座——《海浪——海洋的节拍》
生命与海洋科学学院　2023/10/30 16:54:00
  （钟沛基 2022280095）

时间：2023年11月2日 10:15

地点：丽湖校区 四方楼 北311

主讲人：孙建 教授

　 　　中国海洋大学 海洋与大气学院

2000年获中国海洋大学海洋科学学士学位，2005获得中国海洋大学海洋科学博士学位。主要从事台风浪和风暴潮动力过程、长距离传播的涌浪特征分析和机理，以及海洋锋对海浪SAR成像和海浪要素反演的影响等研究。在海洋科学知名学术期刊发表论文30余篇，主持国家自然科学基金联合基金重点项目与多项国家自然科学基金面上项目，并参与了多个国家重点研发计划与科技重大专项。

 

此讲座由生命与海洋科学学院《海洋科学专题讲座》课程组织。

欢迎广大师生前来聆听！
