北京师范大学-香港浸会大学联合国际学院王元新助理教授主讲题为“网络暴力的社会因素分析”的学术讲座成功举办
传播学院　2021/6/22 16:28:00
  （钟沛基 2022280095）

5月28日，北京师范大学-香港浸会大学联合国际学院人文与社会科学学部助理教授王元新受邀来到传播学院为学院师生主讲了题为“网络暴力的社会因素分析”的学术讲座。本次讲座由传播学院冯广超教授主持。

王元新博士毕业于美国天普大学媒体与传播专业，主要研究方向为新媒体效果、网络暴力、冲突管理等。她在Health Communication, Machine Learning 等国际期刊杂志上发表了多篇研究论文。

王老师向大家介绍了她的最新研究成果——“网络暴力”研究。她从研究背景、理论框架、研究假设、研究发现这四个方面展开阐述。从2015年以后，网络暴力成为一种十分普遍的现象，西方学者倾向于认为网络暴力的诱因是心理因素。但王老师认为，从陈凯歌的电影《搜索》以及我国的社会实际情况来看，网络暴力更多的是一种群体行为，是一种由社会因素诱导出来的社会现象。

王老师通过文献研究发现社会不公与社会暴力之间存在着因果关系。王老师采用实验控制的研究方法，选取了1000多个来自美国和中国的实验样本，设计了12个实验场景，并对场景进行操控，从而对样本的攻击取向进行测量和分析。王老师根据她的研究框架提出了4个研究假设：1.社会分配不公（财务、收入等）对网络暴力的影响；2.程序不公对网络暴力的影响；3.互动性不公对网络暴力的影响；4.社会主导取向对于网络暴力的影响。王老师指出，从数据结果中可以看出大部分假设是成立的，但中国的数据与美国的数据存在些许的差异。

在阐述了人口、种族、婚姻与收入等变量对网络暴力行为的影响后，数据分析得出了三个研究结论：一是网络暴力研究也应考虑社会因素；二是网络暴力涉及到每一个社会领域；三是网络暴力不仅是一种个人行为，更是一种社会行为。

最后，现场的老师们与王老师就研究内容进行了深入的探讨，讲座在大家热烈的掌声中圆满结束。
