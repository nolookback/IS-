“AI时代社会科学研究的未来” 学术讲座在传播学院举办
传播学院　2024/12/5 9:43:00
  （钟沛基 2022280095）

2024年11月25日，新加坡南洋理工大学传播与信息学院原媒体技术教授Richard Ling受邀为传播学院师生做了一场以“AI时代社会科学研究的未来”为主题的学术讲座。

杨洸教授作为主持人，为师生们介绍了Richard Ling教授的学术简历。作为移动传播领域的杰出学者，Ling教授深入研究移动传播、短信和移动电话的社会影响。他在SSCI和SCI期刊上发表了360多篇论文，学术成果备受学界瞩目。2016年，他荣获国际传播学会 International Communication Association ( ICA ) Fellow 荣誉称号，并于2017年被任命为国际顶级传播学期刊 Journal of Computer - Mediated Communication的主编。

随后，Ling教授以人工智能将如何影响社会科学研究为切入点，开始了本次讲座。讲座主要分为四个方面，分别是方法作为研究工具、历史实例、人工智能与理论/方法、媒体研究者的角色。

在谈到“方法作为研究工具”时，Ling教授提出科学是由演绎推理推动的。在理论上，它是由探索一种抽象理念的欲望所推动的，使用新的方法（例如工具）能够改变科学的实践。Ling教授指出新的见解会随着新的研究工具的出现而产生，并特别提醒研究学者需保持“工具理性”的概念。

借助历史实例，教授介绍了计算统计学的发展脉络，物理学与社会科学之间的关联性。他强调，人工智能在社会科学应用中提供了不同 “视角”，类似黑箱，带来更多预测但可能在理论阐释等方面有所欠缺，这促使学术界思考什么是“伪相关”，并需秉持谦逊态度去进行研究。

最后，Ling教授对媒体研究者的角色进行阐述，提出了人工智能的关键问题如解释机制与预测结果的差异、数据本质及偏差影响、生成内容问题，揭发了目前人工智能逐渐融入学术界的现象，并对此延伸到从宏观层面上思考人工智能的利弊，值得我们深入思考和探讨。

本场讲座激发了师生对于数字时代的媒体研究的深刻思考，现场师生表示受益匪浅。讲座在掌声中顺利结束。
