祝贺深圳市医学会第一届基础医学专业委员会学术讲座暨选举、成立会议顺利召开
医学部　2021/4/25 15:47:00
  （钟沛基 2022280095）

2021年4月23日，深圳市医学会第一届基础医学专业委员会学术讲座暨选举、成立会议在中洲圣廷苑酒店正式举行。深圳市人大常委会副主任、市科协主席蒋宇扬教授，深圳市医学会会长蔡志明教授，清华大学生命科学院院长杨茂君教授，深圳大学医学部主任朱卫国，深圳大学医学部副主任许兴智等六十多名委员参加了本次大会。

许兴智教授致开幕词，对到场的六十多名委员及获邀嘉宾的到来表示热烈地欢迎。

本场学术讲座，杨茂君教授用幽默风趣、通俗易懂的语言，向在场的各位专家介绍了他领导的实验室在细胞能量代谢相关蛋白质的结构与功能，并以这些生物大分子的高分辨结构为基础开展小分子化合物筛选及设计的系列重大突破，阐释了基础医学与转化和临床之间的关系。讲座既接地气又扣主题非常的精彩！

蒋宇扬主席专程来指导，并发表重要讲话。他指出，基础研究工作的重要性，希望基础医学专业委员会注重以临床问题为导向推进基础研究，在深圳市医学产业发展上提供方向和指导，从而为深圳医学及相关产业的发展贡献智慧和力量。

在会议主持人深圳市医学会组织管理与学术管理部叶培部长宣读第一届基础医学专业委员会成立批复后，许兴智教授从申请成立基础医学专业委员会背景、基础医学专委会筹备过程、基础医学专委会职责三个方面向在座委员做基础医学专业委员会筹备工作报告。

会议进行了投票选举，并颁发聘书。

第一届基础医学专委会主任委员朱卫国教授表达了各位委员对基础医学专委会支持的感谢，并表示，深大医学部将集全力致力于深圳市基础医学工作的推进，将深圳市基础医学做强做大。

蔡志明教授代表医学会对当选的委员表示祝贺。从医学会地位、学会组成及学术会议举办数量上介绍了医学会。并希望，新成立的基础医学专业委员会在主任委员朱卫国教授的带领下，在各高校科研机构、医疗机构等共同努力下，做出优异的成绩，为深圳市及国家医疗卫生领域出谋划策，为相关条例、立法提供合理化建议，为深圳市和健康中国做出重大的贡献。
