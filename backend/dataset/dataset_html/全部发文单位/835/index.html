时间：8月12日下午4:00

报告人简介：

祖萌，2014年毕业于浙江大学，获得理学学士学位；随后分别于2016年和2021年于康奈尔大学(Cornell University)和格里菲斯大学(Griffith University)获得硕士及博士学位。2021年9月加入Wiley北京办公室，任职Wiley旗下材料类期刊的审稿编辑，负责Advanced Functional Materials, Small Methods及Solar RRL等期刊的稿件同行评审工作。

徐广臣，博士毕业于北京理工大学材料科学与工程学院，博士期间曾赴美国密歇根州立大学（Michigan State University）进行交流访问。2011年9月至今，任职于Wiley出版集团。主要管理和负责Wiley在中国的编辑、出版和合作事务，同时兼任InfoMat、Advanced Science和Small Methods创始顾问主编，Advanced Energy Materials和Small等期刊执行主编。

讲座摘要：

Wiley作为一家有200多年发展历史的公司，始终致力于帮助机构与个人拓展通向成功所需的知识与技能。我们提供科学、技术、医学和人文社科期刊与在线学习、评估与认证相结合的解决方案，帮助高校、研究机构、学术团体、企业、政府以及个人提高学术和专业影响力。作为研究人员，将科研成果整理成文并发表在合适的期刊上是科研工作中非常重要的一个环节。本次报告将着重介绍Wiley旗下的材料科学领域期刊，并从编辑的角度介绍期刊的稿件处理流程，并给出一些在投稿过程中需要注意的事项和技巧，希望对大家有所启发和帮助。

欢迎有兴趣的师生参加！
