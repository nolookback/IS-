深圳大学管理学院周志民教授主讲题为“在线品牌社群研究的基本问题与创新思路”的学术讲座成功举办
传播学院　2021/6/16 15:35:00
  （钟沛基 2022280095）

6月8日中午，深圳大学管理学院副院长周志民教授受邀来到传播学院，主讲题为“在线品牌社群研究的基本问题与创新思路”的学术讲座。本次讲座由传播学院何建平教授主持。

周志民教授主要从事品牌管理研究，他主持了3项国家自然科学基金项目，出版了《品牌管理》等7部专著、译著或教材，发表了80余篇中英文期刊论文，曾荣获广东省哲学社会科学优秀成果一等奖等20余项科研奖，“教育部新世纪优秀人才”、“广东省高校‘千百十工程’省级培养对象”、“广东省十大杰出青年岗位能手”、“深圳市高层次人才地方级领军人才”等多项荣誉称号。

讲座伊始，周志民教授以哈雷摩托车通过车友社群改善经营状况的故事引出了“在线品牌社群（OBC）”的概念。此外，周教授通过对比花粉俱乐部和李宁互动社区，向在场师生阐述了“在线品牌社群”的作用，并提出了“在线品牌社群”建立的关系逻辑：成员-成员关系、成员-社群关系、成员-品牌关系。

接着，周教授将“在线品牌社群”的基本研究问题概括为以下四个方面：第一，如何将访客转化为社群成员；第二，研究社群里成员与成员之间的关系；第三，研究成员与社群的关系；第四，研究社群如何助力品牌发展。同时，周教授围绕这四个方面与现场的老师和同学们分享了他本人及团队最新的研究发现。

然后，周教授认为“在线品牌社群”有关研究的创新思路可以概括为跨界交叉与本土化。同时，他还通过自己的研究向在场的师生展示如何将管理学、心理学和社会学的概念与在线品牌社群的研究相结合。

最后，基于以上分享，周志民教授结合传播学的研究提出“在线品牌社群”未来可以围绕社群当中的信息扩散、创造力、社群的持续热度以及线上与线下品牌社群整合等方向开展研究。

讲座现场气氛热烈，现场的老师和同学们对周教授分享的相关研究产生了浓厚的兴趣，提出了许多问题，周教授一一作答。至此，本次讲座在大家热烈的掌声中圆满结束。
